**コマンドカテゴリ分類**
1. **セッション制御**: /clear（履歴クリア）、/compact（コンテキスト圧縮）、/model（モデル切替）、/memory（メモリ管理）
2. **コンテキスト管理**: /init（CLAUDE.md生成）、/review（コードレビュー）、/rewind（チェックポイント巻き戻し）
3. **MCP管理**: /mcp add・/mcp list・/mcp status
4. **プラグイン**: /plugin install・/plugin list・/plugin info
5. **エージェント**: /agents start・/agents list・/agents stop
6. **リモートセッション**: /remote・/ssh（遠隔操作）
7. **CLIオートメーション**: --print（非インタラクティブ実行）、--allowedTools

**使用頻度別整理**
日常90%を占めるコアコマンド: /clear, /compact, /init, /memory, /model, /review

**発見方法**
`/` 入力後のオートコンプリートで利用可能コマンドが表示される。`/` + 数文字で絞り込み可能。
