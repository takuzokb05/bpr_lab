## 要約
2026年6月15日に施行されたClaude Agent SDKの課金体系変更を解説するTotalum記事。
変更の核心：
- **プログラマティック利用の分離**：Pro/Max/Team/EnterpriseプランのClaude利用制限からAgent SDK・`claude -p`が切り離され、別の月次クレジットプールに移行
- **プラン別クレジット額**：Pro $20/月、Max 5x $100/月、Max 20x $200/月
- **API料金で課金**：クレジット超過分はAnthropicAPIレートで課金
- **対象外となるもの**：対話的Claude Code・Claude Cowork・claude.aiは引き続きサブスクリプション制限内
この変更の背景として2026年4月時点で"claude agent sdk"検索が前年比約50,000%増（月間50→14,800件）という急成長が挙げられている。開発者が既存プランで自動化どこまでできるか把握するのに重要な一次情報。
