## 要約
The New Stack（著名技術メディア）によるClaude Code Routines機能の実践解説。Routinesの定義：「保存されたClaude Code設定（プロンプト・リポジトリ・コネクター）をパッケージ化し、Anthropicのマネージドクラウドインフラで自動実行する機能」。3種トリガー：①Scheduled（1時間毎/夜間/週次/指定日時での定期実行）、②API（HTTPポストで即時起動、CIパイプライン等との統合向け）、③GitHub（PR作成/リリースイベントで自動起動）。重要点：ラップトップが閉じていてもRoutinesは継続実行（Anthropicクラウドが担当）。実ユースケース：夜間テスト実行・定期セキュリティスキャン・PR自動レビュー・日次レポート生成・依存関係更新チェック。The New Stackは技術インフラ分野で高い信頼性を持つメディアであり、本記事のRoutines解説は開発チームのDevOps担当者への説得力が高い。
