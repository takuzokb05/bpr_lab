## 要約
wshobson氏によるオープンソースのマルチハーネス・エージェンティックプラグインマーケットプレイス。Claude Code・Codex CLI・Cursor・OpenCode・GitHub Copilot・Gemini CLIの6ツールに対応した統一プラグインリポジトリ。**コンセプト**: 異なるAIコーディングツール間でスキル・エージェント・ワークフロー定義を横断再利用できる標準フォーマットを提供。チームが複数のAI開発ツールを使い分ける場合に、共通のカスタム命令・スキル・エージェント定義を一元管理できる。**Claude Code固有サポート**: Claude Code skills形式（SKILL.md）・MCPサーバー設定・フック定義をマーケットプレイスフォーマットでパッケージ化。コミュニティ投稿によるスキル・エージェントのキュレーションと品質管理を想定。**背景**: 2026年にAIコーディングツールが多様化し、チーム内で複数ツール並用が増加。ツールごとにカスタマイズを再作成するコストを削減するための実験的エコシステムプロジェクト。Claude Codeのコミュニティマーケットプレイス構想（SitePoint記事で言及）との方向性が一致。
