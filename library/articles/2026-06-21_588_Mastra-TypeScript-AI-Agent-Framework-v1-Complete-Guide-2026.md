## 投稿内容
Mastraは2024年10月に登場したオープンソースのTypeScript AIエージェントフレームワーク。2026年1月にv1.0を正式リリースし、22,000+ GitHub stars・週30万npmダウンロードを達成している。

**開発チーム:**
Gatsby.js（Netlifyが買収した人気Reactフレームワーク）の創設者チーム：Sam Bhagwat（CEO）・Abhi Aiyer（CTO）・Shane Thomas（CPO）。Apache License 2.0でオープンソース公開。

**主要機能:**
- **エージェント**: LLMとツールを使いオープンエンドタスクを推論・実行
- **ワークフロー**: 決定論的・ステートフルなステップパイプライン
- **メモリ**: 会話コンテキストの永続化
- **統一モデルルーター**: 94プロバイダー・3,300超モデル対応（OpenAI・Anthropic・Google等）
- **オブザーバビリティ**: トレース・ログ・評価機能内蔵

**デプロイオプション:**
- Vercel・Netlify・Cloudflareへの自動デプロイ
- Hono・Express・Fastify等の既存サーバーに統合
- スタンドアロンサーバーとして独立デプロイ可能

**TypeScript統一のアドバンテージ:**
フロントエンド・バックエンドをTypeScriptで統一するチームに最適。Claude Agent SDK（Python/TS対応）やLangGraph（Python中心）と異なる選択肢として2026年に急速に採用が拡大。

## 要約
Mastra v1.0（2026年1月リリース）はGatsby創設者チームによるTypeScript専用AIエージェントフレームワーク。22k+ GitHub stars・週30万npmDL。94プロバイダー/3300超モデル統一ルーター、エージェント・ワークフロー・メモリ・オブザーバビリティを内蔵。Apache 2.0。TypeScript開発チーム向けのClaude Agent SDK代替として2026年に急成長。
