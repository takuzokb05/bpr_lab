## 要約
nic9nic9氏によるFX自動売買開発の実体験レポート。LLMを活用した自動売買ボットを継続開発する中で、過去の取引データ・戦略・失敗を記憶するAIツールの必要性に気づき自作に至った過程。技術スタック：Python + Claude API + SQLite（記憶DB）+ MT4/MT5連携。記憶の3層構造：①短期記憶（直近トレード結果・直前エントリー理由）、②長期記憶（戦略別パフォーマンス・通貨ペア別傾向）、③失敗記録（ドローダウン原因・同じミスの繰り返し防止）。実装上の課題：コンテキスト長制限（長期記憶全体をLLMに渡せない）→ベクトル検索で関連記憶を取得する方式に移行。リアルタイム性との問題：LLMの応答速度がスキャルピングに不適→スイングトレード主体に転換。再現可能な実装知見として、FX自動売買×LLM記憶統合の国内希少事例として価値が高い。
