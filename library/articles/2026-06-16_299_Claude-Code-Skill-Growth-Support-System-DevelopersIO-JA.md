# Claude Codeでスキル成長支援＆見守りシステムを構築—DevelopersIO実践

## 要約
DevelopersIO掲載のClaude Codeを使ったエンジニアスキル成長支援システム構築実践記事。Claude CodeのSKILL.md機能を応用し、コードレビュー結果からメンバーの成長ポイントを自動記録・可視化する仕組みを実装。スキルファイルの設計・フックとの連携・成長データのMarkdown蓄積パターンを具体的なコードとともに解説。PostToolUseフックでレビュー結果をキャプチャし、別スキルで成長レポートを生成する連携設計が特徴。Claude Codeを「コード生成ツール」から「チームの学習・成長プロセス支援インフラ」として活用する新しいユースケースであり、スキル・フック組み合わせの参考実装として価値が高い。
