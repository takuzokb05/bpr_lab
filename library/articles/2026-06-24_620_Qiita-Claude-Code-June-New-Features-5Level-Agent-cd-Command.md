## 要約
QiitaでのClaude Code 2026年6月新機能の詳細解説。2大新機能：①Agentツールが最大5段階のネストに対応（サブエージェントがさらにサブエージェントを呼び出す階層構造で、モノレポ・マイクロサービスの複雑なオーケストレーションが可能に）、②/cdコマンドでセッションを維持したままディレクトリ切り替え（従来はcd後にコンテキストが切れていた問題を解消）。5段階ネストの実用例：①ルートエージェントがタスク分析→②ドメイン別サブエージェントに分配→③各ドメインでファイル探索エージェント→④実装エージェント→⑤テストエージェントという完全自律パイプライン。動的ワークフロー（Workflow tool）との組み合わせで数百並列の大規模コードベース処理が現実的に。
