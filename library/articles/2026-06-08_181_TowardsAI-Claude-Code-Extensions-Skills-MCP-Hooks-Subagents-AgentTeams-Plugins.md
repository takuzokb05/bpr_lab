## 要約
Claude Codeの6種拡張システム（Skills/MCP/Hooks/Subagents/Agent Teams/Plugins）を体系的に解説したTowardsAI記事。Skillsは~/.claude/skills/のMarkdownファイルでコンテキスト依存ロード、MCPはJSON-RPC 2.0でGitHub・PostgreSQL等外部ツール連携、Hooksはイベント駆動シェルコマンドで自動実行を実現。Subagentsは独立コンテキストで並列探索し結果を圧縮返却、Agent Teamsは.claude/agents/のYAML定義で役割分担、PluginsはMarketplaceから配布可能なパッケージ。CLAUDE.mdは全セッションロード・スキルはオンデマンドロードの使い分けが重要。
