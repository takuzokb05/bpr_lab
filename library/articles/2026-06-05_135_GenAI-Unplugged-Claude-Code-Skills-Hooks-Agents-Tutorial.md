## 投稿内容
GenAI UnpluggedのClaude Code拡張機能総合チュートリアル。Skills（~/.claude/skills/の.mdファイル、SDKもAPIもビルドステップも不要）、Slash Commands（カスタムコマンド）、Hooks（特定イベントに自動実行するシェルコマンド：ファイル編集後・タスク完了後・入力必要時等）、Subagents（委任した子エージェント）の4機能を体系的に解説。CLAUDE.md: メモリ、Skills: ルーティン、Hooks: 保証、Agents: 委任の4分類フレームワークを提示。CLAUDE.mdに何を書くべきかの判断基準付き。

## 要約
Claude Code拡張機能の4象限フレームワーク（メモリ/ルーティン/保証/委任）が理解の核心。Skills = Markdownファイルだけで作成、Hooks = 特定イベントへのシェルコマンド自動実行、Subagents = 子エージェントへの委任。「CLAUDE.mdに書くか、Skillsに書くか、Hooksで保証するか」の判断基準が実用的。bpr_labのスキル整理と日次収集エージェントのHooks活用設計に直接応用可能。
