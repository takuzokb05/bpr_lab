TradingAgents v0.2.5 released. Key changes: Grounded Sentiment Analyst (now reads real Yahoo News, StockTwits, Reddit data before generating reports—replacing hallucinated social posts), GPT-5.5 model coverage, Qwen/GLM/MiniMax dual-region support with API-key auto-detection, TRADINGAGENTS_* env-var configurability, remote Ollama support, non-US alpha benchmarks, ticker path-traversal hardening.
