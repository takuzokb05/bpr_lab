## 要約
Claude Code v2.1.178（2026年6月16日公開）の主要変更を解説するDevelopersIO記事。パーミッションルールの新構文導入とネストした `.claude` ディレクトリのサポート強化が主な機能追加。サブエージェントおよび認証まわりの不具合も修正。`enforceAvailableModels` オプション追加によりマネージドデプロイでの利用可能モデルをアローリストで制限可能に。エンタープライズプランでは管理者カスタムロール（billing/privacy等の個別付与）が実装された。DevelopersIOはClaude Code全バージョンを継続トラッキングしており、日本語で技術詳細を得る信頼性の高い一次情報源。
