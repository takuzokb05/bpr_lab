# GitHub Copilot for JetBrains: ClaudeをAgent Providerとして利用するパブリックプレビュー開始

- URL: https://github.blog/changelog/2026-06-22-new-features-and-claude-as-agent-provider-preview-in-jetbrains-ides/
- ソース: web
- 言語: en
- テーマ: claude-code
- 取得日: 2026-06-27

## 投稿内容
GitHub公式Changelog（2026年6月22日）の発表。JetBrains IDE向けGitHub Copilotに以下の新機能が追加。最大のハイライトはClaudeをAgentプロバイダーとして利用するパブリックプレビュー開始。その他：組織・エンタープライズレベルのカスタムエージェント対応、Copilot CLIでのメッセージキュー機能、エージェントデバッグログの統合ビュー、モデルピッカーの機能強化、ターンごとのAIクレジット表示、クラウドエージェントの一般提供開始。

## 要約
JetBrains IDEでGitHub CopilotがClaudeをAgentプロバイダーとして使用できるパブリックプレビューが2026年6月22日に開始。これはGitHub CopilotとAnthropicエコシステムの統合深化を示す公式一次情報。Claude Codeユーザーは既存のJetBrains開発環境（IntelliJ IDEA・PyCharm・WebStorm等）からClaude Agentを直接利用できる新経路が開かれる。合わせてエンタープライズ向けカスタムエージェント対応、デバッグログ統合ビュー、ターンごとのAIクレジット表示も追加。クラウドエージェントのGAも完了。VS Code外での主要IDE統合としてJetBrainsが初の公式Claude Agent対応となり、エンタープライズ開発者のClaude採用経路が拡大する。
