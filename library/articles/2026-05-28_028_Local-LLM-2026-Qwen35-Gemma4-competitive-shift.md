# ローカルLLM 2026急進化: Qwen3.5・Gemma 4がノートPCで動作・市場シェアChatGPT64.5%に低下

- URL: https://www.gizmodo.jp/2026/05/2026_local_llm_matome.html
- ソース: web
- 言語: ja
- テーマ: ai-news
- 取得日: 2026-05-28

## 投稿内容
2026年のローカルLLMの進化がすごい。Qwen3.5（アリババ、3月）やGemma 4（Google、4月）があります。Qwen3.5・Gemma 4は一般的なノートPCを含めたさまざまなデバイスで動かせるようにサイズが調整されており、自分のPC環境に合わせたモデルを選びやすくなっています。LLMの競争軸が「モデル単体の性能」だけではなく「そのモデルをどのような環境で、どれだけ使いやすく動かせるか」に移ってきている。ChatGPTの市場シェアが2025年初めの86.7%から2026年1月には64.5%まで低下し、GoogleのGeminiは5.7%から21.5%へと4倍以上に成長しました。

## 要約
Gizmodo Japanによる2026年ローカルLLM進化まとめ（2026年5月）。Qwen3.5（Alibaba、3月リリース）とGemma 4（Google、4月リリース）がノートPC含む多様なデバイスで動作するサイズバリエーションを展開し、量子化技術の進歩により前世代最前線モデルクラスをコンシューマーデバイスで実行可能に。LLM競争軸の構造的変化を分析：「モデル単体性能（ベンチマーク）」から「実行環境・ツール連携・エージェント機能・UI・画像生成・リサーチ機能・ローカル実行しやすさ」へ。市場シェア変動：ChatGPT 86.7%→64.5%（2025年初→2026年1月）、Gemini 5.7%→21.5%に急拡大（4倍超）。LLMをモデルだけでなくエコシステム全体として評価する視点の重要性を指摘。
