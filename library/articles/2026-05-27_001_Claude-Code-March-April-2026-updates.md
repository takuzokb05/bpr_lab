## 投稿内容
Builder.ioによるClaude Code 2026年3〜4月アップデート解説記事。

## 要約
5週間でv2.1.69からv2.1.101まで30本超リリースという過去最大の反復ペース。主要変更：Claude Opus 4.7がMax/Team Premiumのデフォルトモデルに昇格。xhigh effortレベル追加（推奨設定）と/effortスライダーによるインタラクティブ調整。Agent view（`claude agents`コマンド）で全セッションの状態一覧確認が可能に。Opus 4.6の1Mコンテキストウィンドウ（Max/Team/Enterprise）が3月13日に正式提供開始。インタラクティブチャート・ダイアグラム・モバイルアプリ生成機能追加。/simplifyを/code-reviewにリネーム。Pro/Max/EnterpriseのCode w/ Claude 2026イベントで5時間上限が2倍に拡張。/resumeでバックグラウンドセッション再開サポート。api.anthropic.com接続不可時の起動ハング修正。MCP/SDK起動高速化。Claude Codeが単なるコーディングツールからエージェントプラットフォームへ急速に進化中。
