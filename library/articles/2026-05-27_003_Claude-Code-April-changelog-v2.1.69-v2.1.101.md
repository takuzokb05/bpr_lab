## 投稿内容
Apiyi.comによるClaude Code 2026年4月変更履歴の詳細技術解析記事。

## 要約
バージョン別主要変更：/resumeによるバックグラウンドセッション再開サポートが追加（長時間セッションの断絶に対応）。/modelコマンドが現在セッションのみのモデル変更に変更（永続変更ではなくなった—要注意の挙動変更）。api.anthropic.com接続不可時の起動ハング問題を修正。MCP/SDKの起動を高速化。ターミナルのグリッチ・クラッシュ・セッション復元の多数修正。バックグラウンドセッション管理の強化により大規模マルチセッション運用が実用的に。Claude Code利用者は/modelの挙動変更（セッションスコープ化）を特に注意すべき変更点として把握すること。
