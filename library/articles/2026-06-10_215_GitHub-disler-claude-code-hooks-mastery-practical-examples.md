## 要約
GitHubユーザーdislerが公開した「claude-code-hooks-mastery」リポジトリ。Claude CodeのHooks機能（PreToolUse・PostToolUse・UserPromptSubmit・Stop）を徹底的にマスターするための実践的なサンプルと解説集。Hooksの核心は「エージェントループ外で実行される決定論的ガードレール」であり、フォーマット自動適用・lint強制・セキュリティチェック・ログ記録などに使う。hookSpecificOutput.updatedToolOutputを使ったツール出力の差し替えパターンや、settings.jsonへの設定方法など、実際のプロジェクトに即応用可能なコード例を多数収録。Claude Code v2.1系での動作確認済み。
