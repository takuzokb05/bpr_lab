## 要約
builder.io（Steve Sewell）による50個のClaude Code実践Tips集。1Mトークンコンテキスト（Sonnet 4.6 / Opus 4.6）の活用、`/model opus[1m]` でのセッション中途モデル切替、`claude remote-control` によるiOS/Androidからのリモート操作・承認を解説。
`!git status` / `!npm test` でシェルコマンドをコンテキストに直接注入する技法、エイリアス `cc` で全パーミッションプロンプトをスキップする設定も収録。
plan mode（Shift+Tab ×2）→実装（Shift+Tab ×1）の2段階フロー、hooks・CLAUDE.md・subagentsの組合せ設計パターンを体系化。
チーム標準化のアプローチ（shared CLAUDE.md・標準フック・オンボーディング）も詳説。計50項目を日常ワークフロー順に整理し、初心者から上級者まで即実践できる内容。
