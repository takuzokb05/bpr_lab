## 要約
香港大学データサイエンスラボ（HKUDS）が開発する「Your Personal Trading Agent」というコンセプトのオープンソースLLMトレーディングシステム。
主な特徴：
- **マルチ市場対応**：ChinaFutures・GlobalFutures・Forex・Options v2の4エンジンを搭載
- **堅牢なバックテストエンジン**：Monte Carlo・Bootstrap CI・Walk-Forward検証に対応
- **個人投資家向け設計**：大規模ヘッジファンド向けではなく個人が実装・実験できる規模感
- **直近更新**：バックテストエンジン強化、Forexエンジンv2対応
FX自動取引に直接応用可能なForexエンジンを含む点が特筆。TradingAgentsとの違いは個人用途・研究室主導・マルチ資産クラス対応。MT5連携の参考実装として調査価値あり。
