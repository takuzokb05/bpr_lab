# Anthropic-SpaceX Colossus Deal — 300MW/220k GPUs, Claude Code 5-Hour Limit Doubled

## 要約
2026年5月6日、AnthropicはSpaceXとの計算インフラ提携を発表。SpaceX Memphis「Colossus 1」データセンターの全コンピュート容量（300MW超・220,000+ NVIDIA GPU）を月内アクセス可能とした。合わせてClaude Codeの5時間レート制限をPro/Max/Team/Enterprise全プランで2倍に引き上げ、ピーク時間帯のPro/Max制限を撤廃。Claude Opus APIのレート制限も大幅引き上げ。将来的にはSpaceXとの軌道上AIコンピュート研究も検討中。既存のAmazon（最大5GW）・Google/Broadcom（5GW）・Microsoft/NVIDIA（$30B Azure容量）・Fluidstack（$50B）に続く大型コンピュート確保で、Anthropicの学習・推論インフラが急拡大。一次情報：Anthropic公式ブログ。
