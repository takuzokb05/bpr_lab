# Anthropic、Stainlessを$300M超で買収: SDK・MCPサーバー生成基盤を内製化

## 投稿内容
Anthropic Acquires Stainless — Anthropic Official, May 18, 2026. Anthropic has acquired Stainless, the SDK and MCP server tooling company used by hundreds of AI developers. Stainless (founded 2022, NY-based, founder Alex Rattray ex-Stripe) generates production-ready SDKs across TypeScript, Python, Go, Java, Kotlin from API specifications. Anthropic's official SDKs have been generated by Stainless. Stainless customers retain full ownership and modification rights to SDKs generated to date; hosted Stainless products will wind down. Strategic rationale: "The usefulness of agents depends on the quality of what they can connect to." Following the December 2025 Bun acquisition (Claude Code runtime), this represents Anthropic's second tooling-layer acquisition, integrating SDK generation and MCP server creation into core infrastructure.

## 要約
Anthropicが2026年5月18日にSDK自動生成スタートアップStainlessを$300M超で買収したと公式発表。Stainlessは2022年創業（元Stripe Alex Rattray）で、Anthropic全公式SDK（TypeScript・Python・Go・Java・Kotlin）の生成を担ってきた。OpenAI・Google・Cloudflareも同社ツールを利用しており、買収によりこれらライバルへのサービス提供は終了。顧客は生成済みSDKの所有権・改変権を保持。2025年12月のBun買収（Claude Codeランタイム）に続く第2弾の開発ツール層統合。モデル層→ランタイム→SDK生成→MCPサーバー生成へのスタック垂直統合が鮮明。「エージェントの有用性はそれらが接続できるものの質に左右される」という戦略思想を体現する重要M&A。
