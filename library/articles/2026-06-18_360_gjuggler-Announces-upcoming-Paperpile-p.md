## 投稿内容

There are good MCP servers and skills  for literature search -- see GXL's Paperclip, Scite MCP, Elicit MCP.

Paperpile will release plugins for Claude Code & Codex soon.

For literature search in mainstream agents to be really effective, you need (1) a good MCP server, (2) thoughtful skills describing best practices & gotchas to avoid, and (3) an easy way to install that doesn't require forking a repo. For icing on the cake, add (4) a way to get full-text content to your machine so an agent can read it like code.

Plugins finally make all this possible, but they're still a bit under-developed.
