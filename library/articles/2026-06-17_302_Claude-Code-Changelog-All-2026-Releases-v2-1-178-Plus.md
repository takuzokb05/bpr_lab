## 要約
Claude Code 2026年全リリースノートをバージョン別に集約したサードパーティ製リファレンス（毎週更新）。主要追加機能：`/cd`コマンド（セッション移動・プロンプトキャッシュ維持）、ネスト型サブエージェント（最大5階層の背景チェーン）、`--safe-mode`フラグ（全カスタマイズ無効化によるトラブルシューティング）、`fallbackModel`設定（最大3モデルのフォールバックチェーン）。UX改善：NO_FLICKERレンダリングエンジン、Focus View、Writeツール60%高速化、エージェントビュー（複数セッション管理UI）。Claude for AWS（Bedrock統合、フルAPI機能セット）も同期収録。
