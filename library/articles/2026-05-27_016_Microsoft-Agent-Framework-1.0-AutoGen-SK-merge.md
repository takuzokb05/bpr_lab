## 投稿内容
Microsoft Tech Community公式ブログによるMicrosoft Agent Framework 1.0発表記事（2026年4月3日）。

## 要約
2026年4月3日に.NETとPython向けの本番利用可能なオープンソースフレームワークとして正式リリース。AutoGenとSemantic Kernelを統合したMicrosoftの公式エージェントフレームワーク。新機能A2Aプロトコル（Agent-to-Agent）：異なるランタイム間でのエージェント通信を実現（PythonエージェントとNETエージェントが協調可能）。2026年フレームワークランキング：LangGraph（1位）・Claude Agent SDK（2位）・CrewAI（3位）・Microsoft Agent Framework（新参入）。Google ADK（2025年4月リリース）もA2Aプロトコルをネイティブサポート。AutoGen v0.4+（MS公式書き直し版）とAG2（コミュニティ継続版）の分裂が利用者を悩ませる。エージェント間通信の標準化（A2Aプロトコル）が2026年の主要テーマとして浮上。
