## 要約
fazm.aiがClaude Codeデスクトップクライアントから読み取ったAnthropicの最新APIリリースノート（2026年6月）集約記事。主要アップデート：Claude Fable 5（6月9日GA、$10/$50 per Mトークン）・Claude Opus 4.8コンテキストウィンドウ拡張。拒否リクエスト課金廃止（stop_reason: "refusal"は入力トークン不課金）。セルフホスト型サンドボックスのパブリックベータ（ツール実行をユーザーインフラに移動可能）。Claude Platform on AWS開始（IAM認証・AWS課金統合）。Managed Agents向けMCPトンネル（プライベートネットワーク内MCPサーバーへのセキュアアクセス）。Dynamic Workflowsの研究プレビュー開始。Workload Identity Federation（キーレス認証）GA。公式ドキュメント整備：Agent SDK overview・Skills設計ガイドが新規追加。
