## 要約
wmedia.esによるClaude Code + Sonnet 5の実践活用ガイド。Claude Code v2.1.197以降で`sonnet`エイリアスが自動的にSonnet 5に解決される。切替コマンド：`/model sonnet`。新機能`opusplan`モードはOpus 4.8がプランニング段階を担い、Sonnet 5が実行段階を担うハイブリッドアーキテクチャ。プラン別デフォルト：Pro/Team Standard/Enterprise固定シートはSonnet 5がデフォルト、MaxとEnterprise従量課金はOpus 4.8のままで手動変更が必要。Sonnet 5推奨場面：日常的なエージェント業務・スピード重視・Webリサーチ・ブラウンフィールドコードのデバッグ・コスト効率優先。Opus 4.8残す場面：フロンティアレベルのマルチファイルリファクタリング・精度最優先タスク。Sonnet 5の自己検証機能：複雑タスクで明示的プロンプトなしに出力を自己チェック。Claude Codeユーザーが即座にアクションできる実践情報として高価値。
