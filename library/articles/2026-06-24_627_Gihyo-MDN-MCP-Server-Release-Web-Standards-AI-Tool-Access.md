## 要約
技術評論社gihyo.jpによるMDN MCPサーバーリリース速報（2026年6月）。Mozilla公式のMDN Web DocsがMCP互換クライアント向けにMCPサーバーを公式リリース。対応クライアント：VS Code、Zed、Cursor、Claude Code、Codex CLIなど全MCP互換ツール。提供コンテンツ：HTML/CSS/JavaScript/WebAPI等Web標準仕様の最新ドキュメント、ブラウザ互換性テーブル（Baseline対応状況）。重要性：従来はContext7等の非公式サーバーがMDNコンテンツを提供していたが、Mozilla公式サーバーにより一次情報源への直接アクセスが可能に。使用方法：claude mcp add mdn-docs @modelcontextprotocol/server-mdn（例）。Web開発者がClaude Codeでフロントエンド実装する際にMDNを参照しながらコーディングするユースケースの実用性が大幅向上。6月15日リリースで即日利用可能。
