## 投稿内容
TechCrunchがClaude Code Voice Modeの公式ローンチを報道（2026年3月3日）。

**機能概要**
- スペースキー長押しでPush-to-Talk音声入力
- 20言語対応（英語・日本語含む）
- ターミナル向け: 音声→テキスト変換（STT）に特化（会話型音声UIではない）
- 設定不要、クロスプラットフォーム（Mac/Linux/Windows）

**利用可能プラン**
Pro・Max・Team・Enterprise（2026年3月ロールアウト開始、現在広く展開）

**主な活用シーン**
- コーディング中に手を使わずにプロンプト入力
- 長い仕様書・指示の音声入力
- ハンズフリーのコードレビュー依頼
- 複数の作業を並行しながら音声でタスク指示

**技術的特徴**
音声入力はテキストに変換されてからClaude Codeの通常パイプラインに入力される。スペースを離した瞬間に送信。バックグラウンドノイズフィルタリングあり。

日本語でも利用可能であり、日本語プロンプトの音声入力にも対応。
