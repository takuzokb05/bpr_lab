## 投稿内容
Microsoft BUILD 2026（2026年6月2〜3日）公式Agent Frameworkブログ発表。Microsoft Agent Framework 1.0（2026年4月GA）に続くBUILD期間中の新機能一覧。

**MAI Models（新規）**
- MAI-Thinking-1: 35B active parameters、256Kコンテキスト、ゼロ蒸留でMicrosoft独自訓練の推論モデル
- MAI-Image-2.5: text-to-image / image-to-image両対応

**Microsoft IQ（GA）**
- Work IQ: Microsoft 365からのリアルタイム職場知識（メール・ドキュメント・会議・組織関係）
- Foundry IQ: 構造化ビジネスデータグラウンディング
- Fabric IQ Ontology: ビジネス意味論
- Web IQ（新規）: リアルタイムWeb情報取得

**Microsoft Scout（新規Autopilot型エージェント）**
常時稼働の自律エージェントカテゴリ「Autopilots」の最初の製品。OpenClaw + WorkIQ上で構築。Teams/Outlook統合で会議準備・スケジュール競合・ルーティンタスクを自律処理。

**Agent 365 SDK（GA）**
無料・フレームワーク非依存。Microsoft Agent Framework・OpenAI Agents SDK・LangChain・Semantic Kernel・Azure AI Foundry のパッケージを提供。

**Foundry Toolkit for VS Code（GA）**
エージェントテンプレート生成・GitHub Copilot連携・ローカルトレース可視化デバッグ・Foundry Agent Serviceへのデプロイ。

## 要約
Microsoft BUILD 2026（6月2〜3日）公式発表：MAI-Thinking-1推論モデル（35B/256K）、Autopilot型Scout、Microsoft IQ統合知識レイヤー、Agent 365 SDK（LangChain/LangGraph互換）GA、Foundry Toolkit for VS Code GA。AI Agent Framework 1.0から続くMicrosoftエンタープライズAgent戦略の最新展開。
