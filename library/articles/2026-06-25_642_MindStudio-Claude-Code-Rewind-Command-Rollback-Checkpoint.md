## 要約
2026年6月24日のClaude Codeアップデートで追加された`/rewind`コマンドの詳細解説記事。
機能の詳細：
- **巻き戻しの仕組み**：`/clear`実行前の状態に会話コンテキストとコードの両方を復元できる
- **従来の`/clear`との違い**：`/clear`はコンテキストリセット（一方向）だが`/rewind`は任意チェックポイントへの双方向移動
- **ミッドセッション修正**：実装が意図しない方向に進んだ場合、良かった時点まで遡って別アプローチを試せる
- **使用シーン**：アーキテクチャ決定の比較、バグ混入前の状態への復帰、複数アプローチの並列評価
`/rewind`はコードの変更（ファイル編集）も含めて復元可能かどうかという重要な点も解説。長時間セッションの安全性向上に直結する実用的な新機能。
