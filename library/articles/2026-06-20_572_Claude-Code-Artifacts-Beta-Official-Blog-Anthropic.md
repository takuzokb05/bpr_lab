# Claude Code Artifacts ベータ公開 — セッション作業をライブ共有ページに変換

- ソース: web
- 言語: en
- テーマ: claude-code
- 取得日: 2026-06-20

## 要約
Anthropic が 2026年6月18日、Claude Code Artifacts をベータリリース（Team/Enterprise プランのみ）。Claude Code のセッション作業をプライベートURLの自己更新型HTMLページに変換する機能。PR ウォークスルー・システム解説・ダッシュボード・リリースチェックリスト等をチームでリアルタイム共有可能。共有はデフォルト非公開で同組織認証メンバーのみアクセス可能。Enterprise では管理者がロール別アクセス制限・保持期間設定・Compliance API での監査が可能。アーティファクトはセッション進行にあわせて自動更新される点が既存ドキュメント共有との差別化ポイント。「コードを書く」から「作業成果をドキュメント化・共有する」まで一気通貫でエージェントが担う方向性を示す重要アップデート。
