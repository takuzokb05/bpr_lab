## 要約

MindStudioによるClaude Codeワークフローパターンの5段階分類：(1) Sequential、(2) Plan-then-Execute、(3) Multi-Agent Parallel、(4) Supervised Autonomous、(5) Fully Autonomous。各パターンの適用シーン・リスク・コスト特性を定量比較し、「どのパターンを選ぶべきか」の判断フレームワークを提供。Plan-then-Executeは複雑タスクで最もコスト効率が高く、Fully AutonomousはCIパイプライン統合時に有効という実践的知見を含む。ワークフロー選択ミスによる「コスト爆発」防止のためのチェックリストも掲載。CLAUDE.mdへの反映やスキル設計の指針として直接活用可能。
