## 要約
ECC（Everything Claude Code）はClaude Code・Codex・Opencode・Cursor横断で動作するエージェントハーネス最適化システム（OSSリポジトリ）。5機能：Skills（スキル管理）・Instincts（本能的挙動定義）・Memory（コンテキスト記憶）・Security（セキュリティガード）・Research-First Development（研究優先開発）。CLAUDE.mdとスキルディレクトリを体系的に構成するフレームワークを提供し、コンテキスト予算管理・セッション間記憶持続・セキュリティフックの自動設定を含む。複数AIコーディングツール横断で同設定を流用できる設計でベンダーロックイン回避を意識。スキルの再利用性と組み合わせパターンドキュメントが充実している。
