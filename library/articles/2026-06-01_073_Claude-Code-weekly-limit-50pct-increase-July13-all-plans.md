# Claude Code 週次利用制限50%増加 — 全プラン即時適用・7月13日まで

## 要約
Claude Codeの週次利用制限が即時50%増加し、2026年7月13日18:00 PDT（日本時間7月14日10:00）まで全有料プラン（Pro・Max・Team・seat-based Enterprise）に適用。CLI・IDEエクステンション・デスクトップアプリ・Webの全サーフェスに適用、オプトイン不要。
直前に5時間上限が2倍になっており、両施策が同時に重なり、短期ウィンドウで実質2倍・週次上限で1.5倍の効果。
価格据え置きのままレート制限を3段階緩和した背景にはOpenAI Codexとの競争激化がある。
Hacker News でも話題になり、ProユーザーがMax 5x 相当の weekly cap を体験したと報告。
7月13日以降はデフォルト（緩和前の水準）に戻ることが想定され、期間限定の特典として位置付けられている。
