# Claude Code Skillsの使い方と汎用テンプレート公開 SIOS Tech Lab JA

- URL: https://tech-lab.sios.jp/archives/50570
- ソース: web
- 言語: ja
- テーマ: claude-code
- 取得日: 2026-06-13

## 要約
SIOS Tech LabによるClaude Code Skills活用ガイド（日本語）。汎用スキルテンプレートを公開：コードレビュー・テスト生成・ドキュメント作成・リファクタリングの4種類のSKILL.mdテンプレートを無償提供。スキルの配置方法（~/.claude/skills/でグローバル、.claude/skills/でプロジェクトローカル）と命名規則（動詞-名詞形式が自動トリガーしやすい）を解説。実際の業務で効果が出た事例として、PR作成の自動化（/pr-create）、朝会の議事録生成（/standup-summary）を紹介。スキルのデバッグ方法（CLAUDE.mdにスキル一覧を明示する方法）と、スキルが正しくトリガーされない場合のチェックリストも掲載。
