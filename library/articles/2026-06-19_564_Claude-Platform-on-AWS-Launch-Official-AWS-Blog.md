# Introducing Claude Platform on AWS: Anthropic's Native Platform Through Your AWS Account

## 要約
2026年5月、Anthropic が AWS 上で Claude Platform を GA（一般提供）開始したことを告げる AWS 公式ブログ。含まれる機能: Claude Managed Agents・コード実行・Web 検索・Web フェッチ・プロンプトキャッシング・バッチ処理・引用・Files API・Skills・MCP コネクター。認証は AWS IAM、監査ログは CloudTrail、課金は AWS 請求書に一本化。Amazon Bedrock との違いは「Anthropic がサービス運営者でデータが AWS バウンダリ外で処理される」点（Bedrock は AWS がデータ処理者でバウンダリ内）。Claude Platform on AWS では新 API 機能とベータ機能を Anthropic ネイティブ API と同日に受け取れることを保証。AWS ファーストパーティとして初のクラウドプロバイダー統合。
