## 要約
GitHub Changelog（2026年6月30日）によるClaude Sonnet 5のGitHub Copilot GA発表。CLI形式タスクで特に強い性能、プロンプトキャッシュ利用率が高い、低effortでも競争力のあるレイテンシ。ゼロデータリテンション（ZDR）ポリシー適用。対象プラン：Copilot Pro/Pro+/Max/Business/Enterprise。あわせて発表されたClaude Code新機能：(1) Checkpoints（最もリクエストの多かった機能—作業途中の進捗を自動保存し、任意の時点に即時ロールバック可能）、(2) ターミナルインターフェース刷新、(3) ネイティブVS Code拡張のリリース。GitHub Copilot vs Claude Code 2026：Claude Codeは自律タスクと100万トークンコンテキストで優位、Copilotはより多くのIDE対応・GitHub統合・インライン補完で優位。
