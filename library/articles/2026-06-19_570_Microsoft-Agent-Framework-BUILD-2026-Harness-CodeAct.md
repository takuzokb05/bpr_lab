# Microsoft Agent Framework at BUILD 2026: Agent Harness, Hosted Agents, CodeAct

## 要約
Microsoft Agent Framework 公式ブログによる BUILD 2026 発表まとめ。AutoGen と Semantic Kernel を統合して2026年4月3日に GA した Agent Framework 1.0 の BUILD 2026 での新機能発表を報告。主な発表: (1) Agent Harness（Claude Code と同様のエージェント実行環境を .NET/Python に提供）、(2) Hosted Agents（Azure 上でエージェントをサーバーレス実行）、(3) CodeAct（LLM が Python コードを生成・実行することで任意のツールを呼び出すパラダイム）。Graph-based ワークフローと A2A（Agent-to-Agent）プロトコルによる Python↔.NET クロスランタイム連携も発表。LTS（長期サポート）版の安定 API を提供し企業導入を促進。Claude Agent SDK・LangGraph との競合定義も提示。
