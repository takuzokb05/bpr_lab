## 投稿内容
AugmentCodeによるClaude CodeとClaude Agent SDKの用途別比較ガイド。Claude Code: インタラクティブなコーディング補助・IDE統合・既存コードベースの修正に特化。Claude Agent SDK: プロダクション向け長時間実行エージェント・ヘッドレス自動化・CI/CDパイプライン組み込みに適切。SDKはClaude CodeのコアハーネスをPython/TypeScriptライブラリとして提供。選択基準：ヒューマン・イン・ザ・ループが必要→Claude Code、完全自律バッチ処理→SDK。両者の組み合わせ実践例も紹介。

## 要約
Claude CodeとClaude Agent SDKの用途分類ガイド。選択基準が明確：対話的コーディング支援＝Claude Code、ヘッドレス自動化・CI/CD・長時間バッチ処理＝Agent SDK。SDKはClaude Codeの同一ハーネスのライブラリ版。FX自動取引システムのような24時間稼働エージェントはSDK向き。June 15課金変更と合わせて読むと移行・設計方針の判断に直結する。
