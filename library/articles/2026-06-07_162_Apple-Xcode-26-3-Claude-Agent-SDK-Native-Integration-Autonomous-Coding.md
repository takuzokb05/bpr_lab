# Apple Xcode 26.3 Native Claude Agent SDK — Visual Preview Capture, Autonomous Debug Loop

- URL: https://www.anthropic.com/news/apple-xcode-claude-agent-sdk
- ソース: web
- 言語: en
- テーマ: claude-ecosystem
- 取得日: 2026-06-07

## 要約
2026年2月3日、Appleが発表。Xcode 26.3がClaude Agent SDK（Claude Codeのハーネスと同一）のネイティブ統合を実現。主要機能：①Xcode PreviewsのビジュアルキャプチャでClaudeがUIを視覚確認しながらイテレーション、②ビルドログ・クラッシュ・ユニットテストのリアルタイム監視と自動修正ループ（バックグラウンドで継続動作）、③Model Context Protocol（MCP）経由でClaude Code CLIからもXcode機能にアクセス可能（ビジュアルプレビュー含む）。iPhone/Mac/Apple Vision Pro向け開発全般に対応。Apple Developer Program会員向けにRC提供開始、App Store正式リリース予定。Anthropicにとって主要IDEプラットフォームへの統合拡大（VS Code・JetBrainsに続く第3の公式統合）。
