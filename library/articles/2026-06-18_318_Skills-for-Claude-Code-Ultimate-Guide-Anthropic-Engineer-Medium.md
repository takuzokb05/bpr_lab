## 要約
AnthropicエンジニアによるClaude Code Skillsの完全ガイド（Medium）。意思決定フレームワーク：「CLAUDE.mdはメモリ、Skillsはルーティン、Hooksは保証、Agentsは委任」。Skillsは文脈に応じて自動ロードされる軽量マークダウンディレクトリ。実践的原則：まず単一の困難なタスクで成功するまでイテレート→成功アプローチをSkillとして抽出（広いカバレッジから始めない）。Anthropicの内部ベストプラクティスを一次情報として公開した希少な記事。セットアップ5層：CLAUDE.md(プロジェクトルール)→MCPサーバー(外部ツール/データ)→Skills(再利用ワークフロー)→Hooks(自動化・安全チェック)→Subagents(孤立した研究・レビュー作業)。
