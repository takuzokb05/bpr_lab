## 投稿内容
MCP Is Growing Up. MCP SDK monthly downloads grew from ~2M (Nov 2024) to 97M (March 2026)—48x. 5,800+ registered servers. 41% of software companies running MCP in production. The 2026-07-28 Release Candidate brings stateless protocol core (standard HTTP infrastructure, round-robin load balancers, Mcp-Method header routing), Extensions framework (MCP Apps, Tasks), and OAuth/OIDC alignment. Breaking changes included; 10-week validation window for SDK maintainers. Security concern: 14 CVEs (RCE via MCP servers) reported in April 2026.

## 要約
Agentic AI Innovation Forum（AAIF）によるMCP成熟度分析。**定量的成長**: SDK月間DLが2024年11月の200万から2026年3月に9700万（48倍）、登録サーバー5800本超、ソフトウェア企業41%が本番運用中。**2026-07-28 RC主要変更**: ①ステートレスプロトコルコア（通常HTTP、ラウンドロビンLB対応、Mcp-Methodヘッダーでルーティング）②MCP Apps（サーバーレンダリングUI）③Tasksエクステンション（長時間実行）④OAuth/OIDC準拠強化。Breaking changes含む、SDK開発者向け10週間検証ウィンドウ。**課題**: 2026年4月に14件のCVE（MCPサーバー経由RCE）が報告されセキュリティ懸念。エンタープライズガバナンス標準不足。**評価**: ステートレス化でエンタープライズ導入障壁が大幅低下、「AIのUSB-C」という位置づけが2026年に定着しつつある。
