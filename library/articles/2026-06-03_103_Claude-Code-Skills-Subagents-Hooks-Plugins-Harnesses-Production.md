## 投稿内容
A deep dive into every Claude Code extension point — Skills, Subagents, Hooks, Plugins, and Harnesses — for building production-grade multi-agent workflows.

## 要約
Substack「BoringBot」掲載の本番マルチエージェントワークフロー設計ガイド。Skills vs Subagents vs Harnessesの使い分けを明確化：Subagentsは分離されたコンテキストで実行し要約を返す、Skillsはメイン会話でオンデマンドロード、Harnessesはエージェントループを外から制御するメタ構造。2026年のベストClaude Codeスキルランキング：/batch（並列ワークツリー処理）、/simplify（複雑プロンプト再構成）、/debug（トラブルシューティング）、/loop（繰り返し実行）、/claude-api（APIフォーカス作業）。本番投入時の意思決定フロー：まずSubagentsから始め→同じ指示を複数回使うならSkillsへ移行→複数専門家の協業が必要なときはAgent Teamsに移行。
