## 投稿内容
Anthropicが2026年5月14日に発表した課金体系の重大変更に関する解説記事。

## 要約
2026年6月15日から適用の重要課金変更。プログラマティックなClaude利用がサブスクリプションのレートリミットプールから分離され、別の月額クレジット（標準APIリスト価格での従量課金）に移行。対象：Agent SDK呼び出し、claude -pコマンド、Claude Code GitHub Actions、サードパーティエージェント（OpenClawなど）。これまでサブスクリプション内「実質無制限」だったclaud -p実行やGitHub Actions利用が従量課金になる変革的変更。インディーハッカー・個人開発者に月額コスト増の可能性。関連して：Managed Agentsパブリックベータ（セキュアサンドボックス・組み込みツール・SSEストリーミング）、セルフホストサンドボックスパブリックベータ（ツール実行を自環境へ移動、エージェントループはAnthropicインフラ上）、マルチエージェントセッションとOutcomesがパブリックベータ開始。Claude Code利用コストの再計算が必要な最重要情報。
