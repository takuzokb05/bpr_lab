Claude Codeのプラグイン管理、革命的にラクになった

注目ポイントはここ
・`.claude/skills`下で自動読み込み
・マーケット不要でローカル完結
・CLIに `claude plugin init <名前>` で雛形即作成
・`/plugin`コマンドの引数補完が超強化済み

つまり
開発環境のカスタマイズが爆速で始められる

これまでは
マーケット検索 → インストール → 手動設定

今は
ローカルに置くだけ → 自動認識 → CLIで即操作可能

「面倒で後回し」だったプラグイン整備が一気に身近に感じるはず

この変化、知らなかったらかなり損

すぐに `.claude/skills` フォルダを覗いて
`claude plugin init myplugin` を試してみるべき

後で追うより今追った方がいいですよ
