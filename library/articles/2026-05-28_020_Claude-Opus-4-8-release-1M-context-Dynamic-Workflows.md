## 投稿内容
Anthropic releases Claude Opus 4.8 with new 'dynamic workflow' tool. The model ships with a 1M token context window, scores 69.2% on SWE-bench Pro (up from 64.3% on Opus 4.7), and posts 96.7% on USAMO 2026 math. It is roughly four times less likely than Opus 4.7 to allow flaws in code to pass unremarked. Pricing starts at $5 per million input tokens and $25 per million output tokens (same as Opus 4.7). Fast mode is 3× cheaper than previous models' fast mode. Available on Claude API, Amazon Bedrock, Google Cloud Vertex AI, and Microsoft Foundry.

## 要約
Anthropicが2026年5月28日にClaude Opus 4.8をリリース（同日Dynamic Workflows・Effort Controlも発表）。主要ベンチマーク：SWE-bench Pro 69.2%（Opus 4.7比+4.9pt）、USAMO 2026数学96.7%（Opus 4.7は69.3%、+27.4pt）。コードの欠陥見落とし率がOpus 4.7比4分の1に低下。1Mトークンデフォルトコンテキスト、最大128kアウトプットトークン。価格：入力$5/M・出力$25/M（4.7と同額）、Fast mode $10/$25/M（旧版比3倍安・2.5倍速）、プロンプトキャッシュで最大90%コスト削減、バッチ処理で50%削減。Claude CodeではOpus 4.8が並列サブエージェントをスポーンしてコードベース規模のマイグレーションを自律処理。Anthropic IPO競争激化の中での戦略的大型リリース。
