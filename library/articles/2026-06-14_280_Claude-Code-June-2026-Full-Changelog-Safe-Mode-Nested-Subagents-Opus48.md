## 投稿内容
Claude Code June 2026 Update: Safe Mode, Opus 4.8, and Doubled Rate Limits — developer-focused changelog guide covering all June 2026 updates.

Major June 2026 updates:
- Safe Mode: retries once on fallback model when API rejects unexpected non-retryable error (v2.1.169+)
- /cd command added for directory navigation
- Opus 4.8 elevated to default model
- /usage command with per-category breakdown
- v2.1.172 (June 10): nested sub-agents up to 5 levels deep
- Plugin search improvements
- Better Chrome, VSCode, terminal workflow integration
- Fable 5 added then disabled due to government directive
- Bug fixes: image processing errors, remote session sticking, 529 overload fallback
