SkillSpector — AI Agent Skill Security Scanner 🤖💀

Before installing an AI agent skill, scan it like you would any untrusted code.

• Developed by NVIDIA
• Analyzes skills for Claude Code, Codex CLI, Gemini CLI, and other AI agents
• Detects 64 vulnerability patterns across 16 security categories
• Identifies prompt injection, data exfiltration, privilege escalation, supply-chain risks, tool poisoning, memory poisoning, and malicious code
• Static analysis + optional LLM-powered semantic review
• Supports Git repos, ZIP files, URLs, directories, and individual skill files
• Outputs Terminal, JSON, Markdown, and SARIF reports
• Risk scoring with install recommendations (SAFE → DO NOT INSTALL)

Research behind the project found that 26.1% of analyzed AI agent skills contained security vulnerabilities and 5.2% showed indicators of malicious intent.
