## 投稿内容
2025年7月から2026年3月までのClaude Code主要機能の時系列記録。機能がいつ追加されたかを確認するリファレンスとして有用。

**2025年7月:** サブエージェント、Plan Mode、カスタムスラッシュコマンド

**2025年8月〜9月:** チェックポイントとロールバック機能、VS Code拡張機能エクスポート対応

**2025年10月:** Ctrl+Bによる非同期サブエージェント操作（バックグラウンドタスク）

**2025年11月:** Agent Skillsフレームワーク（スキルの初期実装）

**2025年12月:** リモートコントロール（/rc）・定期実行（/loop）・名前付きセッション管理

**2026年1月:** v2.1.0でスコープ付きフック、自動コンパクション（コンテキスト圧縮）、安全な実験用/branch

**2026年2月:** /batchで並列ファイル変換、/simplify（コード簡素化）、Agent Teams（マルチエージェント）、Opus 4.6がデフォルトモデルに

**2026年3月:** /btwでサイドバンド質問（メインコンテキスト汚染なし）、/effortで推論制御（low/medium/high）、MCPエリシテーション（外部ツール承認）対応、Max/TeamプランでMコンテキスト1Mトークン拡張

この記録は「どの機能が本当に新しいのか、どれが確立済みか」を判断する際の基準として使用できる。
