## 要約
Anthropic 公式ドキュメントによる Claude Fable 5 と Claude Mythos 5 の詳細説明（2026年6月9日リリース）。Fable 5 は同社史上最高性能モデルで、テキスト・画像・ファイル対応、1Mトークンコンテキスト、最大128kトークン出力、入力$10/M・出力$50/M の料金設定。Opus 4.8 比でより複雑なタスクで優位。自律コーディング・複雑推論・科学研究に特化した長期間アジェンティック実行、高度ビジョン理解（図表・入れ子テーブル）、自己検証行動を実装。Mythos 5 は Fable 5 と同性能だがサイバー・生物・化学・蒸留リクエストを安全分類器でフラグ立てし承認済みユーザーのみ利用可能。1モデルを2製品に分割した安全設計の詳細を提供。
