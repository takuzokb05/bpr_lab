## 要約
Orca（株式会社オルカ）による CLAUDE.md の実務的な書き方ガイド（2026年）。CLAUDE.md は Claude Code が起動時に自動読み込む「プロジェクト手順書」で、毎回の説明省略・作業品質の均一化に効果的。必須記載項目：テックスタック・ディレクトリ構造・命名規則・build/test/lint コマンド・禁止事項・コーディングスタイル。運用のコツ：500語（日本語なら1000字）以内に収める、「してはいけないこと」を明示する、プロジェクト進行に合わせて継続更新する。また CLAUDE.md の階層配置（プロジェクトルート・サブディレクトリ・グローバル）の使い分けと、チーム共有時の注意点（機密情報を含めない）も解説。
