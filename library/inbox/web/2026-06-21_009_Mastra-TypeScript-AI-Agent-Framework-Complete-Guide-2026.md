## 要約
Mastraは2024年10月にリリースされたオープンソースのTypeScript AIエージェントフレームワーク。2026年1月にv1.0をリリースし、現在22,000+ GitHub stars・週30万npmダウンロードを達成。主な特徴：(1) TypeScriptファーストの設計でReact/Next.js/Node.jsとシームレス統合、(2) 統一モデルルーターで94プロバイダー/3,300超モデルをサポート（Anthropic/OpenAI/Google等）、(3) Vercel・Netlify・Cloudflareへの自動デプロイ、(4) エージェント・ワークフロー・メモリ・ワークスペース・オブザーバビリティを内蔵。Gatsby.js創設者チーム（Sam Bhagwat CEO）が開発、Apache 2.0ライセンス。Claude Agent SDKやLangGraphと異なりTypeScriptバックエンドに最適化された選択肢として注目。
