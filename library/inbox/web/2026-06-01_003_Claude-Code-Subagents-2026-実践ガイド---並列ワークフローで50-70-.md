## 要約
Claude Code のsubagents機能を深掘りした実践ガイド（tembo.io）。Subagentは親エージェントが生成する独立したClaude instanceで、互いに並列実行しながら結果を返す設計。
Background subagents（Ctrl+B）でメイン会話と並列実行が可能。v2.1.117+ からFork mode（CLAUDE_CODE_FORK_SUBAGENT=1）が追加され、親のprompt cacheを共有。
2026年中頃の時点で1開発者あたり4〜8並列worktreeが安定稼働しており、複雑タスクの完了時間を50-70%削減可能と報告。
本番環境では CLAUDE_CODE_SUBAGENT_MODEL 環境変数でコスト上限・コンプライアンス制御が可能。
YAML定義ファイルで再利用可能なsubagent構成を管理し、agent viewダッシュボードで複数セッションを可視化できる。
