## 要約

著名開発者Nader DabitによるClaude Agent SDK完全ガイド（Substack）。ゼロからコードレビューエージェントを構築するハンズオン形式で、コードベース分析→バグ・セキュリティ問題検出→構造化フィードバック生成の3ステップを実装。`pip install claude-agent-sdk`からスタートし、ツール定義・エージェントループ・human-in-the-loop チェックポイントの実装まで網羅。サブエージェント委譲（並列処理）の実装例も含み、ファイル数1,000超のリポジトリを効率的に分析する最適化手法を解説。Agent SDKが「Claude Code SDKと同じハーネス上で動く」ことの実用的な意味（ファイル編集・bash実行・MCP対応が標準搭載）を具体例で示している。
