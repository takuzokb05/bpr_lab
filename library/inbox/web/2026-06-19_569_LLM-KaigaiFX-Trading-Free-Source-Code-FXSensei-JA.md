## 要約
FX Sensei による LLM を活用した海外 FX 自動売買の実装解説記事（ソースコード公開あり）。Codex を用いた FX ボット生成が2026年4月時点で評判が良く、数時間ごとのトレードログを LLM に入力して戦略を改良するイテレーション型の手法を紹介。実装スタックは MetaTrader 5 + Python + LLM API で、MQL5 の制約を Python ブリッジで解決。センチメント分析（ニュースフィード・SNS テキスト）とテクニカル指標を組み合わせた複合シグナル生成の実装例を掲載。バックテスト結果と実運用での差異（スリッページ・約定遅延）についての考察も含む。注意事項として海外 FX 業者の規制リスクと日本の金融庁方針についても言及。
