## 要約
MDN（Mozilla Developer Network）が2026年6月15日に公式MCPサーバーの提供を開始。AIツールがMDNのWebドキュメントを直接モデルコンテキストに取り込めるように。HTML・CSS・JavaScript APIの権威あるリファレンスドキュメントをスクレイピングなしで取得可能。大手リファレンスドキュメントプロバイダーがネイティブMCP統合を実装した重要マイルストーン。Claude CodeやMCP対応クライアントとの統合手順も解説。コーディングエージェントが最新の仕様情報を幻覚なく参照できる実用的なサーバー。
