## 要約
株式会社Uravation による Claude Code 2026年5月アップデートの総整理記事。4大新機能として「Agent view（claude agents コマンド）」「/goal（目標設定）」「/usage のカテゴリ別内訳表示」「/code-review コマンド」の追加を解説。5月28日に登場した Opus 4.8 は SWE-bench Verified を 87.6% → 88.6% に向上させ、/effort の最高設定や Dynamic Workflows との組み合わせで長時間タスクに強化された。Subagent @mention 機能（複数エージェントを並列実行）もアップデートされており、「複数セッションをまとめて回し、達成条件を決めたら自走させる」エージェント運用ツールへの進化を具体的事例とともに紹介。料金対比のコストパフォーマンス分析も掲載。
