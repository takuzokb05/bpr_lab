# QuantaAlpha: An Evolutionary Framework for LLM-Driven Alpha Mining

## 要約
LLM生成クオンタティブアルファファクターを進化的最適化（変異・交叉）でトラジェクトリレベルで反復改善するフレームワーク。仮説生成からコード生成まで一貫したセマンティック整合性を確保。GPT-5.2でCSI 300データを使用し IC=0.0472、ARR=4.68%、最大ドローダウン11.8%を達成。転移ファクターはCSI 500で4年間約40.28%の累積超過リターン、S&P 500で約19.1%を実現しクロスマーケット汎化を確認。バックテストノイズとレジームシフト感度をstructured multi-round探索と複雑さ制約で対処。
