## 要約
Microsoft BUILD 2026（2026年6月2〜3日）の全AI発表をまとめた包括的レビュー（A Guide to Cloud & AI）。主要発表一覧：①**MAI Models**（7システム：MAI-Thinking-1推論モデル35B/256K・MAI-Image-2.5）、②**Microsoft IQ**（Work IQ・Foundry IQ・Fabric IQ・Web IQ統合知識レイヤー）、③**Microsoft Scout**（常時稼働Autopilot型エージェント）、④**Agent 365 SDK GA**（LangChain・OpenAI Agents SDK・LangGraph・SK互換）、⑤**Foundry Toolkit for VS Code GA**（エージェントテンプレート・ローカルデバッグ）、⑥**Majorana 2量子プロセッサ**（前世代比1000倍信頼性・平均寿命20秒）、⑦**GitHub Copilot App プレビュー**（デスクトップ agentic体験）、⑧**Project Rayfin プレビュー**（プロトタイプ→本番の管理バックエンドas-a-service）。AI Agent Framework 1.0（記事110・118）との連続性で読むとMicrosoftのエンタープライズエージェント戦略全体像が理解できる。
