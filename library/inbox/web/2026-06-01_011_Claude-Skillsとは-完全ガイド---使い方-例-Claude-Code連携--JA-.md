## 要約
f2t.jp によるClaude Skills の日本語完全解説ガイド。Anthropicが2025年10月に発表したSkills機能の仕組みから実践まで。
Skills は「再利用可能な専門知識」をClaude Codeに渡す仕組み。.claude/skills/スキル名/SKILL.md に配置し、文脈判断で自動使用またはスラッシュコマンドで呼び出し。
CLAUDE.md との違い: CLAUDE.md=毎回自動読込（常時適用）、skills=必要時のみ（コンテキストトークン節約）。
サポートファイル（テンプレート・出力例・スクリプト）も同フォルダに配置可能。
活用事例: PRマネージャー1人で32人分カバー、広告制作30分→30秒短縮。
2025年10月以降の最新機能として国内でも急速に普及中。
