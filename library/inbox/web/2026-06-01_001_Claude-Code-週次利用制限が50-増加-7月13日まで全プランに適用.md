# Claude Code 週次利用制限が50%増加 7月13日まで全プランに適用

- URL: https://apidog.com/blog/claude-code-weekly-limits-50-percent-increase-july-2026/
- ソース: web
- 言語: en
- テーマ: claude-code
- 取得日: 2026-06-01

## 要約
Claude Code の週次利用制限が即時50%増加し、2026年7月13日18:00 PDT まで全有料プラン（Pro・Max・Team・Enterprise）に適用される。
CLI・IDEエクステンション・デスクトップアプリ・Webの全サーフェスに適用、オプトイン不要。
直前に5時間上限が2倍になっており、両施策が重なり短期ウィンドウで実質2倍・週次上限で1.5倍の効果。
価格据え置きのままレート制限を緩和した背景には、OpenAI Codex との競争激化がある。
Hacker News でも話題になり、pro ユーザーが weekly cap の詳細な数値を報告。
7月13日以降はデフォルトに戻ることが想定され、期間限定の恩恵として位置付けられている。
