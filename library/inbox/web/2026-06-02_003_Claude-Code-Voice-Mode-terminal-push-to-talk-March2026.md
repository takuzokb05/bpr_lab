## 要約
TechCrunchによるClaude Code Voice Mode公式発表レポート（2026年3月3日）。
スペースキー長押しで音声入力ができるPush-to-Talk機能で、20言語対応。ターミナル向け音声インターフェースのため、会話型音声ではなくテキスト変換（STT）に特化。
Pro・Max・Team・Enterpriseプランで利用可能。2026年3月に少数ユーザーへのロールアウト開始し、その後広く展開。
用途：コーディング中に手を使わずにプロンプト入力、長い指示の音声入力、ハンズフリーのコードレビュー依頼など。
Claude Code Docs: voice modeはクロスプラットフォーム対応（Mac/Linux/Windows）で設定不要。ターミナルフォーカス時にスペースキーを長押しすることで起動する。
