## 要約
GitHub Changelog（2026年6月30日）によるClaude Sonnet 5のGitHub Copilot GA発表。Copilotでのコーディングパフォーマンスが向上し、CLI形式タスクで特に強い結果。ゼロデータリテンション（ZDR）ポリシー適用。対象プラン：Copilot Pro/Pro+/Max/Business/Enterprise。あわせてClaude Code新機能：Checkpoints（最もリクエストの多かった機能—進捗保存と即時ロールバック）、ターミナルインターフェース刷新、ネイティブVS Code拡張のリリース。GitHub Copilot vs Claude Code 2026の比較：Claude Codeは自律タスクと100万トークンコンテキストで優位、Copilotはより多くのIDE対応・GitHub統合・インライン補完が有料プランで無料の点で優位。
