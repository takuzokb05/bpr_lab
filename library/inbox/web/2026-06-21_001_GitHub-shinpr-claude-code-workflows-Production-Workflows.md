## 要約
GitHubリポジトリ `shinpr/claude-code-workflows` は、Claude Codeに特化した本番環境対応の開発ワークフロー集。専門AIエージェントで動作するプロダクションレディなワークフローを提供。コードレビュー自動化・テスト実行・デプロイパイプラインなどのワークフローがSKILL.md形式で実装されており、Claude Codeのマルチエージェント機能（subagents）を活用している。開発チームが日常的に使うワークフローをClaude Code上でどう標準化するかの実践的リファレンス。MCP・Hooks・Skillsを統合した構成で、コードベース探索から本番デプロイまでをカバーする。
