## 要約
株式会社ゼロイチラボによるClaude Codeスキル活用術（日本語）。「作る」から「育てる」への運用思想を軸に解説。初期スキル作成（SKILL.md基本構造）→実運用でのフィードバック収集→スキルの反復改善という育成サイクルを提案。実際のプロジェクトで使用しているスキル事例（バグレポート自動生成、API仕様書生成、コードフォーマット確認）を公開。スキルとCLAUDE.mdの役割分担：CLAUDE.mdはプロジェクト全体のコンテキスト、スキルはタスク固有の専門知識と明確に区別。チーム展開のベストプラクティスとして、スキルをGitリポジトリで管理し、CIでSKILL.mdのlintを行う方法を解説。
