# イリノイ州SB 315可決: 米国初・フロンティアモデル第三者監査義務化

- URL: https://www.nbcnews.com/tech/tech-news/illinois-legislature-passes-historic-ai-bill-rcna347191
- ソース: web
- 言語: en
- テーマ: ai-news
- 取得日: 2026-05-28

## 要約
イリノイ州下院が2026年5月27日、SB 315（AI安全対策法）を110-0の全会一致で可決（上院は5月21日可決済み）。米国初・フロンティアモデルへの年次独立第三者監査義務化法案。対象：年収5億ドル超でカタストロフィックリスクが可能なフロンティアモデルを展開するAI企業。義務要件：年次監査・監査結果の公開・リスク評価サマリーの定期提出・セーフティインシデント72時間以内の報告。違反ペナルティ：最大$300万/件（州司法長官が唯一の執行権限、私人訴訟不可）。施行は2027年から2028年に延期。Pritzker知事の署名待ち。OpenaI・Google・Anthropic等が対象になる可能性。
