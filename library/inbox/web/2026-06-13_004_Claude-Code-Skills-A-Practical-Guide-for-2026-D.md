## 要約
DEV Community掲載のClaude Code Skills実践ガイド。スキルの本質的定義：SKILL.mdファイルを持つフォルダで、YAMLフロントマターにname/descriptionを記載し、本文にMarkdown形式の指示を書く。プロンプトがdescriptionにマッチすると自動ロードされる仕組みを解説。スキル vs スラッシュコマンド vs サブエージェントの違いを比較表で整理。実際のSKILL.mdの書き方例（コードレビュー、テスト作成、デプロイスクリプト生成）を提示。スキルをチームで共有する際のGitリポジトリ管理方法、~/.claude/skills/（グローバル）vs .claude/skills/（プロジェクト）の使い分けを解説。
