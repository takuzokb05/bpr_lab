## 要約
2026年6月のClaude Code CLIリリーストレイン(2026.6)で追加された10の新機能を解説するSitePoint記事。主要新機能は：
1. **ネストサブエージェント（3階層）**：親エージェントが子エージェントを最大3レベルまで生成可能、複数モジュール横断タスクの分解に対応
2. **fallbackModelチェーン**：プライマリモデル失敗時の自動フォールバック設定
3. **コミュニティツールマーケットプレイス**：ツールの発見・共有プラットフォーム
4. **エージェント別コスト帰属**：どのサブエージェントにいくらかかったか追跡可能
5. **スコープ付きパーミッション**：エージェント単位の権限制御
その他、エージェントチェックポイント・ストリーミングログ・マルチリポジトリオーケストレーション（ベータ）も含む。GAとベータの機能区分を明確に説明しており実装ガイドとしても有用。
