## 要約
2026年6月22日付のGitHub公式Changelogで、JetBrains IDE向けGitHub Copilotに複数の新機能が追加されたことが発表された。最大のハイライトは**ClaudeをAgentプロバイダーとして利用するパブリックプレビュー**の開始で、JetBrains IDE上でClaudeモデルをエージェントとして直接使用できるようになる。その他の新機能として、組織・エンタープライズレベルのカスタムエージェント対応、Copilot CLIでのメッセージキュー機能、エージェントデバッグログの統合ビュー、モデルピッカーの機能強化、ターンごとのAIクレジット表示、クラウドエージェントの一般提供開始が含まれる。これはGitHub CopilotとAnthropicのエコシステムの統合が深化していることを示す重要な一次情報。Claude Codeユーザーにとって、既存のJetBrains開発環境との連携経路が広がることを意味する。
