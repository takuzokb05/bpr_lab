## 要約
v2.1.169（2026年6月8日）対応のClaude Code CLIの完全テクニカルリファレンス（56,000語超）。5コアシステム解説：(1)設定（CLAUDE.md階層）、(2)パーミッション、(3)フック（25以上のライフサイクルポイント）、(4)MCP、(5)サブエージェント。フックはプロンプトではなく「必ず実行が必要なもの」に使用、MCPはツールチェーン拡張、スキルはドメイン専門知識の自動適用。v2.1.169新機能：--safe-mode（全カスタマイズ無効トラブルシューティング）、/cd（プロンプトキャッシュ保持ディレクトリ変更）、disableBundledSkills（バンドルスキル非表示）。Claude Code as infrastructure思想に基づいた開発ハーネス構築の実践参考書。
