# FX自動売買BotのローカルLLMをqwen3.5:9bからgemma3:12bに変更した話

- ソース: web
- 言語: ja
- テーマ: ai-trading
- 取得日: 2026-06-03

## 要約
個人開発者によるFX自動売買ボットのローカルLLM切替実践レポート。ローカルLLM採用理由：①相場稼働中はボットを止められない（外部API障害リスク排除）、②APIコスト削減、③自社ロジック・市況データの外部送信を避けるセキュリティ配慮。qwen3.5:9bからgemma3:12bへの切替理由と実際のパフォーマンス比較を詳述。EURUSD：LLMに売買プランそのものを作らせるアプローチ、GBPUSD：CatBoostなどの機械学習モデルとLLMを組み合わせたグリッド型ボット。2026年4月時点でCodexが最も評判良い外部LLMとされるが、急速に変化するため評判の良いものを切り替えながら使うことを推奨。ローカルLLMをFX自動売買に組み込む際の実践的知見として希少。
