## 要約
Model Context Protocol公式ブログによる2026年ロードマップ。2026年3月時点でMCPは月間SDKダウンロード9700万件超、GitHubスター81,000+を達成し、Anthropic・OpenAI・Google・Microsoft・AWSの全主要ベンダーがサポート。コミュニティ製MCPサーバーはGitHub・Slack・PostgreSQL・Stripe・Figma・Docker・Kubernetesなど200以上。2026年優先事項：①Streamable HTTP（サーバーをリモートサービスとして実行可能にするトランスポート）、②エンタープライズ展開の課題解決（監査証跡、SSO統合認証、ゲートウェイ動作、設定ポータビリティ）。2026-07-28 RC仕様変更（既報P-017）の背景情報として一次情報。
