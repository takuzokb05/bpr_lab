## 要約
Claude Agent SDKとMCPサーバー接続の実践ガイド。Streamable HTTP（旧HTTP+SSEを置き換え）がクラウドホスト型MCPの標準トランスポートに。OAuth 2.1がリモートMCPサーバーの認証標準として採用。ローカルプロセス・HTTP・SDK内直接実行の3トランスポートタイプの使い分け基準を解説。GitHub・Slack・データベース接続のコード例付き。AgentSDK+MCPの組み合わせで外部API呼び出しをカスタムツール実装なしに実現する方法を紹介。
