## 要約
The New StackによるClaude Code Routines機能の詳細解説。Routinesは「保存されたClaude Code設定（プロンプト・リポジトリ・コネクター）をパッケージ化し、Anthropicのマネージドクラウドインフラで自動実行する機能」。3種類のトリガー：①Scheduled（時間単位/夜間/週次など）、②API（HTTPポストで即時起動）、③GitHub（PR・リリースイベント連動）。重要な示唆：ラップトップを閉じていてもRoutineは継続実行。ユースケース：夜間テスト実行、定期レポート生成、PRレビュー自動化。このセッション自体がRoutines/Scheduled Triggerで起動している実例に相当。
