## 要約
AI顧問社によるClaudeプラン変更（2026年6月15日）の日本語解説。最重要変更点：Claude Agent SDK・`claude -p`（非対話型）の利用が対話型利用制限とは別の「Agent SDKクレジット」から引き落とされるようになった。プランごとのAgent SDKクレジット：Pro $20/月、Max 5x $100/月、Max 20x $200/月、Teamは組織管理者が設定。クレジット枯渇後は標準APIレートへフォールバック（有効にしている場合）。Claude Code通常使用（対話型）には影響なし。対話型のClaude Code・Claude Cowork・Claudeの利用制限は従来通り。日本ユーザーへの影響：APIを組み込んだ自動化ツール・バッチ処理・定期実行スクリプトがAgent SDK扱いになるため、月間使用量の再計算が必要。移行前後の課金差異シミュレーション例と推奨対応手順付き。
