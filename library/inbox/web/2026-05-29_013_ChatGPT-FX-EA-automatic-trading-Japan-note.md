# ChatGPT売買判断を定期連携したFX自動取引EAの開発（国内個人実践事例）

## 要約
ChatGPT（生成AI）の売買判断を定期的にMT5 EAと連携させてFX取引を自動化した国内個人開発者による実践報告。MT5側EAはChatGPTのAPI応答を解析してBUY/SELL/HOLDシグナルを受信し、指定ロットで自動発注する仕組み。ChatGPT側にはOHLC・ボリューム・主要テクニカル指標（RSI・MACD・BB）を提供し、市場状況の総合判断を依頼。定期的なポーリング（15分足・1時間足）でシグナルを更新しトレンドフォローに対応。プロンプト設計のコツとして「具体的な出力フォーマット指定（JSON）」「リスク許容度の明示」「なぜその判断をしたかの理由要求」を挙げる。ChatGPT-5との組み合わせでFX判断精度が向上したと報告。2026年版テンプレートとGoogleColab対応バックテストツールを提供。国内個人がMT5+LLMを組み合わせた具体的な実装として参考価値が高い。
