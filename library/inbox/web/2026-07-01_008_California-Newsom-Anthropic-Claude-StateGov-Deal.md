# カリフォルニア州、Anthropic と提携——全州機関・市区町村に Claude を50%割引で提供（30万人規模）

## 要約

カリフォルニア州知事Gavin Newsomが2026年6月29日に発表。Anthropicと州政府の提携により、全州機関・参加する市区町村（約30万人の州職員）がClaudeを50%割引で利用可能に。Californiaは米国最大のAI州政府展開事例となる。提供窓口は加州技術省新設の「SITeS（Statewide Information Technology Shared Services）ポータル」で、Claudeが最初に登録されたAI生産性ツール。割引に加え、無料ワークフォーストレーニングとAnthropicエンジニアによるGenAI技術支援を提供。既に利用中の機関：カリフォルニア技術省・緊急サービス局（サイバー防衛）、DMV（顧客サービス改善・待機時間削減）、医療サービス局（米国最大のメディケイド機関、内部ワークフロー改善）。同州は2月にClaude Codeのテストユーザーとして提携済み。AnthropicのIPO（2026年10月目標）前の大型政府契約として注目。
