## 要約
TokenMixによるMCP 2026仕様変更の全変更点チェンジログ。9つの主要変更：①ハンドシェイク廃止、②セッションIDヘッダー削除、③_metaフィールド追加、④Extensions framework（reverse-DNS ID方式）、⑤Tasks extension（tasks/get, tasks/update, tasks/cancel）、⑥MCP Apps（初の公式Extension）、⑦Roots非推奨、⑧Sampling非推奨、⑨Logging非推奨。各変更の移行マップ（before/after）付きで、MCPサーバー開発者向けの実用的な移行ガイドとして有用。Tier 1 SDKは10週間以内に対応予定。
