## 要約
AntropicがClaude CodeをどのようにAnthropicチームが利用しているかを説明した公式PDFドキュメント。主要なワークフロー：git worktreeを使った並列マルチセッション、PRレビューでの複数エージェント同時ディスパッチ（ロジック・セキュリティ・パフォーマンスをそれぞれ専門エージェントが担当）、`/permissions`で事前許可した安全コマンドの`.claude/settings.json`管理。Anthropic内部での実際の利用パターンを公式PDFとして開示した一次情報。チームでのClaude Code活用を始める際の参照文書として最も信頼性が高い。
