# Claude Code Auto Mode: 2段階ML分類器で実現した自律コーディング

- テーマ: claude-code
- 取得日: 2026-05-30

## 要約
AnthropicがClaude CodeにAuto Modeを導入した。従来の人間承認プロンプトをML分類器に置き換え、毎ツール呼び出しをリアルタイム評価する。アーキテクチャは2層構成：入力層（ファイル読み込み・シェル出力・Web応答を検査し悪意ある内容をブロック）と実行層（提案アクションをfastフィルタ→詳細分析の2段階分類器で評価）。安全と判定されれば自動実行、曖昧・危険と判定されれば安全パスを探索または人間に確認。サブエージェントワークフロー向けには送出前バリデーションと完了後チェックでプロンプトインジェクションを検出。Proプランに2026年5月中旬から展開、Sonnet 4.6をサポート。SpinnerがPermissionチェック時に赤く変わるUI変更も含む。「コーヒーブレイクで本当に離席できる」水準の自律度を実現。
