# Claude Code Artifacts 公式発表 — AIセッションをライブ共有ページに変換

- URL: https://claude.com/blog/artifacts-in-claude-code
- ソース: web
- 言語: en
- テーマ: claude-code
- 取得日: 2026-06-20

## 要約
Anthropic が 2026年6月18日、Claude Code Artifacts をベータリリース（Team/Enterprise プランのみ）。Claude Code のセッション作業をプライベートURLの自分更新型HTMLページに変換する機能。PR ウォークスルー・システム解説・ダッシュボード・リリースチェックリスト等をチームでリアルタイム共有可能。共有はデフォルト非公開で同組織認証メンバーのみアクセス可能。Enterprise では管理者がロール別アクセス制限・保持期間設定・Compliance APIでの監査が可能。アーティファクトはセッション進行にあわせて自動更新される点が既存ドキュメント共有との違い。Claude.com の Artifacts 機能を Claude Code に拡張した位置付け。
