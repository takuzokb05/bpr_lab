## 要約
Nimbalyst による Claude Code Skills の実践ガイド（2026年版）。スキルの基本構造（SKILL.md + YAML フロントマター）から始まり、プロジェクト別カスタムスキルの設計・チーム展開・CLAUDE.md との使い分けを説明。スキルは「アドバイス的」でClaude が状況に応じて選択・適用するのに対し、Hooks は「決定論的」で必ず実行される点を強調。Skills でアーキテクチャ上の判断を封じ込め、反復作業の質を均一化できる。実際のプロジェクトへの適用例（コードレビュー自動化・PR 説明生成・デプロイチェック）を含む実践的内容。
