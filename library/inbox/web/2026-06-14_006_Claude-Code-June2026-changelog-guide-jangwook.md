## 要約
jangwook.netがまとめた2026年6月のClaude Code変更ログ詳細。主要アップデート: (1) Safe Mode追加（不安定なAPIレスポンス時にフォールバックモデルで1回リトライ）、(2) /cdコマンド追加、(3) Opus 4.8がデフォルトモデルに昇格、(4) /usageコマンドでカテゴリ別使用量確認が可能に、(5) v2.1.172（6月10日）でネストサブエージェントが最大5段階まで対応。その他: プラグイン検索機能改善、Chrome/VSCode/ターミナルワークフロー強化、Fable 5一時追加→政府指令により停止。セッション固着バグ・image処理エラーの修正も含む。開発者向けに変更点を時系列で整理した実用的リファレンス。
