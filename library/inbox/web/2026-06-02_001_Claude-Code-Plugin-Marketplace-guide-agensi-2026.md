## 要約
Claude CodeのPlugin Marketplace（Pluginsディレクトリ）の実践ガイド（agensi.io）。Pluginsは2026年5月にpublic betaとして公開され、スキル・サブエージェント・スラッシュコマンド・フック・MCPサーバーを1パッケージで配布できる新形式。
公式Anthropicマーケットプレイスには101のプラグイン（Anthropic製33、パートナー製68）が収録。GitHubやPlaywright、Supabase、Figma等の有名サービスが対応済み。
`/plugin`コマンド→Discoverタブで検索・インストールが可能。コミュニティ全体で77マーケットプレイス・1,208プラグインが存在（2026年6月1日時点）。
既存のSkills/MCPサーバーとの違い：Pluginsはこれら全てを1ファイルにまとめて配布・発見可能にする包括的フォーマット。
開発者にとってはスキルやMCPサーバーをプラグイン形式にラッピングして公開することで発見可能性が大幅向上する。
