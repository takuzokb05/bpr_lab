## 要約
The New Stackによる「Claude Dreaming」機能の詳細解説。Anthropicが2026年5月6日のCode with Claude 2026イベントで発表したManaged Agents向け新機能。
Dreamingは「スケジュールされたメモリ統合プロセス」で、エージェントが過去セッションを自動レビューし、重要な学習内容を抽出・再構成して次の会話前にメモリを更新する。
抽出対象：①繰り返す間違い、②収束するワークフロー、③チームレベルの嗜好と慣行。メモリの断片化・矛盾・陳腐化を防ぐ「メモリ再構成」機能も含む。
生物学的な睡眠中メモリ統合との類比。現在はデベロッパーアクセスプログラム経由のみ利用可能。
Outcomes機能（タスクの評価ルーブリック定義→自動再試行）と組み合わせることで継続的改善ループを実現。これは「エージェントが経験から学習する」仕組みの実装例として重要な一次情報。
