# Look-Ahead-Bench: A Standardized Benchmark of Look-Ahead Bias in Point-in-Time LLMs for Finance

- URL: https://arxiv.org/abs/2601.13770
- ソース: web
- 言語: en
- テーマ: ai-trading
- 取得日: 2026-06-28

## 要約
金融予測に使用するLLMの時間的ルックアヘッドバイアスを定量化するベンチマーク。時間的に分離した市場レジームにわたる性能劣化をクオンタティブベースラインと比較測定。標準モデル（Llama 3.1 8B/70B、DeepSeek 3.2）はアルファ減衰指標で有意なルックアヘッドバイアスを示した。専門化「ポイントインタイム」モデル（Pitinf-Small/Medium/Large）はモデルサイズとともに汎化が大幅改善。コード付きで再現可能なフレームワークを提供し、バックテスト汚染評価ではなく実際の金融展開に適したLLMを判定する標準手法を確立。
