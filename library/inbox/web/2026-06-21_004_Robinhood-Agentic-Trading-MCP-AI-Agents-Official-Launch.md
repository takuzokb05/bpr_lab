# Robinhood Agentic Trading MCP AI Agents Official Launch

- URL: https://robinhood.com/us/en/newsroom/robinhood-is-now-open-to-agents/
- ソース: web
- 言語: en
- テーマ: ai-trading
- 取得日: 2026-06-21

## 要約
Robinhoodが2026年5月27日、AIエージェントによる自律取引を可能にする「Agentic Trading」を正式ローンチ。MCP（Model Context Protocol）サーバーを介して第三者AIエージェントをRobinhoodの専用ブローカーアカウントに接続し、株式売買を完全自動化できる。主要な安全機能：(1)独立した専用口座（メイン資産は隔離）、(2)ワンタップのキルスイッチで即時切断、(3)不正検知AIがエージェントの指示と実行を照合。株式（equities）のみでスタートし、オプション・暗号資産・先物は順次対応予定。同日「Agentic Credit Card」も発表：エージェントが事前設定の上限内でクレジットカード支払いを実行可能。Robinhood CEOは「agentic financeの始まり」と表現。これはメインストリームの小売投資プラットフォームが実資金への外部AIエージェントアクセスを公式に認めた最初期の事例。
