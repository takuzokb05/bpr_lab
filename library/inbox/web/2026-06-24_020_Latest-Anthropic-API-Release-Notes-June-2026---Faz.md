## 要約
Fazm AIによるAnthropic API 2026年6月リリースノートのキュレーション。主要アップデート：①refusal課金停止（stop_reason: refusalで出力なしのリクエストは非課金）、②サーバーサイドフォールバックベータ（refused requestが同一ラウンドトリップで別モデルに自動リトライ）、③Rate Limits API（管理者が組織・ワークスペース単位のレート制限を照会可能）、④MCP Tunnels（プライベートネットワーク内MCPサーバーへのリモートアクセス、研究プレビュー）、⑤モデル非推奨（Claude Sonnet 4・Opus 4を6月15日廃止）。API利用者は特にモデル移行と新フォールバック機能の活用を要チェック。
