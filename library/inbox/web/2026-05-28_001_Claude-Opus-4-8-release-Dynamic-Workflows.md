## 要約
Anthropicが2026年5月28日にClaude Opus 4.8をリリース。1Mトークンコンテキストウィンドウ、SWE-bench Pro 69.2%（Opus 4.7の64.3%から+4.9pt）、USAMO 2026数学で96.7%（Opus 4.7は69.3%）を達成。コードの欠陥見落とし率がOpus 4.7比で4分の1に低下。ハイブリッド推論モデルで価格は入力$5/Mトークン・出力$25/Mトークン（Opus 4.7と同額）。Fast modeは旧版比3倍安価・2.5倍高速で$10/$25/Mトークン。プロンプトキャッシュで最大90%コスト削減。Anthropicは同日にDynamic Workflows（Claude Code）とEffort Control（claude.ai/Cowork）も同時リリース。IPO競争激化の中での戦略的大型リリース。
