## 要約
iret.media（クラスメソッドグループ）によるClaude Code 4拡張点の実践解説。MCP・Skills・Hooks・Subagentsそれぞれの役割と使い分けを図解。MCP: 外部ツール・API接続、Skills: 繰り返しワークフローのテンプレート化、Hooks: ライフサイクルイベントへの自動応答、Subagents: 並列タスク処理。各拡張点の実装難易度・ユースケース・組み合わせ方を解説。特にHooks（32種類のイベント）の活用例として、コミット前の自動テスト実行、ファイル変更時のlint実行などを紹介。JA圏では数少ない4拡張点を体系的に比較した技術記事。実際のプロジェクト（AWSインフラ構築）での活用事例も含む。
