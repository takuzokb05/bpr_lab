## 要約
Microsoft BUILD 2026（6月2〜3日、サンフランシスコ）での公式Agent Frameworkブログ発表。主要アップデート：①**Microsoft IQ**（Work IQ・Foundry IQ・Fabric IQ・Web IQ統合知識レイヤー）GA、②**Microsoft Scout**（Autopi lotカテゴリ初エージェント・Teams/Outlook統合・会議準備自動化）GA、③**MAI-Thinking-1**（Microsoftが独自訓練した推論モデル・35B active params・256Kコンテキスト）リリース、④**Agent 365 SDK**（LangChain・OpenAI SDK・LangGraph互換）GA、⑤**Foundry Toolkit for VS Code**（エージェントテンプレート・ローカルデバッグ・デプロイ一体化）GA。Microsoft Agent Framework 1.0（記事110）に続くBUILD期間中の追加機能展開の公式一次情報。OpenAI・Anthropicとの競合関係でのMicrosoftのエンタープライズAgent戦略全容が把握できる。
