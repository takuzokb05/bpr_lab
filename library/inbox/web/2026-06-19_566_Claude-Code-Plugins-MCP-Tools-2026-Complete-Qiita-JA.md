Qiita に投稿された Claude Code 強化のためのプラグイン・MCP・ツール網羅リスト（2026年版）。2026年4月時点でセッション間のコンテキスト消失問題を解決するメモリ系プラグインが急増しており、AI 圧縮によるセッション引き継ぎ機能を持つソリューションの実装例を紹介。MCP ツール検索機能（遅延読み込みでコンテキスト使用量を最大95%削減）を活用した最適化パターンも詳解。GitHub・Slack・Google Drive・Notion・PostgreSQL・Linear・Jira 等の主要 MCP サーバー設定手順を一覧化。自社 API を MCP サーバーにラップする実装例（社内 CRM/ERP 連携）も掲載。プラグインの選定基準（パフォーマンス・セキュリティ・メンテナンス頻度）のチェックリスト付き。
