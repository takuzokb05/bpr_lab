## 要約
marmelab（フランスの技術会社）によるClaude Code実践ティップス記事（2026年4月24日）。初日から知っておきたかった教訓を網羅。主要ポイント：①プランモード活用（"dont implement yet"で仕様書を先に作成し、12ステップ仕様書作成に2時間かけて実装時間を6-10時間節約）、②コンテキストウィンドウ管理（サブエージェントでリサーチ分離、新タスクは新セッション開始）、③MCP接続でNotion/Figma/DBなど外部ツール連携、④HooksとCLAUDE.mdの役割分担（Hooksは例外なく実行が必要なアクション向け）、⑤検証は必ずコード外で実施（テスト・スクリーンショット）。スコープが明確な小タスクはプランモード不要。
