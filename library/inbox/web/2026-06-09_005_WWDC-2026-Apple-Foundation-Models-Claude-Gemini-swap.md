# WWDC 2026: Apple Foundation Models now swaps Claude, Gemini without code changes

## 要約
WWDC 2026（2026年6月9日）でAppleがFoundation Modelsフレームワークをサードパーティクラウドプロバイダーに開放。新しいLanguageModelプロトコルでiOS 27/macOS 27/iPadOS 27対応。Anthropic（Claude）、Google（Gemini via Firebase SDK）が実装。開発者はSwift Package Managerの依存関係を更新するだけでプロバイダー変更可能（セッションコード変更不要）。Xcode 27はデュアルエンジン採用：ローカルNeural EngineモデルでリアルタイムSwift提案 + クラウドルーティングで重い分析をClaude/Gemini/OpenAIへ。
