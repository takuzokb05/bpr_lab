## 要約
ZennのimohukeDraw.io, Filesystem, Serenaなどを含む実務エンジニア向けMCPサーバー厳選まとめ。選定基準：①実務で継続使用できる安定性、②セキュリティリスクの低さ、③Claude Code / Claude Desktop両対応。2026年時点で注目の定番サーバー（GitHub公式、AWS公式、MDN公式、Filesystem、Draw.io、Serena等）を用途別に整理。設定例（claude mcp add コマンド）付き。14,000+のMCPサーバーが乱立する中で品質基準を設けて選別しており、実務導入の参考として価値が高い。
