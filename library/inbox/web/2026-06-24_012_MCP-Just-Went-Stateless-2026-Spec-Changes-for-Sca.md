## 要約
Microsoft Tech CommunityによるAzure App Service視点でのMCPステートレス化解説。従来のMCP実装ではWebSocketセッションの維持が必要でAzure App Serviceのスケールアウトに支障があったが、2026仕様でのステートレス化により複数インスタンス間でリクエストを自由にルーティング可能に。具体的なAzure App ServiceへのMCPサーバーデプロイ手順を掲載。認証はEntra IDと統合、Rate Limits APIとの連携方法も解説。Microsoft/Azureエコシステムでのエンタープライズ級MCP導入の実践ガイドとして価値が高い。
