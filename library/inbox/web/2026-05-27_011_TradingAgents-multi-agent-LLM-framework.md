## 要約
UCLA Tauric ResearchによるTradingAgents v0.2.4（2026年4月25日リリース）。LangGraphベースのマルチエージェントLLMフレームワークで実際の証券会社を模倣。5層・約12エージェント構成：ファンダメンタル・センチメント・テクニカルアナリスト×複数、リサーチャー、トレーダー（異なるリスクプロファイル）、リスクマネージャー（7役割）。7つのLLMエージェントが互いに議論しコンセンサスによるトレード決定。LLMバックエンド：GPT・Claude・Gemini・Grokをサポート。パフォーマンス実績：AAPL対象（2024年6〜11月）で累積リターン26.62%（バイアンドホールドの-5.23%を大幅アウトパフォーム）。GitHubスター51,300超・フォーク9,300超（2026年4月時点）。LLMが非構造化情報（決算説明会・ニュース・ソーシャルセンチメント）と構造化価格データを単一意思決定パイプラインで処理できる点が革新的。FX自動取引開発の参考として直接参照すべき重要フレームワーク。
