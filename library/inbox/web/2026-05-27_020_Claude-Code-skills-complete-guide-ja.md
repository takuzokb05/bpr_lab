## 要約
Claude CodeのSkills機能を徹底解説した日本語完全ガイド2026年版。スキルは/スキル名一発で繰り返し業務を自動化できるClaude Code拡張機能。作り方・設計パターン・活用パターンを体系的に解説。「育てる運用」の概念：使い回しながら継続的に改良・精緻化するアプローチ。公式docs（code.claude.com/docs/ja/skills）と対応した内容で日本語解説として有用。具体的活用パターン：コードレビュー自動化、ドキュメント生成、デプロイチェックリスト実行、デバッグプレイブック。Agent Skillsの概念（AI総合研究所の記事とも対応）では複数のサブエージェントを活用した高度な自動化が可能。Cowork（GUIモード）との組み合わせでノンエンジニアでも活用可能。日本語圏のClaude Code Skills情報の主要参照先として価値が高い。
