## 要約
wshobson氏によるオープンソースのマルチハーネス・エージェンティックプラグインマーケットプレイス。Claude Code・Codex CLI・Cursor・OpenCode・GitHub Copilot・Gemini CLIに対応した統一プラグインリポジトリ。各AIコーディングツール間でスキル・エージェント・ワークフローを横断して再利用できる標準化フォーマットを提供。複数のAI開発ツールを使い分けるチームが共通のカスタム命令・スキル・エージェント定義を一元管理できる。Claude Code固有のskills形式・MCP設定・フック定義も含む。AIコーディングツールのエコシステムが分散する中で、マルチツール対応の統一化を目指す実験的プロジェクト。
