## 要約
チームが実際に使用しているClaude Codeセットアップの公開。30のスキルファイル、複数のMCPサーバー、自己学習型フックで構成。自己学習型フックとはセッション終了時にClaudeが得た知見を自動的にCLAUDE.mdへ追記する仕組みで、知識ベースがセッションをまたいで蓄積される。CLAUDE.md設計の具体例、スキルの分類方法（コードレビュー・デプロイ・DB操作等）、MCPサーバーの組み合わせパターンを紹介。チーム運用での知見共有・標準化の実践例として参考度高い。
