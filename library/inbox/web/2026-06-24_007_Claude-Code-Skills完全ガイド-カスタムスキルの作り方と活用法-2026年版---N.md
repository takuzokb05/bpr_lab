## 要約
Nexa CorpによるClaude Codeスキル機能の完全解説。スキルの定義（特定タスクの専門知識・手順をパッケージ化してClaude Codeに渡す機能）から始まり、カスタムスキルの作成手順、.claude/skills/配下への配置方法、CLAUDE.mdからの参照方法を詳説。重要な点は「ユーザーが毎回指示しなくてもClaude Codeが自律的にスキルを選択・使用する」という動作原理。プロジェクト固有のルール・ワークフローをスキルとして定義することで、チーム間での一貫した開発プロセスを実現できる。
