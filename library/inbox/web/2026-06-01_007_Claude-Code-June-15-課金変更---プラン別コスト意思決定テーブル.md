## 要約
6月15日以降のClaude Code課金変更を受け、プランごとの最適選択をマトリクス形式で整理した実用記事。
インタラクティブ利用（CLI・IDE・デスクトップ・Web）はサブスクリプション上限を維持。
プログラム的利用（claude -p、Agent SDK、GitHub Actions、サードパーティエージェント）は新設の月次クレジットプールに移行（フルAPI価格）。
月次クレジット量: Pro $20、Max 5x $100、Max 20x $200、Team Standard $20/seat、Team Premium $100/seat。ロールオーバーなし・共有不可。
claude -p の実際の月次トークン消費量から「クレジットが足りるか」を逆算する計算式を提示。
インタラクティブ利用中心なら影響ほぼゼロ、バッチ処理・自動化が主用途のユーザーは実質値上げとなる分岐点を明示。
