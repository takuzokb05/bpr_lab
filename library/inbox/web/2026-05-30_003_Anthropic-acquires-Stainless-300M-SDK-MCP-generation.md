# Anthropic、Stainlessを$300M超で買収: SDK・MCPサーバー生成基盤を内製化

- URL: https://www.anthropic.com/news/anthropic-acquires-stainless
- ソース: web
- 言語: en
- テーマ: claude-ecosystem
- 取得日: 2026-05-30

## 要約
Anthropicが2026年5月18日、SDK自動生成スタートアップのStainlessを$300M超で買収したと発表。Stainlessは2022年創業（元Stripe社員Alex Rattray）で、Anthropic全公式SDKのTypeScript・Python・Go・Java・Kotlin生成を担ってきた。OpenAI・Google・Cloudflareも同社ツールを使用しており、買収によりこれらのライバルへのサービス提供は終了（顧客は生成済みSDKの所有権・改変権を保持）。Anthropicの戦略的意図は「エージェントが接続できるシステムの拡大」で、モデル層からランタイム・SDK・MCPサーバー生成ツール層へのスタック下降統合。2025年12月のBun買収（Claude Codeの実行ランタイム）に続く第2弾。プラットフォーム・エンジニアリング責任者は「エージェントの有用性はそれらが接続できるものの質に左右される」と述べた。
