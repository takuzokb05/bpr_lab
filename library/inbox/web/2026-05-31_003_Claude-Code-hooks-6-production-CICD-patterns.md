## 要約
Pixelmojo による Claude Code Hooks の本番CI/CDパターン集。Command hooks（直接チェック）・Prompt hooks（セマンティック評価）・Agent hooks（ツールアクセスを伴う深い分析）の3層構造を解説。6つの具体的な生産パターン：セキュリティゲート、ファイル保護、必須レビュー強制、コードフォーマット自動化、通知送信、コマンドバリデーション。PreToolUseによるブロッキングを活用した品質保証ワークフローの実装例あり。CI/CDパイプラインへのHooks統合の実践ガイド。
