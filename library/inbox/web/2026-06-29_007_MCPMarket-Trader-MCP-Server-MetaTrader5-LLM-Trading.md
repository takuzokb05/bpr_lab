# Trader MCP Server: AI LLMでMetaTrader 5を操作するMCPサーバー

## 要約
MCPMarket掲載のTrader MCP Serverは、MetaTrader 5（MT5）をModel Context Protocol経由でAI LLM（Claude・GPT等）から操作可能にするMCPサーバー。機能：リアルタイム価格取得、注文執行（成行・指値・逆指値）、ポジション管理、EA（Expert Advisor）制御、口座情報参照。Claude Codeから直接FX取引を発注・管理するユースケースを実現。MQL5との連携はWebhookブリッジ経由。Claude Code + MCP + MT5の統合でトレードエージェント構築が現実的になってきた具体例。FX自動取引 × LLMの実装参考として高い価値。
