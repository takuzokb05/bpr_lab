## 要約
多くのユーザーが見落としているClaude Codeの9つの隠れた機能を解説した実践記事。主な機能として、Voice Mode（スペース長押しでPush-to-Talk、日本語含む20言語対応）、Remote Control（スマホ・ブラウザからローカルセッション操作）、/loop コマンド（定期実行の自動化）、Auto Mode（安全な自律動作）、1Mコンテキストウィンドウ（Opus 4.6以降、約75万語処理）、Dynamic Workflows（最大1,000並列サブエージェント）、NO_FLICKERレンダリングエンジン（描画速度60%向上）、バックグラウンドエージェント（ウェブ上で複数タスク並行）、/code-review --fix（コードレビュー指摘の自動適用）を紹介。各機能の有効化方法・使用場面・注意点を網羅。特にVoice ModeとRemote Controlの組み合わせによる「外出先からコード修正」の活用法が実用的。
