# NIST Announces AI Agent Standards Initiative for Interoperable and Secure Innovation

- URL: https://www.nist.gov/news-events/news/2026/02/announcing-ai-agent-standards-initiative-interoperable-and-secure
- ソース: web
- 言語: en
- テーマ: ai-news
- 取得日: 2026-06-30

## 要約
米国国立標準技術研究所（NIST）のAI標準革新センター（CAISI）が2026年2月に発表したAIエージェント標準化イニシアティブ。目的：自律行動が可能な次世代AIエージェントを信頼性高く広く採用させ、デジタルエコシステム全体でシームレスに相互運用できるようにする。Singaporeでは同時期にIMDAが世界初のエージェントAI特化型AIガバナンスフレームワークを発表（エージェントIDカードという標準的な開示フォーマットを導入）。米国側はEU AI Actの8月2日高リスクAI義務適用、ホワイトハウスの3月20日AI国家政策フレームワーク（連邦統一AIガバナンスの提言）と連動。Attentiveが2026年6月にエージェントAI企業ガバナンスフレームワークを公表（エージェントごとのユニークID・精密権限スコープ・包括的監査証跡を義務化）。
