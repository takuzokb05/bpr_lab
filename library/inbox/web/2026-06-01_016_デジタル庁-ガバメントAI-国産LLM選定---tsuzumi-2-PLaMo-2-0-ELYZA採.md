# デジタル庁 ガバメントAI 国産LLM選定 — tsuzumi 2・PLaMo 2.0・ELYZA採用

- URL: https://www.watch.impress.co.jp/docs/news/2091358.html
- ソース: web
- 言語: ja
- テーマ: ai-news
- 取得日: 2026-06-01

## 要約
デジタル庁が2026年3月6日、「ガバメントAI」で試用する国産大規模言語モデル（LLM）として3モデルを選定。
選定モデル: NTTデータの「tsuzumi 2」、KDDI/ELYZAの「Llama-3.1-ELYZA-JP-70B」、Preferred Networksの「PLaMo 2.0 Prime」。
選定基準: 日本語処理能力・セキュリティ・データ主権・可用性・コスト。
政府業務特有の高セキュリティ要件（機密情報処理・個人情報保護法対応）を満たすことを重視。
LLMの「国産」採用は政府のAI戦略における国際競争力維持・安全保障的観点から重要な一手。
民間企業にも国産LLM活用事例として波及効果が期待される。
