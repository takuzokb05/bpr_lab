# Microsoft Agent Framework 1.0: AutoGen + Semantic Kernel 統合リリース

## 要約
Microsoftが2026年4月3日にMicrosoft Agent Framework 1.0を正式リリース。.NETとPython対応の本番利用可能なオープンソースフレームワーク。AutoGenとSemantic Kernelを統合したもの。新機能A2Aプロトコル（Agent-to-Agent）により異なるランタイム間でのエージェント通信が可能（Pythonビルドエージェントが.NET環境のエージェントとシームレスに協調できる）。AI agent frameworkランキング：LangGraph（1位）、Claude Agent SDK（2位）、CrewAI（3位）。Googleは2025年4月にADKをリリース（エージェントツリー・Vertex AI統合・A2Aプロトコルネイティブサポート）。AutoGen v0.4+（MS公式）とAG2（コミュニティ継続）の分裂が発生。2026年は異なるフレームワーク間のエージェント協調（A2Aプロトコル）が重要テーマに。Claude Agent SDKが2位に位置することはAnthropicエコシステムの強さを示している。
