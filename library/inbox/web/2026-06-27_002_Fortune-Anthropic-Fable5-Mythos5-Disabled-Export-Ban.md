# Anthropic Disables Fable 5 & Mythos 5 Following U.S. Export Control Directive

- URL: https://fortune.com/2026/06/13/anthropic-disables-fable-mythos-export-controls-national-security-threat/
- ソース: web
- 言語: en
- テーマ: ai-news
- 取得日: 2026-06-27

## 要約
2026年6月13日、米国商務省の国家安全保障輸出管理指令により、Anthropicは最新AIモデル「**Fable 5**」と「**Mythos 5**」への全世界ユーザーのアクセスを停止した。Fortune誌の報道によると、停止対象は米国外のユーザーだけでなく米国内の外国籍者（Anthropic従業員含む）にも及ぶ。政府側は「Fable 5のセーフガードをバイパスするジェイルブレイク技術が存在する」と主張し、具体的にはコードベース読み込み→脆弱性修正を求めるプロンプトが挙げられた。Anthropicは「他モデルにも同様に存在する狭い範囲の脆弱性であり、この基準を適用すれば全フロンティアモデルのデプロイが止まる」と反論。Claude Opus 4.8等の旧モデルには影響なし。この事案はAI規制・輸出管理が商用AIサービスに直接影響する前例となる重大ニュース。
