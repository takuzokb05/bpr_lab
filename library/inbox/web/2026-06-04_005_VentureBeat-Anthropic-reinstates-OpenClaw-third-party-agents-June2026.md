# VentureBeat: Anthropic がOpenClaw・サードパーティAgentのサブスク利用を再開 — 条件付き

- ソース: web
- 言語: en
- テーマ: claude-ecosystem
- 取得日: 2026-06-04

## 要約
VentureBeats独立報道：Anthropicが一時停止していたOpenClaw・サードパーティAIエージェントのClaude Subscriptionでの利用を「条件付き」で再開。条件＝6月15日以降は専用Agent SDKクレジットプール（Pro: $20/月、Max 5x: $100/月、Max 20x: $200/月）から引き落とし、サブスクのインタラクティブ利用枠とは分離。背景：Anthropicは5月中旬に「サブスク価格でAPIコストを無制限吸収できる抜け穴」を塞ぐため一時遮断していたが、ユーザーからの反発が強く、クレジット制限付きで再開した経緯。GitHub Gist（MagnaCapax）が詳細試算を公開：クレジット枠は一部ワークロードで実質12〜175倍のコスト増に相当。6月15日施行まで残り11日という緊急情報。
