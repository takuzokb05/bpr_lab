# Slack向け「Claude Tag」が登場、8月3日までに移行が必要に

- URL: https://www.zaikei.co.jp/article/20260629/858947.html
- ソース: web
- 言語: ja
- テーマ: claude-ecosystem
- 取得日: 2026-06-29

## 要約
Slackに常駐するAnthropicのAI「Claude for Work」が新機能「Claude Tag」にアップグレード。2026年8月3日までに既存ユーザーは移行が必要。Claude Tagは従来のアプリ型と異なり、Slackチャンネル内でメンション（@Claude）するだけで応答する常駐型AI。会話履歴の参照範囲が拡大し、チームコンテキストを維持しながら長期的な作業支援が可能になる。管理者側では権限設定・ログ管理・データ保持ポリシーの見直しが課題となる。AI統合がSaaSの標準機能として定着しつつある動向を示す一次情報。
