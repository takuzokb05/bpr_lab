## 要約
2026年5月14日Anthropic発表の重要な課金体系変更。6月15日からプログラマティックなClaude利用がサブスクリプションのレートリミットプールから分離され、別の月額クレジット（標準APIリスト価格）に移行。対象：Agent SDK呼び出し、claude -pコマンド、Claude Code GitHub Actions、サードパーティエージェント。これまでサブスクリプション内で無制限だったClaude Code GitHub Actionsやclaud -p実行が従量課金になる変革的な変更。開発者・インディーハッカーにとって月額コスト増になり得るため要注意。同時にAnthropic Managed Agentsがパブリックベータ（セキュアサンドボックス・組み込みツール・SSEストリーミング付き）、セルフホストサンドボックスもパブリックベータ開始。Claude Opus 4.7が$5/$25 per MTokで同一価格で提供。この課金変更はClaude Code活用コスト計算の見直しを迫る重要情報。
