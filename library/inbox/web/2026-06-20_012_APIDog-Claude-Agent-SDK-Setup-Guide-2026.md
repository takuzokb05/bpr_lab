## 要約
APIDog による Claude Agent SDK のセットアップガイド（2026年版）。Claude サブスクリプションプランと Agent SDK の6月15日課金変更後の実際のセットアップ手順を解説。Agent SDK は旧称「Claude Code SDK」で2026年3月に改名。ツールループ・ファイルアクセス・Bash・WebSearch・コンテキスト圧縮を SDK が処理する仕組みを説明。Pro（約$20/月クレジット）・Max 5x（約$100/月）・Max 20x（約$200/月）の課金構造と、標準API料金での従量課金の具体的計算方法を提供。Xcode 26.3 との Native Integration にも触れ、Claude Code と Agent SDK の使い分け（対話型 vs プログラマティック）を明確化。
