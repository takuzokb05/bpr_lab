## 要約
Vantagepointによる6月15日課金変更の実務解説。変更の骨子：Claude.ai chat・Claude Code（ターミナル）・Claude Coworkはサブスクリプション枠で継続利用可能。一方、Claude Agent SDK経由のプログラマティックな自律エージェント処理は新設の「クレジットプール」から別途消費。Pro: $20クレジット、Max 5x: $100、Max 20x: $200がスタンダードAPIレートで課金。チームがとるべきアクション：Agent SDKの利用量を事前モニタリング→コスト上限設定→バッチ処理の最適化→不要な自律エージェントの無効化。特に影響が大きいのはCI/CDパイプラインに組み込んだSDKエージェント。移行前後のコスト試算シートと判断フローチャートを提供。
