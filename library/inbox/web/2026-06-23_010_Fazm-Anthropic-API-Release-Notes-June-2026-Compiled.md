## 要約
fazm.aiがClaude Code デスクトップクライアントから読み取ったAnthropicの最新APIリリースノート（2026年6月）をまとめた記事。主要アップデート：Claude Fable 5（2026年6月9日GA）$10/$50 per Mトークン、Claude Opus 4.8のデフォルトコンテキストウィンドウ拡張。拒否リクエストの課金廃止（stop_reason: "refusal"は入力トークン不課金）。セルフホスト型サンドボックスのパブリックベータ開始（ツール実行をユーザーインフラに移動可能）。Claude Platform on AWSの提供開始（AWS IAM認証、AWS課金統合）。Managed Agents向けMCPトンネル（プライベートネットワーク内のMCPサーバーにセキュアアクセス）。Dynamic Workflowsの研究プレビュー開始。Workload Identity Federation（キーレス認証）のGA。新ドキュメント：Agent SDK overview、Skills設計ガイドが公式ドキュメント化。
