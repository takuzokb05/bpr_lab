# TrustTrade: Human-Inspired Selective Consensus Reduces Decision Uncertainty in LLM Trading Agents

- URL: https://arxiv.org/pdf/2603.22567
- ソース: web
- 言語: en
- テーマ: ai-trading
- 取得日: 2026-05-31

## 要約
2026年3月公開のarXiv論文。複数LLMトレーディングエージェント間の意思決定不確実性を削減する「選択的コンセンサス」手法 TrustTrade を提案。人間のトレーダーが信頼できる専門家の意見を選択して参照する行動を模倣し、エージェント間の合意形成を改善。TradingAgentsフレームワーク（7エージェント協調）のような多エージェント系における意思決定品質の向上手法として意義がある。LLMトレーディングシステムの信頼性向上に向けた学術的アプローチ。
