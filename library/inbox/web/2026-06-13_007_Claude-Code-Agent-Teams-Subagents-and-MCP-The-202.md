## 要約
Developers DigistによるClaude Code Agent Teams・サブエージェント・MCP統合の実践プレイブック。5段階ネスト可能なサブエージェントの設計パターン：オーケストレーター→専門エージェント→ツール実行の3層アーキテクチャを解説。bypassPermissions・run_in_backgroundを活用した並列サブエージェント起動でワークフローを50-70%高速化する手法。Agent Teamsの@mention機能でチームメートに特定タスクを委任する方法。MCP統合との組み合わせ：MCPサーバーがツールを提供し、サブエージェントがそれらを並列実行する構成例。コスト管理：サブエージェントのトークン消費が積み重なる問題と、レイヤーごとのトークン予算設定の重要性を定量的に解説。
