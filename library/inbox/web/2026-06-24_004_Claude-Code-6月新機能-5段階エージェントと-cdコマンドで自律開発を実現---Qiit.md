## 要約
QiitaでのClaude Code 6月新機能解説。主な追加機能：①Agentツールが最大5段階ネストに対応し複雑な自律タスクを階層化可能、②/cdコマンドでセッションを維持したままディレクトリ切り替え可能。これによりモノレポ環境での複数パッケージ横断作業が大幅に効率化。5段階エージェントネストはサブエージェントがさらにサブエージェントを呼び出すことで、大規模コードベースの解析・修正を自律的に実行できるようになる。動的ワークフローとの組み合わせで特に効果的。
