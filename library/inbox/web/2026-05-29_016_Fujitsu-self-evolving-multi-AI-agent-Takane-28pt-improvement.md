# 富士通: 業務とともに自己進化するマルチAIエージェント技術開発、Takane精度28ポイント向上

- URL: https://global.fujitsu/ja-jp/pr/news/2026/05/25-01
- ソース: web
- 言語: ja
- テーマ: ai-news
- 取得日: 2026-05-29

## 要約
富士通が2026年5月25日に発表したプレスリリース。複数AIエージェントがチームで業務を実行しながら、実行結果・人のフィードバック・制度改定から継続的かつ安全に自律学習する「自己進化マルチAIエージェント技術」を開発。業務特化LLM「Takane」（富士通独自）の精度が平均28ポイント向上を実証。医療・製造・金融・行政など複数領域に適用。従来は専門家が手動で実施していたプロンプト調整を自動化。安全性確保のため「AIエージェントが失敗理由を検証してから学習」する設計を採用。クラウド・オンプレミス・エッジ環境での運用を目指す。「Fujitsu Kozuchi Enterprise AI Factory」への組み込みを予定。カーネギーメロン大学との共同研究で省電力化も推進。日本大手ITベンダーによる自己進化型AIエージェントの実用化発表として、国内エンタープライズAI動向を示す重要事例。
