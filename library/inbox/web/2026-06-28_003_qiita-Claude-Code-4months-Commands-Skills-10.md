## 要約
4ヶ月間の日常使用からClaude Codeの高価値コマンド・スキル10選。セッション管理: /rewindで5種のロールバック（会話のみ/コードのみ/両方/要約）、/resumeで名前付きセッション復元。カスタマイズ: /insightsがHTML使用レポート生成（摩擦点・機能提案を特定）、/update-configで自然言語によるsettings.json変更、/skill-creatorでスキル説明を反復改善してオートトリガー精度向上、/fewer-permission-promptsでログスキャンしread-only許可リスト自動構築。実用ツール: /copyでフォーマット除去済みレスポンスコピー、/loopでCI監視用インターバル繰り返し、/scheduleでセッション終了後も継続するRoutinesをクラウド登録。
