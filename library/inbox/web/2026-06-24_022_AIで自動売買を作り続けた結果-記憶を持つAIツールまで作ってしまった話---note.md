## 要約
noteのnic9nic9氏によるFX自動売買開発の実体験レポート。LLMを活用した自動売買ボットを継続開発する中で、過去の取引データ・戦略・失敗を記憶するAIツールの必要性に気づき自作に至った過程を詳述。技術スタック：Python + Claude API + SQLite（記憶DB）。記憶の構造：短期記憶（直近トレード）・長期記憶（戦略別パフォーマンス）・失敗記録（ドローダウン原因）の3層。実装上の課題としてコンテキスト長制限、関連記憶の検索精度、リアルタイム性とのトレードオフを挙げる。FX自動売買とLLM記憶の実践的組み合わせとして参考価値が高い。
