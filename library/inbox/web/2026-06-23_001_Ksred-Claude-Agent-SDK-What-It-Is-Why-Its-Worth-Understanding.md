## 要約
Claude Agent SDKはAnthropicがClaude Codeを動かすのと同じハーネスを外部開発者に公開したライブラリ。Python/TypeScriptの2フレーバーで提供され、バンドルされたCLIバイナリ、サブエージェント、セッション管理、MCPサポート、ホスト型実行モデルを持つ。2025年9月にClaude Code SDKから改名。サブエージェントは(1)並列化と(2)コンテキスト分離の2つの目的で使われる。マネージドエージェントレベルではリードエージェントがジョブを分割し専門サブエージェントに委任、並列作業して結果をフィードバック。2026年4月の検索量は14,800（前年比50,000%増）。6月15日以降、Agent SDK利用はサブスクリプション内の別クレジットプール（Pro: $20、Max 5x: $100、Max 20x: $200）から引き落とされる。実装前に「これはClaude Codeで完結するタスクか、外部システムとの統合が必要か」を判断することが重要。
