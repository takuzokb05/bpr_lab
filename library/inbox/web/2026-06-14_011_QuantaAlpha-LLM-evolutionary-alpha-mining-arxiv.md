# QuantaAlpha: LLM駆動アルファ因子マイニングの進化的フレームワーク（arxiv）

- URL: https://arxiv.org/abs/2602.07085
- ソース: web
- 言語: en
- テーマ: ai-trading
- 取得日: 2026-06-14

## 要約
清華大・北大・CAS・CMU・HKUSTの研究者チームによる2026年2月のarXiv論文。QuantaAlphaは各マイニング実行を「軌跡（trajectory）」として扱い、軌跡レベルの突然変異とクロスオーバーで因子を改善する進化的アルファマイニングフレームワーク。既存のLLMエージェントアルファマイニング（AlphaAgent・RD-Agent）と比較して: IC +0.0970・ARR +17.84%の改善（vs RD-Agent）、IC +0.0535・ARR +12.21%の改善（vs AlphaAgent）、最大ドローダウン（MDD）も削減。CSI 300で学習した因子がCSI 500・S&P 500に転用可能で、4年間の累積超過リターン160%・137%を記録。LLMを使った「コントローラブルなマルチラウンド検索」と「実証済み経験の再利用」が可能な点が技術的貢献。GitHub公開済み。
