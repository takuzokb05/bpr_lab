## 要約
Pixelnthingsによるエンジニア以外向けのClaude Code Skills & Hooks実践ガイド。プログラミング知識ゼロから始められる設定手順に特化。**Skills（SOPs）**: Markdownで書いたスキルファイルをClaudeに渡すと「専門知識パッケージ」として機能。`「この作業をスキルにして」`と話しかけるとSKILL.mdを自動生成。モデル主導（Model-invoked）機能でユーザーが毎回指定しなくても自律的にスキルを選択。**Hooks（決定論的自動化）**: 毎回必ず実行させたいタスク（フォーマット、セキュリティチェック、Slack通知）にはHooksを使う。Skillsは「ガイドライン」、Hooksは「ルール」と使い分け。**非エンジニア向け実装例**: 議事録自動作成スキル、Notionページ更新フック、週報生成ルーティン。**今日から始めるステップ**: Claude Codeで「/skillsを作成して」と入力するだけでSKILL.mdファイル自動生成、プログラミング不要。
