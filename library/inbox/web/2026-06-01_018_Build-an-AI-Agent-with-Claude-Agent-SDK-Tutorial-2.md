## 要約
SerpApiによるClaude Agent SDK を使ったAIエージェント構築チュートリアル（2026年版）。
Claude Agent SDKはClaude Codeと同じエージェントループをPython/TypeScriptライブラリとして提供。
主要インターフェース: query() 非同期ジェネレータ（プロンプトを受け取り型付きメッセージをストリーム返却）。
組み込みツール: ファイル読書き・コマンド実行・コード理解・Web検索・Web取得が即利用可能（手動tool実装不要）。
Sessions機能でマルチターン会話の状態管理が可能。MCPサーバーへのファーストクラスサポート。
