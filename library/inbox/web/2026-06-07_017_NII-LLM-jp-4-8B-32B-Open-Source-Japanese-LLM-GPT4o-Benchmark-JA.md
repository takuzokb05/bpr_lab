# 国産LLM「LLM-jp-4」オープンソース公開 — 12兆トークン・GPT-4o超えベンチマーク達成（NII・JA）

- ソース: web
- 言語: ja
- テーマ: ai-news
- 取得日: 2026-06-07

## 要約
国立情報学研究所（NII）が2026年4月3日に発表。「LLM-jp-4 8Bモデル」と「LLM-jp-4 32B-A3Bモデル（MoEアーキテクチャ）」をオープンソースライセンスで公開。学習データ：約12兆トークンの高品質コーパス（日英バランス）。性能：一部ベンチマークでGPT-4oおよびQwen3-8Bを上回る成績を達成。特長：日本語に特化した学習データ設計・商用利用可能なライセンス・研究機関による品質保証。活用可能性：FX自動取引ボットの日本語ニュース感情分析・企業内文書処理・ローカルLLM運用でのAPIコスト削減。グローバル最先端（Claude Opus 4.8/GPT-5.5）には及ばないが、日本語ローカル処理では競争力ある選択肢として浮上。
