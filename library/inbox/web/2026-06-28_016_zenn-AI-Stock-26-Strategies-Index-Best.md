# AIに株の自動売買戦略を片っ端から検証させたら、インデックス最強だった

- テーマ: ai-trading
- 取得日: 2026-06-28

## 要約
Claude AI（OpenClaw経由）を使って26のトレーディング戦略をPython/alpaca-pyで系統的にバックテスト。2007-2025年（中長期）と2020-2025年（デイトレ）でウォークフォワード分析・取引コストモデリングを実施。7つのデイトレ戦略（SMAクロスオーバー・RSI逆張り・ギャップフィル・BBスクイーズ・VWAP逆張り等）と19の中期戦略が全てS&P 500バイ&ホールド（年率10.5%）に負けた。最良パフォーマーのRSI逆張りでも年率4%以上劣後。最も防御的な「200日SMA+20%金」戦略は最大ドローダウンを55.2%→27.7%に削減も、年率リターンが1.9%低下。AIが数ヶ月かかる作業を数時間で完了し、追加の検証ギャップも独自に特定。
