## 要約
Smart-Webtechによる2026年版Claude Codeワークフロー完全ガイド。4フェーズ推奨ワークフロー：①Plan Mode（探索・質問・変更なし）、②実装計画書作成・合意、③コーディング+テスト、④コミット・PR作成。2026年のコンテキスト管理の自動化：Compaction（自動圧縮）・Plan Mode（コンテキスト汚染防止）・Agentツール（並列探索）・Memory（プロジェクト記憶）。MCP活用：`claude mcp add`でNotion/Figma/DB接続。Hooks：確定的な繰り返し処理に最適（フォーマッタ/危険コマンドブロック）。証拠提示の重要性：「成功した」ではなく「テスト出力・コマンド結果・スクリーンショット」を提示させる。実践的なワークフロー設計の参考資料として2026年現在の標準手法を網羅。
