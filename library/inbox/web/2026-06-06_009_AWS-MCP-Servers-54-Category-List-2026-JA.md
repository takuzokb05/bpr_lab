## 要約

2026年4月1日時点でAWSが公開しているオープンソースMCPサーバー54種類をカテゴリ別に整理した日本語まとめ記事。S3・EC2・Lambda・RDS・CloudWatch・CodePipeline等のAWSサービスに対応したMCPサーバーを、「ストレージ」「コンピューティング」「データベース」「監視・ログ」「DevOps」等のカテゴリに分類。各サーバーのGitHubリンク・主要ツール数・認証方式（IAMロール/APIキー）を一覧化。特筆点として、AWS公式サーバーはAWS SDKベースで実装されており、通常のAWS認証情報がそのまま使える点が Claude Code との親和性を高めている。Claude CodeからAWSリソースを直接操作するユースケース構築の出発点として有用。
