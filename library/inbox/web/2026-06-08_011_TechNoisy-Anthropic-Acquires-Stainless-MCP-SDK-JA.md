# AnthropicがStainlessを買収——MCPの生みの親がSDK生成基盤を取り込む意味

## 要約
2026年5月18日、AnthropicはSDK・MCPサーバー生成ツールを手がけるStainlessを買収した。Stainlessは「OpenAPIスペックからSDKを自動生成」するツールで、Python/TypeScript/Go/Java等に対応。買収の戦略的意味：①MCPサーバーの自動生成パイプラインを内製化→サードパーティ開発コスト削減；②既存REST APIをMCPに変換するワークフロー簡素化；③Claude Agent SDKとMCPエコシステムの統合強化。日本国内への影響：国産SaaSのMCP対応コスト低下が加速する可能性。買収金額は非公開。Anthropicの年間収益予測47億ドル（2026年S-1提出前）と合わせ、エコシステム垂直統合の動きが加速。
