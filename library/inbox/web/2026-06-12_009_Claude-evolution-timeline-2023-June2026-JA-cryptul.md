## 要約
Cryptul Insightsが作成したClaude/Claude Code進化の包括的年表（2023年〜2026年6月）。主要マイルストーン: Claude 1.0（2023/3）→ Claude 3系（2024/3）→ Claude Code β（2025/2）→ 正式GA（2025/5）→ Opus 4.8＋Dynamic Workflows（2026/5/28）→ Claude Fable 5（2026/6/9）。Claude Codeのバージョン変遷: v2.1.69（3月）→ v2.1.101（5月）→ v2.1.169（6月9日）→ v2.1.172（6月10日）。各バージョンの主要機能追加（Agent View、Auto Mode、Safe Mode、5段階サブエージェントなど）を時系列で整理。日本語での一次資料として価値高。ビジネス向けにClaude採用のタイミングと戦略を考察。
