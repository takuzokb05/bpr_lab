## 要約
Agent WarsによるMCP 2026リリース候補の解説（2026年6月23日付け一次情報）。Samplingが非推奨化された理由：Sampling機能はサーバーからLLM呼び出しを要求する逆方向のフローで、MCPの主なユースケース（ツール提供）と乖離しており複雑性を増すだけだったため。代替として従来通りのツール定義とレスポンス処理で同等機能を実現可能。Extensions frameworkの導入で機能の独立バージョニングが可能になり、コアプロトコルのスリム化が実現。AI開発者・MCPサーバー開発者は早急な対応計画が必要。
