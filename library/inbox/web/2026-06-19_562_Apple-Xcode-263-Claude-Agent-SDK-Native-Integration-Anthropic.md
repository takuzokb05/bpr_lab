# Apple Xcode 26.3 integrates Claude Agent SDK natively

- URL: https://anthropic.com/news/apple-xcode-claude-agent-sdk
- ソース: web
- 言語: en
- テーマ: claude-ecosystem
- 取得日: 2026-06-19

## 要約
Anthropic 公式ニュース。Apple の Xcode 26.3 が Claude Agent SDK をネイティブ統合したことを発表。Claude Agent SDK は Claude Code を動かすのと同一のハーネスをライブラリとして公開したもの（2026年3月に "Claude Code SDK" から改名）。Xcode 統合により iOS/macOS アプリ開発者が IDE 内から Claude Agent SDK を活用した AI 自動化機能を直接呼び出せるようになる。Claude Code の能力（ファイル読み書き・シェル実行・git 統合）を Apple 開発環境に持ち込む動きとして業界から注目。Anthropic にとっては Apple エコシステムへのリーチ拡大という戦略的意義がある。
