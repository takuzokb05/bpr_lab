## 要約
DevelopersIO掲載のClaude Codeを使ったエンジニアスキル成長支援システムの構築実践記事。Claude Codeのスキル（SKILL.md）機能を応用して、コードレビュー結果からメンバーの成長ポイントを自動記録・可視化する仕組みを構築。スキルファイルの設計・フックとの連携・成長データのMarkdown蓄積パターンを具体的なコードとともに解説。Claude Codeを「コード生成ツール」ではなく「チームの学習・成長プロセス支援インフラ」として活用する新しいユースケースとして参考になる。
