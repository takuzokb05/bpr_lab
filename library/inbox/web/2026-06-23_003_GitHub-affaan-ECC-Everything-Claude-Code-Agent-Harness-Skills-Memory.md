## 要約
ECC（Everything Claude Code）はClaude Code・Codex・Opencode・Cursor向けのエージェントハーネスパフォーマンス最適化システム。Skills（スキル管理）、Instincts（本能的挙動定義）、Memory（コンテキスト記憶）、Security（セキュリティガード）、Research-First Development（研究優先開発）の5機能を統合。GitHub affaan-mが開発したOSSで、複数AIコーディングツール横断で動作する点がユニーク。CLAUDE.mdとスキルディレクトリを体系的に構成するフレームワークを提供。コンテキスト予算管理、セッション間の記憶持続、セキュリティフックの自動設定が含まれる。Claude Code以外のエディタ（Cursor等）でも同じ設定を流用できる設計でベンダーロックイン回避を意識している。スキルの再利用性と組み合わせパターンのドキュメントが充実。
