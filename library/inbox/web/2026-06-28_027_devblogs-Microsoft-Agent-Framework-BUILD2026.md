## 要約
BUILD 2026（2026年6月3日発表）でMicrosoft Agent Framework（2026年4月3日GA）の主要追加機能を発表。Agent Harness: 自動コンテキストコンパクション・組み込みシステム命令・ファイルメモリ・タスク追跡・OpenTelemetry観測可能性を含む本番対応パターン。Foundry Hosted Agents: ゼロスケーリングクラウドデプロイ（永続ファイルシステム状態・セッション別VM分離）。CodeAct: 複数ツール呼び出しを単一Pythonプログラムに統合してHyperlight マイクロVM内で実行、測定値で52.4%のレイテンシ改善・63.9%のトークン削減を実現。追加: GitHub Copilot SDK統合・マルチエージェントトポロジー向けHandoff Orchestration Pattern。
