## 要約
travisvn/awesome-claude-skills は Claude Skills・リソース・ツールのキュレーションリポジトリ。
Chat2AnyLLM 版は2026年6月1日時点で10,006件のスキルを収録（最大規模）。
ComposioHQ 版は1000以上の実運用スキルを整理し、カテゴリ別・ユースケース別に検索可能。
VoltAgent/awesome-agent-skills は Claude Code・Codex・Gemini CLI・Cursor・Opencode互換の1000以上のエージェントスキルを収録。
スキルカテゴリ: Backend Dev・DevOps・Data/Analytics・AI/LLM・Frontend・Version Control・Testing・Security・Documentation・Business Productivity。
Andrej Karpathy の投稿を起点にForrest Changが作成したCLAUDE.mdリポジトリが144kスターを獲得した事例も紹介。
