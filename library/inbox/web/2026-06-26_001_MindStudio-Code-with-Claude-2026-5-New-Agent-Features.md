## 要約
MindStudioがまとめた2026年Claude Codeの5つの主要新エージェント機能解説。①階層型エージェントスポーン（最大5階層・各自独立コンテキスト窓）②Artifacts機能（セッション作業をインタラクティブWebページ化・チーム共有・バージョン履歴）③Agent view（全セッションを一画面管理・ブロック箇所可視化）④/rewindコマンド（セッション途中の任意チェックポイントへ巻き戻し）⑤fallbackModelパラメータ（主モデル失敗時の自動フォールバック）。各機能について実際のユースケースと制限事項を解説。Token compounding問題（深いネストほどコスト指数増）や、Artifacts利用にはTeam/Enterprise planが必要など実践的注意点も含む。
