## 要約

wmedia.esによるClaude Code + Sonnet 5の実践活用ガイド。Claude Code v2.1.197以降でSonnetエイリアスが自動的にSonnet 5に解決される。切り替えコマンドは`/model sonnet`。新機能として`opusplan`モードが追加され、Opus 4.8がプランニング、Sonnet 5が実行を担うハイブリッドアーキテクチャを実現。Sonnet 5の特徴：最もエージェント性の高いSonnetで複雑タスクの自己検証が明示的プロンプトなしに機能。Sonnet 5を選ぶ場面：日常的なエージェント業務、スピード重視、Webリサーチ、ブラウンフィールドコードのデバッグ。Opus 4.8を残す場面：フロンティアレベルのマルチファイルリファクタリング、精度最優先タスク。MaxとEnterprise従量課金プランはOpus 4.8がデフォルトのまま（手動変更必要）。Pro/Team Standard/エンタープライズ固定シートはSonnet 5がデフォルト。
