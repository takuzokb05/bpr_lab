# 知っトク資格クイズ（知ってると助かる知識）

色んな資格の「知ってると助かる知識」を、**超初心者向け**に1問ずつ出題するHTMLツール。
1問ごとに「結論 → なぜ → こう役立つ → おまけ豆知識」の詳しい解説がつき、
解くほど“ちょっと得する”——をコンセプトにしています。

## 使い方

`index.html` をブラウザでダブルクリックで開くだけ。サーバー不要・インストール不要です。

1. 分野を選ぶ（IT・デジタル／日常お役立ち／教養・一般常識／おまかせミックス）
2. 4択で答える（各問に「📜 どの資格の話か」を表示）
3. その場で正誤＋詳しい解説を読む
4. XP・連続正解（🔥コンボ）・正答率で「賢くなった感」を確認
5. 結果画面で達成バッジと自己ベスト更新。70%以上で紙吹雪

## 取り入れたUX（評判のいいクイズアプリの定番）

- **即時の色つきフィードバック**（正解＝緑／不正解＝赤＋正解を提示）
- **XP＋コンボ報酬**：連続正解ほど獲得XPが増える（変動報酬で「もう1問」を誘発）
- **進捗バー**と短いセッション（最大10問）で「できそう」感
- **ストリーク（🔥）** による継続のモチベーション
- **結果画面の達成バッジ・紙吹雪・自己ベスト記録**（localStorage に保存）

## 出題スキーム（在庫が増えても破綻しない）

問題（内蔵数）が増えても、**1回の出題数は固定**（`MAX_PER_SESSION`、既定10問）。
在庫から「どの10問を出すか」を、各問の出題履歴（`localStorage`）をもとに賢く選びます。

優先順位は **B＋間違い復習** 方式：

1. **前回間違えた問題（復習）** … 1セッションの最大半分まで優先的に再出題
2. **まだ出していない問題（未出題）** … 新しい知識を優先
3. 足りなければ、残りの復習 → 正解済みで穴埋め

最後に並びをシャッフルするので「復習が必ず先頭」にはなりません。
各問の `id` は問題文から自動生成（`hashId`）するため、並べ替えても履歴は壊れません。
スタート画面には「クリア◯/全◯問・復習待ち◯問」を表示します。

## 問題の足しかた

該当カテゴリの `questions/<カテゴリ>.js` の `addQuestions(...)` に、次の形を足すだけ。
`category` と `id` は自動で付くので書かなくてOKです。

```js
{
  qual: "ITパスポート",            // この知識が役立つ代表的な資格名（ラベル表示用）
  question: "問題文",
  choices: ["選択肢1", "選択肢2", "選択肢3", "選択肢4"], // 4つ
  answer: 0,                      // 正解の番号（0始まり）
  explanation: {
    conclusion: "ひとことの結論",
    why:        "なぜそうなるのか",
    useful:     "知っているとこう役立つ",
    trivia:     "ついでの豆知識（任意。無くてもOK）"
  }
}
```

新しい分野（カテゴリ）を増やすときは:
1. `questions/_categories.js` の `CATEGORIES` にキーを追加
2. `questions/<キー>.js` を作り `addQuestions("<キー>", [...])` で登録
3. `index.html` に `<script src="questions/<キー>.js"></script>` を1行追加

追加したら `node validate.cjs` で形式チェック。
