# へぇ〜雑学クイズ（資格の超入門）

色んな資格の「知ってると役に立つ」雑学を、**超初心者向け**に1問ずつ出題するHTMLツール。
1問ごとに「結論 → なぜ → こう役立つ → おまけ豆知識」の詳しい解説がつき、
解くほど賢くなった気がする——をコンセプトにしています。

## 使い方

`index.html` をブラウザでダブルクリックで開くだけ。サーバー不要・インストール不要です。

1. 分野を選ぶ（IT・デジタル／日常お役立ち／教養・一般常識／おまかせミックス）
2. 4択で答える
3. その場で正誤＋詳しい解説を読む
4. 連続正解（🔥）と正答率で「賢くなった感」を確認

## 構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | 画面とクイズの進行ロジック |
| `style.css` | 見た目（ダークテーマ・スマホ対応） |
| `questions.js` | 問題データ（**ここを足すだけで出題が増える**） |

## 問題の足しかた

`questions.js` の `QUIZ_DATA` 配列に、次の形のオブジェクトを追加するだけです。

```js
{
  category: "it",                 // CATEGORIES のキー（it / life / culture）
  question: "問題文",
  choices: ["選択肢1", "選択肢2", "選択肢3", "選択肢4"], // 4つ
  answer: 0,                      // 正解の番号（0始まり）
  explanation: {
    conclusion: "ひとことの結論",
    why:        "なぜそうなるのか",
    useful:     "知っているとこう役立つ",
    trivia:     "ついでの豆知識（任意。無くてもOK）"
  }
}
```

新しい分野を増やしたいときは、`questions.js` 冒頭の `CATEGORIES` にキーを追加します。

## 現状（MVP）

- 4択クイズ、3分野 各6問（計18問）＋おまかせミックス
- 1回の出題は最大10問（`index.html` の `MAX_PER_SESSION` で調整可）
- スコア・連続正解・正答率の表示
