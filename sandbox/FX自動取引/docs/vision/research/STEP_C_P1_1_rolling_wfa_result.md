## fold 設計

データ全体を 6 等分し、5 fold を anchored expanding 形式で生成:

```
fold 1: IS [0, 1/6], OOS [1/6, 2/6]
fold 2: IS [0, 2/6], OOS [2/6, 3/6]
fold 3: IS [0, 3/6], OOS [3/6, 4/6]
fold 4: IS [0, 4/6], OOS [4/6, 5/6]
fold 5: IS [0, 5/6], OOS [5/6, 6/6]
```

各 fold OOS は全データの 1/6 ≈ 16.67%。M15 (2y) で 8281 bar、H1 (5y) で 5158 bar、D1 (10y) で 234 bar。

## 観察された「ねじれ」 — 解釈は保留

ねじれは検証を深めるシグナル。各ねじれについて、現時点で結論せず、追加検証で本質を詰める。

### ねじれ1: A クラス (Bonferroni 棄却) が WFA で 5/5

**現象**:
- GBP_JPY M15 CHOP / USD_JPY D1 YZ_vol / GBP_JPY D1 YZ_vol が P0-3 では Bonferroni N=606 で棄却 (Romano-Wolf 12-family のみ通過)
- WFA では全 3 閾値で 5/5 (TR_mean 1.24-1.60)

**並立する解釈**:
- (a) Bonferroni N=606 は試行間 independence 仮定で過度に保守的、WFA の方が実態に即している → 採用可
- (b) WFA は「複数期間で TR>1.0 が再現する」だけで TR の絶対水準は弱い (1.24-1.60) → 経済的有意性 (取引コスト込み) で見れば net- の可能性
- (c) 各 fold の n_high が大きく異なる (例: GBP_JPY D1 YZ_vol で 14-194)、サンプル変動の偶然か実信号か区別困難

**判定保留** — 必要な追加検証:
- **P1-1b**: Mode B 閾値再選定 WFA — fold 間で「最良閾値」のドリフトが小さければ (a) 寄り、大きければ (c) 寄り
- **P1-1e**: 経済的有意性 — 取引コスト込みで net+ になるか

### ねじれ2: EUR_USD M15 CHOP の二検定矛盾 (P0-3🔴 vs WFA🟢)

**現象**:
- P0-3 permutation: p_raw=0.012、Bonferroni N=606 で棄却 (p_bonf=1.0)、Romano-Wolf 12-family でも棄却 (p_rw=0.066)
- WFA: 全 5 fold で TR>1.0 (TR_mean=1.16, TR_min=1.04)

**並立する解釈**:
- (a) permutation は時系列の自己相関を破壊する → null 分布が現実より tight、WFA の方が現実的
- (b) WFA TR_mean=1.16 は permutation null max (1.378) より小さい → ノイズと区別できないが時系列で偶然連続成功した可能性
- (c) 12 期間で偶然ノイズが正方向に並んだ可能性 (multiple comparisons の罠の WFA 版)

**判定保留** — 必要な追加検証:
- **P1-1b**: Mode B WFA でこの閾値が複数 fold で再選定されるか
- 5 fold 全 Up = ノイズで起きる確率 (5 fold が独立なら 1/32 ≈ 3.1%) — 偶然と区別困難

### ねじれ3: EUR_USD H1 YZ_vol の fold 3 クラッシュ (TR=0.41)

**現象**:
- AAA クラス (P0-3 で p_bonf=0, p_rw=0) が、WFA fold 3 で TR=0.41 (n_high=13)
- fold 3 OOS は 2023-11-08 〜 2024-09-06
- 他の 4 fold では TR 1.15-1.81 で安定

**並立する解釈**:
- (a) **レジーム変化説**: 該当期間に EUR_USD のボラ環境が大きく変わり、閾値超過頻度が激減 → 環境センサー併用で救済可
- (b) **カーブフィット説**: 単一 IS:OOS = 75:25 分割で偶然 OOS (=データ後半) が良かっただけ、fold 3 の TR=0.41 は真の「OOS 性能」を反映
- (c) **閾値固定不適合説**: 閾値が時期依存で、固定すると一部期間で機能しない (= Mode B WFA で別の閾値が出る)

**判定保留** — 必要な追加検証:
- **P1-1c**: 該当期間の市場データ実見 (price chart, ボラ環境, 主要イベント)
- **P1-1b**: Mode B WFA で fold 3 IS から最良閾値を再選定し OOS で評価
- 加えて: 「環境センサー併用なら救済」の主張は現時点で無検証 (循環論法回避のため別途 falsify する必要)

### ねじれ4: D1 USD/GBP の WFA 5/5 と H3 の関係

**現象**:
- D1 層は P0-3 で Bonferroni N=606 全滅 → H3 (三層) ★★☆ → ★☆☆ に降格
- WFA で USD_JPY/GBP_JPY が 5/5 通過 → 単独評価では D1 機能あり

**並立する解釈**:
- (a) 「D1 単独で TR>1.0 → D1 層の存在意義あり」 = H3 復活
- (b) 「D1 シグナルは H1 シグナルの遅延版で独立情報を持たない」 = H3 復活ならず (情報冗長)
- (c) 「D1 シグナルは H1 と独立だが、組み合わせても価値追加しない」 = H3 部分的復活

**判定保留** — 必要な追加検証:
- **P1-1d**: 層間相関 (M15/H1/D1 の同時発火率、D1 シグナルが H1 シグナルと独立か)
- **P1-1d**: 補完性 (二層 M15+H1 vs 三層 M15+H1+D1 で OOS 性能差を測る)
- **重要**: D1 単独 WFA 5/5 だけでは H3 (M15/H1/D1 必要十分) の支持にはならない

### ねじれ5: AAA クラスの WFA 不揃い

**現象**:
- AAA 5 閾値中 4 個は WFA 5/5 だが、EUR_USD H1 だけ 4/5
- AA 2 閾値は両方 4/5 (両ファミリー borderline)

**観察**: P0-3 多重補正での p_rw 値と WFA の n_pass は完全に一致しない。検定の指す意味が違う。
- p_rw 小 (= 試行内で最強付近) ≠ WFA 全 fold で TR>1.0
- WFA 全 fold で TR>1.0 ≠ N=606 試行内のノイズを超える

**判定保留** — 両者を統合した「真の信頼度ランキング」は単純化できない。追加検証 (Mode B WFA、経済的有意性) で総合判定する。
