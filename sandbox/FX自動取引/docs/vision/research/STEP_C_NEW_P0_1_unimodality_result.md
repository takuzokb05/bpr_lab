## 検証方法

各 (pair, tf, indicator) で:
1. 20 分位 (5% 刻み, rank-based qcut) に区切る
2. 各分位ごとに |future_return| の median / mean / std / n を計算
3. 形状判定 (線形回帰 + 二次回帰):
   - 線形 R²>0.7 → 単調 (slope sign で増加/減少)
   - 二次 R²>0.7 かつ +0.15 線形比改善 + 二次係数 a の sign → U字 / 逆U字
   - argmax/argmin が端 → 端立ち (右肩/左肩)
   - それ以外 → 不明瞭

## 観察事項

### ① YZ_vol は高ボラほど |return| 大、定説どおり

- M15: P/T 比 2.5-2.7、R²_linear 0.84-0.91 で強い単調性
- H1: P/T 比 2.6-4.1、R²_linear 0.75-0.89
  - USD_JPY H1 は P/T 比 4.10 と最大 — H1 が一番きれいな「単峰」
- D1: P/T 比 2.09-2.95、R²_linear 0.56-0.72
  - サンプル数 1407 と少なくノイズ大
  - EUR_USD/GBP_JPY D1 は「右肩」と判定されたが、これは線形 R² が 0.6 未満で「単調増加」の閾値 (0.7) を割っただけで、形状自体は単調

### ② CHOP は低 CHOP ほど |return| 大、定説どおり (ただし弱い)

- 全て低 CHOP 帯 (q0) で |return| 中央値が最大
- ただし P/T 比は 1.22-1.33 とごく小さく、信号としては弱い
- USD_JPY/EUR_USD M15 CHOP は R²_linear 0.24-0.46 と弱い
- GBP_JPY M15 CHOP のみ R²_linear 0.80 で「単調減少」判定
- → CHOP は「最低端」で取らないと差が出ない。<25 の妥当性を補強する観察

### ③ M15 YZ_vol の Mode B 閾値二極化 (P1-1b) の真因が判明

P1-1b で M15 YZ_vol 3 ペアの Mode B 最良閾値が **10%ile / 90%ile に二極化** した。
本検証で形状は単峰 (単調増加) と確定したので、この二極化は次のいずれか:

- **(α) 90%ile 選択**: 高ボラ帯のサンプルがそのまま TR を押し上げる本来の挙動。MIN_SAMPLE_IS=50 制約を満たす最大値。
- **(β) 10%ile 選択**: `comparison: gt` で 10%ile 閾値だと「high 群 = 90%」となり、low 群 (10%) のサンプルが少なくノイジー。median |return|_low が偶然小さくなれば TR = high/low が膨らむ。

つまり「最良閾値が時期で変わる」のは指標–リターン関係そのものの問題ではなく、**TR = median(high) / median(low) の評価指標が low 群のサイズ・ばらつきに敏感に反応する** ことに起因する。

**示唆**:
- TR の評価式そのものを見直すべき (固定ベンチマーク比 / 順位検定など)
- または分位点で切る方針を捨て、continuous score (YZ_vol の z-score) ベースに移行する検討

### ④ D1 層の頑健性低下 (P0-3🔴, P1-1b 完全崩壊) の真因

D1 層 (3 組) は R²_linear 0.56-0.72 と低め。これは:
- サンプル数が n=1407 (M15: 49691, H1: 30948 と比べ 1/35 - 1/22)
- 各分位 70 サンプル → ノイズ比が大きい
- 形状自体は他と同じ単峰 (単調増加) だが、回帰がフィットしにくい

→ **D1 棄却は形状ではなくサンプル数不足が真因**。Q3 (CHOP <25 検証) で D1 を扱う必要はない。

## 各論: 単峰の中での「最適閾値」の位置

### YZ_vol M15 — 高 80-90%ile が最適 (現 SPEC は 30-80%ile)

- M15 YZ_vol の |return| 中央値の伸びは特に q14-q19 帯 (=70-100%ile) で顕著
- 例: USD_JPY M15 YZ_vol は q14=0.0017 → q19=0.0026 (+53% 急上昇)
- **現 SPEC**: USD_JPY M15 YZ_vol > 30%ile / GBP_JPY > 30%ile (= q6) は単峰の中段選択
- **示唆**: より厳しい 70-80%ile でフィルタすると high 群の TR が上がるはず — Mode B Q2 で確認する論点

### YZ_vol H1 — 90%ile 以上で急増、グリッド上限選定 (P1-1b) と整合

- H1 YZ_vol は q18 → q19 で急上昇 (USD_JPY: 0.00227 → 0.00291、+28%)
- P1-1b で全 fold グリッド上限 (SPEC × 1.5) 選定 = この q19 帯 (≧95%ile) を取りに行っている動き
- **Q2 (グリッド拡張)** で SPEC × 2.0-3.0 まで広げると、本当の頭打ちが見える可能性

### CHOP M15 — 低 CHOP <25 (≒q0 以下) でしか差がつかない

- 全ペアで argmax = q0
- q0 (CHOP min ≒ 25) 以外の分位の median は相互にほぼ平坦
- **示唆**: 「低 CHOP でフィルタ」は q0 帯 (<25-30 程度) の限定的な領域で機能
- **現 SPEC**: USD_JPY <35 / EUR_USD <30 / GBP_JPY <30
- Mode A < 30 (P0-3🔴/A) と Mode B < 25 (CV=0) のうち **<25 のほうが理にかなう** という P1-1b の暫定結論を補強

## 結論と次の行動

### 確定事項

1. **指標–リターン関係は単峰 (単調) — H2 (ボラ・トレンド直交分解) の前提揺らぎは解消**
2. **OPERATING_MODEL § 2-1 の概念設計まで波及するシナリオは消滅** — 指標選定の方向性は維持してよい
3. **Mode B 閾値二極化は形状問題ではなく TR 評価指標のサンプル感度問題** — 真の問題は「TR = median(high)/median(low)」の脆弱性
4. **D1 層脆弱性 (P0-3🔴 / P1-1b 完全崩壊) はサンプル数不足が真因** — 形状自体は問題なし

### Q1 から導かれる次の検証論点

- **(優先度↑) Q2 H1 グリッド拡張** — 単調増加が確定したので、グリッド上限選定の P1-1b 結果はそのまま「より高い閾値が良い」サインと読める。SPEC × {2, 3, 5} まで拡張して頭打ちを見つける価値あり
- **(優先度↑) Q3 CHOP <25 多重補正** — q0 帯でしか差が出ないことが補強。<25 ベースで P0-3 を再実行する論拠強化
- **(新 Q4) TR 評価指標の脆弱性** — 単峰関係の上で TR が時期で揺れるのは、評価式そのものの問題。Spearman 相関や Kendall τ など順位ベースの代替評価が必要かもしれない

### 仮説台帳 (HYPOTHESES_2-1.md) への影響

- **H2 (ボラ-トレンド直交分解) の前提**: 単峰確定で大きく改善。★☆☆ → ★★☆ への昇格候補
- **H3 (M15 補完層) の前提**: 形状自体は単峰だが、CHOP の P/T 比 1.22-1.33 では信号弱め — ★☆☆ 維持か、CHOP を補完層から外す方向
- **H7 (三層生存=採用根拠)**: 試行数 N=606 / 多重補正で生存できるかが問題で、形状とは別 — ★☆☆ 維持
- 他 H6 / H8 — 本検証では影響しない

→ 5 件 ★☆☆ のうち **H2 が改善方向**、残り 4 件は維持。「物語破棄オプション条項」発火条件 (★☆☆ ≥3 件) は引き続き成立。判断は (b) Q2/Q3 後送り推奨を維持。
