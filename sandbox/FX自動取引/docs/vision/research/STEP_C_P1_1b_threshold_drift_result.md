## 個別解析

### 発見1: M15 YZ_vol 3ペアが「両極端」を選ぶ

USD_JPY M15 YZ_vol Mode B fold 別最良閾値:
- fold 1: **90%ile** (0.00108)
- fold 2-5: **10%ile** (0.00024-0.00029)

EUR_USD M15 YZ_vol:
- fold 1, 4: **10%ile**
- fold 2, 3, 5: **90%ile**

GBP_JPY M15 YZ_vol:
- fold 1, 4, 5: **90%ile**
- fold 2, 3: **10%ile**

**含意**:
- IS_TR を最大化する閾値が「最低 10%ile」と「最高 90%ile」で分かれる = 指標 (YZ_vol) と future_return の関係が **単峰でない** (例: 低ボラ域でも高ボラ域でも値動き大、中間ボラだけ小さい)
- これは仮説 H2 (ボラ-トレンド直交分解) や OPERATING_MODEL の「volatile レジーム = 高ボラ」の前提自体を揺るがす
- Mode A 値 (30%/80%ile) は単に「両極の中間値」で、どの fold でも最良ではない

**追加検証**:
- M15 YZ_vol の値域別 future_return 中央値曲線を描画 → 単峰/二峰判定
- これは P1-3 (YZ_vol × CHOP 同時分布) の前段階で必要

### 発見2: H1 YZ_vol 3ペアがグリッド上限を選定

USD_JPY/EUR_USD/GBP_JPY H1 YZ_vol、Mode B fold 別最良閾値:
- 全 3 ペア、fold 2-5 で **グリッド上限 (Mode A × 1.5)** が選ばれた
- fold 1 だけ Mode A 値前後

**含意**:
- グリッド上限が選ばれる = **グリッド外にもっと良い閾値があるかもしれない** (グリッド設計の不適合)
- Mode A SPEC 値 (P0-3 で AAA 最強級と評価) は **最適ではなかった**
- P0-3 多重補正の信頼度評価 (p_rw=0) は「Mode A 値が偶然以上に強い」を示すが、「最強」とは言っていない
- **再検証必要**: グリッドを SPEC 値 × {1.5, 2.0, 2.5, 3.0, 4.0} に拡張して上限が更に伸びるか確認

**Mode B OOS_TR の改善**:
- Mode A: H1 mean OOS_TR ≈ 1.7-1.8
- Mode B: H1 mean OOS_TR = **2.01-2.29** (大幅改善)
- これは「Mode A 閾値より高い閾値の方が選別が効く = 真の signal は更に高ボラ域」を示唆

### 発見3: EUR_USD/GBP_JPY M15 CHOP が真に頑健 — ただし <25 で

- EUR_USD M15 CHOP: Mode B 全 fold で **<25** を選定 (CV=0、完全一致)
- GBP_JPY M15 CHOP: Mode B 4 fold で <25、1 fold で <28 (CV=0.052)

**含意**:
- 「<30」(Mode A 値) ではなく「<25」が真の最良
- CHOP は他指標と違い、グリッド端点ではなく明確な閾値 (25-28) に収束 = **真の signal の存在示唆**
- ただし P0-3 で EUR_USD M15 CHOP は 🔴 棄却推奨だった → **Mode B で <25 を採用すれば結論変わる可能性**
- 追加検証: <25 を P0-3 と同じ permutation で評価し、Bonferroni / Romano-Wolf 通過するか確認

### 発見4: GBP_JPY D1 YZ_vol が完全崩壊 (P1-1b の最重要発見)

Mode A: 5/5 通過 (TR_mean=1.60、A クラス)
Mode B: **n_pass=0/5** (OOS_TR mean=0.84)

| fold | best_grid | IS_TR | OOS_n | OOS_TR |
|---|---|---:|---:|---:|
| 1 | 10%ile | 2.35 | 234 | - (n_high=0) |
| 2 | 30%ile | 1.56 | 234 | - (n_high=0) |
| 3 | 90%ile | 1.72 | 0 | - |
| 4 | 90%ile | 1.65 | 42 | 0.72 |
| 5 | 20%ile | 1.65 | 124 | 0.96 |

**含意**:
- IS で「最良」と選定された閾値が、OOS で全く機能しない (TR<1.0 or OOS_n=0)
- これは **典型的なカーブフィット** の証拠
- Mode A で 5/5 通過していたのは「閾値固定」の偶然(P0-3 採用閾値 55%ile が「中庸」で IS/OOS 両方で平凡な TR を出した)
- **Mode B (より公正な評価) では完全崩壊**

**判定変更**:
- GBP_JPY D1 YZ_vol: A クラス → **採用棄却**

### 発見5: USD_JPY D1 YZ_vol も Mode B で破綻気味

### 発見6: EUR_USD D1 YZ_vol — P0-3 棄却 + Mode B で更に悪い
