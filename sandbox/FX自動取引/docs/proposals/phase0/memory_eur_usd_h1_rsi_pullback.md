# プロポーザル: EUR_USD H1 RSI Pullback (亡き者 BT グリッドで PF>1.0 が 13/20、亡き者本番も唯一プラス)

## 1. 戦略仮説 (1段落)

EUR_USD 主要ペアの H1 タイムフレームで、トレンド方向の RSI 押し目 (RSI<30/35 で long、RSI>70/65 で short) を捕捉する mean-reversion 戦略。エッジ源は (a) EUR_USD のレンジ性 + 流動性の高さによるリバウンド有意性、(b) H1 という「ノイズ少・コスト負担少」の時間軸、(c) 亡き者の世界の実取引で **EUR_USD のみが PF プラス (+3,315 JPY, 13件)** であるという事実数字裏付け。亡き者BTグリッド (`data/backtest_grid_h1_EUR_USD.csv`) 20パラメータ中 13個で OOS PF > 1.0、平均 OOS PF=1.06、最大 1.49、平均DD -3.6%。**パラメータ感度が低くロバスト**。

## 2. 想定エッジ源 [G1-1]

- **構造的優位**: EUR_USD はマクロイベント駆動でレンジ振動が頻発 (ECB/Fed 政策金利差、欧州CPI連動)。RSI が極値到達後、平均回帰する Microstructure 的傾向あり (短期 mean reversion)
- **行動経済的根拠**: 過剰反応 → 反転 (Overreaction → reversal)、Lo & MacKinlay (1990) 系の負の短期自己相関効果
- **データドリブン根拠**: 亡き者の世界の実取引で **EUR_USD のみ唯一プラス** (PF推定 2.37、勝率38.5%、+3,315 JPY)。MTFPullback 戦略でも EUR_USD は機能していた
- 「過去データで効いた」だけではなく、`backtest_grid_h1_EUR_USD.csv` の全 20 グリッド中 13 が OOS PF>1.0、**最悪 OOS でも PF=0.68**。極端な負け側に振れない構造

## 3. シグナル定義 (擬似コード)

```python
# H1 ローソク確定時 (毎時 :00) に判定
def signal(bar_close, history_h1):
    rsi_14 = RSI(history_h1.close, period=14)
    atr_14 = ATR(history_h1, period=14)

    # トレンドフィルタ: EMA50 > EMA200 で長期上昇トレンド中のみ long、逆で short
    ema50 = EMA(history_h1.close, 50)
    ema200 = EMA(history_h1.close, 200)
    trend_up = ema50 > ema200
    trend_down = ema50 < ema200

    if rsi_14 < 30 and trend_up:
        return Signal(direction="long", entry=bar_close,
                     sl=bar_close - atr_14 * 1.5,
                     tp=bar_close + atr_14 * 2.5)
    if rsi_14 > 70 and trend_down:
        return Signal(direction="short", entry=bar_close,
                     sl=bar_close + atr_14 * 1.5,
                     tp=bar_close - atr_14 * 2.5)
    return None
```

**最適パラメータ (BT グリッドより)**: RSI 30/70 + ATR×2.5、または RSI 35/65 + ATR×1.5 (どちらも OOS PF>1.3、平均 56 trades over 5y)

## 4. データ要件 [G1-2]

- **必要データ**: EUR_USD H1 OHLC (既存: `data/mt5_EUR_USD_H1_5y.csv` 5年あり)
- **取得元**: MT5 (`Mt5Client.get_h1_history`)、無料
- **コスト**: ゼロ
- **ラグ**: H1 確定時刻 +5秒以内 (現在の MT5 polling で十分)

## 5. リスクモデル [G1-5]

- **SL**: ATR(14) × 1.5 (BTグリッドより最頻最適)
- **TP**: ATR(14) × 2.5 (RR=1.67、勝率 40% で期待値プラス: 0.4×2.5 - 0.6×1.5 = +0.1R/trade)
- **時間損切り**: 24時間 (亡き者 EUR_USD 平均保持 4-12時間。安全マージン×2)
- **ポジションサイジング**: 口座残高の 1% / 取引 (Kelly 1/4)
- **想定 MaxDD**: BTグリッド最大 -6.0% (full_dd 列の min)、平均 -3.6%。**1.5x 安全係数で MaxDD想定 -9%**
- **テールリスク**: ECB緊急利下げ等で 3σ ジャンプ → SL 滑り想定 +1.5pip スプレッド込みで -1.7R 想定

## 6. 自己改善メカニズム [G0-B] — 必須

### ドリフト検出
- **ローリング20取引PFモニタ**: 直近20トレードで PF を計算、PF<0.9 が3連続更新で warning
- **環境変化シグナル**: EUR_USD H1 ボラ (ATR/価格) が直近50バー rolling 90%ile を上回ったら "high_vol regime"、戦略一時停止
- **KL ダイバージェンス**: 入力分布 (RSI値) の直近1ヶ月分布が学習時(5年BT)分布と KL>0.3 で reweight トリガ

### 自動再最適化
- **月次ローリング再最適化**: 毎月1日 03:00 JST、過去5年データで RSI 閾値 (30,32,35,38) × ATR (1.5,2.0,2.5,3.0,3.5) のグリッドサーチを再実行 (BTグリッドそのものを再生成、20秒で完了)
- **採用条件**: 新パラメータが直近1年 OOS で PF>1.2 を維持していれば自動採用、未満なら直前パラメータ維持
- **アルゴリズム**: ベイズ最適化は不要。BTグリッドが小さい (5x4=20) ので brute-force で十分

### フォールバック
- **直前安全パラメータ保持**: 過去6か月の採用パラメータ履歴を `data/strategy_history.json` に記録
- **PF<0.7 が30トレード連続**: 自動停止 + Slack #ai-alerts 通知 + 1ヶ月の冷却期間
- **冷却期間後**: 過去5年BTで再評価、PF>1.2 ならロット 1/4 で再開、3ヶ月安全なら full lot

### 擬似コード (1段落)
> 毎月初に 5y BTグリッドを再走、ベスト OOS PF パラメータを採用 (PF>1.2 必須)、PF<1.2 なら旧パラメータ維持。毎取引後にローリング20トレードPFを更新、3連続でPF<0.9 なら半量化、PF<0.7 が30トレード連続なら自動停止 → 1ヶ月冷却。環境変化センサ (ATR p90超過) は high_vol regime として一時停止し、p70 復帰で再開。

## 8. WFA / OOS [G1-7]

- **既存 WFA**: `backtest_grid_h1_EUR_USD.csv` で **wf_n_windows=5** の anchored walk-forward 実行済

## 11. リスク・既知の弱点

1. **2024-2026 期間限定の有効性**: BTグリッドは5年だが、ECB金利体制の変化 (例 2027年以降の利上げサイクル終了) でレンジ性が崩れる可能性 → 自己改善メカニズムで月次再評価が必須
2. **トレード頻度の低さ**: 14-56 trades/年 = 月 1-5 件、サンプル評価が遅い → Phase 2 で M15 も合成して頻度を上げる検討
3. **EUR_USD 単一通貨依存**: 通貨分散が効かない → Phase 3 で GBP_USD, AUD_USD への横展開を検証
4. **スプレッド変動リスク**: 重要指標発表時のスプレッド拡大で SL slippage → 経済指標カレンダー API で発表 30 分前後を回避
5. **過去BTの sample 数**: 一部パラメータで OOS trades 13-17件と少ない → Deflated Sharpe で安全マージン

## 撤退条件 (事前明記)

以下の **いずれか** で即撤退:
1. 運用開始から 90日経過時点で実取引 trades < 5 件 (機会喪失)
2. 直近30トレードで PF < 0.95 が確定 (亡き者の二の舞)
3. 累計 PnL < -3% (口座 100万円基準で -30,000 JPY)
4. 機会費用 (年率4% = 月3,300 JPY @ 100万) を 3か月連続で下回る
5. Deflated Sharpe < 0 が WFA 再評価で確定
