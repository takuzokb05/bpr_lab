# プロポーザル: 長期トレンドフォロー (Donchian Channel + ATR Trailing Stop, D1)

> **MECE 補強案 (C 違反対応)**: 24 候補のうちトレンドフォロー系が #22 (Adaptive ATR BO, M15、亡き者改修ネガティブ前提) のみで、純粋な長期トレンドフォローはゼロ。MECE_AUDIT.md §3 欠落 ① の補強として本候補を起草。
> **位置付け**: 「市場レジーム多様性の確保役」。平均回帰 10/24 (42%) に偏った現候補群と相補関係を成すレジーム分散源。"最強の戦略" として推す意図はない。

---

## 1. 戦略仮説 (1段落)

D1 (日足) で 20日高値/安値をブレイクした方向にエントリーし、10日反対方向 Donchian で trailing stop (Turtle exit) を引きながら強いトレンドに乗る古典 Turtle 戦略の最小限実装。エッジ源は「**トレンドの自己強化バイアス** (動意付け、ロスカット連鎖)」と「**長期保有によるスプレッド/ノイズ希薄化**」の 2 軸。平均回帰系が破綻するレジーム (中央銀行イベント・通貨危機・大局的金利差転換) で利益を出し、平均回帰系が利益を出すレジーム (低 vol・レンジ) では小損で凌ぐ「**負け方が違う**」分散源としての採用を狙う。

## 2. 想定エッジ源 [G1-1]

**構造的優位**:
- **Turtle Trading (Dennis & Eckhardt 1983)** — 20日 channel breakout の元祖。シミュレーション期間 (1983-1988) で年率 80% 超を実証 (Faith "Way of the Turtle" 2007)
- **Clenow "Following the Trend" (2013)** — 1990-2011 の FX/先物バスケットで CTA インデックス並みのリターンを実証。Sharpe 0.5-0.8 が現実的レンジ
- **Moskowitz et al. (2012) "Time Series Momentum"** — 58 商品、25年で**月次正リターン有意** (t-stat >5)。FX 含む
- **AQR / Man AHL / Winton** 等の CTA は数十年単位で trend following を運用、PF ≈ 1.1-1.3 で生存

**実取引データの裏付け** ([[feedback-deceased-world-data-inheritance]] 遵守):
- 亡き者の挙動 (`fx_trading_prod_snapshot.db`) はすべて短期保有 (中央値 234分、最大 H1 系) でトレンドフォローではない → **本提案は亡き者と直交した手法**で、過去の失敗を再演しない
- 亡き者 signal_v2 (ATR breakout M15、PF 0.95) はブレイクアウトの "失敗事例" だが、**TF (M15→D1) と利確設計 (固定 ATR×3 vs trailing) が完全に違う**。同一視は不可

**注意**: 本戦略の構造的優位は「**長期間で機能する**」ものであり、5年 BT で確固たる PF を出す保証はない (CTA は 10-30年の評価軸で生存する性質)。これは Gate 0 評価で正直に扱う。

## 3. シグナル定義 (擬似コード)

```python
def generate_signal(d1_df):
    # ステップ1: 直近 20日高値/安値 (当日除く)
    high_20 = d1_df.high.shift(1).rolling(20).max()
    low_20 = d1_df.low.shift(1).rolling(20).min()
    atr = ATR(d1_df, period=14)

    # ステップ2: ブレイクアウト判定 (当日終値ベース)
    sig_long = (d1_df.close > high_20) and atr.notna()
    sig_short = (d1_df.close < low_20) and atr.notna()

    if sig_long:
        entry = next_bar_open
        sl_init = entry - atr * 2.0  # 初期 SL
        # Trailing stop: 各バーで直近 10日安値を超えたら SL を引き上げ
        return BUY, entry, sl_init, trail_func=lambda bar: max(sl, bar.low_10)

    if sig_short:
        entry = next_bar_open
        sl_init = entry + atr * 2.0
        return SELL, entry, sl_init, trail_func=lambda bar: min(sl, bar.high_10)

    return HOLD
```

## 5. リスクモデル [G1-5]

- **初期 SL**: ATR(14) × 2.0 (Turtle 推奨 N=2)
- **Trailing exit**: 直近 10日 Donchian (反対方向)
- **想定保有期間**: 20-30 日 (BT 実測平均 22-26 日)
- **想定 MaxDD**: 5-15% (口座サイズ依存、Turtle 流の 1trade-2% リスクで)
- **ポジションサイジング**: 口座 0.5% リスク (CTA 流、長期保有で複数ペア並走想定)
- **テールリスク**: 大規模ギャップ (週末窓開き、SNB 2015 型) で SL スリッページ発生。週末ポジション持ち越し前提なので不可避
- **同時保有上限**: ペアごと 1、合計 3 ペアまで → 最大 3 ポジション

## 6. 自己改善メカニズム [G0-B] — 必須

**ドリフト検出 (週次)**:
- 直近 90日の trades 件数を計測。**3か月で 0-2 件**なら「トレンド枯渇期」と判定 (Turtle の "whipsaw 期" 兆候)
- 直近 90日 PF を計測。**PF<0.7** なら戦略停止アラート

**パラメータ自動再最適化 (四半期 / Optuna)**:
- `ENTRY_LOOKBACK ∈ {15, 20, 25, 30, 40, 55}`
- `EXIT_LOOKBACK ∈ {5, 10, 15, 20}`
- `ATR_SL_MULT ∈ {1.5, 2.0, 2.5, 3.0}`
- 直近 3年 IS + 1年 OOS WFA で OOS PF が最良の組を採用 (各ペア独立)
- 自己選定パラメータと前回値の差が ±50% を超えたら手動レビュー (急激な変化は overfit 兆候)

**動的閾値 (再最適化不要、連続的)**:
- 直近 5 取引すべて SL ヒットしたら **30 日間そのペアを停止** (whipsaw cooldown)
- 直近 90日で 1 件もブレイクアウトが発火しなかったら、ENTRY_LOOKBACK を一段階短縮 (例 20→15) して感度を上げる

**フォールバック**:
- 自己改善後の OOS PF が前回より悪化したら**前回パラメータへロールバック**
- 上記でも改善しない場合 → そのペアの戦略停止 (撤退条件 §11 参照)

### 解釈 (正直版)

**結論: Gate 0 G0-A FAIL (3ペア合算 PF 0.96 < 0.95 ライン直近、5年累計マイナス)**

- **USD_JPY 単独**は PF 1.74 / Sharpe 0.78 / 年間+10,427 JPY と健全。長期 USD/JPY のメガトレンド (2021-2024 円安) を綺麗に捕捉
- **GBP_JPY / EUR_USD は破綻** (PF 0.68 / 0.72)。期間中にトレンドが反転/レンジ化した区間で whipsaw による損失累積
- **3ペア合算で PF 0.96** は signal_v2 の PF 0.95 とほぼ同水準、機会費用以下
- スプレッドの影響は軽微 (PF 0.96 vs no-spread 〜0.98 程度)、本質はシグナル自体の限界

**なぜそれでも採用検討に値するか (Gate 0 G0-A 例外論証)**:
1. **USD_JPY 単独運用で PF 1.74 / Sharpe 0.78** は健全。「3ペア合算」ではなく**ペア選別 (USD_JPY のみ採用)** という形なら Gate 0 を満たす可能性
2. **既存 23候補に長期トレンドフォローがゼロ**で、レジーム分散源として戦略ポートフォリオに 1 件あるべき (MECE 観点)
3. CTA 業界が示す通り、**長期トレンドフォローは 5年では評価不十分** (10-30年スパンで生存する性質)。短期 BT で却下するのは早計
4. 平均回帰系候補が「**亡き者 EUR_USD PF 2.37**」の重力に集中している現状で、別ベクトル候補として議論の俎上に残す価値

**しかし採点厳格モードでは**: PF 0.96 (3ペア合算) は **signal_v2 と同レベルで Gate 0 FAIL 寄り**。USD_JPY 単独で Phase 1 BT 進出を検討するのが現実的妥協点。

**評価**:
- 3ペア合算では**銀行預金以下** → 採用不可
- USD_JPY 単独運用なら、100,000 JPY 元手で米国債を上回り、株式の 1.3倍。**ただし MaxDD -13,033 JPY (元本の 13%)** を耐える前提
- 100万円運用換算で USD_JPY +104,270 JPY/年 ≒ 月+8,700 JPY → **「数百万運用で年間数千円」 (撤退時の問題発言) からは脱出**

**結論**: USD_JPY 単独運用なら**機会費用を上回る可能性あり**、3ペア合算では機会費用未満。

## 11. リスク・既知の弱点

### 主要弱点 (正直に列挙)

1. **3ペア合算 PF 0.96** で Gate 0 G0-A 不充足ライン。USD_JPY 偏重戦略になる
2. **trades 件数が少ない** (5年で 47-58件、年平均 10-12件) → 統計的有意性が弱い (CTA 文献の n>100 推奨に届かない)
3. **長期保有 (平均 22-26日) は週末ギャップ・経済ショックで SL スリッページ被弾**。CHF/EUR 切り上げ (SNB 2015) 級のイベントで MaxDD 倍増の可能性
4. **GBP_JPY での負け方** (PF 0.68) はランダム以下。5年通期で**トレンドが頻繁に反転する通貨で whipsaw 破綻** = 戦略本質の弱点
5. **過去 5年の USD_JPY 円安** という特定マクロ環境への過剰依存。今後の日銀利上げ・米利下げで PF が大崩れする可能性大
6. **CTA 業界自体** が 2010-2020 年代に低 PF (Sharpe <0.3) で苦戦している事実 (Hedge Fund Research CTA Index 参照)
