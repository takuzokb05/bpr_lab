## 主要 LLM 判断パターン (ペア別)

### USD_JPY (CONFIRM only PF=1.408 を生んだ判定の特徴)

USD_JPY CONFIRM 例 (CSV から抽出した代表的 reasoning パターン):
- 関連通貨 (EUR/USD, GBP/USD) の 24h 変化方向と USD/JPY 方向が**整合的**なときに CONFIRM を出す
- ロンドン・NY セッション中 (流動性確認) で **トレンドが明確**な場合
- ATR に対して **SL が適切**な配置 (狭すぎず広すぎず)

### EUR_USD (なぜ効かなかったか)

- CONFIRM 率がわずか 6.7% で、CONFIRM 群でも勝率 30.4%、PF 0.527 と base 0.723 より悪化。
- LLM が EUR_USD の方向感を高い確信で読み取れていない。逆に CONFIRM を出した時の方が損失化している。
- 仮説: EUR_USD は ATR が JPY系より小さく (平均勝ち 14.4 pips、SL/TP も小さい)、`m15_close_*` の差や `related_*_24h_change_pct` の数値情報だけでは判断が困難。

### 共通の判定特徴

- 全ペアで **NEUTRAL が最多** (46-72%)。LLM が「取ってよい」と判断するケースが多数を占める。
- ただし NEUTRAL の PF はすべて 1.0 未満 (0.730 / 0.944 / 0.801) で、信号フィルタとしては機能していない。
- CONFIRM (確信) を出す件数は限定的だが、CONFIRM 群の PF は USD_JPY と GBP_JPY で base を超える → **LLM の確信度シグナルは部分的に有効**。

## 結論

### 主要発見

1. **CONFIRM only が USD_JPY で +0.40 のゲートを単独 PASS** (PF 1.002 → 1.408, n=94)。GBP_JPY の +0.19 より大幅に改善し、サンプル拡大の効果を確認。
2. **CONFIRM+CONTRADICT そのまま戦略が GBP_JPY/USD_JPY で機能** (両方 PASS)。LLM が「危険・中庸」と判断したものだけ除外する積極的フィルタ。
3. **EUR_USD ではどの戦略も機能せず**。LLM が現状コンテキスト (24h 変化、セッション、ATR) だけでは EUR_USD の方向性を読み解けない。
4. **3ペア合算ゲートは FAIL** (CONFIRM+CONTRADICT そのまま +0.296)。これはペア依存性が原因。EUR_USD を除けば PASS の可能性が高い。

### Phase 2' 進出推奨度

**条件付き推奨**:
- 対象を **USD_JPY と GBP_JPY に限定** すれば CONFIRM only / CONFIRM+CONTRADICT そのまま でゲート PASS。
- EUR_USD は除外するか、別アプローチ (異なるプロンプト、追加コンテキスト) を要する。
- ただし、USD_JPY 27.7% のサンプル達成率では **完全な再現性検証が未完**。残り 72% の検証完了を要する。

### 残作業 (クレジット復活後)

- USD_JPY 残り 1,891 件 (約 $7.8 USD)、EUR_USD 残り 1,753 件 (約 $7.3 USD) の処理。
- ただし EUR_USD は現サンプル 686 で十分 FAIL を結論付けられるため、優先度低。
- **USD_JPY の追加 1,891 件処理を最優先**。CONFIRM only PF 1.408 が大サンプルで維持されるかが Phase 2' 進出の鍵。

### CYCLE2_PLAN.md 撤退条件への当てはめ

- 撤退条件「Phase 0' 候補3本起草後、いずれも PF 差分 +0.3 未達」: **撤退に該当しない**。USD_JPY と GBP_JPY (CONFIRM+CONTRADICT そのまま) で +0.3 以上達成済み。
- ただし「効くペア・効かないペア」がはっきり分かれた点は新規発見であり、ペア絞り込み戦略の必要性を示唆。
