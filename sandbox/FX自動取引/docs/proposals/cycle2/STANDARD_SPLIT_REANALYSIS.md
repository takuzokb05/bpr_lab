### 問題点 (Karen の指摘が示唆していること)

## 6. 推奨アクション

### 結論

**Proposal 3 採用継続 OK**。SPEC v3 起動 OK。ただし以下の修正を強く推奨。

### 推奨修正 1: SPEC v3 の「IS-OOS PF」の出典を明示

現状の `IS PF 1.335 → OOS PF 1.376 (+0.041 改善)` を以下に書き換え:

```markdown
### OOS 性能 (標準分割 3 方式の中央値)
- 時間半々 (共通カットオフ 2025-05-07): IS PF 1.338 → OOS PF 1.377 (+0.039)
- 年単位 (2024 IS / 2025 OOS): IS PF 1.268 → OOS PF 1.629 (+0.361)
- 直近 12 ヶ月 OOS: IS PF 1.338 → OOS PF 1.377 (+0.039)
- 2026 Hold-out PF: 1.304 (n=185, lift vs base +0.497)
- 16 分割の OOS PF 範囲: 1.212 〜 1.481 (中央値 1.377)
- 16 分割の lift vs base 範囲: +0.460 〜 +0.596 (平均 +0.527)

オーバーフィット観察: 16 分割中 OOS PF > IS PF は 7/16 (43.8%)。
IS と OOS は概ね同水準 (中央 PF 1.34〜1.38 で揺れる) であり、OOS が IS を大幅劣化する分割は存在しない。
```

### 推奨修正 2: 2026 Hold-out PF を本番期待値の基準にする

- 2024 IS, 2025 OOS は **訓練済み LLM (claude-sonnet-4) を 2024-2025 データに当てた**。
- 2026 Hold-out PF=1.304 (n=185) は **モデル知識カットオフ以降の純粋な未見データ** → **これが最も本番期待値に近い**。
- 月次 PF 1.30 × 大数法則 (n≥185) → 期待 Sharpe 〜 0.10/月 (改善前 -0.04 から大幅プラス)。

### 推奨修正 3: 撤退基準を「OOS PF < 1.0」ではなく「lift vs base < +0.30」に変更

- 16 分割で OOS PF 範囲 1.21〜1.48 (絶対水準は揺れる)
- しかし lift vs base は +0.460〜+0.596 と極めて安定
- → 撤退基準は「lift < +0.30 が 3 ヶ月連続」が現実的

### Karen への回答

Karen の数字 (1.368 / 1.330 / -0.038) は J メタ分析の特別分割を使うと再現する **事実**。しかしこれは「特別分割が問題を作っている」ではなく **「特別分割でも 16 分割の中の 1 つの ordinary な点に過ぎない」**。

選択バイアスの本質は「OOS > IS となる分割を選んだか」だが、本件は逆で **J メタ分析の per-pair-count-half は OOS > IS だが、共通カットオフでは OOS < IS が多発**。J メタ分析が示した「IS<OOS」は確かにバイアスありだが、それを補正しても **lift vs base が安定して +0.46 以上** ある以上、Proposal 3 のロバスト性は揺るがない。
