## 0. TL;DR

1. **Phase 0 が掲げた「OOS PF 2.46」は再現できなかった**。最新 5 年実データ + 現実コスト (3 pip / RT) で BT すると **M15 PF=0.699 / Sharpe=-1.65 / 累計 -8,875 JPY (2 年, lot 0.01)**。年率 **-0.44%**。
2. **「H1 で大負け / M15 で勝つ」という timeframe 特異性は実証できなかった**。同期間 2 年 H1 で PF=0.587, M15 で PF=0.699。**両方とも構造的に負ける戦略**で、M15 が H1 を凌駕する根拠は **不在**。
3. Phase 0 の BTグリッド (`data/backtest_grid_USD_JPY.csv`) は **yfinance 60 日データ × OOS n=11 件**。私のBTは **MT5 2 年データ × OOS n=67 件**。サンプル数 6 倍、データ品質 (実ブローカー価格) も上、結論は **OOS PF 2.46 はサンプル不足由来のノイズ** だった可能性高い。
4. **ゲート判定: PF / Sharpe / 機会費用 が全て FAIL**。OOS trades >= 30、Black Swan 耐性、損失上限作動の 3 項目だけ PASS だが、これは戦略がそもそも「ほぼ無利益」ゆえに上限まで届かなかっただけ。
5. **history 委員の警告が的中**: 亡き者 USD_JPY H1 PF 0.477 と本 BT M15 PF 0.699 は、どちらも RSI Pullback ロジックの構造的欠陥を示す。**timeframe を変えても結果は変わらず、戦略本体に欠陥がある**。

atr_14 = ATR(M15, 14)
ema50  = EMA(M15 close, 50)
trend_up   = close > ema50
trend_down = close < ema50

**核心的指摘**:
- 「H1 で大負け、M15 で勝つ」 → 実証されず。**両 TF とも負ける**。

- 「日銀政策 / 円介入で M15 が活きる」 → 円介入 2024-07 では取引機会ゼロ (戦略が発動せず)、2024-08 で 1 件 + PnL +477 JPY。**特に M15 が活きる証拠なし**。
- 「M15 高頻度 = スプレッド負担大」 → コスト除外でも PF=0.91。**コスト除いてもエッジなし**。

### 6.5 Phase 0 グリッドの PF 2.46 が再現できなかった理由 (考察)

Phase 1 RSI 検証スクリプト (`scripts/_phase1_rsi_pullback_validation.py`) を確認すると:
- **データソース**: yfinance M15 60 日 (約 5,600 バー)
- **戦略**: MA200 トレンドフィルタ + ADX フィルタ + RR=2.0 (= TP は ATR×SL×2.0)
- **OOS trades**: 11 件のみ (= 統計的根拠なし)
- **Phase 0 候補書きの戦略**: EMA50 + Session Filter + RR=1.5 (= 別物)

つまり Phase 0 候補書きは **「グリッドで実証された設定」と「候補書きで示した擬似コード」が別物**。私が候補書きの擬似コードを忠実に実装した結果 PF 0.70、グリッドの設定 (MA200+RR2.0) を再現してもなお PF 0.65 (下記)。

つまり **Phase 0 グリッド CSV の「OOS PF 2.46」は、yfinance 60 日 × n=11 件の小サンプル偽陽性**である強い疑いがある。

→ M15 2y データでは SNB / Brexit / COVID は **過去カバー不能**。
→ 円介入 2024-07 は USD/JPY 直撃イベントなのに **戦略が発動していない**。RSI ベース戦略はギャップ + 急変動時に「強い RSI 過熱」になる前に方向が反転するため、エントリーチャンスを取り逃がしている可能性。
→ **キルスイッチは無発動** (取引がそもそも少ないため上限を踏まない)。

**Black Swan ストレス耐性**:
- 損失額 < 5% / 月 をクリア (PASS だが、これは **戦略が動かなかった結果**であり、「耐性がある」とは言えない)
- 5 年データで再検証する必要があるが、**現環境では困難**

## 14. 棚上げではなく「廃案」を推奨する理由

棚上げの場合: 「将来データが揃ったら再評価」「パラメータ最適化を頑張ったら活きるかも」。
廃案の場合: 「戦略の構造仮説そのものが棄却された」。

**廃案を選ぶ理由**:
