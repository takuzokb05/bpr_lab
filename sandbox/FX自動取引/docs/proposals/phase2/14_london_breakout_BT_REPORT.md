## 1. 戦略概要

### 1.1 仕様 (Phase 0 からの確定形)

```python
# Stage 1: アジアレンジ計算 (UTC 0:00-7:59)
asia_high = max(high[UTC 0-7])
asia_low  = min(low[UTC 0-7])
range_size = asia_high - asia_low

# Stage 2: ブレイク判定 (UTC 8:00-9:59、2時間限定)
# range_size が ATR 0.3〜5.0 倍の範囲内であること (極端な日除外)
if price > asia_high + buffer_pips:
    if macd_hist > 0 and rsi > rsi_threshold:
        enter_long(SL = entry - ATR×1.5, TP = entry + ATR×3.0)
elif price < asia_low - buffer_pips:
    if macd_hist < 0 and rsi < rsi_threshold:
        enter_short(SL = entry + ATR×1.5, TP = entry - ATR×3.0)

# Stage 3: ニュースフィルタ
# 第1金曜 UTC12-14 (NFP) / 第1木曜 UTC11-14 (ECB/BoE) / 第3水曜 UTC17-19 (FOMC) は回避

# Stage 4: Exit
# TP / SL / UTC 21:00 強制 close
```

## 5. Black Swan ストレステスト

| 事象 | 期間 | データ範囲内 | GBP_USD | EUR_USD | USD_JPY |
|---|---|---|---|---|---|
| Brexit 2016 | 2016-06-23〜30 | **範囲外** | n=0 | n=0 | n=0 |
| COVID 2020-03 | 2020-03-09〜31 | **範囲外** | n=0 | n=0 | n=0 |
| USD/JPY 介入 2024-07 | 2024-07-08〜15 | ✓ | +281 (n=1) | -174 (n=1) | -306 (n=1) |
| USD/JPY 介入 2024-08 | 2024-08-02〜09 | ✓ | +1308 (n=3) | -552 (n=5) | +959 (n=1) |
| 円キャリー巻き戻し 2024-08 | 2024-08-02〜09 | ✓ | (同上) | (同上) | (同上) |

**重大な欠陥**: 5年データ (2021-05〜) では **Brexit と COVID をカバーできない**。本戦略の主役ペア GBP_USD の最大のテールリスクである Brexit (2016-06-24 GBP/USD が 1時間で 11% 下落) を **検証できなかった**。

検証できた介入イベントでは:
- GBP_USD は介入時にむしろプラス (n=3-4 と少ないが偶然の可能性大)
- EUR_USD は介入時に確実にマイナス (本戦略は介入時に弱い)
- USD_JPY は介入時にプラス・マイナス両方発生 (方向性なし)

**キルスイッチ作動チェック**: 個別イベント単位での日次損失は -0.3% 未満。設計上の -1.5% 警告 / -3% 半量 / -5% 停止 のいずれも作動せず。

### 5.1 Brexit カバレッジ欠落の意味

candidate 仕様 §11 (リスク・既知の弱点) の (1) で「2020-03 のコロナショック」「夏季の流動性枯渇でレンジ自体が消滅」が明示されている。**5年データではこの最も致命的なテール (Brexit 2016 / COVID 2020-03) を BT で再現できない**ため、Phase 3 進出を仮に検討するなら **10年データへの拡張が絶対条件**。

## 10. 副次的な学び (Phase 3 への入力)

### 10.1 「実装最速」≠「採用最速」の証拠

Phase 0 提案 §9 で「実装最速 (1-2週)、Phase 1 で最初に検証すべき」と書かれた候補が、**精査 BT で全 PF 0.95 未達**。スピード優先で選んだ手法ほど、エッジ源の薄さが定量的に露呈する。これは **「effort heuristic」とは逆向きの罠** — 簡単な手法は「皆が知っている」ため、実 BT で先行者利益が消える典型。

### 10.2 Optuna 月次再最適化の限界

学習 PF → OOS PF の劣化が 20-25% で常態化。**月次再学習はパラメータドリフトを追跡できているが、戦略仮説の弱さを補填できない**。エッジ源が薄い手法に Optuna をかけても「過適合の月→ぐず崩れ→学習リセット」を繰り返すだけ。

### 10.3 セッション境界仮説の現代的減衰

「アジアレンジは静か → ロンドンで爆発」というセッション構造的非効率は、Phase 0 で「実証済」とされた。本 BT では:
- WF 比率 GBP_USD 41% / EUR_USD 23% / USD_JPY 55% = 構造優位が **GBP_USD で 41% にとどまる** (ランダムなら 50%)
- 2021-2026 のアルゴリズム化進行で、このセッション効果は **既に裁定済み** の可能性が高い
- GitHub の London Breakout 実装が多数あること自体が「皆が知っている = エッジ消失」のシグナル

### 10.4 USD_JPY のみ PF>1.0 の意味

USD_JPY は H1 BT グリッド (`backtest_grid_h1_USD_JPY.csv`) で **PF>1.0 ゼロ** = H1 全般で死んでいる通貨。本戦略で OOS PF 1.17 を出したのは:
- アジア → 東京 → ロンドンの **キャリー軸ペア** (本戦略はキャリー方向と順張り傾向あり)
- 日銀介入の方向性 = JPY 売り (USD/JPY 上昇) と本戦略のロング方向が偶然一致した可能性

これは候補 #2 (USD_JPY M15 RSI Pullback) の論点と独立した発見であり、**USD_JPY 単独運用の London Breakout 単独版**を別候補として切り出すことは理論的に可能だが、**年率 +0.29% では機会費用に届かない**ため Phase 3 候補としても弱い。
