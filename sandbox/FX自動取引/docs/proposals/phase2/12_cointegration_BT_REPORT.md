## 1. 戦略概要 (Phase 0 からの確定)

### 戦略仕様 (擬似コードレベル)

```python
# Step 1: ペア候補生成 (21 組合せ、CHF 系構造変化は除外)
candidates = [(a, b) for a, b in combinations(7 通貨ペア, 2) if {a,b} != {USD_CHF, EUR_USD}]

# Step 2: 週次 cointegration 再評価 (180 営業日窓)
for ts in trading_days:
    if days_since_last_eval >= 7:
        for pair_a, pair_b in candidates:
            sa, sb = closes[ts-180:ts]
            score, p, _ = coint(sa, sb, trend="c")  # statsmodels.tsa.stattools.coint
            beta = OLS(sa, [const, sb]).params[1]   # hedge ratio
            pair_state[(a,b)].is_active = (p < 0.05)
            pair_state[(a,b)].beta = beta

# Step 3: シグナル生成 (毎日)
for each active pair:
    spread = sa[-60:] - beta * sb[-60:]
    z = (spread - mean) / std

    # Entry: |z| >= 2.0
    if z >= 2.0:  short(A, lots=0.10); long(B, lots=0.10*|beta|)
    elif z <= -2.0:  long(A, lots=0.10); short(B, lots=0.10*|beta|)

    # Exit: |z| <= 0.5 (mean reversion) / |z| >= 4.0 (stop) / 30日 (time stop)

# Step 4: キルスイッチ
- 日次 -5% で全クローズ + 翌日まで停止
- 月次 -10% で月停止
```

### ペア / TF / パラメータ

| 通貨ペア | EUR_USD, GBP_USD, AUD_USD, NZD_USD, USD_CHF, USD_CAD, USD_JPY (7 銘柄) |
| 取引ペア候補 | 21 組合せ - EUR_USD/USD_CHF 除外 = **20 組合せ** |
