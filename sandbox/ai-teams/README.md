# AI Council（AI評議会）

**「あなたの脳内に、最強の意思決定チームを。」**

## 🎯 コンセプト

AI Councilは、プロフェッショナルな**討論番組スタイル**の意思決定支援ツールです。

知的で冷静な女性キャスターが司会を務め、異なる専門性を持つAIエージェントたちが構造化された議論を展開。あなたは「番組プロデューサー」としてテーマを投げるだけで、質の高い討論と成果物が得られます。

## ✨ 特徴

- **プロフェッショナルな進行**: 経済ニュース番組のような、落ち着いた知的な討論
- **構造化された議論**: オープニング → 発散 → 収束 → 結論の4フェーズで進行
- **リアルタイム議事録**: 書記AIが議論を即座に構造化されたドキュメントに変換

## 🤖 AIエージェント構成

| エージェント | 役割 | 使用モデル |
|------------|------|-----------|
| 🎤 **AIモデレーター** | 知的な女性キャスター。物腰は柔らかいが進行管理は鉄壁 | chatgpt-4o-latest |
| 📐 **論理担当** | リスク管理の番人。数字とエビデンスを要求 | chatgpt-4o-latest |
| 👽 **アイデア** | 空気を読まない天才。水平思考で別角度の提案 | gemini-3-flash-preview |
| ❤️ **共感担当** | ユーザーの代弁者。感情面をケア | claude-sonnet-4-20250514 |
| 📝 **書記** | 透明な記録者。議論を構造化（UIには非表示） | claude-sonnet-4-20250514 |

## 🚀 セットアップ

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. APIキーの準備

以下のAPIキーが必要です:

- **OpenAI API Key** (GPT-4o用)
  - https://platform.openai.com/api-keys で取得
  
- **Google API Key** (Gemini用)
  - https://aistudio.google.com/app/apikey で取得
  
- **Anthropic API Key** (Claude用)
  - https://console.anthropic.com/settings/keys で取得

### 3. アプリケーションの起動

```bash
streamlit run ai_council.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

## 📖 使い方

### 1. 招集（Setup）
左サイドバーの「作戦司令室」で:
- APIキーを設定
- 参加させたいエージェントをON/OFFで選択

### 2. 号令（Trigger）
中央の「議論アリーナ」で:
- 議題を入力（例: 「空き家の有効活用策について、新規事業案を出せ」）
- 「🚀 議論を開始」ボタンをクリック

### 3. 観戦（Watch）
中央パネルで議論の応酬を観察:
- アイデア担当が突飛な案を出す
- 論理担当が法的リスクを指摘
- 進行役が仲裁し、議論を前へ進める

### 4. 収穫（Harvest）
右側の「ライブ・ボード」で成果物を確認:
- **決定事項**: 合意形成されたポイント
- **懸念点**: リスクや課題
- **ネクストアクション**: 次に取るべき行動

議論が進むにつれ、リアルタイムで議事録が更新されます。

## ⚠️ 注意事項

- APIの利用には各プロバイダーの料金が発生します
- 議論は最大20ターンで自動終了します
- 長時間の議論はAPIレート制限に注意してください
- セッション状態はリロードすると消えます（将来的に永続化予定）
