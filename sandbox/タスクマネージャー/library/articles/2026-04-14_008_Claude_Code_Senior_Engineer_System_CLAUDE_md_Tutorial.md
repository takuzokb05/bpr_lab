## 投稿内容
Tutorial showing how to implement senior-engineer-level behavior in Claude Code using CLAUDE.md. Covers coding conventions, test strategies, CI/CD integration, context management thresholds, and the Plan Mode combination.

## 要約
CLAUDE.mdを活用してシニアエンジニアレベルの振る舞いをClaude Codeに実装するチュートリアル（Dev Genius）。中心概念：**CLAUDE.mdは設定ファイルではなく「AIチームメンバーの職務記述書」**として機能する。実践パターン：①プロジェクト固有コーディング規約の埋め込み②テスト戦略・品質基準の明記③デプロイワークフロー・CI/CDパイプラインの指定。設計上の重要ルール：**200行以内に保つ**（超えると遵守率が急低下）、「やってはいけないこと」の明記（hooksで強制）、**コンテキスト管理閾値の記載**（50-70%=注意、70-90%=/compact、90%+=必須/clear）。Plan Modeとの組み合わせでコード品質を向上させる具体例を提示。CLAUDE.md設計のベストプラクティスを初心者からシニアまで段階的に理解できる構成。
