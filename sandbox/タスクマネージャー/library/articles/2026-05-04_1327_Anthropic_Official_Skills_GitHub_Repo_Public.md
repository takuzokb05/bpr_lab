## 要約
Anthropic公式のAgent Skillsパブリックリポジトリ。2026年3月時点でClaude Code Skillエコシステムは公式Anthropicスキル・検証済みサードパーティスキル・コミュニティ提供スキルを含む。このリポジトリはSKILL.md形式の実装例・テンプレート・ベストプラクティスを提供する公式ソース。/simplify・/batch・/debug・/loop・/claude-apiなどのバンドルスキルのソースコードも参照可能。Capability Uplift型（新能力付与）とEncoded Preference型（チーム固有の作業方式の定型化）という2種のスキル分類の実例が豊富。開発者が独自スキルを作成する際の参考として最も権威ある情報源。PRやIssueを通じたコミュニティ貢献も受け付けている。
