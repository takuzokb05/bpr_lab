## 投稿内容
Claude Code is powerful out of the box and most people never configure it properly

There's an official Anthropic plugin called claude-code-setup that fixes that

It analyzes your project and tells you exactly what to activate: hooks, skills, MCP servers, subagents, and how to configure each one step by step
One command to install:

/plugin install claude-code-setup@claude-plugins-official

Most people skip this and wonder why their setup feels half-finished.
