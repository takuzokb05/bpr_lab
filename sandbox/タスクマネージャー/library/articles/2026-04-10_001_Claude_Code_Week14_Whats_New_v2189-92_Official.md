## 要約
Claude Code の Week 14（2026-03-30〜04-03）リリースノートで、v2.1.89〜v2.1.92 の4バージョンをカバーする公式ドキュメント。主要変更点は①/powerup コマンドによるインタラクティブ学習システム（アニメーションデモ付き）、②MCP ツールごとの結果サイズ上限を最大 500K 文字まで個別設定可能に拡張、③PreToolUse フックに「defer」オプション追加（承認待ち一時停止）、④/cost コマンドのモデル別・キャッシュヒット別コスト内訳表示の強化、⑤alt-screen レンダラーの opt-in（フリッカー解消、スクロールバック仮想化）。Homebrew チャンネル修正、ctrl+e バグ修正も含む。公式一次情報として今週の Claude Code 変更の全容を把握するのに最重要。
