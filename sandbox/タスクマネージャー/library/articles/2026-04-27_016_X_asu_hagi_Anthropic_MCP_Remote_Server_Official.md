## 投稿内容
【速報】AnthropicがMCP「リモートサーバー」を正式サポート——クラウド上でAIエージェントが外部ツールを使える時代へ

━ 何が起きたか ━
Anthropicが「Model Context Protocol（MCP）」のリモートサーバー対応を正式リリース。これまでローカル環境に限られていたMCPが、クラウドサーバー上でも動作可能になった。

━ なぜ重要なのか ━
・AIエージェントがSlack・Gmail・Notionなど外部ツールと「常時接続」できるようになる
・開発者がローカル環境なしで、ウェブ上にMCPを公開・配布できる
・ClaudeがAPIを介して企業の社内システムに直接アクセスする「業務AI」の基盤が整う

━ 何が変わるか ━
① 企業がカスタムMCPを構築→社員全員がClaudeで社内データを参照
② SaaS企業がMCPを公開→ユーザーがAIからサービスを直接操作
③ エージェント同士がMCPを介して連携→人が介在しないワークフローが現実に

詳細→ https://support.anthropic.com/en/articles/11503834-building-custom-integrations-via-remote-mcp-servers

## 要約
AnthropicがMCPリモートサーバーの正式サポートを発表。これまでローカル環境に限定されていたMCPが、クラウドサーバー上で動作可能に。公式ドキュメント：https://support.anthropic.com/en/articles/11503834-building-custom-integrations-via-remote-mcp-servers。影響：①企業がカスタムMCPを構築し社員全員がClaudeで社内データ参照可能、②SaaS企業がMCPを公開してユーザーがAIからサービス直接操作、③エージェント間でMCP連携による非人間介在ワークフロー。Claude Managed Agentsとの組み合わせでSaaSプラットフォームとしてのClaude基盤が完成に近づく。エンタープライズAI導入の「次のフェーズ」を示す重要な公式発表。
