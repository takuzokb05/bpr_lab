## 投稿内容
Microsoft Agent 365が
一般提供されました。

これはAIチャットの新機能ではなく、
企業内のAIエージェントを
管理する管制塔です。

Claude Codeのようなローカルエージェントも
今後の検知対象として名指しされ、
端末、MCPサーバー、ID、到達できる
クラウド資源まで可視化する方向です。

つまり、AIエージェントは「便利ツール」から
「管理対象のIT資産」に変わります。
