## 要約
Claude CodeとClaude Agent SDKを組み合わせたAgent Teams構築の実践ガイド（Medium/Isaac Kargar）。リードエージェントがチームメンバーを作成・作業調整・クリーンアップする階層型協調パターンを解説。Agent SDKの組み込みツール（ファイル読み込み・コマンド実行・コード編集）により、ツール実装なしでエージェントが即動作できる設計を詳述。コードレビューエージェントをゼロから構築するチュートリアル形式で、構造化フィードバック・バグ・セキュリティ問題検出の実装例を示す。プロダクション向けのstate persistence・コストキャップ・サーキットブレーカー・ツール権限管理パターンも網羅。
