## 要約

2026年2月、MicrosoftがAutoGenとSemantic KernelをMicrosoft Agent Frameworkとして統合、RCリリース。
- **統合の背景**: 2つの別々のMicrosoftエージェントライブラリが競合・混乱を引き起こしていたため統合
- **サポート機能**: グラフワークフロー・A2A（Agent-to-Agent）通信・MCP対応・チェックポイント・ストリーミング・Human-in-the-loop
- **AutoGen**: メンテナンスモードに移行（新機能開発停止、バグ修正のみ）
- 既存のAutoGenコードの移行パスとSemantic Kernelとの互換レイヤーを提供

AIエージェントフレームワーク統合の業界トレンド（Microsoft, A2A, MCPなど共通プロトコルへの収束）の象徴。
