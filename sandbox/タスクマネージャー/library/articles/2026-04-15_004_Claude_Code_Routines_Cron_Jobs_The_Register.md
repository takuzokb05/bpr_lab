## 投稿内容
The Register's coverage of Claude Code Routines with critical analysis. Describes Routines as 'mildly clever cron jobs' running on Anthropic's cloud infrastructure. Key points: routines are research preview stage, run in Claude Code's web infrastructure, not locally. Three trigger modes: scheduled, API-triggered, GitHub event-triggered. Limits: Pro 5/day, Max 15/day, Team/Enterprise 25/day. The Register notes this is an early-stage feature with limited execution counts. Also covers the redesigned Claude Code desktop simultaneously released.
