Anthropic acquired Stainless, the company behind every official Anthropic SDK since the early Claude API days.

The deal gives Claude cleaner SDKs, CLIs and MCP server infrastructure, reinforcing that agents need better software connectivity, not just better model intelligence. https://t.co/TbiPLjPiRH
