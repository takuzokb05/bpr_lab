## 投稿内容
the biggest use case that i feel like no one is talking about is HTML as an 'interactive view' for claude code: 

i run my /chief-of-staff skill that lays out my schedule for the day, all the stuff i gotta do, and 10 decisions i gotta make 

by default, i have all of that output into an HTML view i can look at, interact with, and then copy my replies via markdown and paste them back into claude code 

sooooo much of what i do now is using this flow - editing text, updating copy on websites, creating slides, everything is better via html interactive views!!!

## 要約
Claude CodeのスキルアウトプットをHTMLインタラクティブビューとして出力し、そこで操作・編集してからMarkdownでClaude Codeに返すという双方向ワークフロー。/chief-of-staffスキルで1日のスケジュールと意思決定10件をHTML表示させる具体例を紹介。テキスト編集・Webサイトコピー更新・スライド作成など幅広いタスクに応用でき、CLIの出力をブラウザのインタラクティブ性と組み合わせることで作業効率が大幅に向上すると報告されている。
