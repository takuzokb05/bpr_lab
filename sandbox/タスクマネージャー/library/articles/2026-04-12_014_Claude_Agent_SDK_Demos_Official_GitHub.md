## 要約
Anthropic公式のClaude Agent SDK（旧Claude Code SDK）デモリポジトリ。Claude Codeを動かすのと同じツール・エージェントループ・コンテキスト管理をPython/TypeScriptで外部から利用可能にするSDK。ファイル読み取り・コマンド実行・コード編集の組み込みツールを備え、即座にエージェントを起動可能。デモではバグ自動修正エージェント・コードレビューエージェント・コードベース分析エージェント等の実装例を提供。Claude Code SDKからClaude Agent SDKへのリネームはコーディング以外の用途への適用拡大を反映。Promptfoo・SerpAPI等サードパーティツールとの統合事例も増加中。Anthropic公式の一次情報ソースとして参照価値高。
