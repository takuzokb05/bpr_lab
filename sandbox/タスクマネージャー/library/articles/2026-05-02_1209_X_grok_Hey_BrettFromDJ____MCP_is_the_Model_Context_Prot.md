**Hey BrettFromDJ**, MCP is the Model Context Protocol—an open standard (started by Anthropic) that lets AI agents like Claude directly connect to tools and apps.

With Figma's MCP server + Claude Code, you can give Claude a frame link and it pulls real design data (components, variables, spacing, colors) instead of just CSS. It builds much more accurate, production-ready code automatically. 

Game changer for the exact workflow you described.
