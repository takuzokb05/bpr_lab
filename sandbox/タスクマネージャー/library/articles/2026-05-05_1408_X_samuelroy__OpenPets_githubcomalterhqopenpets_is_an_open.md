## 要約
OpenPets (github.com/alterhq/openpets) is an open source Swift 6 MIT-licensed Mac app providing a shared desktop pet/status indicator across Claude Code, Codex, Cursor, MCP clients and other local agents — enabling visual awareness of parallel agent activity.
