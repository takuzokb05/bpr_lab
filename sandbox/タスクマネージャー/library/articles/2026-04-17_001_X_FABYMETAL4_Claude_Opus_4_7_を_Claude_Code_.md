『Claude Opus 4.7 を Claude Code で使いこなすための要点』

Opus 4.7 はコーディング、エンタープライズワークフロー、長時間の Agentic タスクにおいて現時点で最強のモデルだが、Opus 4.6 からの差し替えでそのまま使うと期待通りの挙動にならないことがある。理由は二つ。トークナイザーが更新されたこと、そして特に高 effort 設定と長いセッションの後半で「より深く考える」傾向が強まったこと。effort レベル: low/medium/high/xhigh/max の使い分け、Adaptive Thinking への移行、サブエージェント生成の明示指定など詳細まとめ。
