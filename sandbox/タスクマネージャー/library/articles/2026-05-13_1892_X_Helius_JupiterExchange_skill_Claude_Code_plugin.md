## 要約
HeliusのClaude Code公式プラグインにJupiter Exchange（SolanaのDEXアグリゲーター）スキルが追加されたという正式アナウンス。ユースケースとして取引ターミナルから予測市場まで高性能アプリの構築が可能。JupiterのAPIとHeliusのRPC・取引インフラを組み合わせて使える。いいね24・RT6と同日比較でエンゲージメントが高く、Solana開発者コミュニティでの注目度の高さが伺える。Claude Codeのスキルとしてブロックチェーン金融インフラを統合するというユースケースの具体的な事例として、プラグインエコシステムの広がりを示す重要なアナウンス。
