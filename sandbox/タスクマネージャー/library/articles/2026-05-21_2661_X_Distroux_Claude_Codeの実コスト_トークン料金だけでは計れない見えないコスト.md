## 投稿内容

Claude Code cost takes miss half the bill.

Tokens are a line item.
You also pay for reverted diffs, broken setup, review time, and handoff.

Track:
- accepted lines
- reverted lines
- test pass rate
- time to merge

If cleanup costs more than generation saved, it is not cheaper.

## 要約

Claude Codeのコストはトークン料金だけではない。ロールバックされたdiff、壊れたセットアップ、レビュー時間、整合性のない変更の手動修正コストも考慮が必要という指摘。TCO（総所有コスト）の観点からClaude Code活用を評価するための実践的視点。
