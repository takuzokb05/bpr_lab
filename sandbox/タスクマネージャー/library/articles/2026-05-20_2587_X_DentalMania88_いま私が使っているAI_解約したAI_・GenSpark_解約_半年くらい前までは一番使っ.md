【いま私が使っているAI、解約したAI】

・GenSpark : 解約。半年くらい前までは一番使っていましたが、Claude Codeを使い始めてから使う機会がほぼなくなり解約。

・Cursor : 解約。GenSparkと同様。

・Claude Code : Proプラン（月額200ドル）→Proプラン（月額100ドル）にダウングレード。後述のCodexが良すぎて使う頻度が少し減り、月100ドルのプランで事足りるように。

・ChatGPT : Proプラン（月額16,800円〜）。GPT5.5になって急激に性能が高くなり、特にCodexでのブラウザ操作自動化や画像生成のクオリティは圧巻。システム開発はClaude Code、プラン作成や画像生成、業務自動化はCodexという感じで使い分け。動画編集もCodexでかなりのクオリティで自動化できるようになりました。

・CircleBack : 月額25ドル。最強の議事録ツール。月300本以上ミーティング議事録を録っていたら複数名での不正利用と判定されてしまったのか、先日一時的に利用制限になってしまい、サポートに連絡して全て自分のミーティングで使っていることを証明したら制限解除されました😅

・Canva : 解約。Adobe IllustratorとChatGPTの画像生成の組み合わせで事足りるので出番なしに。

他にも多数のAIツールを試してきていますが、今は上記の形に落ち着きました。AIの進化は驚異的なので、半年後にはまた構成が変わっている可能性はかなりありそう。

ちなみに添付の動画はCodexを使って、約2時間の動画からワンクリックで作成。「どの部分を切り抜くか？」の判断も自動。手作業での編集には劣りますが、これくらいのクオリティがポン出しできるなら十分かなと。凄い時代になりました…！
