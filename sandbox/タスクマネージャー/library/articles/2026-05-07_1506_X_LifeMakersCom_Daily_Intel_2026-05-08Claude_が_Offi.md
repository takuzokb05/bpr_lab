📋 Daily Intel 2026-05-08｜Claude が Office に入る朝、AI エージェントが代理で買い物する境界

■ Claude for Excel / PowerPoint / Word が一般提供
Anthropic が Claude for Microsoft アプリの GA を発表した。Excel・PowerPoint・Word はすべての契約者で使え、Outlook はパブリックベータ。アプリ間を移動する際にも会話のフルコンテキストを保持する設計。仕事で Office を開く時間が、AI とふたりで作業する時間に置き換わる。あなたなら、最初に何を任せるだろう。

■ AWS × Coinbase × Stripe、AI エージェントの決済が始まる
AI エージェントがステーブルコインを使って、データ・API・コンテンツ・サービスを自律的に購入する決済レールが正式ローンチされた。x402 / MCP / agent-to-agent commerce が日常に降りてくる入口。「自分のかわりに AI が買う」境界が、抽象論から実装に移った日として記憶される。

■ OpenAI、GPT-Realtime-2 で音声エージェントが進化
OpenAI が GPT-Realtime-2 を API でリリース。「GPT-5 クラスの推論を音声エージェントに」運ぶモデルで、リアルタイムに聞き、考え、複雑な問題を一緒に解く相棒として位置づけられている。テキスト中心だった AI との対話が、声で並走する形に変わっていく。

■ OpenAI Codex、Chrome 拡張で並列ブラウザ作業
OpenAI が Codex の Chrome 拡張をリリース。複数タブを横断しながら「あなたの代わりにブラウザを動かす」存在として、定型的なリサーチ作業をバックグラウンドで実行する。あなたが重ねていた手作業は、何時間ぶんが代理されるだろう。

■ 米財務長官 Bessent と Trump 来日、福島処理問題が論点化の動きも
iisia 氏の観測。5 月 11 日からの Bessent 米財務長官、続く Trump 来日の可能性、その後の米側からの「福島第一原発の処理問題」論点化の動きが指摘されている。高市政権の一部は把握済みだが大半には寝耳に水との情報。地政学が日常生活の論点に直結しはじめる局面。
