## 要約

Builder.ioによるClaude Code実践50Tips集。CLAUDE.mdは/initで雛形生成後に手作業で精査すること、フックは「必ず実行すべき処理」（フォーマット・リント・セキュリティチェック）に使用すること、MCPよりもCLI（gh・jq・curl等）の方がコンテキスト効率が高いことを強調している。また、Playwright MCPでUI検証フィードバックループを構築する方法やCodex MCPによる二重モデルレビューパターンも紹介されている。
