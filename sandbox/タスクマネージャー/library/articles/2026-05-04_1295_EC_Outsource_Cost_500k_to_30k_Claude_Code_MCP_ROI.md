## 投稿内容
EC運用で外注に月50万払ってた作業、Claude Code + MCPで自前に切り替えたら月3万になった

最近EC参入の相談が増えてるんだけど、外注費を削る前にまずオペレーション全体を見直した方がいい

①商品説明文・SEOタイトル・メタデータの一括生成
Claudeにブランドガイドラインを食わせてSKU単位で回すと、外注ライターより文脈の一貫性が出る。うちは月200SKU分を30分で回してる

②在庫・受注データの分析レポート自動化
MCP経由でスプレッドシートと直接つないで、週次ダッシュボードを自然言語で更新。手動でExcel開く時間がゼロになった

③カスタマー対応の返信ドラフト生成
過去の対応ログを学習させるとトーンまで合わせてくれて、下書きの8割がそのまま使える

月50万→月3万のインパクトはデカいけど、本質は意思決定のスピードの方で。データ見てから判断するまでのリードタイムが半分になったのが一番効いてる

## 要約
@latdayo（フォロワー579）が公開した定量的ROI事例。EC運用で外注に月50万円支払っていた作業をClaude Code + MCPで自前実装に切り替えたところ月3万円（94%削減）に。①ブランドガイドラインをClaudeに食わせ月200SKU分の商品説明・SEOタイトル・メタデータを30分で一括生成（外注ライターより文脈の一貫性あり）②MCP経由でスプレッドシートに直接接続して在庫・受注データの週次ダッシュボード自動生成、などが具体例。Claude Code + MCPの実務ROIを示す希少な一次事例。
