Anthropic 收购 Stainless（SDK 管道控制权）

事件：Anthropic 收购了 Stainless——OpenAI 和 Gemini 都依赖的 SDK 生成管道。Stainless 同时拥有最成熟的 MCP Server 工具栈。

信号：Anthropic 同时掌控了 MCP 协议（开源标准）和最主流的协议实现工具链。虽然 MCP 仍保持开放，但竞争对手是否愿意在一个越来越与单一厂商绑定的协议上投入，成为关键问题。

信息差：Forbes 今天刚发，国内尚无讨论。
