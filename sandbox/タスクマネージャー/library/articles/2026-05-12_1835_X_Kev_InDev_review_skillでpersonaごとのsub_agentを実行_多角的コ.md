## 投稿内容
@delba_oliveira I've been finding myself defining "personas" in a review skill that instructs Claude Code to run a sub agent per-persona to review from that specific lens. 

Idea being that it keeps it focused on that one angle and doesn't return only surface level findings on the first turn.

## 要約
review skillの中でreviewの視点をpersonaとして定義し、Claude Codeにそのpersonaごとのサブエージェントを並列実行させる手法。多角的視点（セキュリティ、パフォーマンス、可読性等）のコードレビューを自動化する方法論として再現性が高い。@delba_oliveira への返信で公開された実践テクニック。
