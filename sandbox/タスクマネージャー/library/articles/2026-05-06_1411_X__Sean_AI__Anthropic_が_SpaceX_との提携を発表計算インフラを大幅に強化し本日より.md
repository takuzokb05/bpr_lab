## 投稿内容
Anthropic が SpaceX との提携を発表。計算インフラを大幅に強化し、本日より使用制限が更新されました。

変更内容

・Claude Code の5時間レート制限を2倍（Pro / Max / Team）
・ピーク時間帯の制限削減を完全廃止（Pro / Max）
・Opus モデルの API レート制限を大幅引き上げ

最近、「すぐ上限に当たる」という声がずっと多かったし、性能は好きなのに、それだけで ChatGPT に戻ってしまう人も多かった。これでだいぶ変わりそう。

## 要約
AnthropicとSpaceXが提携。Claude Codeの5時間レート制限が2倍に（Pro/Max/Team）、ピーク時間帯制限廃止、Opus API引き上げ。「すぐ上限に当たる」という不満が多かったため、今後ChatGPT離れを防ぐ可能性が高い。
