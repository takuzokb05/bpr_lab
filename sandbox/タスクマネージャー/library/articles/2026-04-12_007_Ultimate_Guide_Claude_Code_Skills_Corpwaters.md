## 要約
Claude Code Skillsの包括的解説ガイド。Skillsは「再利用可能な専門知識パッケージ」であり、一度定義すれば何度でも同じワークフローを再現できる。SKILL.mdファイルの構成（YAMLフロントマター＋Markdownコンテンツ）、モデル主導（Model-invoked）とユーザー呼び出しの違い、サブエージェント実行・動的コンテキスト注入のしくみを解説。Agent Skills Open Standardへの準拠により複数AIツール間での互換性を確保。Anthropic公式スキル（frontend-design: 277,000+インストール）を含む2026年のエコシステム全体像を網羅。Skillsの「invocation control」「subagent execution」「dynamic context injection」の詳細なClaude Code固有拡張も解説。
