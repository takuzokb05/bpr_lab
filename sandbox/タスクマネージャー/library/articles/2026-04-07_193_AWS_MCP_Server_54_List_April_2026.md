## 要約

AWSが2026年4月1日時点で公開しているオープンソースMCPサーバー全54種類のカテゴリ別早見表：
- **カバー範囲**: AWS CDK・Lambda・S3・DynamoDB・CloudWatch・Bedrock・SageMaker等の主要サービス
- 用途別分類：コード生成・インフラ管理・モニタリング・ドキュメント参照
- 各サーバーのGitHubリポジトリリンク付き
- **54サーバーという数は2026年4月時点での最大規模のMCPサーバーコレクション**
- AWS環境でClaude Codeを活用する際のMCPサーバー選定ガイドとして即実用

AWS・Anthropic間の連携強化を示す重要シグナル。MCPエコシステムが97百万インストール突破した2026年3月以降、エンタープライズ対応が急速に進んでいる状況を示す。
