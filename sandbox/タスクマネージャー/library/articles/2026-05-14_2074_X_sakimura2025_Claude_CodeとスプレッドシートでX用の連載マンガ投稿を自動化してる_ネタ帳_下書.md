## 投稿内容
Claude CodeとスプレッドシートでX用の連載マンガ投稿を自動化してる。ネタ帳→下書き→投稿までの流れをコード化しただけなんだけど、これが月15本→月90本になった。

今は全部の手順をnoteで無料公開してる。でも正直、あと半年もしたら同じことを「月3980円のテンプレート」とか「有料スクール」で売る人が絶対に出てくる。というか既に準備してる奴ら見かけてる。

俺がやってることって、結局スプレッドシートに「連載マンガのネタ」「キャラ設定」「次話の流れ」をぶち込んで、Claude Codeで「これを投稿文に変換して」って指示するだけなんだ。コスト0。時間15分で1週間分の投稿が完成する。

この仕組みを有料化する理由は、たぶん「有料だと信用度が上がる」「お金を払ったから実行する心理」を使いたいからだと思う。実際そうなんだけど、本当に必要なのはお金じゃなくて「スプレッドシートに何を入れるか」という設計思想だけなんだ。

今このnoteを読んでる人たちは、半年後に「月3万かかる〜」って言われても「いや、無料でいいや」って判断できる状態になってる。その差がでかい。
