## 要約
Anthropicが2026年3月1日にパブリックベータ公開したAdvisor Tool（アドバイザーツール）の公式ドキュメント。コスト効率の高いエグゼキューターモデル（Sonnet 4.6またはHaiku 4.5）がエージェントタスクを端から端まで実行し、解決困難な意思決定に直面した際のみClaude Opus 4.6をアドバイザーとして相談するモデルペアリングパターン。アドバイザーは全会話を読み込み、計画または軌道修正を生成（通常400〜700テキストトークン、計1,400〜1,800トークン）。利用にはbetaヘッダー `advisor-tool-2026-03-01` が必要。コスト削減と高品質推論の両立を実現する重要なAnthropicの新機能。
