## 投稿内容
You can just run Openclaw straight from an android in a Debian shell via Termux. I do this with Claude code and use this with Openclaw as well! Can easily control your entire network via tailscale/wireguard. I mainly prefer my phone it's the easiest interface to use compared to sitting at a desk.

## 要約
AndroidデバイスのTermuxアプリ上でDebian shellを起動し、Claude CodeおよびOpenclawを動作させるセットアップの紹介。Tailscale/WireGuardと組み合わせることでネットワーク全体をスマートフォンから制御できると述べており、デスクより使いやすいインターフェースとして評価している。PCレスでClaude Codeを実行する具体的な方法として、モバイル環境での開発ワークフローを実現する事例。Termux + Debian shell + Claude Codeという構成で、外出先やデスク離席時にも継続開発が可能になる。
