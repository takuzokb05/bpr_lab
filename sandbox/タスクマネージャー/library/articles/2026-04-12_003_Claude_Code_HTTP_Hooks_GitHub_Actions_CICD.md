## 要約
Claude CodeのHTTP Hooks × GitHub Actionsを組み合わせたCI/CD統合の実践ガイド。HTTP Hooksはローカルstdioフックでは不可能な「クラウドネイティブ自動化」を実現し、PRオープン時に自動コードレビュー、コミット毎のセキュリティスキャン、テスト失敗時の自動修正提案が可能に。GitLab CI/CDとの統合パターンも解説。Quality Gates（lint・型チェック・セキュリティスキャン）をPR/MRオープン時に自動実行し、全ゲート通過までマージをブロックする設定例を提供。AI生成コードをプロダクション品質に引き上げるための必須パターン。実際のYAML設定例を多数掲載。
