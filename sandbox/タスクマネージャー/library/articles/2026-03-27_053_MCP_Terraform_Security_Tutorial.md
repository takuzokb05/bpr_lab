## 要約

MCPを使用してClaude CodeをTerraformとAWSに接続し、セキュリティ監査・修正・検証を自動化するハンズオンチュートリアル（2026-03-27公開の最新記事）。MCP経由でClaudeがTerraformファイルを読み取り・脆弱性を特定・修正コードを生成・適用するパイプライン全体を実装。インフラセキュリティへのMCP活用の具体的ユースケースとして参照価値が高い。
