## 要約
2026年5月13日発表。6月15日より Claude Agent SDK・claude -p・Claude Code GitHub Actions・サードパーティエージェント（OpenClaw等）の利用が、インタラクティブ使用枠とは別の専用月額クレジットから引かれる仕組みに変更。クレジット額：Pro $20/月、Max 5x $100、Max 20x $200、Team Premium $100/席。クレジットは毎月リセット（繰越不可）、ユーザー単位・非共有。初回のみオプトインが必要で以後自動更新。超過時はAPIレートで従量課金（有効にした場合）またはサービス停止。ターミナルのClaude Codeインタラクティブ利用は対象外。インディーハッカー・個人開発者への影響を具体的に解説しており実用価値が高い。
