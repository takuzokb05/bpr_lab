## 要約

仕様駆動開発とバグ修正を自動化するClaude Code用スキル+フックセット。
- **新機能フロー**: Requirements→Design→Tasks→Implementationの4フェーズを自動進行
- **バグ修正フロー**: Report→Analyze→Fix→Verifyの4フェーズを自動化
- 各フェーズは自動的に次フェーズを提案・実行する連鎖スキルとして実装
- `.claude/skills/`に配置してすぐ使えるready-to-useな設定

チームに「AIコーディング規律」を持ち込みたい場合のセットアップとして即座に活用可能。実装手順も含む。
