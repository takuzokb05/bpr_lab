## 投稿内容
AnthropicがStainlessを買収したの、かなり戦略的に見える。

単なるSDK生成会社の買収ではなく、API・SDK・MCP serverという「AIエージェントが外部システムに接続する配管」を押さえにいっている。

AI企業の競争軸が、モデル単体から開発者基盤に広がってきた。

## 要約
AnthropicによるStainless（SDKスペックからMCPサーバーを自動生成する会社）の買収を戦略的に分析。単なるSDK生成ツール買収ではなく、API・SDK・MCPという「AIエージェントが外部システムに接続する配管インフラ」全体を掌握しようとする動きと解釈。競争軸がモデル性能単体から開発者基盤（エコシステム）へと移行しつつあることを示す。Stainless買収によりAnthropicは仕様書からMCPサーバーを自動生成するパイプラインを内製化し、パートナーAPIとの統合を大幅に加速できる。
