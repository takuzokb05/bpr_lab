## 投稿内容

Lukilabs が2026年4月19日に公開した新規OSS エージェントフレームワーク「Craft Agents」の公開情報。

## 要約

Lukilabs が2026年4月19日に GitHub で公開した OSS エージェントフレームワーク「Craft Agents」。AIエージェント開発の簡素化を目的として設計。特徴：モジュラー設計でエージェントのツール・メモリ・ループを独立して差し替え可能、MCP プロトコル対応でツール相互運用性を確保、Python ファースト設計で学習コストが低い。2026年のエージェントフレームワーク競争における新興勢力（LangGraph・Claude Agent SDK・CrewAI・AutoGen/AG2・Pydantic AI と競合）。同月の主要OSS公開：Microsoft Agent Framework 1.0（AutoGen + Semantic Kernel 統合、.NET/Python 両対応）。Craft Agents の競合優位：シンプルな API 設計・MCP 標準準拠・小規模チーム向けの低い学習コスト。2026年に300以上のAIエージェントフレームワーク・ツールが存在する状況（awesome-ai-agents-2026 リスト）。
