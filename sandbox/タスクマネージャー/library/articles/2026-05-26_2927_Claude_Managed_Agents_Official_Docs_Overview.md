## 要約
2026年4月8日 Public Beta 公開。Managed Agents はAnthropicがホスト型エージェント実行環境を提供するAPIサービス。セッション管理・コンテナサンドボックス・ビルトインツール（ファイル読み書き・コマンド実行・ブラウザ操作）をAPIとして提供。Claude Agent SDKとの違いは自前インフラ不要な点。ステートフルな長時間セッション・クリーンな再開・サーバーサイドコンテナ状態管理が特徴。Cloudflare連携でグローバル分散実行も可能。Pro/Max/Team/Enterprise 対応。6月15日以降は Agent SDK クレジットから消費。
