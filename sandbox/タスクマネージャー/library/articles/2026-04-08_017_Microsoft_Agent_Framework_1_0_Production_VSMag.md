## 要約
2026年4月6日、MicrosoftがAgent Framework 1.0をリリース。.NETとPython向けの本番稼働対応マルチエージェントAIワークフロー構築SDKで、長期サポート付き安定版として提供。Semantic KernelとAutoGenの両プロジェクトを統合したオープンソースフレームワーク。エンタープライズ展開を主眼に設計されており、実験から本番運用まで一貫したアーキテクチャを提供。マルチエージェントワークフローの定義・デプロイ・監視機能を含む。Claude Code（Anthropic）やGemini（Google）等の競合との差別化はMicrosoft Azure/Office365エコシステムとの深い統合。2026年のAIエージェントフレームワーク競争の重要マイルストーン。企業がClaude Agent SDKかMicrosoft Agent Frameworkかを選択する際の判断材料になる。
