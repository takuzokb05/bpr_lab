## 投稿内容
Anthropic officially announces Claude Code Routines as a research preview on April 14, 2026. A routine is a Claude Code automation configured once — including a prompt, repo, and connectors — then triggered on a schedule, via API, or in response to GitHub events. Routines run on Anthropic's web infrastructure, not on your local machine. Three trigger types: Schedule (hourly/nightly/weekly), API (external call), GitHub Events (PR/release/push). Plan limits: Pro=5/day, Max=15/day, Team/Enterprise=25/day. Examples: nightly backlog triage, deploy smoke checks, weekly report generation.
