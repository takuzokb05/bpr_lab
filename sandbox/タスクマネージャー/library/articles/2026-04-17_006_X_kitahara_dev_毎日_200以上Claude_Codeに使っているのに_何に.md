## 投稿内容
「毎日$200以上Claude Codeに使っているのに、何にトークンが消えているかわからない」という人がツールを自作した。支出の56%が「会話」、実際のコーディング（ファイル編集・書き込み）はわずか21%。npx codeburn で試せる。無料・OSS。

## 要約
@kitahara_dev（フォロワー1,031、いいね1）によるX投稿（2026-04-17収集）。 Claude Codeのトークン消費内訳を可視化するOSSツール「codeburn」の紹介。定量データ（会話56%、コーディング21%）と`npx codeburn`というインストールコマンドを含む実践情報 npx codeburnコマンドで利用可能な新ツール情報。
