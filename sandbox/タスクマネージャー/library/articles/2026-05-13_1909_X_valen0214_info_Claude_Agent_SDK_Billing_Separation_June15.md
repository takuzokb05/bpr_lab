## 要約
Anthropicが2026年6月15日から「プログラム利用専用クレジット」を導入し、チャット利用と自動化・エージェント利用の課金を分離する。対象はClaude Agent SDK、claude -p、Claude Code GitHub Actions、Agent SDK系アプリの4カテゴリ。最も影響を受けるのはAgent並列実行・GitHub Actions自動化・長時間claude -p運用・AIエージェント開発者。著者の分析ではこの変更がAnthropicの収益的限界から来るだけでなく、「Claude Codeを開発基盤化する」という戦略的意思表示でもあるとしている。「ChatGPT＝会話AI / Claude＝AI開発OS」という位置づけの変化を予測した重要な市場分析。
