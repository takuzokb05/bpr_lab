## 要約
Claude Code Routines機能を活用した5つのプロダクション向けワークフローとMCPサーバー設定を解説（Arcade.dev）。(1)インシデントポストモーテム自動起票、(2)オンコールトリアージ→チケット作成、(3)PRエイジングレポート、(4)エクスパンションシグナルスキャン、(5)チェンジログPR生成の各ワークフローのYAML設定例と対応MCPサーバーの接続パターンを詳述。RoutinesがGitHub eventやAPIコールをトリガーとして自動起動できる仕組みも解説。VPS依存の日次収集ルーティンをRoutinesで置き換える際の参考として有用。
