## 投稿内容
1、Claude Codeが普通のAIと根本的に違う理由を理解する
ChatGPTはテキストを返すだけだ。でもClaude Codeはパソコンのファイルを直接書き換えて、ターミナルでコマンドを実行して、外部と通信してデータを送受信できる。AIに悪意はないが仕様として動くから危ない。設定を間違えるとファイルが消え、APIキーが漏れ、勝手に外部にデータが送られる。これを理解してから設定に入る。
