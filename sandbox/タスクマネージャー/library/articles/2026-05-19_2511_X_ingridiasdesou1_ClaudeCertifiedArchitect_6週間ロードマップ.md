## 投稿内容
6週間でClaude Certified Architectになりたいですか？面倒なことはありません！

Week1: Foundations
→コアを学ぶ:
• Claude API
• Model Context Protocol (MCP)
• Claude Code
• Claude 101

Week2: Build
→消費するのをやめて、作り始める:
• Claude Codeを使った実際のプロジェクト
• Agent SDK＋API
• MCPの統合

Week3: 試験を理解する
→プレイする前にゲームを知る:
• 6つの実世界のシナリオ
• 5つのコアドメイン
• 必要なスキル

Week4: プロのように練習する
→本当に重要なことを構築する:
• マルチツールAIエージェント
• Claude Codeでのチームワーク
• データパイプライン+リサーチエージェント

Week5: 自分を試す
→模擬試験を受ける
→850+/1000を目指す

Week6: 時が来た
→本物の試験に挑戦する
→一発勝負。それを大切にする。

## 要約
Claude Certified Architectの資格取得を6週間で達成するためのロードマップ。Week1でClaude API・MCP・Claude Codeの基礎を学び、Week2でAgent SDK＋API・MCP統合の実装へ進む。Week3-4で試験の5つのコアドメイン・6つの実世界シナリオをカバーし、マルチツールAIエージェントやデータパイプラインを構築。Week5の模擬試験では850点以上/1000点を目標とする実践的な計画が示されている。
