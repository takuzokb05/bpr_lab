チーム単位で一気にCodexへ移行し、ワークフローをそのまま引き継げたという事例も出始めている。

問題はモデルの信頼性（ハルシネーション）だが、移行ハードルが大幅に下がったことで「試しに使ってみよう」という人が確実に増えている。

OpenAIがClaude Codeユーザーの離脱を本気で狙っているのがよくわかる。
