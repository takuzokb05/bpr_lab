## 投稿内容
Claude Codeに新機能
「/goal」が追加されました。

AI開発で一番つらいのは、コードを書けないことではなく、" 途中で止まる・勝手に解釈する・不要な変更まで混ぜる " ことです。

/goalは完了条件を設定し、達成までターンをまたいで作業させる機能。CLAUDE.mdには「2回起きたミス」を再発防止ルールとして残す。自律化より先に、条件設計です。

まず1つだけ試すならこれです。

「テスト通過、lint通過、変更範囲、
触ってはいけないファイル」を/goalに書く。
