## 投稿内容
Pwn2Ownのために見つけていたClaude Code Desktop版の0-Prompt RCEが修正されていた

## 要約
Claude Code Desktop版のセキュリティ脆弱性報告。Pwn2Ownコンテスト向けに発見された0-Prompt RCE（Remote Code Execution）が修正済みになったことをセキュリティ研究者が確認。プロンプト不要でリモートコード実行が可能という深刻な脆弱性がパッチされた。Claude Codeをエンタープライズ・チームで使用している場合、Desktopアプリを最新版に更新することが必須。0-Prompt RCEの修正はClaude Code Desktopのセキュリティ成熟度向上の重要なマイルストーン。
