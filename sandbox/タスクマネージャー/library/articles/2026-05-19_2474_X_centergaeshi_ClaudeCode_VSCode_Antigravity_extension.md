## 要約
Antigravity IDEへの移行時にMCP・Skillsの設定は引き継がれるが、Claude Code for VS Code拡張機能がマーケットプレイスに表示されない問題が報告されている。他の拡張機能（vscode-pdf等）はインストール可能なことから、マーケットプレイス変更か一時的な不安定さが原因と推測。同日別ツイート（2056842189850968407）ではその後インストール完了を報告しており、一時的な不安定さだった可能性が高い。Antigravity IDEでClaude Code for VS Codeをセットアップする際の参考情報。
