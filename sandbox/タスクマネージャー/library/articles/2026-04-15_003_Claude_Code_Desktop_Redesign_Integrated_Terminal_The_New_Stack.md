## 投稿内容
The New Stack covers Anthropic's Claude Code desktop app redesign released April 14, 2026. New features: integrated terminal (no external terminal required), faster diff viewer, in-app file editor, expanded preview area with rearrangeable panes. The redesign accompanies the Routines research preview launch. Claude Code can now produce interactive charts, diagrams, and mobile apps within the desktop. Computer Use integration for GUI interaction also available. The redesigned desktop positions Claude Code as an all-in-one agentic coding environment.

## 要約
Anthropicが2026年4月14日にClaude Codeデスクトップを大幅刷新。**統合ターミナル（外部ターミナル不要）・高速diffビューア・アプリ内ファイルエディタ・拡張プレビューエリア**が追加され、再配置可能なペイン構成に。Routinesリサーチプレビューと同日リリース。インタラクティブチャート・ダイアグラム・モバイルインタラクティブアプリの生成もデスクトップ内で完結。Computer Use統合によりGUI操作も可能。Claude Codeがターミナル専用ツールから本格的なオールインワンのエージェントコーディング環境へ進化した転換点。
