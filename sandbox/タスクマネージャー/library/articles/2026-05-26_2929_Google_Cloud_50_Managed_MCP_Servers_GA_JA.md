# Google Cloud 50以上のマネージドMCPサーバーが一般提供開始

## 要約
Google Cloud Next '26 発表。BigQuery・Spanner・Cloud SQL・Vertex AI・Google Maps など GCP 主要サービス 50以上がMCPサーバーとして一般提供（GA）またはプレビューで公開。認証はGCP既存IAM・サービスアカウントをそのまま使用。従来は自前でMCPサーバーを実装・運用する必要があったが、マネージドサービス化でインフラ管理コストがゼロに。Claude Code・Cursor・Windsurf等のMCP対応クライアントから即座に接続可能。MCPエコシステムの企業採用を大幅に加速させる重要発表。
