## 要約

Claude Code SDK（Agent SDK）の日本語解説記事。APIとの違いと具体的な統合パターンに焦点：
- **SDK vs API の違い**: APIは単発リクエスト、SDKはClaude Codeの全機能（ファイル操作・コマンド実行・MCP）を継承したエージェント実行環境
- **CI/CDインテグレーション**: GitHub Actions・GitLab CI・CircleCIへの組み込みコード例
- CLAUDE.mdによるチーム共通ルールをCI環境でも適用する方法
- SaaS UIカスタマイズへの応用（エンドユーザーがコードを書かずにUIを変更させる）

「SDKを使う理由」が分かりやすく説明されており、導入判断の材料として活用できる。
