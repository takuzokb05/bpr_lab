## 投稿内容
Anthropic has released Routines for Claude Code — automations that you schedule and repeat. They run on Anthropic's web infrastructure so your Mac doesn't need to be online. Routines have access to your repos and your connectors. Limits: Pro=5/day, Max=15/day, Team/Enterprise=25/day. New Claude Code Desktop with parallel sessions, integrated terminal, and file editing also launched simultaneously.

## 要約
Anthropicが2026年4月14日（本日）にClaude Code向け新機能「Routines」を正式リリース。スケジュール設定して繰り返し実行できる自律タスク自動化機能。**Anthropicのウェブインフラ上で動作するためMacがオフラインでも継続実行**される点が革新的で、cronジョブやMCPサーバーの自前管理が不要になる。リポジトリやコネクタへのアクセス権を持ち、スケジュードAPIワークフロー・GitHubルーチン等が主な用途。プラン別制限：Pro=1日5件、Max=1日15件、Team/Enterprise=1日25件。並列セッション・統合ターミナル・ファイル編集機能を備えた新デザインのClaude Codeデスクトップも同時リリース。Managed Agentsと並びClaude Codeの「常時稼働エージェント」化を加速する一手。
