## 要約
Claude Codeの4つの拡張システム（MCPサーバー・Skills・カスタムAgents・Hooks）を体系的に解説したMorph社のガイド。MCPはClaude外部ツール接続、Skillsはslashコマンド付きカスタムワークフロー、Agentsは独立コンテキストで動くサブエージェント、Hooksは決定論的ライフサイクル処理という役割分担を明確化。「何を使うべきか」の判断フレームワークを提供しており、MCP/Skills/Agents/Hooksの使い分けに悩む実践者向けの一次情報として価値が高い。Claude Code 2026年版として最新機能（Routines含む）にも言及。
