## 要約
Claude Code v2.1.91以降で使えるUltraplan機能のハンズオンレビュー。CLIから/ultraplanコマンドを実行するとクラウド上のClaude Codeセッションがプランモードで動き、ブラウザで結果を確認・修正・承認できる。ターミナルを占有しないため別作業と並行可能。ブラウザ上でのレビューはインラインコメント・絵文字リアクション・アウトラインサイドバーが使え、特定段落を指定して修正依頼できる。承認後はローカル環境またはクラウドで実行可能。ローカル実行はテスト環境・社内ツール等が必要な場合に推奨。要件：Claude Code v2.1.91以上＋Claude Code on webアカウント＋GitHubリポジトリ。現在リサーチプレビュー段階。実際に試したところ「大規模リファクタリングの計画段階でターミナルを解放できる点」が最大の利点との評価。
