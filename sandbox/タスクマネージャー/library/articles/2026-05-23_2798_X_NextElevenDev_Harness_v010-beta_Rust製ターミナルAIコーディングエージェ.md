After significant work, Harness v0.1.0-beta is now public.
It's a terminal-first, local-first AI coding agent written in Rust with strong emphasis on performance,
memory, and developer control.
Key features:
- Multi-provider TUI (xAI Grok, Claude, OpenAI, local)
- Semantic + project memory with hybrid recall
- Sub-agent swarm support
- Cost tracking, plan mode, MCP, browser automation
- One-liner install script with prebuilt binary support
