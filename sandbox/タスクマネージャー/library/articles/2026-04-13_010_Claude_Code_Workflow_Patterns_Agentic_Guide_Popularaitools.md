## 要約
Claude Codeのエージェント的ワークフローパターンを体系化した2026年版ガイド。探索→計画→実装のフェーズ分割、コンテキスト管理、セッション間引き継ぎのベストプラクティスを解説。「オーケストレーション層」として複数Claudeセッションを協調させるパターン（メインセッション＋専門サブエージェント群）を詳説。非同期実行でClaude Codeをバックグラウンド作業者として活用するパターン、イベント駆動パイプライン（hooksで特定イベントを起点に別処理をトリガー）、並列実行モデル（gitワークツリーごとの独立セッション）の組み合わせを体系化。多くの開発者が「オーケストレーション層・非同期機能・セキュリティガードレール・クロスデバイス継続性」を活用しておらず、単一ツールとしてしか使っていないという問題提起が含まれる。
