## 投稿内容
Osaurus is an open-source LLM harness for Mac that runs models locally or in the cloud. Switches between OpenAI, Anthropic &amp; local models while keeping files &amp; tools on-device. 
20+ native plugins, MCP server, voice support. 112K downloads.
