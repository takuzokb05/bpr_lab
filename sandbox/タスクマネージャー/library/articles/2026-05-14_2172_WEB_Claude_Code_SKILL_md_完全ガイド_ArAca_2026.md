## 要約
Claude CodeのSkill機能においてSKILL.mdファイルの書き方と構成を徹底解説した実践記事。スキルのname（小文字ハイフン区切り推奨、例：pdf-editor）とdescription（最大200文字で具体的なトリガー条件）が最重要フィールドであることを強調。Claude CodeはSKILL名とdescriptionだけでトリガー判定を行い、必要時のみrefereces/内ファイルにアクセスするプログレッシブロード方式。フロントマターのtrigger設定（auto/user/none）の使い分け、supportingファイルの配置方法、実際のSKILL.mdテンプレートを多数掲載。CLAUDE.mdとの役割分担（200行超えたらSkillsや.claude/rules/に分離）も解説。
