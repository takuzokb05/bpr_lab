## 要約
Claude Code Agent Viewは全セッションを一元管理するUIで、エージェントの起動・バックグラウンド送信・状態確認・セッションへの再参入が可能。複数エージェントを並列稼働させながら必要な時だけ介入する「非同期開発フロー」を実現。エージェント名称・最終応答・ステータスが一覧表示され、タイプアヘッド補完でも名前が出る。Claude Opus 4.7がMax/Team Premiumのデフォルトモデルとなり、新たにxhighエフォートレベルと/effortスライダーが追加された。Agent Viewと組み合わせることでCI的な並列タスク実行が可能になる。
