## 要約
ZennのShirochan氏によるAnthropic API新機能解説。4つの新機能：①コード実行ツール（サンドボックスでPythonをネイティブ実行）、②MCPコネクター（APIリクエストにリモートMCPサーバーURLを追加するだけで外部ツール即時利用可能）、③Files API（ファイルをアップロードして会話を跨いで参照、RAG構築コスト削減）、④拡張プロンプトキャッシュ（最大12xのトークンコスト削減）。具体的な実装コードと各機能のユースケースを示す。エージェント開発のインフラが整備されたことで複雑なパイプライン構築が容易に。日本語エンジニア向け最速解説として高評価。
