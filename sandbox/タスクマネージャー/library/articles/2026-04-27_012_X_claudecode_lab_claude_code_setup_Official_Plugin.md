✅プロジェクトを自動分析  
✅おすすめの自動化手法（hooks, skills, MCP servers, subagentsなど）自体を提案

公式ページ：https://claude.com/plugins/claude-code-setup

## 要約
Anthropic公式プラグイン「claude-code-setup」の紹介。Claude Codeの初期設定を自動化するプラグインで、①プロジェクトの自動分析、②Hooks・Skills・MCP servers・Subagentsなど最適な自動化手法の提案、③各設定手順のステップガイドを提供。インストールは `/plugin install claude-code-setup@claude-plugins-official` で実行。新規プロジェクトや新メンバーのオンボーディング時の摩擦を大幅削減。公式プラグインであることから精度・信頼性が高い。Claude Code v2.1.101以降で利用可能。既存プロジェクトにも適用でき、未使用の自動化余地を発掘するのにも有効。
