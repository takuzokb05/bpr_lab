## 要約

Claude Agent SDK入門チュートリアル（コード付き）。
- インストール（`pip install claude-agent-sdk`）から基本エージェントループ実装まで
- **カスタムツール登録**: Pythonデコレータ（`@sdk.tool()`）で関数をMCPツールとして公開する方法
- MCP外部サーバーとの統合（既存MCPサーバーをSDK経由で呼び出す）
- ストリーミングレスポンスとバッチ処理の使い分け

ゼロからエージェント実装を始めるのに必要な最小限のコードサンプルが揃っており、公式ドキュメントの補完として有用。
