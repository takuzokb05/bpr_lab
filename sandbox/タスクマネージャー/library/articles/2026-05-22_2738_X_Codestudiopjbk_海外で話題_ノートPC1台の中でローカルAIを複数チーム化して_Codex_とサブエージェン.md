【海外で話題】
ノートPC1台の中で“ローカルAIを複数チーム化して、
Codex とサブエージェントを同時に走らせる”運用がデモで現実になってます😳使われたのは
https://t.co/SPayaqW98W
Codex CLI と『ローカルでAIモデルを動かす無料デスクトップアプリ』の組み合わせ👇
・MacBook Pro M5 一台で Codex CLI を司令塔に立てる
・ローカル側で Qwen 3.6 を複数同時起動、サブエージェント化
・コードレビューとバグ探索を並列＆オフラインで全部回す
裏で動いてるのは更新版の MLX エンジンで、batching 対応のおかげで複数モデル同時稼働でも体感速度が落ちないのが今回のバズりポイントです。
構成自体は完全に再現可能で、必要なのは Mac M5 と無料デスクトップアプリ、Qwen の重み、Codex CLI のセットアップだけ。
ノートPC1台で『ローカルCI/CD班』が組めるレベルです。
つまり
個人開発でも“ローカル軍団 × Codex 司令塔”で、完全自前のレビュー班が組める時代に入りました。
クラウド推論を全部止めても運用が成立するという意味で、結構な分岐点です。
気になるのは、ローカル軍団に任せる範囲と Codex 本体に任せる範囲の引き方。
