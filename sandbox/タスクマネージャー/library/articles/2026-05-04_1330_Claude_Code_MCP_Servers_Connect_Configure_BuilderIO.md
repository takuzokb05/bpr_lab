## 要約
Builder.ioによるClaude Code MCPサーバーの接続・設定・活用ガイド。3種のトランスポート（stdio・HTTP・SSE）の使い分け、コーディングタスクで最も実用的なMCPサーバー（GitHub・Playwright・Sentry・PostgreSQL/Supabase・Figma）の設定手順を詳細に解説。MCPがClaude Codeの標準ツール（ファイル操作・Shell実行・Web検索）を超えてイシュートラッカー・モニタリングデータ・データベースへのアクセスを可能にする仕組みを説明。設定ファイル（.claude/settings.json）でのMCPサーバー定義方法、認証トークンの安全な管理（環境変数使用）も解説。Builder.ioは実際の大規模プロダクション開発でClaude Codeを活用しており、実践的な視点からの解説として信頼性が高い。
