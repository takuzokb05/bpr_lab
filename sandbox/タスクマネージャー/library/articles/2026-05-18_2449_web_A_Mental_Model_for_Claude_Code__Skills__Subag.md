## 要約
Level Up Coding掲載。Skills・Subagents・Pluginsを「なぜ3つに分かれているか」というメンタルモデルから解説した思考整理記事。
スキル＝再利用可能な指示セット、サブエージェント＝独立コンテキストで動く特化型AI、プラグイン＝スキル+テーマのチーム配布単位と定義。
どの拡張を使うべきかの決断フレームワークを3つの問いとして提示。
実装よりも概念理解を重視した内容で、チーム内教育資料として活用しやすい。
