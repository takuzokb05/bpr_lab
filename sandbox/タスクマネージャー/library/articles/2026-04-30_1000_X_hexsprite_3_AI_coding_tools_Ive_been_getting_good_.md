## 投稿内容
3 AI coding tools I've been getting good mileage out of lately:

🪨 *Caveman* (https://t.co/ms5fJCEktc)
A skill/plugin for Claude Code, Cursor, and Copilot that compresses agent output into "caveman speak". Roughly 75% fewer output tokens with full technical accuracy. Not a gimmick, it meaningfully reduces the verbosity of these tools. One-line install.

🧰 *Plannotator* (https://t.co/cNQ0rjWw2g)
Visual UI for reviewing diffs before opening a PR, doing code reviews, and iterating on Markdown plans over multiple revision cycles. Annotate, suggest replacements, then send structured feedback back in one click. Works with Claude Code and Copilot.

🧹 *Desloppify* (https://t.co/hoa9hpy1dy)
An agent harness that cleans up code smells like dead code, duplication, naming drift, and weak abstractions. Way more effective than a prompt-based approach because it ships with heuristic guardrails and a hard-to-game quality score. Combines mechanical detection with LLM review and runs your agent through a prioritized fix loop. Works with Claude Code, Cursor, and Copilot.

All three are free and open source.
