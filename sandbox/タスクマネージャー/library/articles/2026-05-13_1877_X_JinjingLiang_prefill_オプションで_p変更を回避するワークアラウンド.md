💡Here's a work around for the Claude Code `-p` change

This is actually a hidden option that has been in Claude Code awhile. Simply use `--prefill`

We have use it in @orca_build for months to get around the limitations

## 要約
Claude Codeの`-p`（非インタラクティブモード）が従量課金に移行する変更への具体的なワークアラウンド。`--prefill`という隠しオプションが以前からClaude Codeに存在しており、`-p`の代替として機能するとのこと。投稿者は@orca_buildプロジェクトで数ヶ月間本番利用してきた実績を持つ。新規ユーザーにはあまり知られていない隠しオプションで、自動化スクリプトや非インタラクティブ用途を維持したい開発者にとって即実用可能な情報。公式ドキュメントに載っていない可能性が高いため希少度が高い。
