# Claude Code デスクトップ刷新: 並列セッション・統合ターミナル・ドラッグ&ドロップ

## 要約
Anthropic公式ブログが2026年4月14日に発表したClaude Codeデスクトップアプリ刷新の詳細。複数リポジトリ間でのパラレル作業を前提とした設計に変更。主要機能：(1)マルチセッションサイドバー—全アクティブ/過去セッションを一覧表示し、複数リポジトリ間を切り替え可能；(2)統合ターミナル—テスト・ビルドをセッション横に実行；(3)インアプリファイルエディタ—スポット編集に対応；(4)高速差分ビューア—大規模変更セットでも高パフォーマンス；(5)拡張プレビュー—HTML/PDFをアプリ内で直接開く；(6)ドラッグ&ドロップレイアウト—ターミナル・プレビュー・差分ビューア・チャットを自由に配置。Pro/Max/Team/Enterprise全プランとAPI経由でも利用可能。デスクトップがRoutinesとあわせてリリース。
