## 投稿内容
I've been asking Claude Code to create HTML rather than Markdown files, following Karpathy's advice that voice-in but image-out might be the best way to talk to AI. I'm sticking with Markdown files in the repo for now, but HTML files are great for deep dives and sharing.

## 要約
Karpathyの「音声入力・画像出力がAIとの最良の対話方式」という考え方に触発されたClaude Codeの活用パターン。リポジトリ内のドキュメントはMarkdownで管理しつつ、詳細調査や共有目的のアウトプットはHTMLファイルとして生成させるという使い分け戦略。HTMLは視覚的リッチネスと共有容易性でMarkdownを上回る場面があり、Claude Codeに「output.htmlを作って」と指示するだけで実現できる。出力フォーマットをコンテキストに応じて切り替えるという実践的なワークフロー改善の提案。
