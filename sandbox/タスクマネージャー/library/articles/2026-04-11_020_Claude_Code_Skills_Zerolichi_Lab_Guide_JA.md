## 要約
株式会社ゼロイチラボによる「スキルを育てる運用」に特化したClaude Code Skills実践ガイド。単なる作り方だけでなく継続的改善プロセスに焦点。重要な知見：①スキルの「自動発動」はdescriptionの最適化がトリガー精度を決める②スキルファイルは2,000トークン以内を守りつつ複数ファイルに分割する設計③使用頻度の低いスキルは定期的にアーカイブして肥大化防止④Claude自身にスキルを評価・改善させる「Autoresearch」的運用⑤チーム展開時の権限設計（user-invocable vs disable-model-invocationの使い分け）。実績数値：PRマネージャー1人で32人分のカバー、30分の業務を30秒に短縮。スキルを「作って終わり」ではなく「育て続ける」視点が他記事との差別化ポイント。
