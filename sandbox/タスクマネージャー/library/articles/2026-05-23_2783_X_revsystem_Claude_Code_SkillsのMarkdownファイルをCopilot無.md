## 投稿内容
Microsoft Copilotの無料版でClaude CodeのSkillsが使えないかと試してたら、できた。Gitやローカルファイルを扱う指示を抜いたMarkdownファイルを作って、それをチャットに添付するだけ。呼び出し方はskillによって違うだろうから、そこはCopilotに聞けば答えてくれる。
