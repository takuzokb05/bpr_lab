## 要約
Uravationによる「Claude Security」解説記事（2026年5月版）。Claude Securityは2026年4月30日にPublic Betaとして公開されたセキュリティ特化機能。主な特徴：①コードベース全体を自動スキャンし脆弱性を検出、②検出した脆弱性に対してパッチを自動生成・適用、③ワンストップのセキュリティワークフロー（スキャン→検出→修正）を提供。Claude CodeのHooksやCI/CD統合と組み合わせることでセキュリティチェックの自動化が可能。既存のSASTツール（SonarQube等）との差別化ポイントはLLMベースの文脈理解による精度向上とパッチ自動生成。FX自動取引など本番システム開発においてもセキュリティ品質向上に直結する重要機能。
