# OpenAI shakes up partnership with Microsoft, capping revenue share payments

## 要約
CNBC による Microsoft-OpenAI パートナーシップ改訂の報道（2026年4月27日）。OpenAI はMicrosoftへの収益シェア義務を2030年まで維持するが固定上限を設定。Microsoftの OpenAI への独占ライセンス終了（2032年満了の非独占IPライセンスに変更）。AGI条項の削除により IPO準備の法的整合性が確立。OpenAI の $50Bドル AWS 契約への道筋確立。Bloomberg・Reuters・Computer World 等メディアが一斉報道した業界再編ニュース。エージェントAI市場でのマルチクラウド競争激化を示す。
