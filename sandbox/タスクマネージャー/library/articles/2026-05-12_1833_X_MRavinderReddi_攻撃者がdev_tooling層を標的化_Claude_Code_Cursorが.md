## 投稿内容
Attackers stopped chasing random crypto wallets and started chasing the dev tooling layer because AI agent infrastructure is the new high-value pipe. Hook a single Claude Code or Cursor extension and you exfiltrate codebases from thousands of dev machines silently.

## 要約
ランダムな暗号ウォレットよりもdev tooling layerを狙う攻撃者が増加。AIエージェントインフラが高価値パイプラインとして標的化。単一のClaude Code/Cursor等への侵害で大規模被害の可能性。AIコーディングツールを日常利用するデベロッパーへのセキュリティ意識喚起として重要。
