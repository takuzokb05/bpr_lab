## 投稿内容
GitHub stars does not make an AI tool safe to run.

Before installing a random Claude Code tool, MCP server, Cursor plugin, or agent repo, spend 5 minutes here:

1. Check the author
new account + one viral repo = red flag

2. Check package.json
look for preinstall, postinstall, prepare

3. Search for curl, wget, base64, eval, child_process

4. Open dependencies
install scripts can hide 3 levels down

5. Check releases
precompiled .exe, .dmg, .7z from a new repo = leave

6. Check the star curve
2,000 stars in 4 days is bought attention until proven otherwise

7. Check issues and PRs
real repos have messy bug reports and boring maintainer replies

8. Run first in a sandbox
no main laptop, no real tokens, no wallet, no private repos

GitHub stars are cheap. Your browser cookies, API keys, and private repos are not.
