## 要約
SmartScopeによるClaude Agent SDK入門ガイド。pip install claude-agent-sdk / npm install @anthropic-ai/claude-agent-sdkで10行以下のコードからエージェントを構築できることを実例付きで解説。Read/Write/Edit/Bash/Glob/Grep/WebSearch/WebFetchなどClaude Code同等のビルトインツールが即座に使用可能。Hooks（PreToolUse, PostToolUse, SessionStart等）のコールバックも利用でき、Claude Codeで実証済みの同じエージェントループを活用できる点を強調。Anthropicが2026年初頭に「Claude Code SDK」から「Claude Agent SDK」へリネームした経緯も説明。
