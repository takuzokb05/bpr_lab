# ClaudeCodeのサブエージェント機能で、テスト修正を自動化してみた

- 言語: ja
- テーマ: claude-code

## 要約

コミット時フックでサブエージェントを自動起動し、テスト実行→分析→修正→再検証→レポートまでを完全自動化した実装記事。git pre-commitフックとClaude Codeサブエージェントを組み合わせた実践的なCI/CD自動化の実装例。フック×サブエージェントのパターンを具体的に示した参考事例。
