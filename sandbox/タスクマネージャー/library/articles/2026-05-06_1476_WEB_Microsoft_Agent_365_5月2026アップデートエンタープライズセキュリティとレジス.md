## 要約
Microsoft Agent 365 v2.0のMay 2026アップデート（2026年5月1日）。主な新機能：①レジストリ同期（Registry Sync）：AI管理者がエコシステムパートナーのエージェントプラットフォームをAgent 365に安全に接続・同意し、外部エージェントとメタデータを統合ビューで管理可能に、②エンタープライズセキュリティ・コンプライアンスフレームワークとの統合強化。Microsoft Agent FrameworkはAnthropicのClaude Agent SDK・GoogleのADKと並ぶメジャーなエージェントフレームワークとして確立しており、エンタープライズ向けガバナンス機能の充実が続く。Claude CodeのRoutinesやManaged Agentsと類似するエンタープライズ向けエージェント自動化市場での競争を示す。
