## 投稿内容
CEO Sundar Pichai just announced that 75% of all new code at Google is now AI-generated, up from 50% just last fall.

Every single line is still reviewed and approved by human engineers, but the company is shifting to agentic AI workflows where AI agents handle complex tasks such as large-scale code migrations one project finished six times faster than before.

Google is also using internal tools like Antigravity to prototype entire apps, such as the Gemini macOS version, in just days instead of weeks.

## 要約
Google Cloud Next 2026でSundar Pichai CEO発表。Google社内の新規コードの75%がAI生成（前回2025年秋は50%）。全行が引き続き人間エンジニアのレビュー・承認を経るが、AIエージェントが大規模コードマイグレーション等の複合タスクを自律処理するagentic workflowへシフト中。実績として「ある大規模マイグレーションプロジェクトが従来比6倍速で完了」を挙げた。社内ツール「Antigravity」ではGemini macOS版全体を数週間→数日でプロトタイプ。AIコード生成が実験段階を超え、世界最大規模のソフト開発組織で主流となった実態を示す一次データとして重要。
