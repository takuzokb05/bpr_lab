Claude Code isn’t free.
Codex isn’t fully free.

But Sonnet 4.6 and GPT-5.4 are already available
in Claude & ChatGPT.

What’s missing is "Access to your codebase"

I built        MCP-BRIDGE         to bridge that gap.
Fully Open-source.
It wires ChatGPT & Claude Desktop
