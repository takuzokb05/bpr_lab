## 要約
MindStudio公式ブログによるClaude Code組み込みスキル「/simplify」と「/batch」の詳解記事。/simplifyは変更済みコードをレビューして再利用・品質・効率の観点から問題を発見し自動修正するスキル。/batchは大規模コードベース変更を5〜30の独立ユニットに分解し、バックグラウンドエージェントが並列処理してPRを作成する機能。/batchの内部フロー（コードベース調査→計画提示→承認→並列実行→テスト→PR作成）を詳細に説明。活用例：「全ファイルの型注釈追加」「APIエンドポイント一括リファクタリング」。Claude Code 2.1系の組み込みスキルとして公式ドキュメントに基づく信頼性の高い解説。
