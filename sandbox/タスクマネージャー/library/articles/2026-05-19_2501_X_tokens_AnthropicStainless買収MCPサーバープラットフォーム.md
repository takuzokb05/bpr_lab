## 投稿内容
ICYMI: Anthropic has acquired Stainless, the SDK and MCP server platform that powers its API, bringing core infra in-house. https://t.co/E96kWtStDx

## 要約
AnthropicがStainlessを買収し、自社APIを支えるSDK・MCPサーバープラットフォームをコアインフラとして内製化したことを報告。Stainlessは公式Anthropic SDKの生成エンジンとして機能しており、今回の取得によりAnthropicはSDK生成からMCPサーバー展開までの開発者向けインフラスタックを自社管理下に収めることになる。
