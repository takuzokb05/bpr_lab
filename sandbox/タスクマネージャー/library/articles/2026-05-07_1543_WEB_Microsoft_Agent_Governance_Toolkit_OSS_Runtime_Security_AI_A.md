# Microsoft Agent Governance Toolkit: OSS Runtime Security for AI Agents

## 要約
Microsoftが2026年4月2日にMIT License下でオープンソース公開した「Agent Governance Toolkit」の発表記事。自律的AIエージェントにランタイムセキュリティガバナンスを提供するフレームワーク。主要機能：①エージェントアクションのリアルタイム監査ログ、②ポリシーエンジン（許可/拒否ルール定義）、③コンプライアンスチェック、④インターセプトフック（エージェントの全ツール呼び出しを傍受・検証）。Microsoft Agent Framework 1.0と連携して動作し、LangGraph・Claude Agent SDK等のサードパーティフレームワークにも対応。エンタープライズがAIエージェントを安全に本番運用するための必要コンポーネントとして位置づけられている。
