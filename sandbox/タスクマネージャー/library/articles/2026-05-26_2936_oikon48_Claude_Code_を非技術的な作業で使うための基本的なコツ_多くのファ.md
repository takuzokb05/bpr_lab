## 投稿内容
Claude Code を非技術的な作業で使うための基本的なコツ:

多くのファイルをフォルダに入れて、Claude にスクリプトを書いたり HTML を作ったりできると伝えること。

例⬇️

- 画像やビデオの編集→ スクリプトを書く

- 財務、税務作業など→ PDFに入れる、スクリプトを書く、HTMLを出力

- 医療アドバイス→ PDF＋データに入れる、HTMLを出力

- 書類の記入→スクリプトを書く

- レポートの作成→ HTMLを書く

- 計画の立案→ HTMLを書く

データアクセスも重要なため、Gmail、Google Calendar などのコネクタを追加する。

一方で、多くの人はファイル内のコンテキストの量を過小評価している。
