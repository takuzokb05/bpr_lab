## 投稿内容

Claude Plugins Official 是 Anthropic 官方维护的 Claude Code 插件目录。

它把 skills、agents、hooks、slash commands 和 MCP server 配置集中成可发现、可安装、可复用的插件生态。当前目录里有 35 个官方插件和 15 个外部插件，安装方式也很直接：/plugin install {plugin-name}@claude-plugins-official

👉 https://t.co/6FQY8Vfiyg
