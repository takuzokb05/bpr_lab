## 投稿内容
You can use now Codex in Claude Code 🤯

OpenAI has dropped a plugin that lets you run Codex directly inside Claude Code.

Code reviews, adversarial reviews, background tasks…one command to install:

/plugin install codex@openai-codex

## 要約
OpenAIがClaude Code向けにCodexプラグインを公開したという情報。インストールは`/plugin install codex@openai-codex`の1コマンドで完了。用途としてコードレビュー、adversarial review（対抗的レビュー）、バックグラウンドタスクなどが挙げられている。競合他社のAIモデルをClaude Codeのプラグインエコシステム経由で利用可能にするという点で、Claude Code Marketplaceのオープン性を示す事例。異なるAIモデルを目的別に使い分けるマルチモデルワークフローの実現手段として注目。
