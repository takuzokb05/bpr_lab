## 要約

Engadgetによる2026年3月23日のAnthropicのComputer Use機能発表の報道。Claude CodeとClaude Coworkがデスクトップを操作できるようになり（ファイルを開く・ブラウザナビゲーション・開発ツール起動）、当初はmacOSのClaude Pro/Maxサブスクライバー向けに限定提供。AIエージェントが実際にコンピュータを操作するという大きな転換点を一般向けに解説している。
