# DeepSeek V4 Preview Released: 1.6T Params, 1M Context, 7x Cheaper than GPT-5.5

- URL: https://www.cnbc.com/2026/04/24/deepseek-v4-llm-preview-open-source-ai-competition-china.html
- ソース: web
- 言語: en
- テーマ: ai-news
- 取得日: 2026-04-27

## 要約
中国のDeepSeekが2026年4月24日にDeepSeek-V4-ProとDeepSeek-V4-Flashのプレビュー版をリリース。V4-ProはMoEアーキテクチャで1.6兆パラメータ、デフォルト100万トークンコンテキスト。V4-Flashは2840億パラメータ。主要ベンチマーク：SWE-bench Verified 80.6%（Claude Opus 4.6の80.8%にほぼ並ぶ）、Codeforces 3,206（GPT-5.4の3,168を超え全モデル最高）、MMLU-ProはGPT-5.4と同水準。価格はV4-Pro入力$0.145/MTok（GPT-5.5の約7分の1）、出力$1.74/MTok（同約6分の1）。オープンソースモデルとしてコーディング・数学で全ライバルを上回り、クローズドモデルではGoogle Gemini 3.1-Proのみが上回る。AI競争が激化する中で西側プロプライエタリモデルの価格優位性に直接挑戦する存在となった。
