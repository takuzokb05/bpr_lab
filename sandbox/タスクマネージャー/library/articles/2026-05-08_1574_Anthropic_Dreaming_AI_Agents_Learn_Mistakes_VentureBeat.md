## 投稿内容

VentureBeat による Dreaming 機能の詳細報道。エージェントが自分の過去の失敗から学習するシステムの技術詳細。

## 要約

Dreaming は Claude Managed Agents の新機能（Research Preview）。エージェントが「非稼働時間」に過去セッションのログを振り返り、失敗パターン・改善点を抽出して次回以降に反映する仕組み。技術的には長期メモリストアへの自動書き込みと構造化リフレクションを組み合わせる。Anthropic は「エージェントは人間同様、睡眠中に経験を統合する必要がある」と設計意図を説明。実用上のインパクト：繰り返しタスクの精度向上・手動プロンプト調整の削減・エージェントの自律的な品質改善。Research Preview 段階のため API アクセスは申請制。Outcomes 機能（目標定義→反復実行）と組み合わせることでエージェントの自律改善ループが完成するアーキテクチャ設計。長期エージェント運用の新しいパラダイムとして注目。
