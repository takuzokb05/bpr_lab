## 要約
2026年5月14日、Anthropicが公式発表：Claude Codeの週次利用制限を50%増加。対象期間は2026年7月13日まで（約2ヶ月限定）。対象プラン：全有料プラン（Pro・Max・Team・Enterprise）。5月6日にも5時間レート制限の2倍化・ピーク時間帯制限撤廃が実施済み。Maxプランユーザーには大きな恩恵で、大規模プロジェクト・Routines活用のハードルが下がる。定量情報：週次制限50%増加、7月13日まで、全有料プラン対象。Claude Codeを日常業務で多用しているユーザーには重要な公式制限緩和情報。
