## 要約
Jidong LabによるClaude Agent SDK（旧Claude Code SDK）の詳細解説記事。「Claude Codeをライブラリとして使う」コンセプトを説明。SDK3大優位点：①組み込みツール（ファイル読み書き・コマンド実行のボイラープレート不要）、②MCPサーバーとの深い統合（Playwright・Slack・GitHub等）、③18のフックイベントによるエージェント実行への介入機能。Python・TypeScript両SDKでの実装例を提示し、セッション管理（--resumeフラグ）、権限モード設定（bypassPermissions/acceptEdits等）の使い方を説明。`context-1m-2025-08-07`ベータの2026年4月30日廃止と、Claude Sonnet/Opus 4.6への1Mコンテキスト標準移行も解説。
