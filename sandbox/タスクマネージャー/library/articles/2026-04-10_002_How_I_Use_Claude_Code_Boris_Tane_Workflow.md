## 要約
Boris Tane による Claude Code の実践ワークフロー解説。10〜15 個の並列セッションを git worktree で衝突なく運用する「並列開発」、Plan Mode でまず計画→スタッフエンジニアレビュー→実行という「計画ファースト」アプローチが核心。CLAUDE.md は約 2,500 トークン（約 100 行）に抑え「Claudeが間違えたら CLAUDE.md に追記」ルールを徹底。/rewind でロールバック、テストを外部真実ソースに使う「テスト駆動 AI 開発」、サブエージェントをスペシャリスト化して main コンテキストをクリーンに保つパターンも詳説。定量的なワークフロー設計と具体的なコマンド例が豊富で実践度の高いガイド。
