## 要約

Model Context Protocol公式ブログによる2026年ロードマップ。4つの優先領域：
1. **Streamable HTTP Transport**: ステートレスな水平スケーラブルなリモートサーバー実現（現行SSEの後継）
2. **`.well-known` Capability Discovery**: ライブ接続なしでサーバー機能を事前に問い合わせる標準エンドポイント
3. **Tasks Primitive (SEP-1686)**: 長時間実行タスクのライフサイクル管理（リトライセマンティクス、有効期限ポリシー）
4. **Enterprise Extensions**: 監査ログ・SSO統合認証・ゲートウェイ挙動の標準化

MCPはLinux Foundation AAIF傘下で標準化が進行中。2024年11月の現行仕様からの具体的な変更点を把握するための一次情報。
