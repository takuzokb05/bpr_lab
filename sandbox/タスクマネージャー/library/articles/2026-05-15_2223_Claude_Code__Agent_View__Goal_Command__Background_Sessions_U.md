## 要約

/goalコマンドはターゲット達成まで自動継続し、マルチターン作業を自律化。
Background Sessionsはフォアグラウンドとは独立してバックグラウンド処理を実行する機能。
Claude Opus 4.7が新デフォルト、xhighエフォートレベルとRoutinesスケジュール実行も追加。
Routinesはスケジュール・GitHubイベント・APIの3トリガーでクラウドエージェントを起動できる。
