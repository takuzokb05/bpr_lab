## 要約
VoltAgentが管理する100以上の専門化されたClaude Codeサブエージェントのキュレーションリポジトリ（GitHub）。コードレビュー・テスト生成・ドキュメント作成・セキュリティ監査・パフォーマンス最適化など開発ユースケースを幅広くカバー。各サブエージェントはSKILL.mdと同様のYAMLフロントマター形式で記述されており、即座に流用可能。awesome-claude-codeリスト（hesreallyhim管理）と並ぶコミュニティ主導リソース集。サブエージェント設計の参考パターンとして特に「専門特化型エージェントの定義構造」「タスク分割のベストプラクティス」を学べる。Claudeエージェント機能を組織的に活用したいチーム・個人開発者にとって実践的な出発点となる。
