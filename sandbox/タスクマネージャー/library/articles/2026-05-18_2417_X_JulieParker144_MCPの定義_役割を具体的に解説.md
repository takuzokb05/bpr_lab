## 投稿内容
MCP 

MCP usually stands for Model Context Protocol in the AI/software world.

It’s an open standard introduced by Anthropic to let AI models connect to external tools, data sources, and applications in a standardized way—kind of like giving AI a universal adapter instead of building a custom integration for every app.
Official info:  Model Context Protocol documentation

## 要約
MCPの定義と役割を具体的に解説：AIアプリ（ホスト）が外部データソース・ツール（サーバー）を発見・利用するためのオープン仕様。教育的価値の高い解説。
