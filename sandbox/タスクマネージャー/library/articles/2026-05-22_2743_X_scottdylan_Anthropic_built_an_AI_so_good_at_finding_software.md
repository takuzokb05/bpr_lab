## 投稿内容
Anthropic built an AI so good at finding software flaws they refused to ship it.

So they handed it to Apple, Google, Microsoft, AWS and Cloudflare instead.

One month later:

— 10,000+ zero-days uncovered

— A bug hidden in OpenBSD for 27 years

— Working exploit chains the model wrote unprompted

— Cloudflare’s bug-find rate up tenfold

Anthropic’s own words today: no company, including them, has safeguards strong enough to stop a model like this being misused.

We are watching the offence/defence balance of the entire internet shift in real time — and almost no one is paying attention.



## 要約
AnthropicのProject Glasswing（Claude Mythos）がApple・Google・Microsoft・AWS・Cloudflareと連携し1ヶ月で10,000件超のゼロデイ脆弱性を発見。Cloudflareのバグ発見率が10倍に向上。OpenBSDに27年間潜伏していたバグも発見。モデルが自発的に動作するエクスプロイトチェーンを生成。Anthropic自身が「いかなる企業も誤用を止める十分な安全策を持たない」と公言。パッチ適用が追いつかずメンテナーがAnthropicに減速を要請。
