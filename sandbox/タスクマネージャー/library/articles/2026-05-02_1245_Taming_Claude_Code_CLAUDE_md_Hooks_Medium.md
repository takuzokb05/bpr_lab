## 要約
MediumのBecoming Better誌掲載。CLAUDE.mdとHooksを組み合わせた実践的なClaude Code制御ガイド。CLAUDE.mdはセッション開始時に自動読み込みされる「永続的メモリ」として機能し、Bashコマンド・コードスタイル・ワークフロールールを記載。HooksはPreToolUse/PostToolUse/UserPromptSubmitなど12イベントで発火し、決定論的なガードレールを提供。3つのコアHook（フォーマット自動適用・テスト自動実行・危険コマンドブロック）の設定例を具体的に解説。「エラーが起きたらCLAUDE.mdに書く」という反復的な改善プロセスが価値ある知見。
