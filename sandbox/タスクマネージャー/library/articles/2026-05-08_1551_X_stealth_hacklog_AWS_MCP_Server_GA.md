## 要約
AWS MCP ServerがGA（一般提供）になったことを報告。`call_aws` 1つのコールでAWS全APIが呼び出し可能になり、CloudFormation/EC2/S3などの複雑なインフラ操作をClaude Codeから直接実行できる。インフラ構築の自動化が大幅に進む可能性。AWSコンソールを介さずに自然言語でAWSリソースを操作できることを意味し、AIによるクラウドインフラ管理の実用化が加速する。
