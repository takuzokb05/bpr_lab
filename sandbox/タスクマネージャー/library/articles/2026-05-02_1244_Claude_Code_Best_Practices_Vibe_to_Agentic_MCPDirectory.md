## 要約
mcp.directoryによるClaude Codeベストプラクティス解説。CLAUDE.mdを「チームの集合知」、Hooksを「反射神経」と定義し両者の役割を明確化。/initでスタータCLAUDE.mdを生成し、Claudeがミスをするたびにその教訓をCLAUDE.mdに書き加える運用を推奨。Hooksは「例外なく毎回実行が必要なアクション」に使い、CLAUDE.mdは助言的・設定的な情報に使うという棲み分けを解説。split into `.claude/rules/*.md` with path globs for context-aware loading という高度なテクニックも紹介。MCP統合でNotion・Figma・DBとの連携も解説。
