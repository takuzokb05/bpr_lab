## 要約
MCP（Model Context Protocol）の完全ガイドで、Claude CodeおよびCursorとのツール連携、設計7原則と5つの失敗パターンを解説。MCP Connectorを使ったAnthropicクライアント側のMCP接続方法、ツール定義のJSON スキーマ設計、リソース管理、エラーハンドリングのベストプラクティスを含む。「設計7原則」として粒度・命名・べき等性・エラー通知・認証・バージョン管理・テストの7軸を提示。5つの失敗パターンはToolsの乱用・スキーマ不足・認証不在・エラー飲み込み・テスト省略。エージェントシステム構築に直接応用できる実践的内容。
