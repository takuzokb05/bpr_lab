## 要約

Claude CodeのSkills・Settingsシステムを日常的に使い込んだ実践ノウハウ。
- **3層メモリスコープ**: `user`（グローバル）/ `project`（チーム共有）/ `local`（個人設定）の用途分離
- スキルフォルダ構造（`SKILL.md`, `references/`, `scripts/`）の最適配置
- **トリガー最適化**: descriptionの書き方で自動発動率が大きく変わる実体験レポート
- コンテキストウィンドウコスト：スキルのreferences/に大きなファイルを置くとコンテキスト消費が急増する注意点

毎日使い続けることで気づいた細かいコスト最適化のコツが含まれており、上級者向けの内容。
