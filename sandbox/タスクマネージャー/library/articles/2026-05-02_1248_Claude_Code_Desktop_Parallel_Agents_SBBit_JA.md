# Claude Codeデスクトップ版を刷新、並列エージェント対応を強化

- URL: https://www.sbbit.jp/article/cont1/184536
- ソース: web
- 言語: ja
- テーマ: claude-code
- 取得日: 2026-05-02

## 要約
SB-Biz-ITによるClaude Codeデスクトップアプリ刷新のニュース記事。2026年4月の大型リニューアルで並列エージェントサポートが強化され、Subagent @mention機能でメインセッションから特定タスクを別エージェントに委任できるようになった。複数ファイルの同時編集・リサーチとコーディングの並行実行が実現。ビジネス媒体視点でエンタープライズ活用のインパクトを解説し、Routines（ルーティン）機能による定期実行の組み合わせで「寝ている間も動き続けるClaude Code」を実現する方向性を示す。
