## 要約
Claude Code Routinesは、ローカルPCを必要とせずAnthropicのクラウド上でClaudeを自動実行できる新機能。トリガーは「定時実行」「API呼び出し」「GitHubイベント連動」の3種類が選択可能。外部ツールとの連携も容易とされており、CI/CDや定期バッチ処理など開発ワークフローへの組み込みを想定している。これまで claude -p やAgent SDKで行っていた自動化がサブスクリプション外（API従量課金）になる変更と並行して発表された機能であり、クラウドネイティブなエージェント実行基盤として注目される。
