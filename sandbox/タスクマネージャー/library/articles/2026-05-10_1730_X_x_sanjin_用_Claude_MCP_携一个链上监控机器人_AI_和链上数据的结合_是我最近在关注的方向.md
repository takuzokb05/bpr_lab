## 投稿内容
用 Claude MCP 携一个链上监控机器人

AI 和链上数据的结合，是我最近在关注的方向。

核心驱动是 MCP——Model Context Protocol。Anthropic 推出的协议，让 Claude 可以直接调用外部工具和 API。

MCP 的意义很简单：原来 Claude 只能靠训练数据回答问题，现在它能实时查数据了。

## 要約
Claude MCPを使ってブロックチェーン監視ボットを構築する方法。Dune MCP Server、Bitquery MCP、QuickNode MCP、evmcpなどのツールの活用方法を解説。
