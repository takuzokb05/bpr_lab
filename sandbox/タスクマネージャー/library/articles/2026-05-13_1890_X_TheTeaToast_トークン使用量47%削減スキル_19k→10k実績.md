## 要約
Claude Code / OpenCodeのトークン消費問題を解決するカスタムスキルを公開した技術報告。定量的な効果：同一タスクで19,000トークン→10,000トークンに約47%削減し、かつ出力品質が向上。実装した機能は4つ：①loop prevention（無限ループの防止）、②context budgeting（コンテキスト予算管理）、③output suppression（不要な出力の抑制）、④compressed working memory（圧縮された作業記憶）。エージェントが同じファイルを繰り返し読み込んだり、目的のないループに陥ったりする問題を構造的に解決するスキルとして、トークンコスト削減に直接効果のある実用的なツール。リポジトリが公開されており即試用可能。
