## 要約
GitHubリポジトリ「claude-code-workflows」（shinpr作）。**本番環境対応のClaude Code開発ワークフロー**をSpecialized AIエージェント群で実装したテンプレート集。5つの核心原則：①常時コンテキストを最小化②繰り返し手順をスキルまたはコマンド化③アクティブセッションをコンテキスト汚染から保護④明確な監視・分離を前提に並列化⑤Plan Modeで計画→レビュー→実行の3段階厳守。**.claude/agents/配下のエージェント定義テンプレート**：role・ツールアクセス制限・model指定を組み合わせたスペシャリストエージェント群の設計例を提供。コンテキスト管理の実装例（50/70/90%閾値ルール）、フォーク型並列タスクのパターンも含む。チームでClaude Codeを組織的に活用するための実装リファレンスとして有用なOSSリポジトリ。
