## 要約
ZennのClaude Code公式プラグイン（スキル）の網羅的解説記事。公式スキル17種の分類と使い分けを整理。フロントエンド系：frontend-design（LP・マーケサイト向け美麗デザイン）、frontend-slides（Marpベースのスライド生成）。コードレビュー系：review（PR差分の多角的レビュー）、simplify（変更後の自動リファクタ3エージェント並列）、security-review（依存関係脆弱性スキャン）。開発支援系：init（CLAUDE.md初期生成）、spec-driven-dev（仕様書駆動開発）、error-handling-audit（エラーハンドリング監査）。Claude Code on Web/Desktop対応スキルも増加。重要な知見：公式スキルはdescriptionが充実しており自動起動精度が高い。カスタムスキルを作る前に公式スキルで代替できないか確認することを推奨。github.com/anthropics/claude-code-skills で公式スキルのSKILL.md全文が参照可能。disable-model-invocation: trueの活用パターンも解説。
