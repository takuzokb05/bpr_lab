## 投稿内容
用 Claude MCP 搭一个链上监控机器人

AI 和链上数据的结合，是我最近在关注的方向。

核心驱动是 MCP——Model Context Protocol。Anthropic 推出的协议，让 Claude 可以直接调用外部工具和 API。

MCP 的意义很简单：原来 Claude 只能靠训练数据回答问题，现在它能实时查数据了。

而链上数据是目前最适合接 MCP 的场景之一。

能做什么：监控某个地址的大额转账，查询任意地址的持仓变化，跟踪新代币的部署和交易量，对比跨链资金流动。

现在有哪些工具可用：Dune MCP Server、Bitquery MCP、QuickNode MCP、evmcp
