## 要約
AnthropicのMCP Serverを実際に構築するための実装ガイド。Tools・Resources・Promptsの3つのコアコンセプトを詳説し、Python/TypeScript SDKでのMCPサーバー実装例を提供。Toolsはmodelが呼び出せる関数（JSON Schema定義）、Resourcesはファイルやデータの参照、Promptsはreusableなプロンプトテンプレート。stdio/HTTP/SSEの3種類のトランスポート方式の選択基準も解説。Claude Desktopへのmcp_servers設定方法、デバッグ手法、セキュリティ考慮事項（入力検証・最小権限原則）を含む。MCP server開発の実用的入門として価値が高い。
