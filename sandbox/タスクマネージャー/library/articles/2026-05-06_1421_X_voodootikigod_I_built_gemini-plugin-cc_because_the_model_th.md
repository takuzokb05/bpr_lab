## 投稿内容
I built gemini-plugin-cc because the model that generates your UI usually isn’t the best model to judge if it actually looks good.

Now Gemini works directly inside Claude Code for visual design review, UX critique, and DESIGN.md generation.

https://t.co/aBbDF3I0Vt

## 要約
gemini-plugin-cc: Claude Code内でGemini Vision APIを使ってUIの見た目をレビュー・UXクリティーク・DESIGN.md生成ができるOSSプラグイン。コード生成モデルと視覚評価モデルを分離することでデザイン品質を向上させるアーキテクチャ。
