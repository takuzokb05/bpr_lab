AWS公式ブログ 2026年4月13日週のラウンドアップ。Claude Mythos Bedrock preview、AWS Agent Registry、S3 Files、Amazon Braket Cepheus QPUなど複数の発表を網羅。

## 要約
**Claude Mythosが初めてAmazon Bedrockで公式プレビュー開始**（2026年4月13日）。サイバーセキュリティに特化した新モデルクラスと位置付けられ、ソフトウェア脆弱性特定・コード分析能力に優れる。現在はインターネット重要インフラ企業・オープンソース保守者優先の限定アクセス。同時発表の**AWS Agent Registry**（Amazon Bedrock AgentCore経由）はエージェント・ツール・スキルを組織横断で管理するプライベートカタログ；セマンティック検索・キーワード検索・承認ワークフロー・CloudTrail監査機能付き。追加発表：S3 Files（S3をファイルシステムとしてアクセス可能）・Bedrock IAM主体別コスト配分・Amazon Braket Rigettiの108量子ビットCepheus QPUサポート。Claude Mythosの存在とBedrock上での展開が公式確認された重要な一次情報。
