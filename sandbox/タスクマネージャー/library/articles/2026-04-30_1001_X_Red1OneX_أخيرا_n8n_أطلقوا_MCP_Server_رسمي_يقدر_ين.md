## 投稿内容
أخيراً n8n أطلقوا MCP Server رسمي يقدر ينشئ ويعدل workflows مباشرة، ويشتغل مع أي MCP Client (Claude Code، OpenCode، Cursor، Windsurf، وغيرهم).

اللي يميز الموضوع إنه مو مجرد ربط عادي، مبني خصيصاً يشتغل مع نماذج الذكاء الاصطناعي.
وأهم نقطة: جا معاه TypeScript SDK جديد، يعني الـ workflows صارت تنكتب كـ كود بدل JSON، والتحقق من الأخطاء صار أدق بكثير.

n8n نفسهم ينصحون باستخدام Coding Agents زي Claude Code، لأنها الأفضل في توليد وفحص كود TypeScript. وقريب راح يطلقون Skill رسمي خاص بـ Claude Code يخلي الشغل أسهل.

اللي يشتغل JSON يومياً في n8n بيفهم قيمة هالنقلة. كتابة الـ workflow كـ كود يعني تتبع أسهل للأخطاء، حفظ نسخ حقيقي، وإعادة استخدام أعلى.
