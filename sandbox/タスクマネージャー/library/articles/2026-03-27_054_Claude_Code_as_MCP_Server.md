## 要約

Claude CodeがMCPクライアント（サーバーを消費）としてだけでなく、`claude mcp serve`経由でMCPサーバー自体としても動作できることを解説した記事。これにより他のエージェントがClaude Codeをサブエージェントとして呼び出す「エージェント内エージェント」アーキテクチャが実現する。オープンソースの`claude-code-mcp`（steipete作）も参照されている。
