# NIST、「AIエージェント標準化イニシアティブ」を発表——相互運用可能なエージェントのための標準

- URL: https://www.nist.gov/news-events/news/2026/02/announcing-ai-agent-standards-initiative-interoperable-and-secure
- ソース: web
- 言語: en
- テーマ: ai-news
- 取得日: 2026-05-13

## 要約
NIST（米国標準技術研究所）が2026年2月、AIエージェントの相互運用性・セキュリティのための標準化イニシアティブ（AI Agent Standards Initiative）を発表。対象はLLMベースの自律エージェントで、識別・認証・権限委譲・監査可能性の4領域を標準化する。シンガポールIDA発表の「Agent Identity Cards」（AIエージェントの能力・制限・権限ドメインを開示する標準フォーマット）と並ぶ国際的な標準化の動き。Claude Code・OpenAI Codexなどエージェント開発者にとってコンプライアンス対応が今後必要になる可能性。
