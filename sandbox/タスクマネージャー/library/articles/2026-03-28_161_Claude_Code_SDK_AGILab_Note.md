## 要約

Claude Code SDKの実用ユースケースを3パターンで解説した日本語記事。
- **CIパイプライン**: PRマージ時に自動でコードレビュー・テスト修正・ドキュメント更新を実行するSDKエージェント
- **SaaS UIカスタマイズ**: バックエンドAPI変更なしでエンドユーザーがチャートや画面レイアウトを変更できる仕組み（SDKがUIコードを動的生成）
- **サブエージェント委譲**: 親エージェントが複雑なサブタスクをClaude Code SDKエージェントに丸投げするパターン
