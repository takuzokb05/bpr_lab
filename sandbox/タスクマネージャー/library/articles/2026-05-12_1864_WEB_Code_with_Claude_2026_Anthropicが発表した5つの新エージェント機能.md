## 要約
Anthropicの開発者カンファレンス「Code with Claude 2026」（5月6日）で発表された5つの主要機能をMindStudioが解説。①Dreaming：Claudeが過去セッションを自己検査し改善点を特定する自己進化機能。②Outcomes：ルーブリック記述 → 別エージェントが採点し閾値未達なら再実行するQA自動化。③Multi-Agent Orchestration：リードエージェントがタスク分割・専門サブエージェントに委任・並列実行。④Claude Finance：金融機関向け０種のプリビルドエージェント（ピッチビルダー・決算レビュー等）をManaged Agentとして提供。⑤Add-ins：55以上のプラグインを備えたマーケットプレイス拡充。これらはClaude Code・Coworkのプラグイン、またはManaged Agentとして利用可能。公式発表の一次情報として重要。
