## 要約
2026年4月3日にMicrosoftが正式リリースしたAgent Framework 1.0の完全解説（Uravation）。
MAFはAzure上でAIエージェントを本番稼働させるためのオープンソース統合プラットフォーム。
主要機能：マルチエージェントオーケストレーション・Enterprise SSO統合・監査ログ完備。
MCPをネイティブサポート：外部ツールとのデータ連携をMCP標準プロトコルで実装。
Azure OpenAI・Azure AI Foundry・GitHub Copilotとのシームレスな統合。
競合比較：LangGraph（柔軟性◎）・CrewAI（チーム設計◎）に対し、MAFはEnterpriseガバナンス◎。
Claude Codeとの連携：Claude Agent SDKとMAFの組み合わせパターンが2026年後半のトレンドになる見込み。
