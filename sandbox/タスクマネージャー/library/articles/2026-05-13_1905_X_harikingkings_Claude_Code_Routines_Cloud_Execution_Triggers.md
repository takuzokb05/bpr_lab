## 要約
Claude Code Routinesは、Claude Codeをクラウド上で自動実行できる新機能。最大の特徴はローカルPC不要でAnthropicのクラウドがプロンプト処理を担うこと。トリガーは3種類：(1) 指定時間の定期実行（スケジューリング）、(2) APIコールからの呼び出し、(3) GitHubイベントとの連動。外部ツールとの連携も容易。この機能によりClaude Codeは常時起動マシン不要の本格的なCI/CD・自動化基盤として利用可能になる。Claude Agent SDKやprogrammatic usageの課金変更と合わせると、Anthropicがエージェント自動化をビジネスとして本格化している方向性が見える。
