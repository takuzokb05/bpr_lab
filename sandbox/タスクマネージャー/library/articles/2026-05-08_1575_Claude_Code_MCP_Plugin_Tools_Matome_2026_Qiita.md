## 投稿内容

Qiita の総合まとめ記事。2026年時点の MCP エコシステム状況とカテゴリ別おすすめサーバー一覧。

## 要約

2026年時点で公開MCPサーバーは3,000以上、プラグインは9,000以上にまで拡大。カテゴリ別おすすめ：開発系（GitHub MCP、Playwright MCP、PostgreSQL MCP）、生産性（Google Workspace MCP、Slack MCP、Notion MCP）、検索（Exa MCP、Brave Search MCP）、ファイル操作（Filesystem MCP）。Claude Code × MCP の実務活用例：SEO記事を Claude Code + Strapi MCP で半自動化し所要時間90分→15分に短縮、Figma MCP でデザイントークンを直接参照してコーディング、GitHub MCP でイシュー監視→自動PR作成。設定方法：.mcp.json（プロジェクト共有、--scope project）と ~/.claude.json（個人設定、--scope local）の2層構造。セキュリティ：MCP Tool Poisoning（悪意あるサーバーが隠しコマンドを埋め込む攻撃）に注意し、信頼できるサーバーのみ追加。
