## 要約
Level Up Coding（Medium系）掲載の実践ガイド。上級エンジニアが実際に使う12のパターンを解説。主なパターン：①外部テスト（ユニットテスト）によるClaude自己検証ループ（コンテキスト枯渇時でも外部ソースが正確性を保証）、②CLAUDE.mdでのアーキテクチャ決定記録（ADR）管理、③サブエージェントへの並列タスク委任（独立タスクのコンテキスト分離）、④/plan modeで探索と実装を分離（変更前に全決定を合意、決定精度を80%→100%近くに向上）、⑤MCPでNotion/Figma/DBに直接アクセス、⑥hooks.jsonでの決定論的強制（pre-commit tests等、advisoryでなく強制力を持つ）、⑦Skill.mdでドメイン知識をパッケージ化。機能を5〜10倍速で出荷する具体的なワークフロー設計の参考に。
