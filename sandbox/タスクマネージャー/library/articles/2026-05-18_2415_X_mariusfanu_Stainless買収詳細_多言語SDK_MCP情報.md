## 投稿内容
Anthropic is acquiring Stainless, a company specializing in SDK and MCP server tooling, to strengthen Claude’s ability to connect to external systems and tools. Stainless has generated the official SDKs for Claude since the early API days, supporting multiple languages like TypeScript, Python, Go, Java, and Kotlin, and is trusted by hundreds of companies for fast, reliable, native-feeling developer integrations.

## 要約
Stainless買収の技術的詳細：Python/TypeScript/Go/Ruby/Java/C#の6言語SDK、MCPサーバー仕様からの自動生成機能が含まれる一次情報として重要。
