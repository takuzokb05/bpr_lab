Pointed my Claude Code to @HeyGen's HyperFrames repo and had it build out a skill to create videos like this one direct from the app or CLI. Created it incredibly fast and the video itself didn't cost anything outside of the Claude usage.  

This was just a basic smoke test, so very little prompting or strategy went into the video itself, but plan on connecting this to some other skills we have made that will feed it with everything it needs to create strong DR video ads. 

Another unplanned benefit that came from this was the /gsap skill it installs significantly leveled up the animation capabilities of my landing page skill, so it can now do some really slick stuff when building out pages.
