## 投稿内容
📋 Daily Intel 5/9｜GPT-Realtime-2 で音声 AI が日常へ／ Anthropic Institute 設立／ Figure F.03 量産／ Codex Chrome 拡張

土曜の AI フロンティア動向。技術発表を「自分の生活がどう変わるか」で読み替える。

■ 音声 AI、日常コミュニケーションの主役へ

OpenAI が GPT-Realtime-2 を API 公開。GPT-5 級の推論を備えた音声エージェントが「聞いて、考えて、解く対話相手」として実用段階に。sama 氏は「若い世代と高齢者は音声を選び、中間層はタイプを選ぶ。文脈量が多いときほど音声が選ばれている」と観察を共有。文字入力中心だった AI 利用が、家庭・運転・歩行中といった日常導線に染み込み始める転換点。

■ Anthropic Institute（TAI）設立、AI と人類の経済・知識への影響を研究

Anthropic が研究機関 TAI を発表。研究領域は (1) 経済普及、(2) 脅威と耐性、(3) 実環境での AI 挙動、(4) AI 駆動 R&D の 4 本柱。同時にオープンソースアラインメントツール Petri を meridianlabs に寄贈。学習と仕事の現場で AI を使う個人にとって「設計者がどこを見ているか」が見える材料となる。

■ Figure F.03、量産 120 日でスケール

Figure_robot は「BotQ で F.03 量産を 120 日でスケールアップ完了」と発表。ヒューマノイドが「研究室から量産工場の現場へ届いた日」が刻まれ、家庭・労働代替への現実的距離が一段階近づいた。

■ Codex Chrome 拡張、AI が背景でタブ並列作業

OpenAI Codex が macOS / Windows の Chrome で稼働。並列タブで反復ブラウザ作業をこなし、構造化ページ間の遷移・データ入力を内部でコード生成して実行する。日々の調査・予約・購買・申請といった「ブラウザでの作業」が、AI に委譲される時代の入口。

■ Claude が Microsoft Office / Outlook ネイティブ統合

Anthropic は Claude for Excel／PowerPoint／Word を一般提供開始、Outlook を public beta に。アプリ間で「会話の文脈」を持ち越して動く方向は、個人の知的生産物が「単一の対話流」で繋がっていく未来を予感させる。

■ Algorithm が世界を動かす時代、AlphaEvolve が証明短縮に貢献

GoogleDeepMind の Gemini 駆動コーディングエージェント AlphaEvolve が、自然法則から物流ルートまであらゆるアルゴリズム発見を加速。20 年来の数学定理を短縮した実例が共有された。アルゴリズムが「個人が委ねる相手」になる地平が見え始める。

技術ニュースの底流は「個人の判断・購買・労働の代理化」と「家庭への身体的 AI の到達」。今日からどう生活に取り入れるか、どこは手元に残すか、を考える土曜朝でありたい。
