# Claude Opus 4.7 正式リリース: SWE-bench Pro 64.3%、xhigh effort、タスクバジェットβ

- ソース: web
- 言語: en
- テーマ: claude-ecosystem
- 取得日: 2026-04-17

## 要約
Anthropicが2026年4月16日にClaude Opus 4.7を一般公開。SWE-bench Proスコアが53.4%（Opus 4.6）→64.3%（+10.9pt）、SWE-bench Verifiedが80.8%→87.6%、CursorBenchが58%→70%と大幅向上。新機能として「xhigh」努力レベル（従来のhigh以上の推論コスト上限設定）、タスクバジェット（ベータ）、Claude Codeの/ultrareviewコマンドを追加。画像解像度も最大3.75MPに拡張（Opus 4.6は1.15MP）。価格はOpus 4.6と同じ$5/$25（入力/出力）で据え置き。重要な変更点としてOpus 4.7は新tokenizer採用により、同じ入力テキストでOpus 4.6比1.0〜1.35倍のトークン数になることに注意。コンテキストウィンドウ1Mトークン、最大出力128K、ナレッジカットオフは2026年1月。長時間自律エージェントタスクでの改善が最も顕著。
