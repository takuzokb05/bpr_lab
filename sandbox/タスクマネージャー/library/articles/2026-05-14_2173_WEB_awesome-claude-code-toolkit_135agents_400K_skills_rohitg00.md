## 要約
GitHubで2026年2月にトレンド1位を獲得したClaude Code向け包括ツールキット。135エージェント、35精選スキル（SkillKit経由で40万以上）、42コマンド、176+プラグイン、20フック、15ルール、7テンプレート、14 MCP設定、26コンパニオンアプリ、52エコシステムエントリーを収録。Claude Code公式awesome-claude-codeリポジトリ（hesreallyhim）よりも実装寄りで、即座に使えるスキル・フック・MCPの組み合わせを提供。エビデンスベースのLLM開発プラクティスとして19+ガード/ライフサイクルフック、調査ワークフロー、フリートインデックスを含む。Claude Codeをプロダクション利用する開発者にとって重要なリソース。
