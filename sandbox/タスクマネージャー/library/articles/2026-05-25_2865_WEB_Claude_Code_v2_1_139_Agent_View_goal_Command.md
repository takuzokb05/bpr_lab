# Claude Code v2.1.139: Agent View & /goal Command

- テーマ: claude-code

## 要約
Claude Code v2.1.139がリリースされ、2つの主要新機能が追加された。**Agent View**（Research Preview）は`claude agents`コマンドで全セッションを一画面に表示するCLIダッシュボード。running/waiting/doneのステータスをアイコン表示し、スペースキーでpeek・Enterで接続・インラインリプライが可能。**`/goal`コマンド**は完了条件を設定するとHaikuモデルが毎ターン後に条件達成を評価し、未達の場合は自動で次のターンを開始する自律ループ機能。ターン数・トークン使用量・経過時間をオーバーレイパネルでリアルタイム表示。Fast ModeがOpus 4.7をデフォルトに変更。Pro/Max/Team/Enterprise/API全プランで利用可能。長時間・複数セッション並列実行ワークフローを根本から変える重要アップデート。
