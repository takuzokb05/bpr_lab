# AWS MCP Server 一般提供開始：AI AgentにAWS全サービスへのセキュアアクセスを提供

- URL: https://aws.amazon.com/about-aws/whats-new/2026/05/aws-mcp-server/
- ソース: web
- 言語: en
- テーマ: claude-ecosystem
- 取得日: 2026-05-06

## 要約
AWSが「AWS MCP Server」の一般提供（GA）を発表（2026年5月6日）。Model Context Protocol経由でAI coding agentにAWSサービスへのセキュアかつ監査可能なアクセスを提供する。主な特徴：①単一のMCPツールから任意のAWS APIを呼び出し可能（ファイルアップロードや長時間実行タスクも含む）、②サンドボックス環境でPythonスクリプトを実行しAWSサービスに対するマルチステップ操作が可能、③IAMガードレール・CloudWatch・CloudTrailによる制御と可視化、④追加費用なし（使用したAWSリソースのみ課金）。Claude CodeなどのAI agentからAWSインフラを直接操作できる実用性の高いMCPサーバーとして注目。米国バージニア北部・フランクフルトリージョンで利用可能。
