## 要約

Claude Codeのフック機能を体系解説した2026年版ガイド。CLAUDE.mdは「アドバイザリー（約80%遵守）」であるのに対しフックは「決定論的で100%実行保証」という違いを明確化。MCPツール呼び出しにもフックが適用でき、ツール名は「mcp__<server>__<tool>」パターンで指定する。フックを活用した無人実行（credential漏洩防止・誤ブランチへのpush防止）パターンも紹介されている。
