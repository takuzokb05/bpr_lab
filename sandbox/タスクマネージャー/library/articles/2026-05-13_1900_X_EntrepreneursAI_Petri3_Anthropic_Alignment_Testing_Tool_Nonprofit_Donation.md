## 投稿内容
Anthropic just donated its open-source alignment testing tool to a nonprofit. Here's what Petri 3.0 does.

→ Tests AI models for deception, sycophancy, and cooperation with harmful requests
→ Used on every Claude model since Sonnet 4.5
→ UK's AI Security Institute uses it as a major evaluation tool

What's new in 3.0:

→ Adaptability — auditor and target models can be tweaked separately
→ Realism — "Dish" add-on uses real system prompts so models can't detect they're being tested
→ Depth — integrated with Bloom for in-depth behavioral assessments

Now donated to Meridian Labs, an AI evaluation nonprofit. Independent of any lab.

## 要約
AnthropicがオープンソースのAIアライメント評価ツール「Petri 3.0」を非営利団体Meridian Labsに移管した。PetriはClaude Sonnet 4.5以降の全モデルで使用されており、英国AI安全保障研究所（AI Security Institute）でも主要評価ツールとして採用されている。バージョン3.0の主な新機能は3点：(1) 監査モデルとターゲットモデルを個別調整できる「Adaptability」、(2) 実際のシステムプロンプトを使用しモデルがテスト中であることを検出できない「Dish」アドオン、(3) Bloomとの統合による深度行動評価。非営利団体への移管により、特定ラボから独立した中立的評価基盤となる。AI安全性評価のエコシステムに具体的影響を与える重要な動き。
