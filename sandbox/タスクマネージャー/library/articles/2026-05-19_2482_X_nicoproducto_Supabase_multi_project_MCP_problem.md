## 投稿内容
Pregunta técnica

Cómo están haciendo para manejar múltiples proyectos de Supabase desde su MCP en Claude Code?

Me resulta muy tedioso tener que desconectar y conectar, alguna solución práctica? Realmente me vuelvo loco al pedo

Mi caso de uso es:

Tengo diferentes proyectos de Supabase en diferentes cuentas y quiero acceder a ellos por el mismo MCP de Supabase desde mi cuenta de Claude Code, no quiero cambiar de cuenta pero igualmente tengo que desconectar el MCP y volver a conectarlo al proyecto que necesito

No escala en el tiempo esto

## 要約
Claude CodeのMCPで異なるアカウントに属する複数のSupabaseプロジェクトを切り替える際、MCPの接続を都度切断・再接続しなければならない問題を指摘する技術的な問い。同一のClaude CodeアカウントからアカウントをまたいだSupabaseプロジェクトに横断的にアクセスしたいというユースケースで、現状の接続管理が実用的にスケールしないという実践的な課題。リプライが5件ついており、同様の課題を持つユーザーが複数いることが示唆される。MCPの接続管理・マルチテナント対応の設計に関する未解決問題として記録価値がある。
