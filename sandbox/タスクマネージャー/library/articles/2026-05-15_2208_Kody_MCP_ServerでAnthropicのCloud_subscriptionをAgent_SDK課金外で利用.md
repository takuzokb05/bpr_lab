## 投稿内容
One cool thing about how 🐨 Kody is just an MCP server is that I can use my Cloud subscription with my AI assistant without worrying about Anthropic pulling the rug and charging me extra beyond my subscription for something like open OpenClaw.

## 要約
@kentcdodds（317K followers）によるMCP活用の実践的洞察。
Kody（Kodybot）がMCPサーバーとして実装されているため、自分のCloud subscriptionのAIアシスタントから利用可能。
AnthropicがAgent SDK課金を追加しても、MCP経由のサブスクリプション利用には影響しない。
「ラグプルリスクなし」：サブスクリプション枠内でMCPサーバーを使い続けられる設計の重要性。
6月15日課金変更への対応策としてMCPサーバー経由利用が有効であることを示す。
