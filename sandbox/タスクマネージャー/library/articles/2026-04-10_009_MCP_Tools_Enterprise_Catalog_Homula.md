## 要約
株式会社 homula による MCP サーバー完全カタログ（2026年最新版）。エンタープライズ利用を前提にカテゴリ別（開発・データ・コミュニケーション・業務系・クラウドインフラ）で整理された MCP サーバー一覧。注目エントリとして Salesforce 公式 MCP（20以上のツール、2026年2月 Hosted Beta 公開）、AWS MCP サーバー群（54種）、Google Colab MCP、Slack MCP、Go 静的解析 MCP などを列挙。各サーバーの認証方式・対応プロトコルバージョン・エンタープライズ適性を比較整理しており、Claude Code の MCP 設定時の選定資料として活用価値が高い。同社がこのカタログをベースに CLAUDE.md/AGENTS.md のチーム設計パターンも解説。
