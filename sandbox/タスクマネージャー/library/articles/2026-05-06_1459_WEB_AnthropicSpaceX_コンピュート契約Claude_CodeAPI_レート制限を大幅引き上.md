# Anthropic・SpaceX コンピュート契約：Claude Code・API レート制限を大幅引き上げ

- URL: https://www.anthropic.com/news/higher-limits-spacex
- ソース: web
- 言語: en
- テーマ: claude-code / claude-ecosystem

## 要約
AnthropicがSpaceXとのコンピュートパートナーシップを発表（2026年5月6日）。SpaceXのColossus 1データセンターから300MW超（NVIDIA GPU 22万基以上）の計算能力を月内に獲得。これにより即座に複数の制限引き上げを実施：①Claude Code 5時間レート制限をPro/Max/Team/Enterprise全プランで2倍に拡張、②Pro/Maxのピーク時間帯制限削減を完全廃止、③Claude Opus モデルのAPIレート制限を「大幅引き上げ」（Tier 1で入力トークン/分1500%増・出力900%増）。Amazon・Googleとの既存契約に加え規模拡大。長期的には軌道上コンピュートへの拡張も示唆。Code w/ Claude 2026イベント（SF, 5/6）の基調講演で発表された重要インフラ強化策。
