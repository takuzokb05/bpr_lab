## 投稿内容
If you are using Claude Code as a VS Code Extension STOP.

Claude Code in the Desktop app has Auto mode for permissions and can read terminal output automatically.

Saves so much time.

## 要約
Claude CodeをVS Code Extensionとして使うことへの警告。Desktop Appはパーミッション管理のAuto Modeとターミナル出力の自動読み込み機能を持ち、VS Code Extensionより大幅に効率的であることを指摘。Permission Auto Modeにより毎回の承認ダイアログを省略でき、ターミナル出力の自動取り込みによりエラーフィードバックループが高速化する。Desktop版に切り替えるだけで作業時間を大幅短縮できるというシンプルだが影響の大きいアドバイス。
