## 要約
TauricResearch（UCLA発）のTradingAgents v0.2.4リリース詳細レポート（2026年4月25日）。
LangGraphベースのマルチエージェントLLM金融取引フレームワーク。Apache 2.0ライセンス・オープンソース。
アーキテクチャ：アナリストチーム×4 → Bull/Bear討論 → トレーダー（3段階評価） → リスク管理（積極/保守/中立） → ポートフォリオマネージャー（5段階）という5層構造。
v0.2.4新機能：Pydantic構造化出力、LangGraphチェックポイント再開、永続的意思決定ログ、DeepSeek/Qwen/GLM/Azureプロバイダー追加、Docker対応。
バックテスト結果：AAPL/GOOGL/AMAZONで23〜27%累積リターン。研究用途向けで投資アドバイスではない旨明記。
GPT-5.x・Gemini 3.x・Claude 4.x・Grok 4.xに対応しマルチLLM環境での検証が可能。
