## 要約

ChatGPT（生成AI）を使ってMT4/MT5 EA（Expert Advisor）コードを自然言語から生成する実践的なチュートリアル。
- **プロンプト設計**: 「MACDがゴールデンクロスしたらエントリー、RSI 70超で利確」をMQL5コードに変換させる具体的なプロンプト例
- **反復改善フロー**: 生成コードのコンパイルエラーをそのまま貼り付けてAIに修正させるデバッグサイクル
- MA crossover・RSI・MACD条件の実装例をコード付きで提示
- MetaEditorのAIアシスタント（OpenAIベース・無料）との使い分け

コーディング経験が少ないトレーダーがEA開発に入門するためのステップバイステップガイドとして実用的。
