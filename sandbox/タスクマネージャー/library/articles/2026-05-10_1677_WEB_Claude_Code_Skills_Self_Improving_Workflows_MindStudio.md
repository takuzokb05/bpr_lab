## 要約
MindStudioによる自己改善型AIワークフロー構築ガイド。learnings.mdファイルを活用した継続学習の仕組みが核心。
スキル実行後に「何がうまくいった/いかなかった」を構造化メモとしてlearnings.mdに追記し、次回実行時の参照ソースにする。
Skill Vetter（スキル品質評価ツール）との組み合わせで定期的な品質チェックを自動化。
複数スキル間の依存関係管理・スキルグラフ（Skill Graphs）の概念も解説。
定量データ：self-improving skills導入後、繰り返し作業の修正率が73%低下した事例。
継続的な改善サイクルを自律的に回すエージェント設計のベストプラクティス。
