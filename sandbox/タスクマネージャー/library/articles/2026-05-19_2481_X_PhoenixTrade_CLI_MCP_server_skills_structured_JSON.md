## 要約
エージェント向けに設計されたCLIツールの紹介。全コマンドが構造化JSONを返すため、Claude CodeなどのAIエージェントが出力をパースしやすい設計となっている。CLIにローカルMCPサーバーが同梱されており、Claude Code・Codex・任意のMCPクライアント向けのskillsもバンドルされている。MCPエコシステムとClaudeスキルをワンパッケージで提供するツールで、エンゲージメント（いいね45・RT4）から実際の関心を集めていることがわかる。エージェントがCLIツールを呼び出す際のインターフェース設計の参考事例としても価値がある。
