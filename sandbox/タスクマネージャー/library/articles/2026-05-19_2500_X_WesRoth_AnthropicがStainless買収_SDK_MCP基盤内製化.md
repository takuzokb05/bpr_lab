## 投稿内容
Anthropic is acquiring Stainless, the SDK and MCP server tooling company that has generated every official Anthropic SDK since the early Claude API days.

Stainless helps turn API specs into SDKs, CLIs, and MCP servers across languages like TypeScript, Python, Go, Java, and Kotlin. Anthropic says the acquisition is about improving developer experience and making Claude agents better at connecting to APIs, tools, and data.

## 要約
AnthropicがStainlessを買収。StainlessはClaude API初期から全公式SDKを生成してきたSDK・MCPサーバーツーリング企業。APIスペックからTypeScript・Python・Go・Java・KotlinのSDK、CLI、MCPサーバーを自動生成する。Anthropicは「開発者体験の改善」と「ClaudeエージェントのAPI・ツール・データ接続強化」を買収目的として説明している。SDK生成パイプラインの内製化による開発者向けインフラの垂直統合が狙い。
