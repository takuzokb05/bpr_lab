## 投稿内容
クロコさん（Claude Code）を --remote-control でiPhoneから使っているとき、（おそらくサブエージェントが）WebSearch中にドメインへのアクセスの承認を求めてくる選択肢がiPhone側にやってこない。ずっとアイコンが点滅状態になる。ターミナルの方では選択肢が出てた。



## 要約
Claude Codeを`--remote-control`オプションでiPhoneから使用中、サブエージェントがWebSearch実行時にドメインアクセスの承認ダイアログがiPhone側に表示されず、ターミナル側のみに出る不具合を報告。アイコンが点滅したまま操作不能になる現象で、リモートコントロール時のUI/UX上の既知バグとして有用な一次情報。
