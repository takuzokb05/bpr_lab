# Claude Code March 2026: All Updates from /loop to Voice Mode

## 要約

2026年3月時点でのClaude Codeの主要アップデートを網羅的にまとめた記事。バージョン2.1.63〜2.1.76にかけて毎週のように新機能が追加された。Voice Mode（スペースキーで音声入力）、`/loop`スケジュールタスク、`/effort`コマンド（3段階の努力レベル設定）、`/context`の改善（コンテキスト消費量の可視化）、Auto Memory（カスタムディレクトリでの自動記憶）が主な新機能として紹介されている。チーム向けには並列エージェント活用やモデル選択のガイドラインも提供されている。
