## 要約

Claude Code自身にSKILL.mdを自動生成させるメタワークフローの紹介。
- 「今やった作業をスキルにして」というプロンプト一言でSKILL.mdが自動生成される手順
- 実例：SPAのレンダリング済みHTMLを取得するスキルの自動生成プロセス
- 生成されたSKILL.mdを人間がレビュー・調整する具体的なポイント（allowed-tools、description改善箇所）
- スキル作成の心理的ハードル（「自分でSKILL.mdを書けない」）をゼロにする手法

「スキルはClaudeに書いてもらう」という発想転換がスキルエコシステムの普及加速につながる。
