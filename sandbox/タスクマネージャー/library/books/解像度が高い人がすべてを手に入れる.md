# 解像度が高い人がすべてを手に入れる

## なぜこの本を読むのか

久保さんは「課題発見や仕組化を考えるのが好き」で、DIA評価も「柔軟な意思決定10.0」「事実に基づく判断10.0」。相手の話を聞いて方向性を見極める嗅覚はある。

問題は、**相談を受けたときに「ってことは〜ですか？」と仮説を出す瞬発力が足りない**こと。今のコミュニケーションスタイルは「Yes, Andで返す」「押しつけない」「仮説形式で提案」— 器はできている。足りないのは、相手の話を聞いた瞬間に**頭の中でツリー構造を展開して「次に聞くべきこと」を特定する力**。

この本は「解像度」を「思考のピクセル密度」と定義し、51問のクイズで具体化⇔抽象化の往復運動を鍛える。あなたの「叩き台→対話素材」の型に、**聞く側の解像度を上げるトレーニング**を追加できる。

## 要点3行

1. **解像度 = 思考のピクセル密度**。物事が細かく見えている人は、同じ話を聞いても「99%の人には見えない問い」を立てられる
2. **具体化思考（分解）+ 抽象化思考（共通点抽出）+ 両者の往復 = 解像度の3層構造**。どれか一つでは不十分で、行き来する力が最も高度
3. **トレーニングは「5つの型」**。抽象↔具体・相違点探し・分類・要点抽出・原因分析。日常の場面で繰り返すことで思考の筋力がつく

## 章別ノート

### 第1章: 解像度がすべて（p50-78）

- 「解像度が低い人」= 話に具体性がなく、ふわっとしている。誰でも言える安易な意見しか出せない
- 「解像度が高い人」= 物事が細かく見えている。ユニークで鋭い洞察。わかりやすく伝えられる
- 3つの具体シーン: SCENE 1（上司への提案）、SCENE 2（会議）、SCENE 3（商談）— いずれも「具体性」の有無で結果が分かれる
- **「見ている世界が違う人」= 99%の人には見えないものが見えている人**
- 「見ている世界」は「思考のレンズの解像度」で決まる

### 第2章: どうしたら解像度は高まるのか（p79-100）

- 解像度が高い人の3つの特徴:
  1. **物事が細かく見えている**（具体化思考）
  2. **ユニークで鋭い洞察を得ている**（抽象化思考）
  3. **物事をわかりやすく伝えられる**（具体⇔抽象の往復）
- 具体化 = ツリー構造での分解。例: 生物→動物/植物→哺乳類/魚類→サイ/ライオン→クロサイ/シロサイ
- 「収益向上」の具体化ツリー: 売上（集客数/客単価）× コスト（固定費/変動費）→ さらに分解
- **メソッド**: 具体化思考で「画素数を増やす」＝ 物事をより細かい単位で見えるようにする

### 第3章: 基本トレーニング — 具体化思考（p101-190）

**トレーニング1: 抽象↔具体**
- 問題1: 「免許がなくても運転できる乗り物を5つ挙げよ」→ 自転車、三輪車、車いす、一輪車...
- 抽象概念から具体を列挙する。**頭の中でツリーを下に降りる練習**

**トレーニング2: 相違点探し**
- 問題5: 「空港の書店と繁華街の書店の違いを5つ挙げよ」
- 同じ「書店」でも場所が違えば客層・品揃え・滞在時間が違う
- **「なぜ違いがあるのか」まで問うと、さらに深い理解が得られる**

**トレーニング3: 原因分析（段階的分解）**
- 問題12: 「身長・体重・筋肉量が大谷翔平と同じなのに、バットに当たらない」→ 身体スペックが同じでも「身体の使い方」が違う
- ツリー分解: 身体の使い方 → 下半身（脚/腰）+ 上半身（胴体/頭）→ 各部位の具体的な問題
- **表面的な原因に飛びつかず、分解して本当の原因を特定する**

**実践トレーニング**
- 問題16: 「あなたの商品の顧客像を具体化せよ」— 自分の仕事に直結する問い

### 第4章: 抽象化思考（p191-253）

- **分類トレーニング**: 電車広告を2つに分類せよ。分類基準を自分で設定する力
- **共通点探しトレーニング**: 「AとBの共通点は？」→ 表面的な共通点ではなく、構造的な共通点を見つける
- **要点抽出トレーニング**: 複数の話から「背景に隠れた本当の気持ち」を掴む
- 問題36: 「老舗和菓子屋の競合はどこか？」→ 同業だけが競合ではない。「消費者のニーズ」を抽象化すると、まったく異なる業種が競合になる
- **「なぜ？」を問う習慣**: 「なんでここに傷があるんだろう」「なぜすぐ汚れるんだろう」— 普通なら気にしないところに「なぜ」を立てられるか

### 第5章: 具体⇔抽象トレーニング — 調整力（p254-300）

- 第3章（具体化）+ 第4章（抽象化）を統合する章
- **「最具体」と「最抽象」の間を行き来する力**
- 具体⇔抽象思考 = 生物（最抽象）⇔ クロサイ（最具体）の間のどの粒度でも語れる
- **比喩トレーニング**: 「ChatGPTをおばあちゃんに説明する」「マーケティングを別の言葉で言い換える」
- 「99%の人には見えないもの」を「99%の人にも見えるようにする」力 = **伝える力の本質**

## 内省ワーク（問い5つ）

1. **直近で相談を受けた場面を思い出す。自分は何回「ってことは？」と聞いた？** ゼロなら、頭の中でツリーを展開せずに聞いていた可能性が高い。次回は「1回だけ」を目標にする

2. **「業務がうまく回らない」と相談されたら、あなたは頭の中で何を展開するか？** この本の言葉で言えば、「収益向上ツリー」のような分解構造を持っているか？持っていないテーマがあれば、それがあなたの「解像度が低い領域」

3. **あなたのコミュニケーションスタイル「Yes, And」は、実は「具体⇔抽象の往復」と相性が良い。** 相手の具体的な話を受けて（Yes）、抽象化して返す（And）ことで、「ってことは〜ですよね」が自然に出るはず。なぜ今それができていないのか？

4. **「答えを持つことの落とし穴」と「解像度」の関係。** AIで答えが出せるようになった今、あなたに求められているのは「答え」ではなく「相手の話の解像度を上げる問い」ではないか？ それは具体化？抽象化？それとも両方？

5. **係長試験の面接で「なぜそう考えるのですか？」と聞かれたら、何階層まで掘れるか？** 1階層（「こうだからです」）で止まるか、3階層（「こうだから→それはつまり→具体的には」）まで展開できるか
