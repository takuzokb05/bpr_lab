# 最強リーダーの「話す力」

## なぜこの本を読むのか

久保さんの課題は「答えが見えた瞬間に一直線で解を出してしまう」こと。堀江さんのように「ってことは〜ですか？」と問いを横展開して、相手が自分で辿り着く余地を作りたい。

この本は「話す力」を3階層で定義している:
- 第1階層: 好感を獲得する（一般向け）
- 第2階層: 分かりやすく伝える（ビジネスパーソン向け）
- **第3階層: 影響力を持つ（リーダー向け）**

あなたに必要なのは第3階層。「正しいことを正しく伝える」のは第2階層で、あなたは既にできている。第3階層は「相手を動かす」— つまり相手の中に変化を起こすこと。答えを渡すのではなく、相手が自ら動き出す状態を作る技術。

## 要点3行

1. **セルフ・パペット** — 「素の自分」と「リーダーの自分」を切り分けて操る。24時間リーダーでいる必要はない。場面に応じて意識的に切り替える
2. **第3階層の5要素**: 注目を集める → 高揚感を高める → 信頼感を与える → 基準を示す → 器の大きさを見せる。この順番で積み上げる
3. **レトリカル・クエスチョン + 沈黙** — 疑問形で投げて、3秒黙る。この組み合わせだけで相手の中にメッセージが浸透する

## 章別ノート

### 第1章: セルフ・パペット（p28-60）

- 話す力の3階層: 好感獲得 → 分かりやすく伝える → **影響力を持つ**
- 第3階層のスキルは5つの要素: 注目 → 高揚感 → 信頼感 → 基準 → 器
- **セルフ・パペット**: 優れたリーダーは「素の自分」と「リーダーの自分」を切り分ける（図5）
- メルケル元首相: 金髪ショートカット + 単色丸襟スーツ = 「スタイル」を確定させている
- セルフ・パペットの4段階: 確定 → 使い分け → 自動化 → 融合

### 第2章: 注目を集める（p61-100）

**5つのスキル:**
1. **場で制する** — 入室の瞬間から存在感を示す
2. **名を付ける** — 曖昧なものに名前を与えて認知させる
3. **レトリカル・クエスチョン** — 答えを求めない疑問形で注目を集める
4. **沈黙する** — 3秒の間で浸透させる
5. **キーフレーズを刻み込む** — 繰り返しで記憶に焼き付ける

- レトリカル・クエスチョンの例: 「4人に1人のお客様に何らかの不満を持っている」→ これを「4人に1人が不満を持っているとしたら、あなたはどう思いますか？」と疑問形にするだけで注目度が変わる
- 沈黙 + レトリカル・クエスチョンの組み合わせが最強
- キーフレーズの3テクニック: ①繰り返す ②場で制する ③レトリカル・クエスチョンと組み合わせる

### 第3章: 高揚感を高める（p101-140）

- エネルギー伝達: リーダーの本気度は無意識に伝わる
- キーフレーズの強調: 高く・大きく・ゆっくり（図10、豊田章男氏の例）
- スタディ（動画分析）→ モデリング → 実践の3段階トレーニング

### 第4章: 信頼感を与える（p141-200）

- **事前分析3P2M**: People/Purpose/Promotion/Market/Message
- 聞き手の「興味」×「理解度」マトリクス（図11）で話し方を変える
- 接続詞の使い方: 「しかし」の後に来る内容が主張。否定→肯定の順で信頼感
- **PREP法**: Point→Reason→Example→Point。ただしリーダーの話し方はPREPだけでは不十分
- 音声マネジメント: 抑揚・速度・間のコントロール
- ジェスチャー: 左右対称の両手ジェスチャーが信頼感を生む（図12、豊田章男氏）
- インボックスジェスチャー: 日本人リーダーにおすすめ（図13）

### 第5章: 基準を示す（p201-240）

- **能動態で語る**: 「つくられた制度」→「つくった制度」。主語を自分にする
- **定義する**: 状況に意味づけする行為。定義 = 宣言・標準・指針であり、縛りではない
- **言い切る**: 判断基準を示すことでメンバーが自由に動ける
- **約束する**: リーダーが約束を口にすることで、組織の方向性が明確になる
- 数字・固有名詞を活用: 「たくさん」→「3つ」。「いろいろ」→「具体名」

### 第6章: 器の大きさを見せる（p241-272）

- **弱みを見せる**: 意図的な自己開示がリーダーの器を示す
- **予想外の「まさか！」をつくる**: 期待を裏切る行動で記憶に残る
- **プライベートの感情をあえて見せる**: 完璧なリーダー像を崩すことで親近感

## 内省ワーク（問い5つ）

1. **あなたの「セルフ・パペット」は何種類あるか？** 「答えを出す自分」「叩き台を作る自分」「バランサーの自分」...。相談を受けるとき、どのパペットが前に出ているか意識したことはあるか？

2. **直近の相談対応で「沈黙」を使ったことはあるか？** 答えが見えた瞬間に3秒黙ることで、「答え」を「問い」に変換する時間が生まれる。その3秒は長いか短いか？

3. **AIへの指示が「ふわっとしている」のは、能動態を使っていないからではないか？** 「〜してほしいな」→「〜を実装してください。条件は3つ」。主語と動詞を明確にするだけで精度が変わるか試してみる

4. **堀江さんの「ってことは〜ですか？」は、この本の言葉で言えば「レトリカル・クエスチョン + 定義する」の組み合わせではないか？** 相手の話を一度「定義」して（抽象化）、それを疑問形で返す（レトリカル・クエスチョン）。この2ステップを意識的にやれるか？

5. **「弱みを見せる」は天然の武器。では「基準を示す」は意識的にやっているか？** DX推進で「基準」を示しているか？「うちの課のDXとはこういうことです」と一文で定義できるか？

## 明日から使えるアクション

- [ ] 次に相談を受けたとき、答えが見えた瞬間に**3秒黙る**。その3秒で「答え」を「ってことは〜ですか？」に変換する
- [ ] AIへの指示を書くとき、**主語と動詞を明確にする**。「〜してほしい」→「〇〇を△△してください」
- [ ] 次の会議で発言するとき、**1回だけレトリカル・クエスチョンを使う**。「これって〜ということですかね？」
- [ ] 自分のDX推進の取り組みを**1文で定義する**。「私がやっているDX推進とは〇〇です」
- [ ] 堀江さんの支援スキーム①の場面を想定して、**3P2Mで事前分析してみる**（People: 相談者は誰か / Purpose: 何を求めているか / Promotion: 距離感は？）

## 過去の読書との接続

### 「イシューからはじめよ」との接続
- 「基準を示す」= イシューの「仮説ドリブン」。先に基準（仮説）を立てて、それを検証する
- 「定義する」= イシューの「イシュー度の見極め」。「この問題をこう定義する」が出発点

### 「解像度が高い人がすべてを手に入れる」との接続
- **レトリカル・クエスチョン = 具体⇔抽象の往復を疑問形でやる**。相手の具体的な話を抽象化して「ってことは〜？」と返す
- **沈黙 = 解像度を上げるための「待ち」**。自分の解像度は高くても、相手が追いつくまで待つ技術
- **セルフ・パペット = 「早押しクイズの自分」と「相手に答えさせるクイズの自分」の切り替え**

### 堀江さんの実践との接続
- 堀江さんは「セルフ・パペット」を無意識に使い分けている。「ってことは〜ですか？」は「問いを投げるパペット」
- 堀江さんの「荒くても間違っていてもいい」= この本の「弱みを見せる」。完璧な答えを出さないことがリーダーシップ
- 堀江さんの支援スキーム①〜⑥ = この本の「3P2M + PREP + 定義する」のプロセス化
