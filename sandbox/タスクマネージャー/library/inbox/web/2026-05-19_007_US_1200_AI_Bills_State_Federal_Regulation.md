# 米国のAI規制: 1200本のAI法案と連邦・州の綱引き

- URL: https://fortune.com/2026/05/15/ai-policy-patchwork-state-federal-regulation-framework-sonnenfeld-marcus/
- ソース: web
- 言語: en
- テーマ: ai-news
- 取得日: 2026-05-19

## 要約
Fortune誌が報じる米国AI規制の現状。現在1200本以上のAI法案が州レベルで存在し、テスト基準が統一されていない「パッチワーク」状態。ホワイトハウスは2026年3月に連邦統一AI規制のブループリントを提示し、州法への連邦プリエンプション（優先）を求める。コロラド州は2026年5月9日にSB 189を可決し既存AI規制を改正（2027年1月1日施行）。FederalレベルではNISTにAI生成コンテンツの透かし・指紋・来歴メタデータガイドライン策定を義務付ける「Protecting Consumers From Deceptive AI Act」が4月23日提出。
