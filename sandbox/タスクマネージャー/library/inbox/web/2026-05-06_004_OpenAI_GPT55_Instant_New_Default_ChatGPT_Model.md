# OpenAI、GPT-5.5 Instantをリリース：ChatGPTの新デフォルトモデルへ

## 要約
OpenAIが「GPT-5.5 Instant」をリリース（2026年5月5日）、GPT-5.3 Instantに置き換わる新デフォルトChatGPTモデル。主要ベンチマーク：AIME 2025数学テスト81.2点（旧65.4点）、MMMU-Proマルチモーダル推論76点（旧69.2点）。「法律・医学・金融などの機密分野でハルシネーションを削減」と発表。新機能として検索ツール・ファイル・Gmailを活用したコンテキスト管理の強化、メモリソースの表示・削除機能を追加。Plus/ProユーザーはWebで即利用可能。開発者向けにはAPI経由で「chat-latest」として提供。Claude Opus 4.7と同じ週にリリースされフロンティアモデル競争が激化していることを示す。
