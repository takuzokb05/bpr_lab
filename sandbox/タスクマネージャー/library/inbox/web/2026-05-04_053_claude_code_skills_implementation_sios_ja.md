## 要約
SIOS Tech LabによるClaude Code Skills実装ガイド（ローカルツール統合編）。スクリプトとの組み合わせによるCapability Uplift型スキルの実装に特化した解説。SKILL.mdのscripts/ディレクトリへのシェルスクリプト・Pythonスクリプト配置方法、スキルからのスクリプト呼び出し構文、stdin/stdout経由のデータ受け渡しを詳しく説明。実践例：AWS CLIラッパースキル（複雑なAWS操作を自然言語で実行）・社内ツール統合スキル（VPN環境内のAPIへのアクセス）・データベース操作スキル（SQLを自動生成して実行）。エラーハンドリングとスクリプトの堅牢化、環境変数の安全な取り扱いも解説。SIOSはLinux/OSS専門の日本企業でOSレベルの技術知識を活かした深い解説が特徴。
