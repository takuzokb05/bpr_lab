# PrismML debuts 1-bit LLM in bid to free AI from the cloud

- ソース: web
- 言語: en
- テーマ: ai-news

## 要約
2026年4月4日、The RegisterによるPrismML「Bonsai」モデルファミリー発表記事。各重みを16bit・32bit浮動小数点ではなく符号（+/-）のみで表現する1-bitアーキテクチャ。「クラウドからAIを解放する」というコンセプトで、エッジデバイスでの完全ローカル実行を目標とする。技術的意義：モデルサイズの劇的削減（従来比90%以上のメモリ削減が可能）によりスマートフォン・IoT機器での推論が現実的に。ただし精度とのトレードオフは評価中。AppleM1/M2 MacBook上でのローカル実行デモを公開。LLMのオンデバイス化・プライバシー保護AIの文脈で重要な技術的マイルストーン。クラウド依存からの脱却はFX自動取引エージェントのローカル実行にも将来的に関連。
