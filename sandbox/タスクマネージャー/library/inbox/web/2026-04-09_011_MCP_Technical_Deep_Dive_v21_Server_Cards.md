## 要約
2026年4月公開のMCP技術深掘り記事。MCP v2.1の主要仕様変更を解説：①Server Cards（/.well-known/mcp.json経由のサーバーメタデータ公開、SEP-1649準拠）、②Elicitation API（エージェントフローでのストリーミング処理対応）、③MCP TypeScript SDK v1.27のストリーミングメソッド追加。Server Cardsはサーバー接続前にCapabilities・認証要件・利用可能ツールを自動検出できる仕様で、クライアントの自動設定・セキュリティ事前検証・UI高速ハイドレーションを実現。MCP v2.1ではWorking Groupsがプロトコル開発の主体となりロードマップが「日付ベース」から「優先度ベース」に変更。2026年3月時点でMCPは累計9700万インストールを突破し、実質的な標準インフラへ。
