## 要約
AItmplによるClaude Code Skills事前作成テンプレートとコンフィグの一覧サイト。70以上の既製スキルテンプレートを提供し、コピー・ペーストで即座に使用可能。Frontend Design・Code Review・Documentation Generator・Test Writer・Security Audit等、開発業務の主要タスクをカバーするスキルテンプレートが揃っている。各テンプレートにはSKILL.mdのフロントマター（name・description・triggers）と本文の指示が含まれ、チームのプロジェクトに合わせてカスタマイズできる。スキル作成の学習コストを大幅に削減できるリソースとして重要。2026年3月以降の最新スキル形式（YAMLフロントマター付きSKILL.md）に対応。プロジェクトへの即戦力スキル導入に最適。
