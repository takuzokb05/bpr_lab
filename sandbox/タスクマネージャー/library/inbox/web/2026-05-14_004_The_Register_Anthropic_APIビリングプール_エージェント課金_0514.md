## 要約
2026年5月14日付けThe Register報道。Anthropicが6月15日よりAgent SDK等のプログラム的API利用をサブスクリプションの対話的使用枠から切り離し、API課金プールに移行することを報道。The Registerの視点では「ユーザーがAIエージェントを多用するほど追加コストが生じる仕組み」として批判的に分析。OpenClawや第三者エージェントアプリも対象。API標準レートでの従量課金が適用されるため、ヘビーエージェント利用者には実質値上げとなる可能性を指摘。本日付けの一次報道であり速報性が高い。
