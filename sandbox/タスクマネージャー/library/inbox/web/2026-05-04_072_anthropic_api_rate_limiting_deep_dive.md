## 要約
johal.inによるAnthropic API 2026-03-01レート制限変更の技術的深掘り分析。2026年3月1日に実施された固定ウィンドウから60秒ローリングウィンドウへのレート制限変更が多くの既存ワークロードを破壊した問題を詳しく分析。固定ウィンドウ（旧）：60秒ごとにリセットされる単純なカウンタ→バーストトラフィックが許容された。ローリングウィンドウ（新）：直近60秒間のリクエスト・トークン数を追跡→均一なレートが要求される。影響を受けたパターン（バッチ処理の冒頭集中・指数バックオフロジックの誤動作）と修正が必要なコードパターンを具体例で示す。vLLM 0.4を使ったレート制限回避の技術的手法も解説。APIを本番利用する開発者に特に重要な技術情報。
