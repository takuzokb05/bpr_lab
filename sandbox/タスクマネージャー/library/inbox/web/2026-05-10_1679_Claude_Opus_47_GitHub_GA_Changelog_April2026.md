## 要約
GitHubがClaude Opus 4.7の一般提供開始をChangelog上で発表（2026年4月16日）。
GitHub CopilotおよびGitHub ActionsへのClaude Opus 4.7統合が正式GA。
主要機能：1Mトークンコンテキスト・128k最大出力・高解像度画像（2576px/3.75MP）・Task Budgets・Adaptive Thinking。
GitHub上での利用料金はOpus 4.6と同額（入力$5/M・出力$25/Mトークン）。
コーディング性能でOpus 4.6比で顕著に向上、特に困難なタスクでの改善が大きいと言及。
Amazon Bedrock・Google Cloud Vertex AI・Microsoft Foundryでも同時提供開始。
