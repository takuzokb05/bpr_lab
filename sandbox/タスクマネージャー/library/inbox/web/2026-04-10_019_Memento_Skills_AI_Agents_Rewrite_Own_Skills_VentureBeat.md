## 要約
VentureBeat による Memento-Skills フレームワークの紹介記事。複数大学の研究者が開発した Memento-Skills は、AI エージェントがモデル再トレーニングなしに自律的にスキルを書き換え・改善できる仕組みを実現。エージェントが経験から学んで SKILL.md 相当のスキル記述を自己更新するアーキテクチャは、Claude Code の Skills エコシステム（SKILL.md）に類似した構造を持つ。LangGraph ベースで実装され、スキルの「メモリ」として外部ストレージを活用。エージェントの自律的改善という AGI 的な能力を小規模に実現したフレームワークとして注目。Claude Code の Skills 設計を考える上での参考アーキテクチャとしても有用。
