## 要約
Pasquale Pillitteri氏による業務自動化に使えるオープンソースAIエージェントフレームワーク10選（2026年版）。「自分の仕事を自動化するために使える」という実用的な視点から厳選。フレームワーク一覧：①LangGraph（グラフベース・複雑ワークフロー向け）②CrewAI（役割ベース・チームシミュレーション）③AutoGen（マルチエージェント会話）④OpenHands（元OpenDevin・コーディングエージェント特化）⑤Phidata（PythonネイティブのAIエージェントフレームワーク）⑥Claude Agent SDK（Anthropic）⑦SWE-agent（GitHubイシュー自動解決特化）⑧Aider（ターミナルAIコーディング）⑨Devin OSS（クローンプロジェクト）⑩Pydantic AI（型安全エージェント）。各フレームワークの実際の使用例・GitHubスター数・メンテナンス状況を評価。業務自動化を始める実務者向けの入門比較として有用。
