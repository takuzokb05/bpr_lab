## 要約
100以上の専門化されたClaude Codeサブエージェントを収録したキュレーションリポジトリ。開発ユースケースを幅広くカバーしており、コードレビュー、テスト生成、ドキュメント作成、セキュリティ監査、パフォーマンス最適化など多様なサブエージェント定義が含まれる。awesome-claude-codeリストと並ぶコミュニティ主導のリソース集として位置づけられる。VoltAgentが管理するGitHubリポジトリであり、各サブエージェントはSKILL.mdと同様のフロントマター形式で記述。Claudeのエージェント機能を組織的に活用したいチームや個人開発者が即座に参照・流用できる実践的なリソース。
