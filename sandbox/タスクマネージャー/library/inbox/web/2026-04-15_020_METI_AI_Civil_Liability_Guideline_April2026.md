# AI利活用における民事責任の解釈適用に関する手引きを公表（経済産業省）

- ソース: web
- 言語: ja
- テーマ: ai-news

## 要約
経済産業省が2026年4月9日に公表した「AI民事責任手引き」（第1.0版）。**補助型AI（人間判断支援）と代替型AI（人間判断代替）の2類型**で不法行為法・製造物責任法上の責任分界を整理した実務的ガイドライン。早稲田大学教授を座長とする有識者研究会がパブコメ経て策定した一次情報。AI事業者の法的予測可能性向上と紛争解決円滑化が目的。AI開発・提供・利用に関わる企業の法務・コンプライアンス担当者にとって必読の公式文書。日本のAI規制実務における重要な一次情報として保存価値が高い。
