## 要約
2026年4月8日にPublic Betaとして公開されたClaude Managed Agentsの公式ドキュメント。Managed AgentsはAnthropicが提供するホスト型エージェント実行環境で、セッション管理・コンテナサンドボックス・ビルトインツール（ファイル読み書き・コマンド実行・ブラウザ操作）をAPIとして提供。Claude Agent SDKとの違いは「自前インフラが不要」である点。ステートフルな長時間セッション、クリーンな再開機能、サーバーサイドでのコンテナ状態管理が特徴。クイックスタート・イベントストリーミング・ツール構成の3セクションが充実。Cloudflare との連携でグローバル分散実行も可能。Pro/Max/Team/Enterprise で利用可能（6月15日以降は Agent SDK クレジットから消費）。
