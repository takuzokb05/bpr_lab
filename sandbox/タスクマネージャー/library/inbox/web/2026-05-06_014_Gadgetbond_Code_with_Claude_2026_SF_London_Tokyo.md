## 要約
Anthropicの年次開発者イベント「Code with Claude 2026」の詳細情報。今年は3都市展開：サンフランシスコ（5月6日・ライブストリーム配信あり）・ロンドン（5月19日）・東京（6月10日）。3つのトラック：①Researchトラック（現在の能力と今後の方向性）、②Claude Platformトラック（本番グレードエージェントの構築）、③Claude Codeトラック（大規模運用・長期タスク・マルチリポジトリ作業）。対象：ソフトウェア開発者・エンジニア・技術系リーダー。東京開催（6/10）は日本の開発者にとって重要なオフライン参加機会。前回（2025年）から規模拡大・都市数増加で開発者エコシステムへの本格投資を示す。
