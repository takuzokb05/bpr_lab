## 要約
SitePointによるMCPの包括的ガイド。アーキテクチャ解説：JSON-RPC 2.0でホスト（Claude等）とサーバー間通信、トランスポートはstdio（ローカル）またはHTTP/SSE（リモート）。2026年時点で97Mダウンロード/月・81,000+ GitHub stars・10,000+公開MCPサーバー・全主要AIベンダー対応。認証はOAuth 2.1（2025年6月仕様から）。2026年ロードマップの重点：ステートレス化（水平スケール対応）・エージェント間通信・ガバナンス成熟化。実装ガイドはTypeScript SDK使用でハローワールドからDB接続まで。セキュリティ注意点：MCPサーバーは信頼できるソースのみ使うこと、自己ホストが望ましい。
