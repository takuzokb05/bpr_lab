## 要約
Claude Code v2.1.86から利用可能になったクラウド型コードレビュー機能`/ultrareview`の解説。2026年4月16日（Opus 4.7と同日）にリリース。通常の`/review`との差分は「フリートエージェントがリモートサンドボックスで動作し、発見されたバグを再現してから報告する検証優先アーキテクチャ」。Logic Specialist・Security Auditor・Performance Optimizer・Verification Leadの4専門エージェントが並列稼働し偽陽性を大幅削減。Pro/Maxサブスクライバーには3回分の無料試用（2026-05-05まで）、以降は変更サイズに応じて$5〜$20。認証はAPI KeyではなくClaude.aiアカウントが必須。推奨ユースケース：認証ロジック変更・データマイグレーション・アーキテクチャリファクタリング・クリティカルPRの最終レビュー。
