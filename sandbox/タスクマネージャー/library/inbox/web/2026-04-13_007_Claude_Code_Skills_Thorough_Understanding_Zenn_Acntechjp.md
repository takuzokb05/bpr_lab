## 要約
Claude CodeのSkills機能を作成例を通じて体系的に理解するZenn記事。SKILL.mdのYAMLフロントマター（name/description）がClaudeによるスキル自動起動の判断基準となる仕組みを解説。Progressive Disclosure（段階的開示）がSkills設計の核心思想であり、「必要な情報を必要なときだけ読み込む」原則を実装するためのディレクトリ構造・ファイル分割の具体例を掲載。スキル肥大化を防ぐためのリファクタリングパターン（87%削減事例あり）も紹介。CLAUDE.mdとSKILL.mdの役割分担、スキルのテスト方法、チームでの共有方法まで実践的に解説されており、日本語でのSkills入門として価値が高い。
