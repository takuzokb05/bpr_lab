## 要約
実プロジェクトから得たClaude Codeベストプラクティス集。著者Ran the Builderが複数のプロダクション環境での経験から導出。重要な知見：①プランニングファースト原則（Plan Mode→スタッフエンジニアレビュー→実行の3段階）、②コンテキストウィンドウ管理（リサーチはサブエージェントに委譲し汚染防止）、③CLAUDE.mdの最小化（毎回確認する質問の答えのみ記述）、④スキルによる専門知識の分離、⑤フックによる品質ゲート（linting・型チェック・セキュリティスキャンの自動化）。特にコンテキスト管理の失敗談（1000行超のCLAUDE.mdが逆効果だった経験）が具体的で参考になる。プロダクション品質を求める開発者向け。
