# Microsoft and OpenAI Restructure Partnership to End AGI Exclusivity

## 要約
2026年4月27日に発表された Microsoft-OpenAI パートナーシップ大幅改訂の詳細分析。変更点：①独占性終了（OpenAIはAzure優先だがどのクラウドでもサービス可能）②Microsoftの収益シェア支払い停止（逆にOpenAIはMicrosoftに20%の収益シェアを2030年まで支払うが上限設定）③IP ライセンスが非独占・2032年満了に変更④AGI条項が契約から削除⑤OpenAIの$50B AWS 契約の法的道筋が確立。OpenAI が公開利益法人へ転換しIPO準備。エージェントAI市場での多クラウド競争が本格化することを意味する重要な業界再編。
