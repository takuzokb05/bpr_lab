## 要約
Q1 2026にClaude Codeが出荷した全主要機能の横断まとめ。Remote Control（外部からセッション接続）、Dispatch（バックグラウンドタスク実行）、Channels（Telegram/Discord通知）、Computer Use（GUI操作）、Auto Mode（中間PermissionMode）、AutoDream（スケジュール実行）の6機能が特に重要。Remote Controlはセッションを外部から制御できるLambda関数的なアーキテクチャ転換。Computer Useはスクリーンショット→推論→マウス/キーボード操作のループ。これらはClaude Codeが「使い捨てセッション」から「常駐型エージェント」へシフトしたことを示す。各機能の設計思想と実用シナリオをQ1全体で俯瞰できる点で一次情報として価値が高い。
