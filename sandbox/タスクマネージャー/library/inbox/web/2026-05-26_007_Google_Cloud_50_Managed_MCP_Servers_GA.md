## 要約
Google Cloud Next '26 で発表され、Google Cloud サービス向けの50以上のフルマネージドMCPサーバーが一般提供（GA）及びプレビューとして公開された。BigQuery・Spanner・Cloud SQL・Vertex AI・Google MapsなどGCPの主要サービスがMCP経由でAIから直接利用可能になる。認証はGoogle Cloudの既存IAM・サービスアカウントをそのまま使用。従来は自前でMCPサーバーを実装・運用する必要があったが、マネージドサービスとして提供されることでインフラ管理コストがゼロになる。Claude CodeやCursor・Windsurf等のMCP対応クライアントから即座に接続可能。MCPエコシステムの企業採用を大幅に加速させるインパクトがある。
