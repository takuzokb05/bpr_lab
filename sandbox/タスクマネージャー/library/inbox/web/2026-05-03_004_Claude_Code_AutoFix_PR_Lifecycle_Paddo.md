# Claude Code Auto-Fix: The PR That Fixes Itself

## 要約
Claude Code の /autofix-pr 機能のライフサイクル詳細解説。ブランチをプッシュして /autofix-pr を実行すると、セッション全体がクラウドに転送される。Claude Code on the web がPR自動修正モードで引き継ぎ、CI ログとレビューコメントを監視しながら修正コミットを自動プッシュし続ける。会話履歴・ファイル編集・推論コンテキストが引き継がれるため、ローカルセッションと同じ質の修正が可能。CI が緑になるまで自律的に動作する。ローカルPC が不要になる新しい開発パターンを詳細に解説。
