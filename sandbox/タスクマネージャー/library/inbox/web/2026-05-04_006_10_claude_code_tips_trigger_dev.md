## 要約
Trigger.devエンジニアリングブログによる、あまり知られていないClaude Code Tips 10選。公式ドキュメントには記載されていない実践的な知見を中心に構成。コンテキストウィンドウ管理の重要性（会話全体がウィンドウに蓄積される仕組み）、/rewindコマンド（ESC ESC）で過去の良い状態に戻る方法、複雑タスクは最初のステップのみを指示して段階的に進める戦略を解説。/initコマンドによるCLAUDE.md自動生成とその重要性を強調。ステータスラインのカスタマイズ（モデル・ブランチ・未コミット数・トークン使用量の表示）など開発環境を整える実用的な設定も紹介。Trigger.devという実際のプロダクションコードベースでの経験に基づく信頼性の高い情報。
