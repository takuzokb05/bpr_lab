# 国産LLM「LLM-jp-4」公開 — 約12兆トークン学習でGPT-4o超え（NII公式）

- URL: https://www.nii.ac.jp/news/release/2026/0403.html
- ソース: web
- 言語: ja
- テーマ: ai-news
- 取得日: 2026-04-15

## 要約
国立情報学研究所（NII）が2026年4月3日に公式公開した国産LLM「LLM-jp-4」。**約12兆トークンの良質コーパスで学習し、日本語MT-BenchでGPT-4o（7.29）を上回るスコア7.54（8Bモデル）・7.82（32B-A3B）を達成**。英語MT-BenchでもGPT-4o同等以上。8Bモデルと32B-A3B（MoEアーキテクチャ）をオープンソースライセンスで公開。今後は32Bモデルと332B-A31B（MoE、超大規模）を2026年度中に公開予定。国産LLMが主要外国モデルを定量的に超えた歴史的マイルストーンとして、日本語AI基盤の自立化に向けた重要な一歩。
