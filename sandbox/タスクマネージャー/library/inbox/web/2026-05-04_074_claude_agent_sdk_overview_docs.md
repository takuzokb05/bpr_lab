## 要約
Anthropic公式Claude Code DocsのAgent SDK概要ページ。Claude Code SDKから名称変更されたAgent SDK（2025年9月）の機能と使い方を公式に解説。Agent SDKはClaude Codeを動かすエージェントループ・ツール・コンテキスト管理をPythonおよびTypeScriptでプログラマブルに提供するライブラリ。Python 0.1.48・TypeScript 0.2.71が2026年3月時点の最新版。組み込みツール（File Read・File Edit・Bash実行・Web検索）とカスタムツールの追加方法を説明。query()とstream()の2つの主要インターフェースの違いと使い分け、マルチターン会話の管理、プロンプトキャッシュの自動活用（入力コスト最大12倍削減）などの重要機能を解説。Agent SDK活用の出発点として不可欠な公式ドキュメント。
