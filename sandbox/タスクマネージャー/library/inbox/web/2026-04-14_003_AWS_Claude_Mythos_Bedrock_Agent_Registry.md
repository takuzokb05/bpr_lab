## 要約
2026年4月13日のAWS公式ブログ。Anthropicの最新モデル「Claude Mythos」がAmazon Bedrockでプレビュー開始。サイバーセキュリティ特化モデルとして位置付けられ、ソフトウェア脆弱性特定・コード分析に優れる。現在はインターネット重要企業・オープンソース保守者優先の限定アクセス。同時発表のAWS Agent Registry（Amazon Bedrock AgentCore経由）はエージェント・ツール・スキルを一元管理するプライベートカタログで、セマンティック検索・キーワード検索・承認ワークフロー・CloudTrail監査を備える。S3 Files（S3をファイルシステムとしてアクセス）、Bedrock IAM主体別コスト配分も発表。Claude Mythosの存在とBedrock上での展開が初めて公式確認された重要な一次情報。
