## 要約
Daniel Avila氏によるClaude Code学習ロードマップ記事。「どこから始めればよいか」という初学者の疑問に答える構造化された学習パスを提供。入門→基礎習得→中級→上級の段階別に必要なスキルと学習リソースを整理。入門（インストール・基本コマンド・最初のCLAUDE.md）→基礎（Plan mode・/clear管理・基本MCP設定）→中級（カスタムスキル作成・フック設定・並列worktree）→上級（サブエージェント・Managed Agents・CI/CD統合）という体系的なロードマップ。各段階での達成目標と推奨練習プロジェクトが明確で、自己評価しながら進められる設計。Claude Codeの急速な機能追加に伴う学習の優先順位付けに役立つ実用的なガイド。
