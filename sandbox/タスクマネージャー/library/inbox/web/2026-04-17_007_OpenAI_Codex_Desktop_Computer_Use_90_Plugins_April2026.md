## 要約
OpenAIが2026年4月16日に「Codex for (almost) everything」を公開。Claude Codeとの競合製品の大型アップデート。主な変更点：(1)バックグラウンドコンピュータ使用—CodexがMacの他アプリを見て・クリックして・入力する機能が中心機能に昇格、複数エージェントが並列実行中に継続；(2)90+追加プラグイン—Atlassian Rovo・CircleCI・CodeRabbit・GitLab Issues・Microsoft Suite・Neon by Databricks・Remotion・Render・Superpowersなど；(3)インブラウザ機能強化；(4)画像生成ネイティブ対応。macOSは2026年2月2日、Windows対応は3月4日に開始。GPT-6リリース予定（2026年4月14日予定→遅延中）に合わせた布石と見られる。Claude Code Routinesと異なりローカル実行中心。エンジニアリングツール以外（デザイン・文書作成）への展開が特徴的。
