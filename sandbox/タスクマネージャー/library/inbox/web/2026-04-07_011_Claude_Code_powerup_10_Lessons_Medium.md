## 要約
Claude Code /powerupの10レッスン全てを実際に受講した体験レポート。1年以上の毎日使用者でも7/10のレッスンで新発見があったという驚きの実験結果から始まる。カバーされるトピック：コンテキスト管理・Hooks・MCP・サブエージェント・/loopコマンド等。各レッスンは3〜10分でアニメーションデモ付き。著者が特に価値を感じたのは「知っていたが使っていなかった機能」に気づく体験。/powerupを使う最大の理由は「機能を実際に動く状態で見せてくれる」点で、ドキュメント読書よりはるかに定着率が高い。Claude Code入門者だけでなく経験者の棚卸しにも適している。
