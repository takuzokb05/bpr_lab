## 要約
SIOS Tech Labによる「3フェーズ開発」メソッドの解説。計画（Planning）→実装（Implementation）→検証（Verification）の3段階フレームワークをClaude Codeに適用する実践ガイド。計画フェーズ：/planコマンドでClaude に設計を出力させ、人間がレビュー・承認してから実装開始。実装フェーズ：承認済みの計画に基づいてClaude が自律実装、人間は中間チェックポイントでのみ関与。検証フェーズ：自動テスト・コードレビュー・統合テストの自動化でClaude 自身が品質検証。このフローにより人間の入力の質がAI出力の品質に直結するという「ゴミイン・ゴミアウト」問題への対処法を提示。SIOSの実際のプロジェクトでの適用事例も含む実証的な内容。
