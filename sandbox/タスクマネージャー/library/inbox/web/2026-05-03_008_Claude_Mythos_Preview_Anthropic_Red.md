## 要約
Anthropic が2026年4月7日に発表した次世代モデル Claude Mythos Preview の詳細。サイバーセキュリティ・自律コーディング・長期エージェントタスクに特化した新しいインテリジェンスクラス。主要な能力：全主要OSと主要ブラウザの数千のゼロデイ脆弱性を自律検出（数十年間見逃されていたものを含む）。Project Glasswing を立ち上げて重要ソフトウェアのセキュリティ強化に活用。アクセスは防御的サイバーセキュリティ用途を優先したゲーテッドリサーチプレビューのみ。AWS Bedrock および Google Cloud Vertex AI でも Preview 提供中。英国 AISI による能力評価結果も公開。Claude Code から Mythos Preview を呼び出してコードを読み→仮説を立て→バグレポートを出力するワークフローが示されている。
