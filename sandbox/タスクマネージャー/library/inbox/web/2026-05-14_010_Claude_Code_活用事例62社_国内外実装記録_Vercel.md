## 要約
国内・海外62社のClaude Code実装事例を集約したコミュニティサイト。各社が実際にどのようなワークフロー・スキル・フック・CLAUDE.md設定でClaude Codeを活用しているかを具体的に記録。Stripe（1,370名への全社展開、Scala→Java 1万行移行を4日に短縮）など大企業事例も含む。非エンジニアの業務自動化（議事録、データ集計）から、マーケティング自動化（190項目Google/Meta Ads監査を5分以内）まで幅広い活用事例をカバー。Claude Codeを活用したFX自動取引システム構築の参考事例として有用であり、実装アーキテクチャのパターン把握に活用できる。
