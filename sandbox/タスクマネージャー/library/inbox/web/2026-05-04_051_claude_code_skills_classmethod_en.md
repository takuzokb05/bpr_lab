## 要約
Classmethod（AWS専門コンサルティング会社）のDevelopersIOによるClaude Code Skills紹介記事。社内の内部勉強会スライドを公開した形式で、クラウドコンサルティング会社の実際の活用視点が含まれる。AWS環境でのClaude Code Skills活用（CloudFormation生成・AWS CDK最適化・セキュリティ監査スキル）が特に詳しい。SKILL.mdのYAMLフロントマター構造、Capability Uplift vs Encoded Preference の使い分け、チーム内でのスキル共有方法を解説。日本のAWSコンサルタントが実務で使用するスキルの実装例を複数含む。Classmethodは日本最大級のAWSパートナーであり、エンタープライズでのClaude Code実活用事例として信頼性が高い。
