## 要約
Claude Code v2.1.90（2026年4月1日リリース）の詳細解説。/powerupコマンドの正式ローンチが目玉。/powerupはターミナル内インタラクティブチュートリアルで10個のレッスン（コンテキスト管理・Hooks・MCP・サブエージェント・/loopコマンド等）をアニメーションデモ付きで提供。他の主な変更：セッション再開の安定性向上、スクロール修正、PowerShellパーミッション問題の修正、プライバシー強化。/release-notesが新しい対話式バージョンピッカーに変更。開発者は/powerupを「静的ドキュメントを読む必要なく機能を学べる」ユーザーオンボーディング革命と評価。毎日使っているユーザーでも7/10レッスンで新発見があったとの報告あり。
