## 要約
DEV Communityに掲載されたClaude Code Skills & Plugins 2026年版完全ガイド。スキルの構造（SKILL.mdのYAMLフロントマター）から人気プラグインまで網羅。特筆すべきはスキルの普遍的互換性：同一のSKILL.mdファイルがClaude Code、Cursor、Gemini CLI、Codex CLI、Antigravity IDEで動作する点。disable-model-invocation: trueで手動のみ起動、user-invocable: falseでClaude自動呼び出しの制御が可能。2,000トークン以内のスキルが最もパフォーマンスが高く、フォーカスを絞ることがベストプラクティスとされる。2026年3月時点で公式Anthropicスキル・サードパーティ認証スキル・コミュニティスキル数千件がエコシステムを形成。
