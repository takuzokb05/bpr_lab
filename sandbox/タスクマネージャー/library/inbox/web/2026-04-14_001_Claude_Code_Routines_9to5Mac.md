## 要約
Anthropicが2026年4月14日にClaude Code向け新機能「Routines」を正式リリース。スケジュール設定して繰り返し実行できる自律タスク。Anthropicのウェブインフラ上で動作するためMacがオフラインでも継続実行可能。cronジョブやMCPサーバーの自前管理が不要になる。リポジトリやコネクタへのアクセス権を持ち、スケジュードAPIワークフロー・GitHubルーチンなどが対象。利用制限はPro=1日5件、Max=1日15件、Team/Enterprise=1日25件。並列セッション・統合ターミナル・ファイル編集を備えた新デザインのClaude Codeデスクトップも同時リリース。
