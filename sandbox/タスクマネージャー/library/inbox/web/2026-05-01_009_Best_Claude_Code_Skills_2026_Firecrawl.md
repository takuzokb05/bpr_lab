## 要約
Firecrawl（スクレイピング/AI向けデータ収集ツール会社）によるClaude Code Skills実践レビュー。SKILL.mdはMarkdown形式の手順書であり、Claude Codeが「いつ・どのように」タスクを実行するかを教えるメタプログラミング的仕組み。記事では実際に試した有用スキルを評価。注目スキル：(1) `/simplify`—コード過剰エンジニアリング検出・リファクタ。(2) `/review`—PRマージ前の自動品質チェック。(3) `claude-code-caveman-mode`—不要なコンテキストを削除してトークンを節約しながらコードの意味を保持。(4) `/init`—プロジェクト解析からCLAUDE.mdを自動生成。スキルの自己改善ループ（Observe→Inspect→Amend→Evaluate）も解説。新スキル作成の最小テンプレートと公式スキルディレクトリへの投稿方法も記載。
