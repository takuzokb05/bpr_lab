## 要約
GitButler（Git管理ツール）公式ドキュメントのClaude Codeフック統合ガイド。GitButlerとClaude Codeを連携させることでAI駆動のGitワークフローを実現する方法を解説。コミット前のコード品質チェック・コンフリクト解決支援・PRデスクリプション自動生成・ブランチ戦略の提案などのユースケースを説明。GitButlerのVirtual Branches機能とClaude Codeフックを組み合わせた高度なワークフロー（自動マージリゾルブ・スマートコミット分割）も紹介。GitとClaude Codeの統合パターンとして特に価値があり、Git操作を伴うCI/CDパイプラインでのClaude Code活用を検討しているチームに有益。GitButlerという実際の製品に組み込まれた実例として信頼性が高い。
