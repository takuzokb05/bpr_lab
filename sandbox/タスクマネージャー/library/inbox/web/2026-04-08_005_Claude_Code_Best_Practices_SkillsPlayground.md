## 要約
Skills Playgroundによる包括的なClaude Codeベストプラクティスガイド。CLAUDE.md・スキル・フックの三層構造を体系的に整理。「CLAUDE.mdはadvisory（約80%遵守）、フックはdeterministic（100%実行）」という原則が核心で、何をどこに置くかの判断基準を明確化している。チーム開発においてリポジトリ内にshared skillsコレクションを維持することで、全員のClaude出力が一貫性を保てると指摘。CLAUDE.mdの過剰記述を避け「これを削除するとClaudeが間違いを犯すか」のテストで精査する手法を推奨。階層的CLAUDE.md（プロジェクトルート+サブディレクトリ）の優先度ルールも解説されており、モノレポ環境での実践的な設定方法が参考になる。
