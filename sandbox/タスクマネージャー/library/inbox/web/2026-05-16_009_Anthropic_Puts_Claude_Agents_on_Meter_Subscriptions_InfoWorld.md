# Anthropic Puts Claude Agents on a Meter Across Its Subscriptions

## 要約
InfoWorldがAnthropicのAgent SDK課金分離を業界アナリスト視点で分析。従来のサブスク「使い放題」モデルからAgent用途を切り出すことで、Anthropicは収益の予測可能性を高めつつ過大利用を防止する。この動きはOpenAIやGoogleが既に採用しているAPI-over-subscription課金モデルに近づく。エンタープライズ顧客には個別のAgent利用枠を交渉する形になる見込み。IntelやSalesforceなどの大手企業が大規模なAgent統合を進める中、Anthropicが課金モデルをエンタープライズ対応にシフトしていることを示す重要な動き。6月15日退役モデル（sonnet-4-20250514、opus-4-20250514）の置き換え先も整理されている。
