## 要約
Claude Code Routinesが2026年4月14日にリサーチプレビューで公開された。Routinesは「プロンプト＋リポジトリ＋コネクター」を1パッケージにまとめてAnthropicのクラウドで自動実行できる機能。スケジュール（hourly/nightly/weekly）、HTTP API、GitHub webhookの3種トリガーをサポート。ローカルcronやサーバー管理が不要で、バグトリアージ・ドキュメントドリフト検出・PR自動生成などのユースケースに対応。実行制限はPro:5回/日、Max:15回/日、Team/Enterprise:25回/日。InfoQがAnthropicの公式アナウンスをもとに技術的背景を詳述。従来のローカルツールからクラウド自動化プラットフォームへの転換点として重要。
