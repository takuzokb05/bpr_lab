## 要約
Anthropic公式のAgent SDK MCP接続ドキュメント。Claude Agent SDKからMCPサーバーに接続する方法をPython・TypeScript両方のコード例付きで解説。MCPサーバーのライフサイクル管理（接続・ツール発見・切断）、stdio/HTTP両トランスポートのAgent SDK内での使用方法、複数MCPサーバーの同時接続パターンを説明。SDKのquery()およびstream()インターフェースとMCPツールの統合方法、エラーハンドリングのベストプラクティスも含む。Managed Agents環境でのMCP設定方法との比較も提供。公式ドキュメントとして最高の信頼性を持ち、Agent SDKを使ったアプリケーション開発でMCPを活用する際の一次参照先として重要。
