# Zed Editor: What Anthropic's New Claude Billing Means for Zed Users

## 要約
ZedエディタチームによるAnthropicの6月15日課金変更に対する公式見解ブログ。ZedはAnthropicのAgent SDK経由でAPIを呼び出すためプログラマティック利用に分類され、6月15日以降はAgent SDKクレジットプールから消費される。Zedチームは変更内容の詳細を解説しつつ、ユーザーへの影響を最小化するための設定ガイダンスを提供。API直接利用（Anthropic APIキー設定）に切り替えることで課金プールを自己管理できる選択肢も説明。IDE/エディタがAgent SDK課金体系に包含されるケーススタディとして、Claude Code以外のツールがいかに影響を受けるかを理解する上で一次情報として重要。
