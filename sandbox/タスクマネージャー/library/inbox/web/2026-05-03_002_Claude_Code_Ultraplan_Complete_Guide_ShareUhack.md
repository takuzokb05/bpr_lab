## 要約
Claude Code の新機能 Ultraplan の完全ガイド。Ultraplan はクラウドのコンテナランタイムにプランニングをオフロードし、ターミナルを即座に解放する仕組み。プランが完成したら GitHub PR スタイルのインターフェースでブラウザからレビュー可能。Monitor ツール・/loop・/team-onboarding・/autofix-pr との連携方法を解説。実際のコスト・ワークフロー・体験談を含む実践的内容。複雑なアーキテクチャ設計タスクや長期計画タスクで特に有効。ローカルターミナルをブロックしない非同期計画立案が最大の利点。
