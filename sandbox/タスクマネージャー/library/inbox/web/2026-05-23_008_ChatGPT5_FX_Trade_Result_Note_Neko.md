## 要約
OpenAI API経由でGPT-5をFXトレードに実験投入した個人の実践記録。結果はマイナス収益となり、LLMが過去データに基づく予測に限界があり相場の急変対応が苦手な点を実体験で確認。プロンプト設計・ポジション管理ルール・損切り閾値の設定を試行錯誤した詳細ログを公開。「LLMは予測エンジンではなく分析補助ツール」という結論に至り、最終的にシグナル生成→人間判断→執行の分離アーキテクチャへ移行。AI自動売買の現実的な失敗事例として一次情報価値が高い。
