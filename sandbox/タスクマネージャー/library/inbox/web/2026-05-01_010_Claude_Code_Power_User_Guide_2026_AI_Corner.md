## 要約
上級ユーザー向けClaude Code完全活用ガイド2026年版。主要テーマ：(1) Plan Modeの徹底活用—複雑タスクでは全てPlan Modeから開始し、実装前に方針を確定させる。(2) コンテキスト管理—90%以上のコンテキスト使用時はパフォーマンスが著しく低下、400k tokens超えではOpusの追加コンテキストが逆効果になりうる。(3) スキルの設計—週1回以上行う一貫したアウトプットが必要な作業をスキル化する原則。(4) ツール管理—MCPは「Notion・Figma・GitHub」など実際に差分を生む数個のみ導入し、`gh` CLIでClaude自身がPR/Issue/CIを操作できる環境を整備。(5) コンテキスト枯渇時のリセット戦略。Anthropic内部チームの実践例（マーケティング部門がSkillsで作業時間を最大83%削減）を複数引用。
