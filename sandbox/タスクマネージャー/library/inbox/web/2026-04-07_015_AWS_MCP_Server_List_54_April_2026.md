## 要約
AWSが2026年4月1日時点で公開しているオープンソースMCPサーバー全54種類をカテゴリ別に整理した早見表記事。AWS CDK・Lambda・S3・DynamoDB・CloudWatch・Bedrock等の各サービスに対応するMCPサーバーが揃っており、コード生成・インフラ管理・モニタリング・ドキュメント参照等の用途別に選べる。各サーバーのGitHubリポジトリリンク付き。AWS環境でClaude Codeを活用する際のMCPサーバー選定ガイドとして即座に実用できる。2026年4月時点の最新情報であり、AWSとAnthropicの連携が加速している状況を示す一次資料。
