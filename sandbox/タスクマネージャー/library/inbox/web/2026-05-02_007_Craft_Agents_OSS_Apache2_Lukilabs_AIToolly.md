## 要約
2026年5月2日、LukilabsがApache 2.0ライセンスで「Craft Agents」をOSS公開。AIToollyが速報。本日リリースの新フレームワークで、AIエージェントをモジュール式に構築できる設計。MicrosoftのAgent Framework 1.0（4月3日リリース）に続くエンタープライズ向けエージェントフレームワーク参入。LangGraph・CrewAI・AutoGenと競合するがApache 2.0の商用フレンドリーなライセンスが差別化点。詳細なAPIと実装パターンは公開リポジトリに順次追加予定。エージェントSDKエコシステムの多様化が加速している証左。
