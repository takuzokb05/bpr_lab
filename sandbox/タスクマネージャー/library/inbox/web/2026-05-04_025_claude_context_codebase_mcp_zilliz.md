## 要約
Zilliz（Milvusベクターデータベースの開発元）によるClaude Code向けコードベース全体検索MCPサーバー。ベクター検索技術を活用してコードベース全体をClaude Codeのコンテキストに取り込む。「Make entire codebase the context for any coding agent」というコンセプトで、コンテキストウィンドウの制限を超えた大規模コードベースへのアクセスを実現。コードのセマンティック検索・関連ファイル・関数の高精度な取得が可能で、Claudeが必要な情報を効率的に見つけられるようになる。Milvus/Zillizのベクターストア技術を基盤とした高速な類似度検索が特徴。大規模モノレポや複雑なコードベースでの開発効率を大幅に向上させる重要ツール。
