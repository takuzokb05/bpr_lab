## 要約
Anthropic公式サポートセンターによるClaude Codeパワーユーザー向けTips集。公式ドキュメントとして信頼性が高い。Context7 MCPを使ったオンデマンドドキュメント取得、カスタムステータスライン設定によるモデル・ブランチ・トークン使用量の可視化、並列git worktreeセッションの活用を公式推奨として解説。/clearの頻繁な使用、Plan modeの活用、段階的タスク分割の重要性を公式が認める形で説明。CLAUDE.mdの適切な管理（プロジェクトコンテキスト・コーディング規約・検証手順の記述）もカバー。Anthropic公式が推奨する使い方を把握できる一次情報として重要度が高い。パワーユーザーが最大限の生産性を引き出すための設定と習慣が体系的にまとめられている。
