# makeshop ECにMCPサーバー対応とSkillsを同時提供 AIエージェント連携で販売分析自動化

- URL: https://www.magazine.makeshop.jp/news20260317-2/
- ソース: web
- 言語: ja
- テーマ: claude-ecosystem
- 取得日: 2026-05-15

## 要約
makeshop by GMOが2026年3月17日にMCPサーバー対応とSkills機能を同時リリース（公式プレスリリース）。
MCPサーバーにより、AIエージェントがEC店舗データへ自然言語でアクセス可能になった。
Skills機能には「リピーター分析」「競合比較」「販売戦略立案」などECノウハウが内包。
技術構成：Anthropic Claude + MCP + makeshop APIの3層統合。
効果：店舗オーナーがチャットで「先月の売れ筋は？」と聞くだけでAIがデータ取得・分析・提案。
国産ECプラットフォームがMCP+Skillsを本番で実装した国内初事例として注目度高い。
他のSaaS事業者への波及効果も期待されるパイオニア事例。
