## 要約
Claude Code の3大拡張システム（Hooks・Subagents・Skills）を横断的に解説した完全ガイド。使い分け基準：Hooks = ライフサイクルイベントで必ず同じ方法で実行すべきアクション（決定論的）、Subagents = タスクを独立したコンテキストで委任、Skills = Claudeが自分で適用方法を決める知識・ワークフロー（.md ファイル）。実際のユースケースごとのどれを使うべきかの判断フローを提供。3システムの組み合わせパターン（例：Hookでpre-commitチェック + Subagentでコードレビュー + Skillでドキュメント生成）も解説。
