## 要約
GitHub の ariadng が公開した metatrader-mcp-server。Model Context Protocol (MCP) を通じて AI エージェント（Claude Code など）が MetaTrader プラットフォーム上で取引操作を直接実行できるようにする OSS ツール。利用可能な MCP ツールは口座情報取得・注文発注・ポジション管理・価格データ取得・EA 実行など。Claude Code から直接 MT4/MT5 に接続して自律的に取引ロジックを実行できるアーキテクチャを実現。MCP エコシステムと FX 自動取引を繋ぐ具体的なブリッジツールとして FX 自動取引開発に直接関連する重要リソース。Python ベースの実装でカスタマイズ可能。
