## 要約
2026年5月、Jama Softwareがエンジニアリング要件管理ソフトJama ConnectのMCPサーバーを発表。「エンジニアリング管理ソフトウェアとしてMCPサーバーを提供する最初のソフトウェア」と位置付け。これにより、Claude Code等からJama Connectの要件・仕様・トレーサビリティデータに直接アクセス可能になる。エンタープライズ向けMCPエコシステムの拡大を示す具体例であり、製造・航空宇宙・医療機器など規制産業でのAI活用加速が期待される　2026年のMCPエコシステムが「500以上の公開サーバー」に拡大する中でのエンタープライズ参入例として重要。
