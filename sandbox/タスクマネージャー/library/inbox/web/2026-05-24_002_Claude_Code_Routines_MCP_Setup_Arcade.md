## 要約
Claude Code Routines機能を活用した5つのプロダクション向けワークフローと、MCPサーバーの組み合わせ設定を解説した実践記事。具体的なワークフローとして(1)インシデントポストモーテム自動起票、(2)オンコールトリアージ→チケット作成、(3)PRエイジングレポート、(4)エクスパンションシグナルスキャン、(5)チェンジログPR生成を紹介。各ワークフローのYAML設定例とMCPサーバーの接続パターンを詳述。Routinesが特定のGitHub eventやAPIコールをトリガーとして自動起動できる仕組みも説明。
