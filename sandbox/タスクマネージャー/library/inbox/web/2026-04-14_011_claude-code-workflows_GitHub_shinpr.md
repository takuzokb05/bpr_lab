## 要約
GitHubリポジトリ「claude-code-workflows」（作者: shinpr）。本番環境対応のClaude Code開発ワークフローをSpecialized AIエージェント群で実装したテンプレート集。5つの核心原則：①常時コンテキストを最小化②繰り返し手順をスキルまたはコマンド化③アクティブセッションをコンテキスト汚染から保護④明確な監視と分離を前提に並列化⑤Plan Modeで計画→レビュー→実行の3段階を厳守。エージェント定義例（.claude/agents/配下）：role・ツールアクセス・modelを個別設定したスペシャリストエージェントの設計テンプレートを提供。実際の開発プロジェクトですぐに使える設計パターンを提供しており、Claude Codeを組織的に活用するためのリファレンス実装として有用。
