## 要約
Anthropicの「Code w/ Claude 2026」（SF, 5月6日）で発表されたClaude Code新機能の公式セッションページ。主要発表内容：①デスクトップアプリGA（マルチウィンドウ・スクロール改善・全画面モード）、②/ultrarview機能によるクラウド上の並列マルチエージェントコードレビュー、③CI自動修正機能（GitHub Actions連携）、④Claude Managed Agentsとの統合によるエンタープライズワークフロー、⑤Cowork機能による複数エージェントの協調作業。Code w/ Claude 2026はサンフランシスコ（5/6）・ロンドン（5/19）・東京（6/10）の3都市で開催。開発者向けの最重要年次イベントとして位置づけ。
