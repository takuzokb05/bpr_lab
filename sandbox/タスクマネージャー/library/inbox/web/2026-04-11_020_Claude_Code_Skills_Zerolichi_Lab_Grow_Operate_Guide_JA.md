## 要約
株式会社ゼロイチラボによる「スキルを育てる運用」に特化したClaude Code Skills実践ガイド。単なる作り方だけでなく、スキルの継続的改善プロセスに焦点。重要な知見：①スキルの「自動発動」の仕組み（descriptionの最適化がトリガー精度を決める）②スキルファイルの最大効果は2,000トークン以内という公式推奨を守りつつ複数ファイルに分割する設計③使用頻度の低いスキルは定期的にアーカイブして肥大化防止④Claude自身にスキルを評価・改善させる「Autoresearch」的運用⑤チーム展開時の権限設計（user-invocable vs disable-model-invocationの使い分け）。PRマネージャー1人で32人分のカバー・30分の業務を30秒に短縮という実際の成果数値も紹介。
