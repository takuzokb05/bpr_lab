## 要約
Paul Goldsmith-Pinkham（Yale大学経済学助教授）によるClaude Code研究者向けセットアップガイド。経済学・データサイエンス研究での活用を念頭に置いた非エンジニア向けの入門。インストールからCLAUDE.md設定・MCP接続まで研究者目線でステップバイステップ解説。特にデータ分析・統計処理・論文執筆補助へのClaude Code活用パターンが充実。研究コードのリファクタリング・文献管理ツールとのMCP連携・LaTeX文書生成の自動化などの実用例を含む。研究者がコーディングツールとしてのAIを日常的に使いこなすための習慣と設定を解説。非エンジニアのセットアップ体験として具体的な躓きポイントと解決策が詳しく記述されており、幅広い読者層に有益。
