## 要約
Engadgetによる2026年5月6日のAnthropicとSpaceX契約報道。AnthropicはSpaceXのColossus 1データセンター（220,000+ NVIDIA GPU、300MW超）の全計算能力を利用する契約を締結。これによりClaude Code 5時間レート制限をPro/Max/Team/Enterprise全プランで2倍化。ピーク時間帯の制限削減も完全廃止（Pro/Max）。Claude Opus API rate limitも大幅引き上げ。ユーザーから長期間「すぐ上限に達する」という不満が多く寄せられていたため、今回の改善はエンタープライズ採用を加速させると分析。Anthropicは将来的に「複数ギガワット」の軌道上コンピュートを目指すと示唆。
