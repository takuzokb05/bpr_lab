## 要約
Claude Code 2026年の主要アップデートをまとめた包括的概説記事。
Agent View（全セッション一覧表示）、/goal コマンド（完了条件到達まで自律継続）、Fast ModeのOpus 4.7デフォルト化、Rewindメニューの「Summarize up to here」機能が追加。
バックグラウンドセッション（--bg）がフォアグラウンドセッションと共存可能になりワークフローが拡張。
レート制限が2倍に拡大されエンタープライズ利用が加速。
2月にRemote Control（スマホ・ブラウザからローカルCC操作）、3月にVoice Mode（スペース長押しで20言語音声入力）など大型機能が続々投入されている。
