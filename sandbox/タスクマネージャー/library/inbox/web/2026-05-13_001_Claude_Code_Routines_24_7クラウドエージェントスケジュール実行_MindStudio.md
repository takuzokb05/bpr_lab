## 要約
Claude Code Routinesは、Anthropic管理インフラ上でクラウド実行されるスケジューリング機能。ローカルPCを起動し続けなくてもエージェントが動作し続ける。設定方法は「スケジュール定義→タスク記述→放置」の3ステップ。マルチエージェントによる並列サブエージェントオーケストレーションもRoutines内で動作可能で、時間・イベント・APIトリガーの3種類のトリガーに対応。Claude Code on the Webから設定でき、既存のclaude project設定と統合される。実業務への適用として、毎晩データ集計→レポート生成→Slack通知という完全自動化パイプラインが紹介されている。
