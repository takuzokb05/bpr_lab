## 要約
TurboDocxによるAPIプロジェクト向けCLAUDE.md作成ガイド。「実際に機能するCLAUDE.mdの書き方」というタイトルが示す通り、理論より実践を重視した内容。APIプロジェクト特有の注意点：認証パターン（OAuth・APIキー・JWT）の記述方法、エラーハンドリング規約（リトライロジック・指数バックオフ）、Webhook処理のデバッグ手順をCLAUDE.mdに含める理由と方法。「APIドキュメント自体は書かない（Claudeは読めるから）、チーム固有のパターンだけ記述する」という重要原則を強調。実際のAPI開発プロジェクトで使えるCLAUDE.mdテンプレートを提供。TurboDocxはドキュメント生成ツールで、技術文書作成の専門知識を活かした実用的な内容。
