## 要約
Gen AI Unpluggedによる包括的なClaude Codeチュートリアル。Skills・スラッシュコマンド・Hooks・サブエージェント・CLAUDE.mdを一冊で解説。
各要素の使い分け指針：Skillsは再利用可能なプロセス、Hooksは決定論的な強制実行、サブエージェントは並列処理と文脈分離。
CLAUDE.mdに書くべき内容と書くべきでない内容の明確な分類（200行以上になったらSkillへ移行）。
/initコマンドによる自動CLAUDE.md生成、Adaptive Memory機能との組み合わせ方を説明。
実践的なワークフロー例：TDD強制・仕様書ドリブン開発・マルチエージェント並列テスト。
