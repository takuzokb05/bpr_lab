## 要約
Yahoo!ニュース掲載（窓の杜転載）のClaude Code「ルーチン（Routines）」機能発表記事。ルーチンはスケジュール・API・GitHubイベントによってトリガーされる自動実行の新機能。具体的には①毎日決まった時間にコードレビューを自動実行 ②GitHubのPRオープンをトリガーにCI修正を自動化 ③定期的なバグレポート集計と要約生成などが実現可能に。従来の手動実行・Hooks（イベント駆動）に加え、Routinesという第三の自動化レイヤーが加わったことでClaude Codeの自律性が大幅に向上。Windows向けの「窓の杜」という一般PC向けメディアで紹介されたことで、エンジニア以外への認知度向上も示している。2026年4月の重要アップデートとして注目度の高い情報。
