## 要約
6月15日の Anthropic 課金変更を「Claude Agentsをメーター制に移行」と評した解説記事。変更前は無制限に近かったプログラマティック使用が固定クレジット+超過分API課金へ。特にMax 20xプランの$200クレジットでも重いエージェントループには数時間で消費する懸念。indie hackerや個人開発者への影響が大きく、Agentループのコスト計算をこれまで以上に意識する必要がある。Claude Code を自動化スクリプトのように使っていたユーザーは「インタラクティブ利用か否か」の定義を確認すること。影響を受けるケースの具体例と影響しないケースを整理。
