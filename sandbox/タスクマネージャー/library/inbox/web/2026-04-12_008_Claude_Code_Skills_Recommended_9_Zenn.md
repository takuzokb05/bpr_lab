## 要約
Claude Code Skillsの用途別おすすめ選定ガイド（Zenn日本語記事）。初めてSkillsを導入する人向けに「結局何を入れればいいか」を用途別に整理。コーディング支援・コードレビュー・テスト生成・ドキュメント作成・デバッグ等の用途ごとに推奨スキルを紹介。各スキルのインストールコマンド・特徴・実際の動作例を記載。「モデル主導」スキルの自動起動条件の設定方法も解説。日本語コミュニティでのClaude Code活用における実践的な入門資料として有用。
