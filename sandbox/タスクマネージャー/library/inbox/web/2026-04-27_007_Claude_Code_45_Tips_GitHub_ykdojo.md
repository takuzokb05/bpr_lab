## 要約
GitHubリポジトリ「ykdojo/claude-code-tips」が公開したClaude Code 45のTipsコレクション。基礎から高度なテクニックまで網羅し、特に注目すべき内容：①カスタムステータスラインスクリプト（現在のコンテキスト使用量・エージェント状態をターミナルに常時表示）、②システムプロンプトを半分に削減する方法（コンテキスト効率化）、③Gemini CLIをClaude Codeのサブエージェントとして使用（無料のGemini APIでコスト最適化）、④Claude CodeをDockerコンテナ内で自律実行（セキュリティ隔離・並列実行）、⑤dxプラグインの活用方法。「dx」プラグインはClaude CodeのDX（開発者体験）を向上させるコミュニティプラグイン。Gemini CLIをminionとして使う手法は、コスト削減しつつサブエージェント並列化を維持できる実践的アーキテクチャとして注目。Claude Code実運用者にとって即効性の高いtipsが多い。
