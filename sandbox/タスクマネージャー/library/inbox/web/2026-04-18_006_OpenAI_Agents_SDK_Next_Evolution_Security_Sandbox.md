## 要約
OpenAIが2026年4月15日にAgents SDKの大型アップデートを発表。主要新機能：①ネイティブサンドボックス実行（ファイル・コマンド・コード編集を制御された環境で実行）、②モデルネイティブハーネス（Claude Managed AgentsのAnthropicに対抗する位置づけ）、③拡張セキュリティ機能（実行権限の細粒度制御）、④長期タスクのデュラブルエージェントパターン。Claude Agent SDKと直接競合する機能セット。AI開発者コミュニティにとってエージェントSDK選択肢が広がる重要リリース。マルチプロバイダー対応でClaude・Geminiモデルとも統合可能。
