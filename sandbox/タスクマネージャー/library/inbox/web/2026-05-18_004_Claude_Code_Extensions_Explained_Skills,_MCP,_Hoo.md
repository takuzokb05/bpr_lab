## 要約
Towards AI掲載。Claude Codeの6つの拡張機能（Skills/MCP/Hooks/Subagents/Agent Teams/Plugins）を横断比較した解説。
各拡張の用途・設定方法・具体例を一記事でカバーしているため全体像把握に最適。
Agent Teamsは複数サブエージェントをオーケストレーションする新機能で、2026年春に追加。
Pluginsはスキルとカスタムテーマを組み合わせてチーム配布できる仕組み。スキルとMCPの使い分け判断フローチャートも掲載。
