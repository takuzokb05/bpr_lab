# ソフトバンク、Oracle Alloy採用「Cloud PF Type A」で国産LLM「Sarashina」活用AI提供へ（2026年6月〜）

- ソース: web
- 言語: ja
- テーマ: ai-news

## 要約
2026年4月16日付ソフトバンク公式プレスリリース。Oracle Alloyを採用した「Cloud PF Type A」上で国産LLM「Sarashina」を活用した生成AIサービスを2026年6月から順次提供開始。企業・自治体のデータ主権ニーズに対応し、国内データセンター完結の処理を実現するのが特徴。SarashinaはSB Intuitionsが開発した日本語特化LLMで、金融・行政・医療など機密性が高い業種向けに設計。Oracle Cloudとの協業で国産LLMのエンタープライズ提供基盤が整備される。デジタル庁のガバメントAI選定（PLaMo・NEC等）と並ぶ国産LLM商用化の重要事例。一次情報（公式PR）。
