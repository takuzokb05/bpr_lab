# GPT・Claude・GeminiなどLLM12種で投資の銘柄選定能力を検証 S&P500全11セクター

- URL: https://ai-data-base.com/archives/100282
- ソース: web
- 言語: ja
- テーマ: ai-trading
- 取得日: 2026-05-04

## 要約
AI Data BaseによるLLM12種のS&P500銘柄選定能力検証レポート。GPT・Claude・Gemini・Llama・Mistral等12種類のLLMにS&P500の全11セクターで銘柄選定を行わせた比較実験。評価方法：各LLMに同一プロンプト（各セクターのトップ5銘柄を選定・理由を説明）を与え、選定銘柄の実際の1年後リターンを評価。主な発見：①推論能力の高いモデル（Claude Opus・GPT-4o）が一般的に高い選定精度②テクノロジー・ヘルスケアセクターでの精度が高く、エネルギー・素材セクターでは低い③マクロ経済変数の考慮不足が共通の弱点。重要な注意：過去1年のデータのため統計的有意性は限られる。「LLM12種中、実際の市場リターンを安定して上回ったモデルはなかった」という率直な結論も含む。
