# 生成AIエージェント基盤「つなぎAI」の販売開始について

## 要約
株式会社ニーズウェルによるプレスリリース（PR TIMES掲載）。生成AIエージェント基盤「つなぎAI」の国内販売開始を発表。
特徴：既存業務システム（ERPやCRM）とAIエージェントを「つなぐ」ためのミドルウェア基盤で、ノーコードでMCP連携が可能。
対象ユーザー：ITリソースが限られた中小企業・非IT部門のDX推進担当者。
価格は月額制SaaS型。MCP普及に伴う国内エンタープライズAI連携市場への参入事例として注目。
