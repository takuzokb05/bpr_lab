# Claude Opus 4.7 新機能: xhigh Effortレベルとビジョン強化

## 要約
Claude Opus 4.7（2026年4月16日GA）の公式新機能ドキュメント。新エフォートレベル「xhigh」を追加：high（既存）とmax（最大）の間に位置し、アジェンティックコーディング・多段階推論・トレードオフ分析に最適。Claude Code v2.1.117からxhighが全プランのデフォルト。Claude API経由では`budget_tokens`で制御可能。Max/Team Premiumプランでは4.7がデフォルトモデルに変更。Fast modeもOpus 4.7に対応（`speed: "fast"`, `model: "claude-opus-4-7"`, betaヘッダ`fast-mode-2026-02-01`）。
