## 要約
LightNodeによるTradingAgents + VPS + MT5を使った24時間FX自動売買ボット構築チュートリアル。実装手順：①TradingAgentsフレームワーク（LangGraph基盤、7役割エージェント）のVPSへのデプロイ、②MT5との接続設定（MetaTrader 5 Python API経由）、③エージェントロール（ファンダメンタル分析/センチメント/ニュース/テクニカル/リスク管理/トレーダー）の設定、④スケジューリングとシグナル生成からオーダー実行までのパイプライン。使用LLM：GPT-5.x、Gemini 3.x、Claude 4.x、Grok 4.xから選択可。FX自動売買とLLMマルチエージェントの組み合わせに関する一次実装情報として価値高い。リスク警告：研究目的。本プロジェクトのFX自動取引開発への直接応用可能。
