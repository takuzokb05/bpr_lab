## 要約
NousResearchによるHermes Agentのv0.7.0（"The Resilience Release"、2026年4月3日）の詳細解説。MITライセンスのオープンソースAIエージェントフレームワークで、2026年2月リリース後2ヶ月でGitHubスター33,000+、フォーク4,200+、142人以上のコントリビューターを獲得。「閉じた学習ループ」が最大の特徴：タスク完了→自動的にスキルドキュメント生成→次回再利用、というサイクルでエージェントが自己改善する。v0.7.0では安定性とセキュリティが大幅強化。Claude Codeとの比較でオープンソース・セルフホスト可能な点が評価されている。FX自動取引や情報収集エージェントの基盤として、Claude APIと組み合わせた活用も考えられる有望なフレームワーク。
