## 要約
Nimbalystによるクロード・コード・スキルの実践ガイド（2026年版）。スキルの本質的定義：名前・description・条件のセットで、フォルダ内にSKILL.md（手順書）＋サポートファイル（テンプレート・スクリプト）で構成。重要な仕組み：スキルは「常時オン」ではなく自動発見（Auto-discovery）方式で、Claudeがメッセージとdescriptionを照合し、マッチした場合のみ全内容をロード。descriptionの書き方がスキル発動率を決める重要ポイント。Agent Skills Open Standardに準拠し複数ツールで再利用可能。Skill Creator（Anthropic公式）を使った対話的スキル作成方法も解説。スキルを自作する際の実践的ガイドとして参照価値高。
