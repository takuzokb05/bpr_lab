## 要約
shanraisshan氏によるClaude Codeベストプラクティス集のGitHubリポジトリ。「vibe codingからagentic engineeringへ」というコンセプトで、場当たり的なAI利用から体系的なエンジニアリングプラクティスへの移行を解説。実際のCLAUDE.mdテンプレートを含み、コード規約・プロジェクト構造・ツール設定・ワークフロー定義のベストプラクティスを実例で示す。複数のClaude Codeセッションを効率的に管理するためのgit worktree活用パターン、サブエージェント委任戦略、Hooksによる自動化フローの設定例も収録。コミュニティへの貢献も活発でIssues・PRを通じた継続的な知見の集積が特徴。実際のプロダクション開発での使用を想定した実践的な内容。
