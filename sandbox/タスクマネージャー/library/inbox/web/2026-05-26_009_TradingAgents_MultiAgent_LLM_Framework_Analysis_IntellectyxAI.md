## 要約
TauricResearch製 TradingAgents フレームワーク（arxiv:2412.20138）の詳細解説。実際の証券会社の組織構造を模倣したマルチエージェント設計：ファンダメンタル分析エージェント・センチメント分析エージェント・テクニカル分析エージェント・Bull研究者・Bear研究者・リスク管理チーム・複数リスクプロファイルのトレーダーエージェント。LangGraph ベースで構築され OpenAI/Anthropic/Google/DeepSeek/Qwen など主要LLMプロバイダーに対応。単一モデル比較でCumulative Returns・Sharpe Ratio・Maximum Drawdownの全指標で優位。FX・株式・暗号資産の各市場での実験結果を含む。エージェント間の動的議論がデシジョン品質を高める点が新規性。
