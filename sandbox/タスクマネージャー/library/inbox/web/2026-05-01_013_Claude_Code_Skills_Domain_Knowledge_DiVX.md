# Claude CodeのSkills機能でドメイン知識をAIに自動でコンテキストエンジニアリングさせる

- 取得日: 2026-05-01

## 要約
株式会社divxによるClaude Code Skills機能の実践解説。Skillsを「ドメイン知識のコンテキストエンジニアリング自動化ツール」として位置づける独自の視点が特徴。SKILL.mdにビジネスドメイン固有の知識・手順・判断ルールを記述することで、Claude Codeがスキル実行時に必要な文脈を自律的に収集・解釈する。特定のSPAサイト内容をClaude Code自身にスクレイピングさせてSkillを自動生成するユースケースを実演。スキル設計のポイント：(1) 「何をするか」より「なぜそのように判断するか」の記述を重視。(2) スキルのテストケースをSKILL.md内に含める。(3) Claudeにスキルの自己評価を促すフィードバックループの設計。非エンジニアでもMarkdown記述でスキル作成可能な具体例付き。日本語ユーザー向けの事例として実践的価値が高い。
