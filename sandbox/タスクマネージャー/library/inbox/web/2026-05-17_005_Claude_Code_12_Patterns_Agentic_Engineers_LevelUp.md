## 要約
Level Up Coding（Medium系）掲載の実践ガイド。上級エンジニアが実際に使う12のパターンを解説。主なパターン：①外部テスト（ユニットテスト）によるClaude自己検証ループ、②CLAUDE.mdでのアーキテクチャ決定記録（ADR）管理、③サブエージェントへの並列タスク委任（独立タスクのコンテキスト分離）、④/plan modeで探索と実装を分離（変更前に全決定を合意）、⑤MCPでNotion/Figma/DBに直接アクセス、⑥hooks.jsonでの決定論的強制（pre-commit tests等）、⑦Skill.mdでドメイン知識のパッケージ化。プランニングが80%の決定ポイントを100%近くに押し上げるという定量的主張も含む。中級から上級Claude Codeユーザー向けの実装中心コンテンツ。
