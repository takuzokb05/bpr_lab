## 要約
MarkAICodeによるClaude Code MCPインテグレーション解説。v2.1.91でMCPツール結果サイズ上限が500,000文字に拡張されたことが大きなアップデートとして強調されている。MCPのTool Search（遅延ロード）機能によりコンテキスト使用量を最大95%削減できる点も解説。HTTP transportがリモートサービス接続の推奨方式であること、stdio transportがローカルプロセス向けであることを整理。Claude Code設定でのMCPサーバー追加（claude mcp add）の実践手順、認証が必要なサービス（GitHub、Slack等）への対応方法、エンタープライズ環境でのMCPサーバーのセルフホスティング構成を詳説。2026年時点でMCPが「AI向けUSB-C」として業界標準化しつつある現状を背景に、実装ガイドとして充実している。
