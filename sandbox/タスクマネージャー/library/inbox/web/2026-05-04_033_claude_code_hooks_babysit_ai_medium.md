## 要約
Lakshmi Narasimhan氏による「AIの保守を辞めるためにフックを使え」というメッセージ性の強いMedium記事。「Claudeが何をしているか30秒ごとに確認する」という状況を解消するためにフックが有効という主張。StopフックとNotificationフックを使った完了通知（osascript/notify-send）で作業から離れた間の完了検知を実現する方法を実装例付きで解説。PreToolUseフックによるBashコマンドの事前検査（危険なrm -rf等のブロック）でAIを「信頼して任せる」状態への移行を提案。フックがCLAUDE.mdの指示より優先される「確定的制御」の重要性を強調。実際のワークフロー改善体験に基づく説得力のある内容。
