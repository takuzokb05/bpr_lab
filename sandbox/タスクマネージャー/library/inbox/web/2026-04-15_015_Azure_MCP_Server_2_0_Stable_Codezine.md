# Azure MCP Server 2.0が安定版として公開、自己ホスト型エージェンティック自動化を強化（CodeZine）

## 要約
Microsoftが2026年4月10日にAzure MCP Server 2.0安定版をリリース（CodeZine報道）。**自己ホスト型MCPサーバーとして複数ツールセット同時利用時のレスポンス・運用効率が向上**。Azure OpenAI/Cognitive Services/Storage等へのMCP接続が安定化し、エンタープライズSSO統合・監査ログ機能も強化。Claude CodeやCursorからAzureサービスをMCP経由で安全に利用できる基盤として注目。AWSに続きMicrosoftもMCPサーバーエコシステムを本格的に整備しており、エンタープライズAI基盤のMCP標準化が加速していることを示す。
