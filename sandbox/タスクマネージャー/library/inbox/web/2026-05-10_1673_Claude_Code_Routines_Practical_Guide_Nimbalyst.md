トリガーは「スケジュール」「GitHubイベント」「API呼び出し」の3種類。
設定方法・テンプレート例・CI/CD連携・Webhook設定まで網羅。
コード品質チェック・定期リファクタリング・依存関係アップデートなどのユースケースを具体的に解説。
Pro/Max以上のユーザーが利用可能で、Claude Code Webとモバイルから管理できる。
