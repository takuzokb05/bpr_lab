## 要約
Anthropic公式のManaged Agentsクイックスタートドキュメント。初めてManaged Agentsを使う開発者向けのステップバイステップ導入ガイド。APIキーの設定から最初のManaged Agentsリクエスト送信・SSEストリーミングでの結果受信まで、Python/TypeScriptの実際のコード例で説明。エージェントの定義方法（system prompt・tools・model設定）、タスクの送信（sessions APIエンドポイント）、リアルタイムイベントのストリーミング処理（SSEパース）を10分以内に試せる構成。無料tier（限定ツール・標準サンドボックス）と本番tier（フルツールセット・強化サンドボックス・SLA付き）の違いも説明。Managed Agents導入の最初のステップとして参照すべき公式ドキュメント。
