## 要約
Claude Agent SDKを使ったプロダクションAIエージェント実装チュートリアル。Claude Codeを動かすのと同じ内部機構（エージェントループ・ツール呼出・文脈管理）をSDKとして公開している点を解説。Python/TypeScriptの両実装例を掲載。Bedrock・Vertex AI・Azure経由のクラウドルーティング設定も網羅。マルチステップ問題推論→ツール呼出→エラー回復のループ動作をデバッグログで可視化。SerpAPI連携・コードレビューエージェント・データパイプラインエージェントの3ユースケース実装。
