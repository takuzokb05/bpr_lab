## 要約
Claude Code の /ultraplan コマンドと /autofix-pr の組み合わせ解説。/ultraplan を実行するとクラウドコンテナでプランが生成され、レビュー後にそのままPRが自動作成される。/autofix-pr はカレントブランチのオープンPRを検出し、Claude Code on the web で自動修正を有効化する。CI 失敗やレビューコメントを自動で監視し修正をプッシュし続ける。会話履歴・ファイル編集・推論の全コンテキストをクラウドに引き継ぐ。ローカル完了後に /autofix-pr を実行するだけで完全ハンズオフのPR管理が実現する実践的ガイド。
