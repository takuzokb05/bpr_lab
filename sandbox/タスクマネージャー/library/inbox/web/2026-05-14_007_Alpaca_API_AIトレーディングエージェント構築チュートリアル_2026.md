## 要約
TradewinkによるAlpaca API（株式・仮想通貨ブローカー）とLLMを組み合わせたAIトレーディングエージェント構築の実践チュートリアル（2026年版）。市場データ分析・注文執行・ポートフォリオ管理を自律的に行うエージェントの構築手順を段階的に解説。Claude CodeなどのエージェントコーディングツールがAlgorithmic tradingのプロジェクト開発を加速していることに言及。Alpacaのペーパートレード（模擬取引）環境での安全なテスト方法も含む。FX自動取引システム開発の参照事例として有用で、特にLLM×ブローカーAPIの接続パターンは自プロジェクトへの応用可能性が高い。
