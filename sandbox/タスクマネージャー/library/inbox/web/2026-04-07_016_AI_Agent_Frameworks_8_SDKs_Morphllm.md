## 要約
2026年の主要AIエージェントフレームワーク8つを比較した技術記事。フレームワーク：Claude Agent SDK・OpenAI Agents SDK・Google ADK・LangGraph・CrewAI・AutoGen・LlamaIndex・Goose（Block）。OpenAIはSwarmをプロダクション向けAgents SDKに置換（3月）、GoogleはADKを4言語（Python/TypeScript/Java/Go）でリリース（4月）、AnthropicはAgent SDK名称変更で「より広い野心」を示す。Google ADKの差別化：A2A（Agent-to-Agent）プロトコルネイティブ対応・Google Cloud Consoleにビジュアルエージェントデザイナー。MCPは9700万インストール突破（2026年3月）でインフラ標準化。ACP（Agent Communication Protocol）がLinux Foundation傘下のAgentic AI Foundationで策定中。フレームワーク選択基準のトレードオフを中立的に整理。
