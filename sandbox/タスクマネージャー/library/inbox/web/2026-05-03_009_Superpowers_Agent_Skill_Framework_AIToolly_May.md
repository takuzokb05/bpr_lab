## 要約
Jesse Vincent（Prime Radiant）が開発した Superpowers フレームワークの最新動向（2026年5月3日）。GitHub Stars 174,000超（7ヶ月で）。Anthropic マーケットプレイスプラグインとして2026年初頭に公開。コーディングエージェント向けのコンポーザブルな SKILL.md ファイルセット。真のRed/Green TDD・YAGNI・DRYを強制。Claude Code・Codex・Cursor・Gemini CLI・OpenCode・Copilot CLI で動作。仕様書作成→チャンク確認→コーディングの厳格なワークフローを実装。`obra/superpowers` リポジトリから利用可能。エージェントが「すぐにコードを書く」のでなく「何を作るか明確にする」設計思想が特徴的。
