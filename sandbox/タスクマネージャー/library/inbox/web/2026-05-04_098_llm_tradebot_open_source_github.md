## 要約
EthanAlgoXによるオープンソースLLMトレーディングボットのGitHubリポジトリ。複数のLLMエージェントを使ってリアルタイムの市場状況に適応する取引戦略の最適化システム。アーキテクチャ：Market Analysis Agent（技術指標・チャートパターン分析）・Sentiment Agent（ニュース・SNS感情分析）・Risk Management Agent（ポジションサイジング・ドローダウン制御）・Execution Agent（注文実行・スリッページ最適化）。対応LLM：OpenAI GPT・Anthropic Claude・Google Gemini・ローカルモデル（Ollama経由）。バックテスト機能・リアルタイム取引・ペーパートレードモードを装備。研究目的のオープンソースプロジェクトで金融アドバイスではないことを明記。LLMを活用した自律型取引システムのアーキテクチャ参考として重要。
