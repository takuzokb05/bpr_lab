## 要約
2026年4月14日リサーチプレビューとして公開されたClaude Code Routinesの公式ドキュメント。Routinesはスケジュール・Webhook・GitHubイベントの3種のトリガーで無人クラウドエージェントを実行する仕組み。claude.ai/code/routinesまたはCLIの/scheduleで作成可能。全有料プラン（Pro/Max/Team/Enterprise）で利用可能。GitHub Actionsとの違いは、判断が必要な作業（PRコメント評価、アラートとコミットの関連付けなど）をAIが担う点。Anthropicインフラ上で動作するためローカル環境不要。
