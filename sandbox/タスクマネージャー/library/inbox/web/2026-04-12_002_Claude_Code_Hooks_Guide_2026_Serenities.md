## 要約
2026年版Claude Code Hooksの完全ガイド。2026年初頭にリリースされたHooksはClaude Codeのライフサイクルの特定ポイントで自動実行されるコマンド・プロンプト・エージェント。PreToolUse（ファイル編集・コマンド実行前に承認/拒否）、PostToolUse、Notification、Stop等12種類のイベントを解説。CLAUDE.mdがアドバイザリ（遵守率80%）なのに対し、Hooksは決定論的（100%実行）という違いを強調。リンター・フォーマッター・セキュリティスキャンをHooksで強制実行するパターンを紹介。
