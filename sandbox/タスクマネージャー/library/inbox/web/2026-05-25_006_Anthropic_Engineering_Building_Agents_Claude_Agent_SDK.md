## 要約
AnthropicエンジニアリングブログによるClaude Agent SDK公式解説。Claude Codeを動かすのと同じ内部機構を使って本番AIエージェントを構築する方法を解説。Python・TypeScript両対応、pip/npmでインストール可能。組み込みツール（ファイル読み書き・コマンド実行・コード編集）を標準装備。エージェントループは「コンテキスト収集→アクション実行→作業検証→繰り返し」の4ステップ。コードレビューエージェントをゼロから構築するチュートリアルを収録。状態永続化・コストキャップ・サーキットブレーカー・ツールパーミッショニング・マルチエージェントオーケストレーションの本番パターンも詳説。6月15日以降の課金変更への対応方針も言及。一次情報として最重要ドキュメントの一つ。
