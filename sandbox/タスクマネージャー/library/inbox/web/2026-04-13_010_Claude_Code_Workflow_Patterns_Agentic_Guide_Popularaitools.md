## 要約
Claude Codeのエージェント的ワークフローパターンを体系化した2026年版ガイド。探索→計画→実装のフェーズ分割、コンテキスト管理、セッション間引き継ぎのベストプラクティスを解説。特に「オーケストレーション層」として複数Claudeセッションを協調させるパターン（メインセッション＋専門サブエージェント）を詳説。非同期実行でClaude Codeをバックグラウンド作業者として活用するパターンも掲載。イベント駆動パイプライン（hooksで特定イベントを起点に別処理をトリガー）と並列実行モデル（gitワークツリーごとの独立セッション）の組み合わせが先進的ワークフローの鍵。規模スケールアップに向けた設計思想も含む。
