## 要約
Claude Code Routinesの実践的ガイド。Web sessionはインタラクティブなクラウドセッション、Routinesはトリガー起動の無人クラウドセッションと明確に区別して解説。3種のトリガー（スケジュール・Webhookモニタリングツール・GitHubイベント）の設定方法を具体的に説明。Routinesとは「保存済みClaude Codeプロンプトを無人クラウドエージェントとして動作させる仕組み」と定義。PRの判断が必要なコメント対応・バックログ整理など、GitHub Actionsが苦手な"判断を要する作業"をカバーする。
