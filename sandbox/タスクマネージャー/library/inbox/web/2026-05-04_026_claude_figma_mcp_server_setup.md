## 要約
Figma公式ヘルプセンターによるClaude Code + Figma MCPサーバー設定ガイド。デザイン→コード変換ワークフローを実現するFigma MCPサーバーのセットアップ手順を公式説明。FigmaのデザインデータをClaude Codeから直接参照し、コンポーネント・スタイル・レイアウト情報を取得してコードを生成できる。設定方法はSettings.jsonへのMCPサーバー定義追加のみで、Figmaのパーソナルアクセストークンを環境変数で設定する。これによりデザイナーとエンジニアの協業ワークフローが大幅に改善され、Figmaデザインを見ながらClaude Codeに実装を指示できる。Figma公式によるサポート記事であり信頼性が最高。デザイン統合が必要な開発チームには必須のMCP設定。
