## 要約
karan氏によるClaude Codeフック集のGitHubリポジトリ。「コピー・貼り付け・カスタマイズ」という使いやすさを重視したフックのコレクション。PreToolUse（実行前チェック）・PostToolUse（実行後処理）・Stop・Notificationの各イベントタイプに対応したフックが含まれる。実用的なフック例：ファイル書き込み後の自動フォーマッタ実行（Prettier/gofmt）、.envファイルへの書き込みブロック、Bashコマンドの事前検査・承認、完了時のデスクトップ通知（macOS/Linux対応）。設定はsettings.jsonへのJSON定義で行い、シェルスクリプト・HTTP endpoint・Claudeプロンプトの3種類のフックタイプをサポート。Claude Codeの自動化を大幅に強化するリソース。
