## 要約
Claude Code向けのMCPサーバー・プラグインを網羅した2026年版完全ガイド。MCPサーバーはツール定義・名前・JSONスキーマをエクスポーズするプロセスで、Claude Codeはそのクライアントとして機能する構造を解説。推奨スターターパック（GitHub + Filesystem + Context）で80%のコーディングワークフローをカバー。3〜5サーバーを上限として始め、コンテキストウィンドウ消費を管理する実践的アドバイスも掲載。プラグインの/plugin screensで事前にコマンド・エージェント・スキル・フック・MCP/LSPサーバーを確認できる新機能も紹介。
