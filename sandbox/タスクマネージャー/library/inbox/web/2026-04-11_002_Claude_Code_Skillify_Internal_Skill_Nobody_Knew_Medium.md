## 要約
Medium記事。Claude Codeに隠された内部スキル「/skillify」の発見レポート。/skillifyは既存のワークフローや繰り返しパターンを分析し、自動的にSKILL.mdファイルを生成するコマンド。エンジニアが手動でスキルを記述する手間を大幅に削減。発見者David Veselýが実験した結果、/skillifyはセッション履歴から「よく使うパターン」を抽出し、適切なフロントマター（description、allowed-tools等）まで自動設定する。非エンジニアにとってスキル作成の敷居を下げる可能性があり、Claude Codeエコシステムの自己増殖的成長を促す機能として注目。2026年3月公開。
