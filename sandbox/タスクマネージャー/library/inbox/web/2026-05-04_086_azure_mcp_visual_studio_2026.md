## 要約
Microsoft公式Visual Studioブログによる「Azure MCPサーバーがVisual Studio 2026に組み込み」の発表記事。Visual Studio 2026にAzure MCPサーバーツールが標準搭載され、IDEから直接Azure ResourcesへのAIエージェントアクセスが可能になった。使用可能なAzure MCPツール：Azure Resource Manager（リソースのCRUD）・Azure Blob Storage・Azure SQL・Azure Functions・Azure App Service管理。GitHub CopilotやClaude Codeなどのエージェントがこれらのツールを使ってAzureインフラを直接操作・管理できる。設定不要でVisual Studio 2026インストール時から利用可能。MCPがIDEに組み込まれた重要なマイルストーンとして、AI開発環境の統合が加速していることを示す重要記事。
