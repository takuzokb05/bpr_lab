# AnthropicがOpenClaw・サードパーティエージェント利用を課金付きで復活

- URL: https://www.dataworldbank.net/2026/05/13/anthropic-reinstates-openclaw-and-third-party-agent-usage-on-claude-subscriptions-with-a-catch/
- ソース: web
- 言語: en
- テーマ: claude-ecosystem
- 取得日: 2026-05-14

## 要約
2026年5月13日報道。AnthropicがOpenClawおよびサードパーティエージェントのClaudeサブスクリプション利用を「条件付きで」再開。一時期サブスクリプション枠から除外されていたが、6月15日からの新クレジット制（月額$20〜$200）の導入により、再び利用可能に。ただしインタラクティブ利用枠とは完全分離されるため「catch（落とし穴）」あり。OpenClaw以外にも同様のAgentアプリが対象。プログラム的Claude利用の正式な位置付け変更を報じた一次情報。
