## 要約
MicrosoftによるAgent Framework v1.0の公式ブログ発表。Semantic Kernel基盤＋AutoGenオーケストレーション概念を統合した.NET/Python向けプロダクション対応オープンソースフレームワーク。2025年10月にプレビュー公開されていたものが正式版（長期サポート付き）に。安定したAPIセットと本番環境への移行を支援する機能を提供。.NETとPythonの両言語でサポート。Claude等複数LLMプロバイダに対応。エンタープライズ向けのマルチエージェントオーケストレーション基盤として位置づけ。OSSとしてGitHub公開。
