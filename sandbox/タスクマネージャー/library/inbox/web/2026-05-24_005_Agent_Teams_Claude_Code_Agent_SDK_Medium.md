## 要約
Claude CodeとClaude Agent SDKを組み合わせたAgent Teams構築の実践ガイド（Medium/Isaac Kargar）。リードエージェントがチームメンバーを作成し、作業を調整・交換・クリーンアップする階層型協調パターンを解説。Agent SDKの組み込みツール（ファイル読み込み・コマンド実行・コード編集）により、ツール実装なしでエージェントが即座に動作できる点を強調。コードレビューエージェントをゼロから構築するチュートリアル形式で、構造化フィードバック生成・バグ・セキュリティ問題検出の実装例を示す。プロダクション向けのstate persistence・コストキャップ・サーキットブレーカー設計も網羅。
