今週の重要トピック

Claude Code

Stainless買収とSDK/MCP垂直統合

今週最大の構造的イベント。AnthropicがStainless（$300M+）を買収したことで、Claude SDKの品質管理とMCPサーバー自動生成の両方が内製化された。Stainlessは競合他社（OpenAI、Google、Cloudflare、Meta）のSDK生成も担当しており、買収は単なる技術獲得ではなく、競合のサプライチェーンを断つ戦略的手筋だった。ホステッドプロダクトは廃止予定で、競合他社はSDK自動同期機能を失うことになる。

技術的な長期インパクトは「OpenAPI spec → MCPサーバー自動生成」の内製化で、今後は品質が保証されたMCPサーバーが加速度的に増える。Agent SDKに新たに追加されたSelf-hosted Sandboxes（public beta）とMCP Tunnels（research preview）も、Stainless買収と同じ方向を向いている——エージェントが安全に到達できる範囲を最大化するというインフラ戦略だ。

CLAUDE.md設計の科学——80行/200行/500語の閾値

今週、Claude Codeの運用知見は「経験則」から「定量的指針」に進化した。最も重要な発見はCLAUDE.mdの遵守率に関する閾値データだ。80行を超えるとルールが無視され始め、200行を超えると大ブロックが丸ごとスキップされ、500語を超えると遵守率が崩壊する。対策として、ルールには「why（理由）」を記載することで曖昧なケースでの判断精度が向上する。

三者の役割分担も明確化された。CLAUDE.mdはadvisoryなコンテキスト（指針）、Hooksはhard rule（必ず実行される決定論的処理）、Skillsはreusableなmulti-step job（再利用可能な手順書）。これにより「何をどこに書くべきか」が初めて体系的に整理された。CLAUDE.mdベストプラクティスの記事だけで週間10件以上が公開されており、コミュニティの関心度の高さが伺える。

/goalコマンドとRoutines——自律エージェントの実用段階

/goalコマンドの実戦投入レポートが今週複数出た。テスト→デバッグを目標達成まで自動ループする機能で、MCPツールのファジング自動化にも活用されている。Routinesは「毎晩のデータ集計→レポート生成」のようなスケジュール駆動の自動実行を可能にし、PCを閉じてもクラウドで処理が継続する。InfoQとclaudefa.stがRoutinesの詳細解説を公開し、No-Code自動化ツールの代替として明確に位置づけた。

Stopフック＋linter/type-checkパターンも注目に値する。フックで型チェックやlint失敗をエージェントにパイプバックするだけで、半壊した編集をコミット前に自動修正できる。.claude/rules/ディレクトリによるチーム判断基準の共有、Explore→Plan→Implementのサブエージェント分業パターンも「個人ツール」から「チーム開発基盤」への転換を示している。

CLI戦争は四つ巴に——Antigravity 2.0参入

Google I/O 2026でAntigravity 2.0（Google版Claude Code）が発表され、Claude Code / Codex / Grok Build / Antigravityの四つ巴になった。Antigravityは即座に週次クォータ3倍緩和で対抗したが、日本語エンコードでフリーズする問題や/modelコマンドの互換性など初期段階の荒削りさも目立つ。一方でClaude Codeの競争優位は「Skills/MCP/hooksのフォーク困難な防御壁」にあるという分析が説得力を持ち、20人チーム検証による最適リポジトリ構成も「Claude Code＝深い統合を前提とした開発基盤」という位置づけを裏付けた。

接続: Antigravityの参入はGemini 3.5 Flashの4倍速性能と合わせてGoogle陣営の攻勢を示すが、FX自動取引のようなCLAUDE.md/hooks/skillsに深く依存したワークフローでは移行コストが高い。

Ecosystem / MCP

MCPがLinux Foundation標準に——オープンガバナンスへの転換

今週のMCP最大ニュースはLinux Foundation標準への正式採択。Anthropic・OpenAI・Microsoft・Googleが共同運営体制に移行し、MCPは単一企業の所有物ではなくなった。これは「Anthropicが支配するプロトコルに依存して大丈夫か？」という企業のリスク懸念を根本的に解消する動きで、エンタープライズ採用の加速が見込まれる。先週のSDKダウンロード9,700万件/月に続き、今週はGitHubスター81,000超も突破した。

MCPトンネルとSelf-hosted Sandboxes

Anthropicが発表したMCPトンネル（tunnel.anthropic.com経由）は、ファイアウォール内部のMCPサーバーにパブリックインターネットを経由せず安全に接続できる仕組みだ。これまでエンタープライズ環境では法務・セキュリティ部門が「内部サーバーをインターネットに晒すリスク」を理由にMCP導入をブロックしていたが、この障壁が技術的に解消された。Self-hosted Sandboxesと合わせて、「エージェントを安全に動かすインフラ」が整ったことになる。

エコシステム拡大: プラグイン公式ディレクトリ + 永続メモリ

Claude Plugins公式ディレクトリがリリースされ、/plugin installで35本の公式プラグインと15本の外部プラグインを一括管理できるようになった。Skills/Agents/Hooks/Slash commands/MCPの全レイヤーを統合する動きだ。

コミュニティ発のagentmemory（14.9K stars、OSS）は特筆に値する。知識グラフ・confidence scoring・ハイブリッド検索を備えた永続メモリで、Claude Code・Cursor・Gemini CLIなど任意のMCPクライアントに対応する。Anthropicが発表した「Dreaming（夢見）」技術——AIエージェントがセッション間で過去の挙動を振り返りパターンを発見する仕組み——と合わせて、「セッション間の記憶喪失」問題に複数の解法が同時に浮上した週だった。

接続: 永続メモリはFX自動取引のバックテスト結果蓄積に直結する。過去の取引判断とその結果をセッション間で保持できれば、戦略の継続的改善サイクルが組める。
