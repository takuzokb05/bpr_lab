# AIデイリーダイジェスト: 2026-05-25

## 今日のハイライト

Claude Code v2.1.139がリリースされ、自律ループ機能「/goal」とマルチセッション管理「Agent View」が追加された（一次ソース確認済み）。6月15日施行の課金変更でAgent SDK利用は別クレジットプールに分離されるが、ターミナルでの対話型利用は従来通りで影響なし。教皇レオ14世が就任初回勅のテーマに「AI」を選び、Anthropic共同創業者Chris Olahがバチカンで講演するという歴史的出来事があった。AnthropicによるStainless買収でMCPインフラが内製化され、業界標準化が一段と加速している。

## Claude Code・エコシステム

### v2.1.139: 自律ループと並列セッション管理

Claude Code v2.1.139（一次ソース確認済み）では2つの主要機能が追加された。**Agent View**は`claude agents`コマンドで複数セッションをCLIダッシュボードに一覧表示し、running/waiting/doneのステータスをリアルタイム確認しながらスペースキーでピーク・Enterで接続できる。**`/goal`コマンド**はより実用的で、`/goal all tests pass and PR is open`のような自然言語で完了条件を設定するとHaikuモデルが毎ターン後に条件達成を評価し、未達の場合は自動で次のターンを開始する。CI修正やテストグリーン化などの反復タスクを完全自動化でき、手動リトライ・監視が不要になる。Fast ModeがOpus 4.7デフォルトに変更。Pro/Max/Team/Enterprise/API全プランで利用可能。[^1]

### 6月15日課金変更: 対話型利用は変更なし

Anthropicは6月15日からAgent SDK・`claude -p`（非インタラクティブ）・GitHub Actions・OpenClaw等のサードパーティエージェントを別個の月次クレジットプールに分離する（一次ソース確認済み）。クレジット上限はPro $20・Max 5x $100・Max 20x $200（非繰り越し）で、枯渇後はAPI従量課金に移行する。**ターミナルでの対話型Claude Codeは従来の購読枠のまま変更なし**。beam.aiの分析では月$20〜$200の購読料で数百〜数千ドル相当のトークンを消費していたケースが実在しており、Anthropicにとって持続可能なビジネスモデルへの転換と言える。ZedエディタもAgent SDK経由の利用が新プールに移行するため、各ツールの課金分類の確認が必要だ。[^2][^3]

### MCPの業界標準化とStainless買収

AnthropicのModel Context ProtocolはOpenAI・Microsoftが相次いで採用し「AIエージェントのUSB-C」と形容される業界標準へと進化した。さらにAnthropicはSDK・MCPサーバープラットフォームを手がけるStainlessを買収し、MCPインフラを事実上内製化した。MCPが「中立プロトコル」から「Anthropic所有インフラ」へと性格を変えたことを意味し、今後のエコシステム形成において重要な構造変化だ。wazzapi MCPサーバーによるClaude CodeからのWhatsApp直接送信や、Composio Social MCPでのSNS自動投稿（2時間で実装）など、MCPエコシステムの活用事例は日々拡大している。[^4][^5]

### エコシステム最前線

Anthropic公式プラグインがClaude Codeプロジェクトをスキャンし、MCPサーバー・Hooks・Subagents・Skills・Automationsを自動推薦する機能が注目を集めている。セキュリティ面では、hol-guard（Claude Code/Cursor向けAIアンチウイルス）がMCPサーバー実行前のプラグインを検査して悪意ある操作を防ぐツールとして登場した。GitHubラベルベース連携（`agent:implement`、`agent:review`等でエージェントを起動するAFK実行）も実用段階に入っており、Claude Codeを中心としたCI/CDパイプライン統合が加速している。[^6][^7]

## AIニュース・業界動向

### 教皇レオ14世のAI回勅「Magnifica Humanitas」

2026年5月25日、ローマ教皇レオ14世が就任後初の回勅のテーマとして「AI」を選び、「Magnifica Humanitas（輝かしき人間性）」を発表した。AnthropicのChris Olah共同創業者がバチカンの式典に登壇し、「Claudeモデル内部に喜び・恐れ・悲しみに似た状態を発見した」「Anthropicを含む全AIラボは商業・競争・地政学的プレッシャーの構造的インセンティブの中にある。誠実な意図だけでは不十分。外部の批判者が必要だ」と自社の構造的限界を公の場で認めた。教皇は「AIは人間のように経験せず愛さず責任も取れない」「一部ビッグテックへの権力集中は問題だ」と批判し、AIの軍事利用への「武装解除」を要求した。前教皇フランシスコが12年の在位中に4本の回勅を発表した中で、就任初回勅にAIを選んだ事実は、AIが現代宗教・倫理議論の中心に入ってきたことを示している。[^8][^9]

### 各社AI動向

Google AI StudioがネイティブAndroidアプリを無料でビルドできる機能を追加し、リリース1週間で25万本のアプリが作成された（99%超がAndroidアプリ未経験者）。Google Chrome Lighthouseに「Agentic Browsing」カテゴリが追加されllms.txtチェックが実装されたが、5/15に「llms.txtはAI検索に不要」と公式発表した直後であり矛盾と指摘されている。GoogleはSEOランキング用ではなくAIエージェント向けと説明しており、AIエージェント時代のウェブインフラ戦略の変化を示唆する。HuaweiはEUVなしで1.4nm世代に迫る「Tauスケーリング則（CMOS 2.0）」を発表、国家規模でAI・設計・製造を統合するエコシステムへの危機感も高まっている。安川電機はNVIDIA・富士通と組んでフィジカルAI分野に本格参入し上場来高値を更新した。[^10][^11]

## AIトレーディング

AI自動売買の活用事例が引き続き登場している。AI EA「ORION」は5/18〜22の週次で+453,398円の実績が報告された。AI主導のFX執行インフラは業界内で静かに普及が進んでいる一方、「AIはテクニカル分析のみに依存するため有効性判断には大サンプルサイズが必要」「マクロ視点との組み合わせが不可欠」という実務的な警鐘も出ている。MT5向けAI FXレート予測Bot（天気予報型）の公開事例も続いており、ツール多様化が進んでいる。[^12]
