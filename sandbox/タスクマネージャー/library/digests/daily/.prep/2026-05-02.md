AnthropicのARRが2026年5月に$44Bへ到達（一次ソース確認済み）、Claude Codeのエンタープライズ需要が主因。TradingView全データをClaude CodeがChrome DevTools経由で直接読む78ツールMCPが公開され、FX自動売買との連携が現実になった。CLAUDE.md・Hooks・Skillsを体系化するベストプラクティス記事が複数発行され、Claude Codeの「使いこなし格差」が広がっている。AIエージェントが「便利ツール」から企業IT資産の「管理対象」へ格上げされるトレンドも進行中。

## Claude Code・エコシステム

### Anthropic ARRが1年で44倍、Claude Codeが牽引

AnthropicのARRは2025年1月の$1Bから2026年5月時点で$44Bへ拡大した（一次ソース確認済み・SemiAnalysisレポート）。月単位でフェーズが変わる成長速度で、先月だけで$14B増（+47%）。牽引役はClaude Codeのエンタープライズ採用で、年間$1M以上を投じる企業がすでに1,000社を超えたとされる。あるエンジニアは「FTE SWEとして月$5〜1万ドル使うのは難しくない」と証言しており、AI PRが爆発的に増え、その下流でのレビュー・リファクタ・技術的負債への対応がさらなるトークン消費を呼ぶ連鎖構造が起きている。[^arr]

### CLAUDE.md・Hooks・Skillsの体系化が加速

複数の実践記事が同日に発行され、Claude Codeの設定アーキテクチャに関する知識が収束しつつある。共通する結論は「CLAUDE.mdは助言的な永続メモリ、Hooksは決定論的なガードレール」という棲み分けだ。CLAUDE.mdをエラーのたびに更新するフィードバックループと、`PreToolUse`/`PostToolUse`で危険コマンドをブロックするHooksの組み合わせが推奨される。さらに`.claude/rules/*.md`にパスglobで分割するテクニックや、Skills（=YAML+Markdown形式のスキルファイル）をADRやTDDルールと組み合わせることでエージェントの逸脱を「レール」として物理的に防ぐアプローチが注目を集めている。[^best1][^best2][^best3]

### Claude Agent SDKがGitLab・Bitbucket対応、OpenTelemetry統合

Claude Agent SDK Python（May 2026版）のGitHubリポジトリが更新。`--from-pr`フラグがGitLab MR・Bitbucket PRをサポートし、SessionStoreプロトコル（S3/Redis/Postgres対応）、W3Cトレースコンテキストによる分散トレーシングが追加された。Claude Opus 4.7にはv0.2.111以降が必要。[^sdk]

### Codex移行機能でClaude Codeのスイッチングコストが消える

OpenAIのCodexにClaude Code環境を数クリックでインポートできる機能が登場した。設定・スキル・MCPサーバー・直近30日のチャットまで持ち込める。Claude Codeユーザーにとって「移行障壁の消滅」を意味し、逆説的にClaudeを使い続ける理由の質が問われる局面になった。[^codex]

### Blender、AnthropicのCorporate Patronを単発寄付に格下げ

Blenderがコミュニティの反発を受けてAnthropicとの継続的なスポンサー契約を見直し、単発寄付に変更したことが海外で賞賛を集めた。AI企業との距離感という「スタンス」を明示したことが評価された事例。クリエイターコミュニティとAI企業の関係が今後も問われる論点だ。[^blender]

## AIニュース・業界動向

### GPT-5.5とClaude Mythos、強力なサイバー能力で規制議論

英国政府機関の検証によりGPT-5.5が「ネットワーク完全乗っ取り攻撃」を自律的に実行できる能力を持つと判明。Anthropicが限定公開（Project Glasswing）している「Claude Mythos」に続く2例目で、一部指標では上回る数値も出ているという。なおGPT-5.5は4月23日リリースだが「全開放」ではなく有料プラン限定、MythosはProject Glasswingで一部アクセス制限中（一次ソース確認済み・partially_confirmed）。[^weekly]

### Zed 1.0登場、Electron廃棄・GPUシェーダー描画で「速いエディタ」路線

Atom開発チームが5年と100万行をかけて書き直したRust製エディタZed 1.0がリリース。GPUシェーダーで画面描画し、AIエージェント並列実行に対応。Cursorとは異なり「圧倒的な速度」をUSPとした選択肢が企業向けZed for Businessと同時登場した。[^zed]

### Microsoft Agent 365一般提供、AIエージェントが「IT管理対象」へ

MicrosoftがAgent 365を正式提供開始。「AIチャットの新機能」ではなく、企業内AIエージェントを統合管理する管制塔として位置づける。Claude Codeのようなローカルエージェントも将来の検知対象として明示され、端末・MCPサーバー・IDまで可視化する方向性が示された。公務員・エンタープライズ環境でClaude Codeを使う場合、組織のIT管理ポリシーとの兼ね合いを今から調べておくべき動向だ。[^agent365]

### コロラド州AI規制SB189、施行を2027年1月に延期

コロラド州のAI規制法施行日が2026年6月から2027年1月へ延期される修正が提案（一次情報・Colorado Sun）。AI技術の使用方法・リスク・学習データの開示義務化と消費者への通知要件は維持。規制の実装準備期間として半年延長された形で、企業のAIコンプライアンス計画に直接影響する。[^colorado]

### ハイパースケーラーのデータセンター投資額、予測を大幅超過

Q1決算発表でAMZNでもデータセンター投資が+55%と、昨年12月時点の+37%予測を大幅に超えた。Huawei Ascend 950PRが量産開始し中国AI市場の60%獲得を目指す一方、TSMC1.3nmプロセス量産は2029年開始予定。AI設備投資は加速フェーズが続いており、半導体メモリ不足の懸念も浮上している。[^infra]

## AIトレーディング

### TradingView MCP 78ツール公開、Chrome DevToolsで生データを直接取得

FX自動売買を運用する立場として最注目の情報。`tradesdontlie/tradingview-mcp`が公開され、TradingView DesktopとClaude Codeを直接接続する78ツールのMCPサーバーが登場した。スクリーンショット解析ではなくChrome DevTools Protocol経由でローソク足・出来高・インジケーターの生データをコードレベルで取得する仕組み。自然言語でPine Scriptを生成・自動デバッグし、BitGet等の取引所APIと接続すれば自動売買まで完結できる構成だ。[^tvmcp]

### LLMアルゴ取引ボット比較：Claude Sonnet 4.6がSharpe比1.94でトップ

QuantLabsによるLLM比較検証でClaude Sonnet 4.6が487%リターン・Sharpe比1.94を達成しGPT-5.5・Gemini 3.1 Ultra・DeepSeek V4等を上回った。高頻度取引は依然として非LLMフレームワークが必要だが、スイング戦略・マクロシグナル戦略では実用水準に達したと評価されている。[^llmtrading]

### TradingAgents v0.2.4、LangGraphベース12エージェント構成で取引会社を模倣

GitHub Stars 53,000超のTradingAgentsがv0.2.4にアップデート。ファンダ・センチメント・ニュース・テクニカルの4並列アナリストとBull vs Bearの議論システムを持つ。DeepSeek/Qwen/GLM/Azure対応、永続的意思決定ログ追加。ただしGoogle株3ヶ月検証ではバイアンドホールド（19.1%）に対しLLMベストが18.1%と僅差。過信禁物。[^ta]
