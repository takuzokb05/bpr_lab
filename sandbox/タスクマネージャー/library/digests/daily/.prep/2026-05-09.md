## Claude Code・エコシステム

### Claude Code 制限2倍：SpaceXとの計算提携が背景

最大のトピックはClaude Codeの利用制限が2倍に拡大したことだ。AnthropicとSpaceXが22万GPU・300MWの計算リソースで提携し、Pro/Max/Teamプランのレート制限が引き上げられた。日経xTechやnoteでも解説記事が出ており、「インフラ勝負が始まった」という認識が広がっている。ヘビーユーザーにとっては直接的な恩恵で、これまで制限に引っかかっていた長時間エージェント実行が現実的になる。

Claude Constitution（訓練データ）の公開もこの週のニュースに含まれており、透明性強化の動きと並行している。Anthropic ARRが$300億を超え（Air Street Capital確認）、Fortune 10企業の8社が採用・500社が$100万超の利用というスケールはエコシステムの厚みを示す。

### Claude for Microsoft 365 GA・Apple iOS 27への組み込み

2026年5月7日、Claude for Microsoft 365がWord・Excel・PowerPointのアドインとして一般公開された（**一次ソース確認済み**）。OutlookはパブリックベータでGAではない点に注意。Anthropic有料プランが前提になるため、コスト設計を先に考える必要がある。組織内でCopilotとClaudeが並走する時代が来たと言えるが、使い分けの設計が先決だ。

Apple iOS 27では「Extensions」フレームワークによりSiriのバックエンドにClaude・Gemini・ChatGPTを選択できる機能が開発中（**一次ソース確認済み**：9to5Mac・MacRumors）。正式発表はWWDC（6/8）予定。スマートフォンがエージェントハーネスになる転換点として注目に値する。

### Anthropic×Google Cloud：$2000億コミット

The Informationが2026-05-05に報道したAnthropicとGoogle Cloudの5年間$2000億支出コミットが確認されている（**一次ソース確認済み**）。2027年開始予定で、Google TPUはNVIDIA比40〜50%安という価格優位性が契約の動機となっている。計算コスト構造が変わると、API利用料にも中長期で影響が出る可能性がある。
