## Claude Code・エコシステム

### SpaceX提携でレート制限2倍 ── 需要と供給の完璧な一致

AnthropicのコンピュートボトルネックがSpaceXとの提携で解消された。SpaceXはColossus 2で次世代Grokの訓練に移行済みでColossus 1が空いたタイミングと、Anthropicのレートリミットのきつさがちょうどマッチした形だ。イーロン・マスクはAnthropicの安全性アプローチを認めた上での提携であることも発言しており、「OpenAI共同創業者がライバルに計算資源を提供する」という構図となっている。その結果、Claude Pro/Max/Teamの5時間制限が実質2倍に拡大し、Claude Codeヘビーユーザーには直接的な恩恵となった。

### 20コマンドリファレンス ＆ 金融データMCP

Claude Codeの実践知識として20コマンドが整理された。Esc（タスク停止・会話履歴保持）、/rewind（前のステップへ）、/compact（コンテキスト圧縮）、/insights（セッション統計）が特に実用度が高い。ただしClaude Code Web・Mac版ではcompact実行が確認できないという落とし穴も指摘されており、ターミナル版との挙動差異に注意が必要だ。

金融データMCPを1コマンドで接続することで、17,000銘柄の株価・暗号資産価格・財務諸表をリアルタイム取得できる環境が公開された。FX自動取引運用者にとってはClaudeをマーケットデータ分析パイプラインに組み込む現実的な選択肢になりつつある。

### Claude for Microsoft 365 GA（一次ソース確認済み）

Word・Excel・PowerPointのアドインが一般公開（2026-05-07）。OutlookはGAに達しておらずパブリックベータ段階（一次ソース確認済み）。Anthropic有料プランが前提で追加課金の有無は契約形態ごとに確認が必要。公務員業務でのWord文書作成・Excel分析に現実的な選択肢となった。CopilotとClaudeが並走する時代の到来であり、使い分けの設計が先に必要だ。

### Anthropicの巨大インフラ戦略と業界地位

Anthropic×Google Cloudの5年・$2000億コミットが複数ソースで確認された（一次ソース確認済み）。Google TPUはNvidia比40〜50%安という数字も検証済みで、コスト効率と計算量を両立する選択だ。加えてBlackstoneとの$15億新会社設立でPE傘下数百社へのClaude実装を推進する動きも同時進行している。

AnthropicのARRは$300億超でOpenAIを上回った（OpenAI約$250億）。従業員一人当たり収益は$9M対$5.6Mでリードしており、Anthropic Instituteの設立も発表された。Apple iOS 27の「Extensions」フレームワークでClaude・Gemini・ChatGPTをSiriのバックエンドとして選択可能になるリークも確認され（WWDC 6/8での正式発表予定）、iPhoneが「スワップ可能なエンジンを持つ最初のコンシューマー向けエージェントハーネス」になるとの分析も出ている。
