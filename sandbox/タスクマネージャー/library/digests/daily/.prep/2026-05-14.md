## Claude Code・エコシステム

### 利用制限50%増とその背景（一次ソース確認済み）

Anthropicが2026年5月13日に公式発表した週次利用制限の50%増加が、日本のClaude Codeユーザーの間で大きく話題になった。7月13日18時PDT終了の期間限定措置で、5月6日に実施された5時間制限の2倍（100%増）と同時適用される。背景はSpaceXのColossusコンピュートとの提携と、OpenAI Codexへの競合対応。「溜まってたタスクが一気に解消できそう」との声が多い一方、「利用制限が上がった裏に何かある」と懸念する声も。

### 課金体系の刷新：6月15日から別枠化

6月15日からClaude Agent SDK・`claude -p`・Claude Code GitHub Actions・OpenClaw等サードパーティエージェントの利用が、インタラクティブ利用枠とは別の専用月額クレジットから引かれる仕組みに変更される。クレジット額はPro $20/月、Max 5x $100、Max 20x $200。超過時はAPIレートで従量課金または停止。「power userがClaude Codeとclaude -pを同一サブスクで使えるのが魅力だったのに」という批判と、「個人がGitHub Actionsを少し回す程度ならOAuthで足りる」という冷静な分析が並立している。The Register、TechnoEdge等が批判的に報道。

### MicrosoftがClaude Code撤退（一次ソース確認済み）

MicrosoftのExperiences & Devices部門（Windows/M365/Teams/Surface担当）が2026年6月30日までに社内Claude Codeライセンスを廃止し、GitHub Copilot CLIへ移行することがTechRadar・Windows Centralなどで確認された。社内開発者はClaude Codeを好んでいたが財務・戦略上の理由による決定。Claude自体はCopilot CLI経由で引き続き利用可能で、AnthropicとMicrosoftの関係が完全に切れるわけではない。この報道を受けてClaude Codeを80〜100席規模でキャンセルする企業も出た。

### Grok Build参入でCLI戦争が本格化

xAIがGrok BuildのベータをSuperGrok Heavy（月300ドル）加入者に向けて公開した。8エージェントの並列処理、plan mode（diff形式で確認）、sub-agentループをネイティブ実装。「Claude Codeはそのためにhooksが必要」という比較も出ており、機能面での差別化が焦点。Grok 4.3のコーディング精度は未知数だが「AIコーディングCLI戦争が本格的に始まった」という評価は共通している。

### エコシステムの広がり：MCP・スキル・活用事例

MCP（Model Context Protocol）の月間SDKダウンロード数が9,700万件に到達し、SAP・Google・各社が相次いで対応。Googleは公式スキルパッケージ13個をオープンソース化し、`npx skills add google/skills`一発でClaude Code・Cursor・Copilotに導入可能になった。moomoo証券が「Moomoo API Skill」を発表、Claude Codeのスキル機能を使って自然言語だけで米国株の自動売買を設計・実行できる業界初のサービスを開始した。国内・海外62社のClaude Code実装事例サイト（claude-code-cases.vercel.app）も公開され、Stripe（1,370名展開・Scala→Java 1万行移行を4日に短縮）などの詳細事例が参照できる。

SKILL.mdの実践ガイド（ar-aca.tech）とawesome-claude-code-toolkit（135エージェント・40万+スキル・176プラグイン）が同日公開されており、ヘビーユーザー向けの実装リソースが一気に充実した。Claude Code内の`.claude`フォルダをプロジェクトの「OS」として活用するアプローチも注目を集めている。

## AIニュース・業界動向

### Anthropicの企業価値急騰と大型資本連携

Anthropicが約9,000億ドル評価での300億ドル調達を協議中と報じられた。OpenAIの3月調達時の評価額（8,540億ドル）を超える規模。法人市場ではRampの支出データによりAnthropicがOpenAIを上回り、2025年5月以降でシェアを4倍に拡大したとの報告も出た。5月3〜4日に発表されたGoldman Sachs・Blackstone・Hellman & Friedman等との合弁会社（総額約15億ドル）は一次ソース確認済み。中規模企業へのClaude実装を支援するエンジニアリングサービス会社で、コンサルティングレイヤーが本命の競争軸になりつつある構図がはっきりした。

### Anthropic×Gates Foundation：公共インフラへの展開

AnthropicとBill & Melinda Gates Foundationが2億ドル規模でClaudeを医療・教育・農業・経済支援へ展開すると発表した。低中所得国のワクチン研究、K-12教育の個別対応、小規模農家の生産性向上などが対象。「便利なチャットAIから人類の公共インフラへ」という段階変化を象徴するニュースとして注目が集まった。

### AIインフラ競争：軌道上データセンター

GoogleがSpaceXとの軌道上データセンター交渉を進めていると報じられた。Google自身のProject Suncatcher（2027年初頭に試験衛星を打ち上げ予定）に加え、AnthropicはすでにSpaceX Colossusとのコンピュート提携を締結済み。AI計算資源の宇宙展開が現実の競争軸になってきた。

### Grok Build・GPT-5.6・Gemini Intelligence

CLI AIコーディングの競争はClaude Code、OpenAI Codex（2ヶ月無料キャンペーン開始）、Grok Buildが三巴の様相に。OpenAIはGPT-5.6の内部テストを開始し来月中のリリースを目指す。GoogleはGemini IntelligenceをAndroid向けに発表し、GooglebookノートPCでのGemini搭載も確定、Dell・HP・Lenovoが今後数ヶ月以内に発売予定。

### 規制・地政学

米財務長官ベッセントが、トランプ・習近平会談後に米中AI安全協議を開始すると表明。AnthropicはClaude Mythos Previewへの中国政府のアクセス要求を拒否。EU向けにはOpenAIがサイバーセキュリティAIへのアクセスを提供する一方、AnthropicはEUとの合意に至っていない。トランプ政権の国家AI政策フレームワーク（7本柱）では州AI法の連邦法による先占が盛り込まれ、AIエージェント商用展開の規制環境が整備されつつある。

Klarna事例が話題を呼んだ：700名のカスタマーサービスをAIに置き換えたが顧客満足度が急落し、CEOが「やり過ぎた」と認め人間の採用を再開した。AI代替の実証事例として、公務員職場でのAI導入議論にも関連する。
