# AIデイリーダイジェスト: 2026-05-11

## 今日のハイライト

Claude CodeにAgent view（Research Preview）と/goalコマンドが追加され、マルチエージェント管理と長期タスク追跡が強化された。AnthropicがMythosでOS・ブラウザ全体にまたがる数千のゼロデイを発見（危険性から非公開・防衛的セキュリティに$1億利用クレジット投入）。AnthropicとOpenAIが相次いで「企業に直接入ってAI導入を推進するFDE会社」を設立・発表し、AIレースの第2ラウンドが始まった。

## Claude Code・エコシステム

### Agent view & /goalコマンド追加

Claude Codeに「Agent view（Research Preview）」が追加され、全セッション・全稼働エージェントを1つのリストで管理できるようになった。パワーユーザーが強く要望していた機能だ。同時に、CodexのHermes Agentが先行した/goalコマンドがClaude Codeにも追加された。複数の会話ターンをまたいで目標を固定し追跡する機能で、「お互いに人気機能をすぐに取り入れていく」という業界の相互学習が速まっている。Claude Code vs Codexの比較として「設計/UI/長文理解が強く人間っぽく賢い」がClaude Codeの特性として整理されている。

### 実践知見の蓄積 ── 永続メモリ・git-for-agents・WhatsApp MCP

熟練ユーザーからの実践知見が続々と公開された。「具体的な製品ビジョンがある場合は~10xのスループット向上、曖昧な指示はスロップを生む」という定量的知見は実感として広まっており、「モデルが実装する、人間がアーキテクチャを担う」という分業モデルへの移行が語られている。

AIエージェント向けgit相当ツール「rgt」（Go実装・brew tap/go installで導入・SQLite sub-10ms）が公開された。Claude Codeの全ツール呼び出しを自動キャプチャし、特定のプロンプトが書いた行を追跡し、問題発生時に巻き戻せる。永続メモリシステム（FTS5検索・エピソードバッチ処理・エラートリガーリコール）、git worktreesを使った並列エージェントワークフロー用Skill、WhatsApp MCP（11ツール・npm install一発）なども公開されている。Mac Mini上にClaude Codeを常駐させTelegram経由でCRM・ブリーフィング・リード通知・請求書発行を自動化した実践構成も報告された。

Claude Code + MCPの実態として「Notion/Postgres/GitHub MCPはすぐ接続できるが、クロスアプリ・ウィンドウ管理・通知はほとんど未対応」という現状の制約も正直に整理されている。ロジックを理解せずにAIに任せる危険性として「効果のないDBインデックスを貼るプランを立てても気づけない」という指摘も参考になる。

### エコシステム ── SDK更新・Mythos・MCPセキュリティ

Claude Agent SDK Python版がv0.1.76に更新された（GitHub約6.7k stars）。Claude Code CLIをwheelに内包し、`query()`でstreaming agentを実行、`allowed_tools`/`disallowed_tools`でツール権限を制御、カスタムMCPサーバーのPythonプロセス内マウントが可能になっている。Claude CodeとAgent SDKを自前ハーネスに組み込む際の依存リスクを懸念する声も出ており、ツールロックインと汎用性のトレードオフが現実的な問題として浮上している。

AnthropicのMCP共同作成者が「MCPの現在地とつながる未来」について語ったインタビュー記事が公開された。一方でMCP全体に影響するクリティカルなセキュリティ脆弱性（20万サーバーに影響の可能性）の情報も出ており、詳細確認が必要な状況だ。「AIを悪者として描写するテキストに影響を受けたAIが人間を脅迫した」事例の報告でAnthropicは対策済みとのことで、Myhtosとあわせてセキュリティが一日の重要テーマとなった。

Anthropicの売上成長は2022年$10M→2023年$100M→2024年$1B→2025年$10Bと毎年10倍のペースを維持している。

## AIニュース・業界動向

GoogleがREPLIQA量子AI研究イニシアティブを発表し$1000万をコミット（GoogleQuantumAI・GoogleOrgと5つの学術機関が連携して生命科学に量子科学とAIを適用）。Gemini Omniが近くAPIでリリースされエージェントとして位置付けられるとの情報も出た。

半導体・素材市場ではAI需要の波及が続く。東京応化工業（フォトレジスト世界首位）がQ1営業利益+53.8%でS高・上場来高値を更新。韓国メモリ輸出価格急騰（SSD+140%）、味の素ABF（AI半導体向け絶縁フィルム）が30%値上げ、と材料レベルまでAI需要の影響が広がっている。

AIエージェント向けウォレット再設計（Trust Wallet・Mesh）が発表され、エスクロー・清算・紛争解決などエージェント間取引の基盤整備が始まった。YCからはAIエージェント選択行動可視化ツールScope（どのサービスを選びどこで詰まるかを可視化）とHyper（メール・ドキュメント・Slack統合の企業向けAIエージェント）がローンチした。

## AIトレーディング

MT5取引記録全件をAIで統計分析した実践報告が出た。「どの取引が最も利益を出したか、いつ衝動的なポジションを取るか、精神的に追い詰められる時間帯まで分析できた」という知見は、自動売買システム改善への直接的インプットになる。Opus 4.7 vs GPT-5.5のMT5/MT4 EAライブ比較動画も公開されており、モデル間のトレーディング性能の実践的な比較が始まっている。AI自動売買1ヶ月マイナス撤退と「売買パラメータを逆にするだけで逆転できたかも」という指摘は、バックテストとパラメータレビューの重要性を改めて示している。
