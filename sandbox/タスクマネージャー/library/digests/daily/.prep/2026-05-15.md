## Claude Code・エコシステム

### Agent View・/goal・Background Sessions

Geeky Gadgetsの公式解説によると、今週のClaude Codeアップデートで**Agent View**（全セッション一覧管理）、**`/goal`**（目標達成まで自動継続）、**`/loop`**（繰り返し実行）、**Background Sessions**（フォアグラウンドから独立した並列処理）、**Routines**（スケジュール・GitHubイベント・APIトリガー起動）が追加された。Pro/Max/Team/Enterpriseユーザーへ順次展開中。現在リミットが一時的に約50%増加しリセット済みとの一次報告もある。

`/codex:adversarial-review`スキルとClaude Codeを組み合わせた相互レビュー構成では、800Kトークン上限に当たった場合にコンテキストをクリアしてCodexで修正→CCで再レビューというフローで対処できる実践知見が共有されている。

### 実践：CLAUDE.md設計と6ヶ月の知見

6ヶ月間Claude Codeを使い込んだ開発者による実践ガイド（Medium）が注目を集めた。**禁止事項・承認フロー・スタイルガイドをCLAUDE.mdに明記**することで失敗率が激減し、PreToolUse hookでwrite前に自動lint・PostToolUseでテスト実行を強制する構成が効果的とされる。rtk CLIでコマンド出力をLLMコンテキスト投入前に圧縮することでコスト約40%削減も実現できる。デザイナー向けには「18のショートカット＆コマンド」記事も公開され、非エンジニアがClaude Codeを日常利用できることを示した（note/大原かほ）。

### スキルエコシステムの成熟

Claude Codeスキルを使った実用ユースケースが広がっている。WebflowサイトをAstroに変換するスキル、STEP→STL変換スキル（Codex/Claude Code共用）、Kody MCP Serverを使ってCloudサブスクリプション内でエージェントを動かす設計パターンなど、ツール横断的な実装例が増えている。UnityのMCP対応によりClaude CodeからUnityを直接操作できるようにもなった。AWSがClaude Code入門ワークショップをZennで無料公開したことも、エンタープライズ採用加速のシグナルだ。

VS CodeでAgentic Browser UseのMCPディスコネクト問題が報告されており（@AnthPB）、長時間・大規模操作での接続安定性は引き続き課題として残っている。

### 6月15日：Agent SDK課金分離と旧モデル廃止

Anthropicが6月15日から**Agent SDKの課金を分離**する（The New Stack詳報）。新クレジット枠はPro $20/月・Max 5x $100/月・Max 20x $200/月（毎月リセット・繰越なし）で、Agent SDK・`claude -p`コマンド・Claude Code GitHub Actions・サードパーティSDKアプリが対象。同日、claude-sonnet-4とclaude-opus-4がAPIから引退しSonnet 4.6・Opus 4.7が新デフォルトになる。移行手順として`claude mcp list`で依存確認、モデルIDを1行修正、クレジット残量モニタリング設定が推奨される。

### Anthropicエコシステム：インフラ・資金・提携

**一次ソース確認済み**: Anthropicは5月6日、SpaceXのColossus 1データセンターをリース（年間50〜60億ドル、GPU 222,000基・H100/H200/GB200含む）。Claude Codeの5時間レート制限が翌日から2倍に拡大した。この影響でxAI（Grok）側のGPUリソースが大幅縮小し、Grok Imageが制限状態に陥っているという報告も出ている（背景は確認済み）。

**一次ソース確認済み**: Anthropicの評価額は9,000億ドルで$300億規模の調達ラウンドが進行中（Bloomberg・TechCrunch・CNBC確認）。ゲイツ財団との2億ドル・4年間提携（医療・教育・農業支援）も発表。中小企業向け「Claude for Small Business」がQuickBooks・HubSpot・Canvaなど15種のSaaSと連携して公開され、makeshop by GMOもMCPサーバー対応を本番リリースするなど、MCPを核にしたエコシステムの商用実装が加速している。

**一次ソース確認済み**: xAIのGrok Buildは5月14日発表。200万トークンコンテキストは正確だが、「16エージェント」はアーキテクチャの呼称で実際の並列実行は最大8エージェント、$99/月は6ヶ月の導入割引（通常$300/月）。SWE-Bench Verified 70.8%スコアも公表済み。Claude Code Pro（$20/月）の5倍の価格差をどう評価するかが焦点だ。

GPT-5.5がChatGPTの新デフォルトに、GeminiはUltra（2Mコンテキスト）とFlash-Liteを投入しGemini 2.0系を6月1日EOLに。中国4社（GLM-5.1・MiniMax M2.7・Kimi K2.6・DeepSeek V4）がオープンウェイトでリリース、SubQがトランスフォーマー非使用のsubquadratic LLMを$2,900万シードで投入するなどアーキテクチャ多様化の兆しも出ている。EU AI法が2026年8月2日に施行される点は、公務員・情シス担当者として今から確認しておくべき動向だ。

### Vibe Coding法的リスクと本番メトリクス

「Vibe coderが訴訟される時代」という警告が2,600以上のいいねを集めた。AI生成アプリが利用規約・プライバシーポリシー・GDPR・アクセシビリティをスキップしたまま本番リリースされるケースが増えている。組織内AIツール展開を担う立場でも、「小さく試す」前にコンプライアンスチェックリストを整備する視点は重要だ。本番後のAIシステム持続性を測る「5つのメトリクス」（多くのチームが1〜2つしか追跡せず6ヶ月後に気づく）も、長期稼働システムを運用する上での教訓として有用だ。

## AIトレーディング

WunderTradingがAgentic Trading（エージェント型AIトレード）の概念整理を公開。従来のルールベースbotとの最大の違いは「市場変化に応じた戦略の動的変更能力」で、LLMエンジン・マーケットデータフィード・リスクマネージャー・ブローカーAPIの4層構成が基本アーキテクチャ。実装例としてLangGraph+ブローカーAPI構成が紹介されており、感情分析→ポジション調整、異常検知→自動ヘッジといったユースケースが示されている。

Freqtradeユーザーから「PineScriptで書いた指標をClaude Codeに投げるだけでバックテスト検証が数分で完結、TradingViewを手動で開く理由がなくなった」という実践報告も共有された。FX戦略開発サイクルの短縮ツールとして、Claude Codeの実用性が自動取引開発者の間で具体的に語られ始めている。
