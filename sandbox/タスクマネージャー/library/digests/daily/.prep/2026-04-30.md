## Claude Code・エコシステム

### Claude SecurityとCursorの応酬

AnthropicはClaude Code Securityをリブランドした **Claude Security** をパブリックベータへ移行し、AIを用いた脆弱性発見・修復の迅速化を発表した。特筆すべきはCursorがわずか2時間後に自社のAIセキュリティハーネスを対抗発表したことで、開発者向けAIセキュリティ市場での競争が一気に顕在化した。[^crn]

### Claude Code実践ノウハウが集中

本日はClaude Codeの実践投稿が多数流入した。`.claude/` フォルダを適切に育てること（CLAUDE.md・AGENTS.md・スキルファイル）の重要性が複数の発信者から指摘され、Skills→MCP→プラグインの3層理解が「ガチ実務活用」への道とされている。[^rasuku]

指示を英語で投げると性能が上がるという実用論も話題。ただし共有リポジトリではコメント・コミットメッセージが英語になるデメリットも指摘されており、個人リポと共有リポで戦略を分けるのが現実解とされる。[^kitahara]

月20ドルのClaude設定で3人分の開発チーム相当（5エージェント・57コマンド・100以上のサブエージェント・1日47分節約）を実現した事例も共有。「コーダーのためのツール」から「業務設計ができる人向け」へとポジショニングが変わりつつある。[^gippp]

### MCPエコシステムの拡張

**n8n** が公式MCPサーバーをリリース。Claude Code・Cursor・Windsurfなど主要クライアントと接続でき、TypeScript SDKによりワークフローがJSON→コードへと移行可能になった。**Autodesk Fusion** にClaude MCPコネクタが統合され、3D設計環境から自然言語でAIを操作できるようになった。金融分析用途では17,000銘柄・30年分データをClaudeで自然言語検索できるFinancialDatasets MCP Serverも登場している。[^autodesk]

**CVE MCP Server** も注目。NVD・CISA KEV・GitHub・VirusTotalなど21の外部APIを統合し、自然言語でリスク評価（EPSS 35%・KEV 30%・CVSS 20%・PoC 15%の重み付け）を行う。スコア76以上は緊急対応対象と自動分類され、DevSecOps用途で実用的。[^cve]

## AIニュース・業界動向

### ホワイトハウスがMythos拡大に反対（一次ソース確認済み）

Bloomberg・WSJが2026-04-30に報道。AnthropicはMythosへのアクセスをProject Glasswing参加企業（Amazon・Apple・Google・Microsoft・NVIDIA・JPMorgan等）から70社追加し合計約120組織へ拡大しようとしているが、ホワイトハウスはサイバーセキュリティリスクと計算能力不足を理由に反対している。国防総省契約を静かに進めてきたAnthropicが国内展開で制約に直面するという矛盾が浮上しており、OpenAIとの政策面での対比が鮮明になっている。[^mythos]

### AnthropicのBun買収とZig問題（一次ソース確認済み）

2025年12月にAnthropicが買収したBun（Zig製）が、AIアシスタントにより4倍のコンパイル速度向上を達成しながら、Zigコミュニティの「AI生成コード禁止」ポリシーを理由にアップストリームへの貢献を見送った。AIツールを積極活用する企業文化とOSSエコシステムの規範が衝突する事例として注目される。[^bun]

### AI投資・半導体ブーム続く

米国の実質コンピュータ固定投資は年率約3540億ドル（前年比+64%・ChatGPT登場比+133%）と記録的水準。Google・Microsoft・Meta・Amazonの4社合計インフラ投資額は今年7250億ドルに達する見込みで、ほぼすべてがAIデータセンター向けだ。サムスン半導体部門の26年1〜3月期営業利益は前年同期比約49倍の5兆7000億円（四半期ベース過去最高）。TSMCは1.4nm世代（A14）を2028年・派生A13/A12を2029年投入予定のロードマップを公表した。[^tsmc]

一方でテック企業は巨額AI投資と並行して人員削減も加速しており、3月のテックレイオフは2年間で月間最多となる4万5800人。KKRは100億ドル超を調達し、元AWS CEOが率いるHelix Digital Infrastructureを設立、AIデータセンター・電力インフラへ参入する。[^layoff]

### AIエージェント認証の標準化へ

FIDOアライアンスが「Agentic Authentication Technical Working Group」を設立し、AIエージェントによる認証の標準化に着手。Agent Payment Protocol（AP2）はPayments Technical Working Groupへ移管された。AIエージェントが外部サービスと接続する際のセキュリティ規格が今後1〜2年で整備される見通し。[^fido]

## AIトレーディング

EAからAIエージェントへという潮流が記事化された（BrokerXplorer）。従来型EAはルールベースの固定ロジックだが、2026年現在のAIエージェントは市場環境への動的適応が可能とされている。国内ではAIでEAを制御するシステムでドル円2円幅の利確事例も報告されており、AI×EAのハイブリッド構成が普及期に入った可能性がある。[^hide]

GogoJungle1位EA「LTAF_USDJPY_H1」についてAI診断72点という分析が公開されており、FX自動売買評価にAIを使う手法も広がっている。なお「Savage Pips AI」の広告投稿が複数見られるが、検証情報がなく参考外とする。
