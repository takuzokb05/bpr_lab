20コマンドリファレンス ＆ 金融データMCP

Claude Codeの実践知識として20コマンドが整理された。Esc（タスク停止・会話履歴保持）、/rewind（前のステップへ）、/compact（コンテキスト圧縮）、/insights（セッション統計）が特に実用度が高い。ただしClaude Code Web・Mac版ではcompact実行が確認できないという落とし穴も指摘されており、ターミナル版との挙動差異に注意が必要だ。

金融データMCPを1コマンドで接続することで、17,000銘柄の株価・暗号資産価格・財務諸表をリアルタイム取得できる環境が公開された。FX自動取引運用者にとってはClaudeをマーケットデータ分析パイプラインに組み込む現実的な選択肢になりつつある。
