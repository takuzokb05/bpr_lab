Agent SDK課金分離：6月15日のXデー（一次ソース確認済み）

Anthropicが5月13日に発表した課金分離ポリシーの実装まで約3週間となり、実務影響分析が集中した。6月15日以降、`claude -p`・Claude Code GitHub Actions・Agent SDK経由の呼び出しはサブスクリプション枠から切り離され、Pro $20/月・Max 5x $100/月・Max 20x $200/月の独立クレジットプールへ移行する（ロールオーバーなし）。軽量ワークロードで実効12倍、重量ワークロードで最大175倍の価格上昇との試算もある。[^1]

インタラクティブな手動Claude Code利用（ターミナル対話）はサブスクリプション枠のまま変わらない点が鍵だ。自動化スクリプト・GitHub Actionsとして呼ぶ「非インタラクティブ利用」だけが影響を受ける。対策として推奨されているのは①直接APIキーへの切り替え、②Haikuへのダウングレード、③バッチ処理化による呼び出し削減、④予算上限の設定の4点。6月15日前のクレジット申請手続きも公式推奨されている。[^2][^3]

AIトレーディング

TradingAgents（arxiv:2412.20138）は証券会社の組織構造を模倣したマルチエージェントFXフレームワーク。ファンダメンタル・センチメント・テクニカルの各分析エージェントがBull/Bear研究者とリスク管理チームと議論し、複数リスクプロファイルのトレーダーエージェントが最終判断する設計。LangGraphベースでAnthropicを含む主要LLM全対応。単一モデルとの比較でCumulative Returns・Sharpe Ratio・Maximum Drawdownの全指標で優位という実験結果が報告されており、既存の自動売買EAとのアーキテクチャ比較の参考になる。[^9]

EA Agent Studio（今週末公開予定）がアプリ上からバックテストできる機能を追加。EAをサーバー側のMT5で処理し、AIがパフォーマンスを判断・分析する仕組みで、スマホでEA作成後にすぐ検証できる設計。現行の開発→検証サイクルを短縮できるか注目したい。
