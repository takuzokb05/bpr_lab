Claude Code・エコシステム

デスクトップ刷新とRoutinesの本格展開

Anthropicが2026年4月14日にClaude Codeデスクトップ版を大幅刷新し、並列エージェントによるマルチタスク対応を導入した（一次ソース確認済み）。同時発表のRoutinesは、プロンプト・リポジトリ・コネクターを1パッケージにまとめてクラウドで自動実行できる仕組みで、スケジュール・GitHubイベント・APIの3トリガーをサポートする。デスクトップ上で複数タスクを並列処理するエージェント型ワークフローが実質的に解禁された形だ。

Claude Code自体の使い方も急速に進化している。21エージェントチームが4並列スプリントを走らせ、アイデアからApp Store提出まで72分で完了した事例が話題に。UCLのリバースエンジニアリング調査ではコードベースの1.6%しかLLMモデルでなく、98.4%がエージェントループ（ハーネス）であると判明した。創設者のBoris Chernyは2026年に自分では一行もコードを書いておらず、Claude Codeのコードベースは最終的に約100行に収束すると予測している。GitHubの全公開コミットの4%がすでにClaude Code由来で、これは1ヶ月で倍増した数値だ。AmazonはClaude CodeとOpenAI Codexを全エンジニアに社内展開済み。

Claude Codeヘビーユーザーとして注目のアップデート：`/claude-api`スキルが標準搭載され、モデル移行・プロンプトキャッシュ・Managed Agentsの操作をツール内で完結できるようになった。また、PixelDojoがMCPで130以上のメディアモデルをワンコマンド（`npx @pixeldojo/mcp init`）で接続可能にし、MonidはOpenRouter的なエージェントツールマーケットプレイスとして単一スキルで数百ツールへのアクセスを提供している。

Anthropic エコシステムの拡大

Claude Agent SDKが本番運用パターンの標準的なリファレンスとして確立されつつある。LangGraphに次ぐAnthropicネイティブの最速成長フレームワークと評価されており、`query()`呼び出しのプロンプトキャッシュ無効化バグが修正され入力トークンコストが最大12x削減された。

Anthropicは金融サービス向けに10本のAIエージェントテンプレートをリリース（ピッチブック作成・KYCスクリーニング・月末決算・バリュエーションレビューなどMicrosoft 365直接統合対応）。Claude Opus 4.7がVals AI Finance Agent Benchmarkで64.37%で業界トップ。Anthropic自体も9,000億ドル評価（Bloomberg 4/29）での資金調達交渉中と報じられ、年間売上は300億ドルに到達している。さらにGoogle Cloudへの200億ドルコミットメントが発表され、AWS TrainiumよりGTP TPUを主要学習基盤として優先する姿勢が鮮明になった。
