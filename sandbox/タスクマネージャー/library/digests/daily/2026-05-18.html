Claude Code・エコシステム

Stainless買収：SDK/MCP基盤の垂直統合

最大のニュースはAnthropicによるStainless買収（$300M+推計）。Stainlessは初期Claude APIから全6言語のSDK（Python/TypeScript/Go/Java/Kotlin等）を自動生成してきたチームで、驚くべきことにOpenAI・Google Gemini・Mistralなど競合AIのSDKも同社が担当していた。

Anthropic公式（@AnthropicAI）が発表した意義は大きく、競合他社が主要なSDK生成ベンダーを失ったという業界再編でもある。技術的な長期価値は「MCPサーバー仕様駆動化（OpenAPI spec→MCPサーバー自動生成）」の一次化で、今後は規格ドリフトのない高品質なMCPサーバーが増える見通しだ。「エージェントが接続できる範囲を広げる布石」という評価が多く、フロンティアの競争がモデル性能からインフラ接続性に移ったことを象徴する買収といえる。

Agent SDK課金変更 6/15（一次ソース確認済み）

2026年6月15日から、Agent SDK・claude -p・CC GitHub Actionsが通常利用枠から切り離され、別クレジットプールへ移行する。Pro $20 / Max5x $100 / Max20x $200が月額上限で、実質12〜150倍超の値上げとの分析もある（TheRegister/VentureBeat）。「AIエージェントはサブスク内で無限に回せる」という前提を今すぐ更新する必要がある。FX自動取引などでCIのAgent SDK・GitHub Actionsを活用している場合、6/15前にコスト試算と設計見直しが必須だ。

公式プラグインと操作術の成熟

`claude-code-setup`公式プラグインがプロジェクトをスキャンしてhooks・skills・MCP・subagentsの最適設定を自動推薦する機能を静かにリリース。Code with Claude SF 2026の発表（Agent View・新claudes agentsフラグ群・Mobile Push通知・Routines）が実装段階に入ったことと合わせ、初期設定のコストが大きく下がりつつある。

実践的操作術もXで蓄積が進む。「詰まったら箇条書きで整理させる」「なぜ止まったか聞く」という二段デバッグ手法は即日使える知見だ。大規模コードベース（100万行超）での成功要件として「モデル性能よりハーネス設計（CLAUDE.md・hooks・skills・subagents）」「専任agent manager」という組織視点の整理も共有された。

Cursorとの比較では「長期プロジェクトの文脈保持はCC優位、インライン編集はCursor」という使い分けが定着。APIコール削減（161ターン→52ターン）の具体手法、対話モードとCI実行モードの制限差異も実務で押さえておくべき情報だ。なお、会社コードをAPI送信する際のIPプライバシーリスクへの警告も出ており、機密性の高いコードベースでは利用規約・データ保持ポリシーの確認を推奨する。
