# AI Teams - プロフェッショナルAI討論チャット

**「あなたのチームに、最強のAIブレーンを。」**

## 🎯 コンセプト

AI Teamsは、Slack/Teamsライクなインターフェースで、プロフェッショナルなAI討論を実現するチャットアプリです。

複数のAIエージェントが構造化された議論を展開し、リアルタイムで議事録を生成。全ての会話と成果物はSQLiteに永続化され、いつでも振り返ることができます。

## ✨ 主な機能

### 🔄 データ永続化
- **APIキー**: 一度設定すれば自動保存。毎回入力不要
- **会議履歴**: 全ての討論と議事録をSQLiteに保存
- **エージェント管理**: カスタムAIキャラクターを作成・編集可能

### 💬 Slack/Teamsライクなチャット
- **ルーム機能**: 会議ごとに独立したチャットルーム
- **参加メンバー選択**: 会議ごとに必要なエージェントだけを招集
- **リアルタイム議事録**: 右パネルに常に最新の成果物を表示

### 🎤 プロフェッショナルな進行
- **4フェーズ構造**: Opening → Divergence → Convergence → Conclusion
- **司会AI**: 知的な女性キャスターが議論を適切に進行
- **ストリーミング表示**: タイプライター風のリアルタイム表示（実装予定）

## 🚀 セットアップ

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. アプリケーションの起動

```bash
streamlit run app.py
```

### 3. 初回設定

1. ブラウザで `http://localhost:8502` が開きます
2. APIキーを入力（一度だけ）:
   - OpenAI API Key
   - Google API Key  
   - Anthropic API Key
3. 「保存して開始」をクリック

## 📖 使い方

### 新規会議の作成

1. 左サイドバーの「➕ 新規会議」をクリック
2. 会議名を入力（例: 「新規事業アイデア検討」）
3. 参加させたいAIエージェントを選択
4. 「作成」をクリック

### 討論の開始

1. メッセージ入力欄に議題を入力
2. 「🚀 送信」をクリック
3. AIエージェントたちが自動的に討論を開始
4. 右側の議事録パネルでリアルタイムに成果物を確認

### 過去の会議を見る

- 左サイドバーの「📂 過去の会議」から選択
- 全ての会話履歴と議事録が復元されます

## 🛠️ カスタマイズ

### カスタムエージェントの作成

1. 左サイドバー「⚙️ エージェント管理」を開く
2. 「エージェント一覧・編集」をクリック
3. 新しいエージェントを追加:
   - 名前とアイコン
   - 性格・役割（システムプロンプト）
   - 使用するAIモデル
